OsmAnd Parking plugin is a free plugin working with the main OsmAnd Maps and Navigation app on both Android and iOS devices. It keeps the information about the exact location of your car and helps you keep track of time. All you have to do is set the type of parking you are using (with or without a time limit) and set the alarm if needed.
