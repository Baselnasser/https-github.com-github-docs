&lt;p&gt;Syntax Highlight for &lt;a href="http://basicengine.org/manual.html"&gt;BASIC Engine BASIC&lt;/a&gt; for the &lt;a href="https://atom.io"&gt;Atom Editor&lt;/a&gt;.
Based on &lt;a href="https://atom.io/packages/language-cbmbasic"&gt;Atom BASIC syntax highlighting for the C64&lt;/a&gt; by &lt;a href="https://github.com/sjsyrek"&gt;Steven J. Syrek&lt;/a&gt;.&lt;/p&gt;
&lt;p&gt;&lt;img src="https://i.github-camo.com/433761200c26d5af7281248aced0a45917acc32a/68747470733a2f2f64616d69616e76696c612e636f6d2f6261736963656e67696e652f626562617369632d73796e7461782e706e67" alt="BASIC Engine Syntax Highlight" data-canonical-src="https://damianvila.com/basicengine/bebasic-syntax.png"&gt;&lt;/p&gt;
&lt;p&gt;This syntax highlighter supports the BASIC available in the &lt;a href="http://basicengine.org/"&gt;BASIC Engine&lt;/a&gt; Single Board Computer.&lt;/p&gt;
&lt;p&gt;Contributions are welcome.&lt;/p&gt;
&lt;h2&gt;
&lt;a href="#what-is-the-basic-engine-computer" aria-hidden="true"&gt;&lt;/a&gt;What is the BASIC Engine computer?&lt;/h2&gt;
&lt;p&gt;The &lt;a href="https://github.com/uli/basicengine-pcb"&gt;BASIC Engine&lt;/a&gt; is a very low-cost single-board home computer with advanced 2D color graphics and sound capabilities, roughly comparable to late-1980s or early-1990s computers and video game consoles. It can be built at home without special skills or tools and using readily available components for under 10 Euros in parts, or mass-produced for even less.&lt;/p&gt;
&lt;p&gt;&lt;img src="https://i.github-camo.com/fe312619239292684786cb23015688c53b623ae8/687474703a2f2f6261736963656e67696e652e6f72672f696d616765732f73637265656e5f626f6f742e706e67" alt="BASIC Engine Startup Screen" data-canonical-src="http://basicengine.org/images/screen_boot.png"&gt;&lt;/p&gt;

&lt;a href="#cursor-index" aria-hidden="true"&gt;&lt;/a&gt;Cursor Index&lt;/h1&gt;
&lt;p&gt;This is the package &lt;a href="https://atom.io/packages/cursor-index"&gt;cursor-index&lt;/a&gt;
for the &lt;a href="https://atom.io/"&gt;Atom&lt;/a&gt; text editor.&lt;br&gt;
Display the character index of the cursor (i.e. &lt;em&gt;&lt;strong&gt;cursor index&lt;/strong&gt;&lt;/em&gt;) in the status bar.&lt;/p&gt;
&lt;p&gt;The &lt;em&gt;&lt;strong&gt;cursor index&lt;/strong&gt;&lt;/em&gt; provided by this package is a 1D character index (not byte index), while&lt;br&gt;
the &lt;em&gt;&lt;strong&gt;cursor position&lt;/strong&gt;&lt;/em&gt; in the Atom core is a 2D point formed by a pair of row and column number.&lt;/p&gt;
&lt;h3&gt;
&lt;a href="#installation" aria-hidden="true"&gt;&lt;/a&gt;Installation&lt;/h3&gt;
&lt;pre&gt;&lt;code&gt;apm install cursor-index
&lt;/code&gt;&lt;/pre&gt;
&lt;h3&gt;
&lt;a href="#usage" aria-hidden="true"&gt;&lt;/a&gt;Usage&lt;/h3&gt;
&lt;p&gt;The cursor index of the text cursor for the active pane&lt;br&gt;
will be shown on the left of the status bar after installation.&lt;/p&gt;
&lt;p&gt;Clicking on it pops up a dialog to enter the cursor index to be navigated,&lt;br&gt;
just like clicking on the cursor position.&lt;br&gt;
Pressing ALT-G is the corresponding short-cut. (Can be disabled in Settings)&lt;/p&gt;
&lt;h3&gt;
&lt;a href="#settings" aria-hidden="true"&gt;&lt;/a&gt;Settings&lt;/h3&gt;
&lt;ol&gt;
&lt;li&gt;Can be toggled on/off (Default: &lt;code&gt;true&lt;/code&gt; for on)&lt;/li&gt;
&lt;li&gt;Zero/one-based index can be toggled (Default: &lt;code&gt;true&lt;/code&gt; for zero-based)&lt;/li&gt;
&lt;li&gt;Key-binding for ALT-G can be toggled on/off (Default: &lt;code&gt;true&lt;/code&gt; for on)&lt;/li&gt;
&lt;/ol&gt;

&lt;p&gt;Testing package for Atom that allows to run TestCafe tests.&lt;/p&gt;
&lt;p&gt;&lt;a href="https://devexpress.github.io/testcafe/"&gt;TestCafe&lt;/a&gt; is a node.js tool to automate end-to-end web testing.&lt;/p&gt;
&lt;h2&gt;
&lt;a href="#install" aria-hidden="true"&gt;&lt;/a&gt;Install&lt;/h2&gt;
&lt;p&gt;You can install via apm: &lt;code&gt;apm install testcafe-runner&lt;/code&gt;.&lt;/p&gt;
&lt;p&gt;Or navigate to the install tab in Atom’s settings view, and search for &lt;code&gt;testcafe-runner&lt;/code&gt;.&lt;/p&gt;
&lt;h2&gt;
&lt;a href="#features" aria-hidden="true"&gt;&lt;/a&gt;Features&lt;/h2&gt;
&lt;h3&gt;
&lt;a href="#quick-and-easy-run" aria-hidden="true"&gt;&lt;/a&gt;Quick and Easy Run&lt;/h3&gt;
&lt;p&gt;This extension allows to run TestCafe test quick and easy from Atom context menu, with hotkeys or atom commands panel.
Step by step:&lt;/p&gt;
&lt;ul&gt;
&lt;li&gt;
&lt;p&gt;Add project folder to Atom (&lt;code&gt;Important&lt;/code&gt;: project root directory will must contain a &lt;code&gt;package.json&lt;/code&gt; and &lt;code&gt;node_modules&lt;/code&gt; folder)&lt;/p&gt;
&lt;/li&gt;
&lt;li&gt;
&lt;p&gt;Highlight the line with the declaration of the test or fixture.&lt;/p&gt;
&lt;/li&gt;
&lt;li&gt;
&lt;p&gt;Choose one of the test launch options:&lt;/p&gt;
&lt;ul&gt;
&lt;li&gt;In the context menu: &lt;code&gt;Run TestCafe&lt;/code&gt; (or &lt;code&gt;cmd+ctrl+alt+t&lt;/code&gt;)&lt;/li&gt;

&lt;a href="#gui" aria-hidden="true"&gt;&lt;/a&gt;gui&lt;/h4&gt;
&lt;ol&gt;
&lt;li&gt;launch atom&lt;/li&gt;
&lt;li&gt;use ctrl + , on windows and linux or cmd + , on mac to enter settings&lt;/li&gt;
&lt;li&gt;navigate to the search box under the &lt;code&gt;install&lt;/code&gt; tab on the left&lt;/li&gt;
&lt;li&gt;enter &lt;code&gt;atom-figlet&lt;/code&gt; and press enter
&lt;/li&gt;
&lt;/ol&gt;
&lt;h3&gt;
&lt;a href="#usage" aria-hidden="true"&gt;&lt;/a&gt;usage&lt;/h3&gt;
&lt;hr&gt;
&lt;p&gt;select the text you wish to convert and run the &lt;code&gt;atom-figlet:convert&lt;/code&gt; command&lt;/p&gt;
&lt;h5&gt;
&lt;a href="#default-keybind" aria-hidden="true"&gt;&lt;/a&gt;default keybind&lt;/h5&gt;
&lt;p&gt;press ctrl + alt + f&lt;/p&gt;
&lt;h5&gt;
&lt;a href="#using-the-command-palette" aria-hidden="true"&gt;&lt;/a&gt;using the command palette&lt;/h5&gt;
&lt;p&gt;open with ctrl/cmd + shift + p, search for &lt;code&gt;atom-figlet:convert&lt;/code&gt; and press enter&lt;/p&gt;
&lt;h5&gt;
&lt;a href="#from-the-menu-bar" aria-hidden="true"&gt;&lt;/a&gt;from the menu bar&lt;/h5&gt;
&lt;p&gt;select &lt;code&gt;Packages &amp;gt; FIGlet &amp;gt; Convert to FIGlet&lt;/code&gt;&lt;/p&gt;
&lt;h3&gt;
&lt;a href="#settings" aria-hidden="true"&gt;&lt;/a&gt;settings&lt;/h3&gt;
&lt;hr&gt;

&lt;ul&gt;
&lt;li&gt;[x] disable ctrl + alt + f under the package settings&lt;/li&gt;
&lt;li&gt;[x] add your new keybind to your keymap file&lt;/li&gt;
&lt;/ul&gt;
&lt;pre&gt;&lt;code&gt;'atom-workspace':
  '&amp;lt;your-keybind-here&amp;gt;': 'atom-figlet:convert'
