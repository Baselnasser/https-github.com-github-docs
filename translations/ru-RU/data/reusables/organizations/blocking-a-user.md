Можно заблокировать пользователя на неопределенный срок или на определенный период времени. При блокировке на определенный период времени пользователь автоматически будет разблокирован после истечения выбранного времени. При блокировке на неопределенный срок можно разблокировать пользователя вручную в любое время. Дополнительные сведения см. в разделе [Разблокирование пользователя из организации](/communities/maintaining-your-safety-on-github/unblocking-a-user-from-your-organization).
