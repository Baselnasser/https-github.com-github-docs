1. При необходимости отфильтруйте список оповещений. Вы можете щелкнуть несколько фильтров в раскрывающемся меню фильтров, чтобы сузить поиск. Вы также можете ввести квалификаторы поиска в поле **Репозитории поиска**. Дополнительные сведения о доступных квалификаторах см. в разделе [Фильтрация оповещений в разделе «Общие сведения о безопасности»](/code-security/security-overview/filtering-alerts-in-the-security-overview).
