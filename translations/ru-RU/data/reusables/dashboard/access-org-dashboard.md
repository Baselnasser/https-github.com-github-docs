1. Чтобы получить доступ к странице панели мониторинга, в верхнем левом углу любой страницы на {% data variables.product.product_name %} щелкните {% octicon "mark-github" aria-label="The github octocat logo" %}.
2. В левом верхнем углу страницы щелкните свое имя пользователя и используйте раскрывающееся меню, чтобы выбрать нужную организацию.
