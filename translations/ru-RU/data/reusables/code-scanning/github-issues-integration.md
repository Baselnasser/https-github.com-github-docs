Оповещения {% data variables.product.prodname_code_scanning_capc %} интегрируются со списками задач {% data variables.product.prodname_github_issues %}, чтобы упростить определение приоритетов и отслеживание оповещений со всеми задачами разработки. Дополнительную информацию о проблемах см. в разделе [Сведения о проблемах](/issues/tracking-your-work-with-issues/about-issues).

Чтобы отслеживать оповещение проверки кода в проблеме, добавьте URL-адрес для оповещения в качестве элемента списка задач в ошибке. Дополнительную информацию о списках задач см. в разделе [Сведения о списках задач](/issues/tracking-your-work-with-issues/about-task-lists).
