Важно выбрать соответствующую причину в раскрывающемся меню, так как это может влиять на то, будет ли запрос включен в будущий анализ. {% ifversion comment-dismissed-code-scanning-alert %}При необходимости можно добавить комментарий, объясняющий контекст закрытия оповещения. Комментарий о закрытии добавляется на временную шкалу оповещений и может использоваться в качестве обоснования для аудита или отчетов. Вы можете получить или сохранить комментарий с помощью REST API сканирования кода. Комментарий содержится в `dismissed_comment` для конечной точки `alerts/{alert_number}`. Дополнительные сведения см. в статье [Проверка кода](/rest/code-scanning#update-a-code-scanning-alert).
{% endif %}
