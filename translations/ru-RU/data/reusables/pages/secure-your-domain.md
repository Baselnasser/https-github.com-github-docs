Если ваш сайт {% data variables.product.prodname_pages %} отключен, но для него настроен личный домен, существует риск перехвата домена. Если личный домен настраивался с использованием поставщика DNS пока как ваш сайт отключен, это может привести к тому, что другой пользователь разместит сайт в одном из ваших поддоменов.

Проверка вашего личного домена запрещает другим пользователям GitHub использовать ваш домен со своими репозиториями. Если домен не проверен и ваш сайт {% data variables.product.prodname_pages %} отключен, необходимо немедленно обновить или удалить записи DNS с помощью поставщика DNS.
