7. Чтобы создать пару ключей для репликации, используйте команду `ghe-repl-setup` с IP-адресом основного устройства и скопируйте возвращенный ею открытый ключ.
  ```shell
  $ ghe-repl-setup PRIMARY_IP
  ```
