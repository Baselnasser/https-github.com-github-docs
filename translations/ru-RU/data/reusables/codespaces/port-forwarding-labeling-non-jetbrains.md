## Маркировка порта

При открытии пространства кода в браузере или в классическом приложении {% data variables.product.prodname_vscode_shortname %} можно пометить перенаправленный порт, чтобы упростить его идентификацию в списке.

{% data reusables.codespaces.navigate-to-ports-tab %}
1. Наведите указатель мыши на порт, который требуется маркировать, а затем щелкните значок метки.
  ![Значок метки для порта](/assets/images/help/codespaces/label-icon.png) {% data reusables.codespaces.type-port-label %}
