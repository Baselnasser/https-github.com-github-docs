Домен {% data variables.product.prodname_dotcom %} использует для переадресации портов, может время от времени меняться. Если вы ссылаетесь на переадресованный порт в коде, например в тесте, рекомендуется использовать переменную среды вместо жесткого кодирования URL-адреса. Дополнительные сведения см. в разделе [Переменные среды по умолчанию для codespace](/codespaces/developing-in-codespaces/default-environment-variables-for-your-codespace#list-of-default-environment-variables).
