1. Выберите владельца для нового репозитория, введите имя репозитория, выберите предпочитаемый параметр конфиденциальности и щелкните **Создать репозиторий из шаблона**.
1. На главной странице созданного репозитория нажмите кнопку **{% octicon "code" aria-label="The code icon" %} Code (Код)** и выберите вкладку **Codespaces** .

   ![Кнопка "Создать codespace"](/assets/images/help/codespaces/new-codespace-button.png)

1. На вкладке **Codespaces** щелкните **Создать codespace в main**.
