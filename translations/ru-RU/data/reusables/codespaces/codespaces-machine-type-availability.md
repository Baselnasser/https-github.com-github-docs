Выбор доступных типов машин может быть ограничен рядом факторов. Это может быть политика, настроенная для вашей организации, или минимальная спецификация типа компьютера для репозитория. Дополнительные сведения см. в разделах [Ограничение доступа к типам компьютеров](/codespaces/managing-codespaces-for-your-organization/restricting-access-to-machine-types) и [Задание минимальной спецификации для компьютеров codespace](/codespaces/setting-up-your-project-for-codespaces/setting-a-minimum-specification-for-codespace-machines).
