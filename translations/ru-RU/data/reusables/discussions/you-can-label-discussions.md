Чтобы более детально упорядочивать обсуждения, можно применять метки. Например, с помощью меток можно указать состояние обсуждения, чтобы повысить эффективность рассмотрения. У каждого репозитория есть один общий набор меток для проблем, запросов на вытягивание и обсуждений. Дополнительные сведения см. в статье "[Управление метками](/github/managing-your-work-on-github/managing-labels)".
