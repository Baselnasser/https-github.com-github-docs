Если в репозитории есть защищенные ветви, вы не можете изменять или отправлять файлы в таких ветвях с помощью {% data variables.product.prodname_dotcom %}. Дополнительные сведения см. в разделе [Сведения о защищенных ветвях](/articles/about-protected-branches).

Вы можете использовать {% data variables.product.prodname_desktop %}, чтобы перенести изменения в новую ветвь и зафиксировать их. Дополнительные сведения см. в статье [Фиксация и проверка изменений в проекте](/desktop/contributing-to-projects/committing-and-reviewing-changes-to-your-project).
