Если определено несколько переменных среды с одинаковым именем, {% data variables.product.prodname_dotcom %} использует наиболее конкретную из них. Например, при выполнении шага определенная на нем переменная среды будет переопределять переменные задания и рабочего процесса с тем же именем. Определенная для задания переменная при его выполнении будет переопределять одноименную переменную рабочего процесса.
