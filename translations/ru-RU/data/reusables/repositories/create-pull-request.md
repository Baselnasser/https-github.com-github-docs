1. Чтобы создать запрос на вытягивание, готовый к проверке, нажмите кнопку **Создать запрос на вытягивание**.
Чтобы создать черновик запроса на вытягивание, нажмите на раскрывающийся список и выберите **Создать черновик запроса на вытягивание**, а затем **Черновик запроса на вытягивание**. Дополнительные сведения о черновиках запросов на вытягивание см. в разделах [Сведения о запросах на вытягивание](/articles/about-pull-requests#draft-pull-requests) и ![Кнопка создания запроса на вытягивание](/assets/images/help/pull_requests/pullrequest-send.png).
