{% ifversion not ghae %} Если в репозитории есть поддерживаемый манифест зависимостей{% ifversion fpt or ghec %} (и если вы настроили граф зависимостей, если это частный репозиторий){% endif %}, каждый раз, когда {% data variables.product.product_name %} будет обнаруживать уязвимую зависимость в репозитории, вы будете получать еженедельный почтовый дайджест. Вы также можете настроить оповещения системы безопасности в виде веб-уведомлений, отдельных уведомлений по электронной почте, ежедневных почтовых дайджестов или оповещений в интерфейсе {% data variables.product.product_name %}. Дополнительные сведения см. в статье "[Сведения о {% data variables.product.prodname_dependabot_alerts %}](/github/managing-security-vulnerabilities/about-alerts-for-vulnerable-dependencies)".
