Кроме того, можно использовать API отправки зависимостей (бета-версия) для отправки зависимостей из выбранного диспетчера пакетов или экосистемы, даже если экосистема не поддерживается графом зависимостей для анализа файлов манифеста или блокировки. Граф зависимостей будет отображать отправленные зависимости, сгруппированные по экосистеме, но отдельно от зависимостей, проанализированных из файлов манифеста или блокировки. Дополнительные сведения об API отправки зависимостей см. в разделе [Использование API отправки зависимостей](/code-security/supply-chain-security/understanding-your-software-supply-chain/using-the-dependency-submission-api).
