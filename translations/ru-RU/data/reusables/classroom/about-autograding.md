Вы можете использовать автоматическую классификацию для автоматической проверки работы учащегося по заданию в {% data variables.product.prodname_classroom %}. Тесты настраиваются для задания, и тесты выполняются немедленно каждый раз, когда учащийся отправляет их в репозиторий заданий на {% данных variables.location.product_location %}. Учащийся может просматривать результаты теста, вносить изменения и отправлять новые результаты.
