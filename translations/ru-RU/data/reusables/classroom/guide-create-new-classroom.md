1. В списке организаций выберите организацию, которую хотите использовать для своей аудитории. При необходимости создайте новую организацию. Дополнительные сведения см. в статье [Создание новой организации с нуля](/organizations/collaborating-with-groups-in-organizations/creating-a-new-organization-from-scratch).
  ![Организация в списке организаций для создания новой аудитории](/assets/images/help/classroom/click-organization.png)
1. Введите имя для аудитории.
  ![Текстовое поле для ввода имени аудитории](/assets/images/help/classroom/type-classroom-name.png)
1. Нажмите кнопку **Создать аудиторию**.
  ![Текстовое поле для ввода имени аудитории](/assets/images/help/classroom/click-create-classroom-button.png)
