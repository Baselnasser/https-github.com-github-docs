## <a name="autolinks"></a>Автоматические ссылки

Чтобы упростить рабочий процесс, можно использовать API для добавления автоматических ссылок на внешние ресурсы, такие как проблемы JIRA и билеты Zendesk. Дополнительные сведения см. в статье "[Настройка автоматических ссылок для ссылок на внешние ресурсы](/github/administering-a-repository/configuring-autolinks-to-reference-external-resources)".

Для использования API автоматических ссылок {% data variables.product.prodname_github_apps %} требуются разрешения администратора репозитория с доступом на чтение или запись.
