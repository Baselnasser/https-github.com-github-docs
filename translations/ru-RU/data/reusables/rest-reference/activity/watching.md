## <a name="watching"></a>Наблюдение

При наблюдении за репозиторием пользователь регистрируется для получения уведомлений о новых обсуждениях, а также о событиях в веб-канале активности пользователя. Простые закладки репозитория см. в разделе [Репозиторий в главной роли](/rest/reference/activity#starring).
