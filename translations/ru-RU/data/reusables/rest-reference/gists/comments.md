## <a name="comments"></a>Комментарии

### <a name="custom-media-types-for-gist-comments"></a>Пользовательские типы мультимедиа для комментариев Gist

Это поддерживаемые типы мультимедиа для комментариев gist.

    application/vnd.github.VERSION.raw
    application/vnd.github.VERSION.base64

Дополнительные сведения о типах мультимедиа см. в разделе [Пользовательские типы мультимедиа](/rest/overview/media-types).
