- При необходимости параметры автомасштабирования для каждого пула.
  - Минимальное количество неактивных средства выполнения тестов для пула
  - Активное время ожидания: количество минут, в течение которых средство выполнения тестов должно оставаться активным в пуле, прежде чем пул будет сокращен до количества неактивных средств
