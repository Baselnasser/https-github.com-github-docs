В репозитории создайте файл с именем `.github/workflows/YOUR_WORKFLOW.yml`, где `YOUR_WORKFLOW` — это любое имя на ваш выбор. Это файл рабочего процесса. Дополнительные сведения о создании новых файлов в GitHub см. в разделе [Создание новых файлов](/github/managing-files-in-a-repository/creating-new-files).
