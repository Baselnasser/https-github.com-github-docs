Используйте `jobs.<job_id>.defaults` для создания `map` с параметрами по умолчанию, которые будут применяться ко всем шагам задания. Вы также можете задать параметры по умолчанию для всего рабочего процесса. Дополнительные сведения см. на веб-сайте [`defaults`](/actions/using-workflows/workflow-syntax-for-github-actions#defaults).
