**Предупреждение.** Единственными поддерживаемыми поставщиками хранилища S3 являются Amazon S3 и шлюз MinIO для NAS. Другие поставщики хранилища, совместимые с API S3, не поддерживаются. Дополнительные сведения о поддерживаемых поставщиках хранилища см. в разделе [Начало работы с GitHub Actions для сервера GitHub Enterprise](/admin/github-actions/getting-started-with-github-actions-for-github-enterprise-server#external-storage-requirements).
