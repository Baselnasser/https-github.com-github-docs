Если определено несколько параметров по умолчанию с одинаковым именем, {% data variables.product.prodname_dotcom %} использует наиболее конкретный из них. Например, параметр по умолчанию, указанный в задании, переопределит параметр по умолчанию с тем же именем, указанным в рабочем процессе.
