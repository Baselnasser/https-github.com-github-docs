{% data variables.product.prodname_actions %} предоставляются бесплатно для стандартных средств выполнения, размещенных на {% data variables.product.prodname_dotcom %} в общедоступных репозиториях, и для локальных средств выполнения. Для частных репозиториев каждая учетная запись {% data variables.product.prodname_dotcom %} получает определенное количество бесплатных минут и место в хранилище для использования со средствами выполнения, размещенными на {% data variables.product.prodname_dotcom %}, в зависимости от продукта, который применяется для этой учетной записи. Использование с превышением предусмотренных объемов регулируется предельными суммами расходов.
