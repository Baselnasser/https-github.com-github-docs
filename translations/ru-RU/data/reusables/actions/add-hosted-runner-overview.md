Вы можете выбрать операционную систему и конфигурацию оборудования из списка доступных вариантов. При развертывании новых экземпляров этого средства выполнения с помощью автомасштабирования они будут использовать ту же операционную систему и конфигурацию оборудования, которую вы определили здесь.

Вы также можете определить метки для средства выполнения, чтобы рабочие процессы могли отправлять задания в средства выполнения для обработки (с помощью `runs-on`). Новые средства выполнения автоматически назначаются группе по умолчанию, или вы можете выбрать, к какой группе должны присоединяться средства выполнения во время создания. Кроме того, после регистрации средства выполнения вы можете изменить его группу. Дополнительные сведения см. в разделе [Управление доступом к {% data variables.actions.hosted_runner %}](/actions/using-github-hosted-runners/controlling-access-to-larger-runners).
