Каждый раз при создании выпуска в {% data variables.product.product_name %} можно активировать рабочий процесс для публикации образа. Рабочий процесс в приведенном ниже примере выполняется при активации события `release` с типом действия `created`. Дополнительные сведения о событии `release` см. в статье "[События, активирующие рабочие процессы](/actions/reference/events-that-trigger-workflows#release)".
