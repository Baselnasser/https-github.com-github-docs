**Примечание**. По соображениям безопасности общедоступные репозитории не могут использовать средства выполнения в группе средств выполнения по умолчанию, но это поведение можно переопределить в параметрах такой группы. Дополнительные сведения см. в разделе [Управление доступом к средствам выполнения тестов локального размещения с помощью групп](/actions/hosting-your-own-runners/managing-access-to-self-hosted-runners-using-groups#changing-the-access-policy-of-a-self-hosted-runner-group).
