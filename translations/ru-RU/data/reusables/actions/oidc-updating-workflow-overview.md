Чтобы добавить интеграцию OIDC в рабочие процессы облачного развертывания, необходимо добавить следующие изменения кода:

- Предоставьте разрешение на получение токена от поставщика OIDC {% data variables.product.prodname_dotcom %}:
  - Для рабочего процесса требуется параметр `permissions` с заданным значением `id-token`. Это позволяет получать токен OIDC из каждого задания в рабочем процессе. Если необходимо получить токен OIDC только для одного задания, такое разрешение можно установить в этом задании.
- Запросите JSON Web Token (JWT) от поставщика OIDC {% data variables.product.prodname_dotcom %} и предоставьте его поставщику облачных служб для получения маркера доступа:
  - Вы можете использовать набор средств Actions для получения токенов в задании или использовать официальное действие, созданное поставщиком облачных служб, для извлечения JWT и получения маркера доступа из облака.
