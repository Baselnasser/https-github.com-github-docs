{% data variables.product.product_name %} предлагает начальные рабочие процессы развертывания для нескольких популярных служб, таких как веб-приложение Azure. Чтобы узнать, как начать работу с начальным рабочим процессом, см. раздел [Использование начальных рабочих процессов](/actions/learn-github-actions/using-starter-workflows) или [просмотрите полный список начальных рабочих процессов развертывания](https://github.com/actions/starter-workflows/tree/main/deployments). Можно также ознакомиться с более подробными руководствами по конкретным рабочим процессам развертывания, такими как [Развертывание в Службе приложений Azure](/actions/deployment/deploying-to-azure-app-service).

Многие поставщики служб также предлагают действия на {% data variables.product.prodname_marketplace %} для развертывания в своей службе. Полный список см. в разделе [{% data variables.product.prodname_marketplace %}](https://github.com/marketplace?category=deployment&type=actions).
