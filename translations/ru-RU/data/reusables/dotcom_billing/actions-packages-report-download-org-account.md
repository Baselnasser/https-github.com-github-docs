1. При необходимости рядом с пунктом "Выставление счетов и планы" щелкните **Получить отчет об использовании**, чтобы отправить отчет об использовании хранилища для {% data variables.product.prodname_actions %}, {% data variables.product.prodname_registry %} и {% data variables.product.prodname_codespaces %} в формате CSV. Это сообщение электронной почты отправляется на основной адрес электронной почты вашей учетной записи. Вы можете выбрать период отчета: 7, 30, 90 или 180 дней.
   ![Загрузить отчет в формате CSV](/assets/images/help/billing/actions-packages-report-download.png)
