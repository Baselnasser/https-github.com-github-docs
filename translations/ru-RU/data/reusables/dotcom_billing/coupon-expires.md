Если вы используете купон для оплаты подписки, по истечении срока действия купона будет взиматься полная стоимость подписки посредством выбранного метода оплаты. Если у вас нет сохраненного метода оплаты, ваша учетная запись будет понижена до {% data variables.product.prodname_free_user %} для личных учетных записей или {% data variables.product.prodname_free_team %} для организаций.
