1. Чтобы настроить {% data variables.product.prodname_copilot %}, откройте Neovim и введите приведенную ниже команду.
  
   ```
   :Copilot setup
   ```
  
1. Включите {% данных variables.product.prodname_copilot %} в конфигурации Neovim или с помощью команды Neovim.
  
   ```
   :Copilot enable
   ```
