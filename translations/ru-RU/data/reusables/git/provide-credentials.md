1. При первом использовании Git для клонирования репозитория или доступа к данным в репозитории, который вы уже клонировали, Git запрашивает учетные данные. Укажите {% данных variables.product.pat_generic %} для учетной записи с доступом к репозиторию.

   Git кэширует {% данных variables.product.pat_generic %} для каталога, в который вы находитесь, и вы сможете получать доступ к данным репозитория и записывать их в {% данных variables.location.product_location %}, используя правильную учетную запись.
