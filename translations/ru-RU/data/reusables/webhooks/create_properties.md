Ключ | Тип | Описание
----|------|-------------
`ref`|`string` | Ресурс [`git ref`](/rest/reference/git#get-a-reference).
`ref_type`|`string` | Тип объекта Git ref, созданного в репозитории. Может быть либо `branch`, либо `tag`.
`master_branch`|`string` | Имя ветви по умолчанию в репозитории (обычно это `main`).
`description`|`string` | Текущее описание репозитория.
