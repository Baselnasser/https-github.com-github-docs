Ключ | Тип | Описание
----|------|-------------
`action`|`string` | Действие, которое было выполнено. Возможные значения:<ul><li> `added` — пользователь принимает приглашение в репозиторий.</li><li>`removed` — пользователь удален из числа участников совместной работы в репозитории.</li><li>`edited` — изменены разрешения участника совместной работы для пользователя. </li></ul>
