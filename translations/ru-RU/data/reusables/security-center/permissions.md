Владельцы и руководители службы безопасности организации могут получить доступ к обзору безопасности для организаций{% ifversion ghec or ghes > 3.4 or ghae-issue-6199 %} и просматривать репозитории своей организации с помощью обзора безопасности корпоративного уровня. Владельцы предприятия могут использовать обзор безопасности корпоративного уровня для просмотра всех репозиториев в организациях предприятия{% endif %}. Участники команды могут просматривать общие сведения о безопасности для репозиториев, для которых у команды есть права администратора.
