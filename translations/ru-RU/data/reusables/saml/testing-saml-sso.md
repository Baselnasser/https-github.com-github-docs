При настройке единого входа SAML в организации можно протестировать свою реализацию, не затрагивая участников организации, для этого нужно оставить флажок **Require SAML SSO authentication for all members of the _название организации_ organization** (Требовать проверку подлинности единого входа SAML для всех участников организации "название организации") снятым.
