| Действие | Описание
|------------------|-------------------
| `external_group.delete` | Активируется при удалении группы Okta. Дополнительные сведения см. в разделе [Сопоставление групп Okta с командами](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams).
| `external_group.link` | Активируется при сопоставлении группы Okta с вашей командой {% data variables.product.prodname_ghe_managed %}. Дополнительные сведения см. в разделе [Сопоставление групп Okta с командами](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams).
| `external_group.provision` | Активируется при сопоставлении группы Okta с командой в {% data variables.product.prodname_ghe_managed %}. Дополнительные сведения см. в разделе [Сопоставление групп Okta с командами](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams).
| `external_group.unlink` | Активируется, когда группа Okta не сопоставлена с вашей командой {% data variables.product.prodname_ghe_managed %}. Дополнительные сведения см. в разделе [Сопоставление групп Okta с командами](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams).
| `external_group.update` | Активируется при обновлении параметров группы Okta. Дополнительные сведения см. в разделе [Сопоставление групп Okta с командами](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams).
