Когда {% data variables.projects.projects_v1_board %} закрывается, если {% data variables.projects.projects_v1_board %} имеет автоматизированные рабочие процессы, они будут приостановлены. Если {% data variables.projects.projects_v1_board %} открывается повторно, вы можете синхронизировать автоматизацию, и тогда положение карточек на доске обновляется в соответствии с параметрами автоматизации, настроенными для проекта. Дополнительные сведения см. в статье [Повторное открытие закрытого компонента "{% data variables.product.prodname_project_v1 %}"](/articles/reopening-a-closed-project-board) или [Закрытие компонента "{% data variables.product.prodname_project_v1 %}"](/articles/closing-a-project-board).
