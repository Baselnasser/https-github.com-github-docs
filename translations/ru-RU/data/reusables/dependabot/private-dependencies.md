В настоящее время {% data variables.product.prodname_dependabot_version_updates %} не поддерживает файлы манифеста или блокировки, содержащие какие-либо частные зависимости Git или частные реестры Git. Это связано с тем, что при выполнении обновлений версии {% data variables.product.prodname_dependabot %} должны иметь возможность разрешать все зависимости из их источника, чтобы убедиться, что обновления версий выполнены успешно.
