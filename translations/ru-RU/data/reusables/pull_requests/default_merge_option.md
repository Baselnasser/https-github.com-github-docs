Если щелкнуть параметр запроса на **вытягивание** слиянием по умолчанию на {% данных variables.location.product_location %}, все фиксации из ветви компонентов добавляются в базовую ветвь в фиксации слияния. Запрос на вытягивание объединяется с помощью [ параметра `--no-ff`](https://git-scm.com/docs/git-merge#_fast_forward_merge).

Чтобы объединить запросы на вытягивание, необходимо иметь [разрешения на запись](/articles/repository-permission-levels-for-an-organization/) в репозиторий.
