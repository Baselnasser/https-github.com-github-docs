### Пользовательские типы мультимедиа

Этот интерфейс API поддерживает следующие типы мультимедиа:

    application/vnd.github.raw+json
    application/vnd.github.text+json
    application/vnd.github.html+json
    application/vnd.github.full+json

Дополнительные сведения о типах мультимедиа см. в разделе [Пользовательские типы мультимедиа](/rest/overview/media-types).
