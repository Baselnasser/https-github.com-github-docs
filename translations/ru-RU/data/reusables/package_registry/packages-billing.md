За использование {% data variables.product.prodname_registry %} не взимается плата для общедоступных пакетов. Для частных пакетов каждая учетная запись на {% данных variables.location.product_location %} получает определенное количество свободного хранилища и передачи данных в зависимости от продукта, используемого с учетной записью. Использование с превышением предусмотренных объемов регулируется предельными суммами расходов.
