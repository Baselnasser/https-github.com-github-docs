При первой публикации пакета для параметра видимости по умолчанию выбирается закрытый доступ. Если пакет связан с репозиторием, видимость пакета зависит от видимости репозитория. Сведения об изменении видимости или настройке разрешений доступа см. в разделе [Настройка контроля доступа и видимости пакета](/packages/learn-github-packages/configuring-a-packages-access-control-and-visibility).
