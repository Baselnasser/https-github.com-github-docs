Сначала вам может быть предложено авторизовать приложение OAuth.{% ifversion not ghae %} Если для вашей учетной записи или организации требуется [двухфакторная проверка подлинности](/github/authenticating-to-github/securing-your-account-with-two-factor-authentication-2fa), вам также потребуется пройти запрос двухфакторной проверки подлинности.{% endif %}
