1. Если для {% data variables.product.prodname_enterprise %} настроена двухфакторная проверка подлинности, выполните одно из указанных ниже действий.
    - Если настроена двухфакторная проверка подлинности по SMS, получите код проверки в SMS-сообщении.
    - Если настроена двухфакторная проверка подлинности через приложение с одноразовыми паролями с ограниченным сроком действия, создайте код проверки.

  После этого введите код проверки в запросе в {% data variables.product.prodname_desktop %} и щелкните **Войти**.
    ![Поле для ввода кода двухфакторной проверки подлинности](/assets/images/help/desktop/mac-2fa-code-prompt.png)
