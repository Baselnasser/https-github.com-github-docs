1. Чтобы обновить версию проекта до последнего выпуска, используйте ветвь `stable`, выполнив команду `git checkout stable`.

  ```shell
  git checkout stable
  ```
  
  А если необходимо использовать конкретную версию проекта, выполните следующую команду, заменив `X.Y.Z` на нужную версию выпуска.

  ```shell
  $ git checkout vX.Y.Z
  ```
