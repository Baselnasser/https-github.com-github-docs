1. Для проверки, что обновление установлено, выполните следующую команду.
  ```shell
  ./bin/ghe-backup --version
  ```

1. Чтобы проверить SSH-подключение настроенных продуктов {% data variables.product.prodname_ghe_server %}, выполните следующую команду.
  ```shell
  ./bin/ghe-host-check
  ```
