По умолчанию отображаются только события за последние три месяца. Чтобы увидеть старые события, необходимо указать диапазон дат с параметром `created`. Для получения дополнительной информации см. раздел [Основные сведения о различных ролях](/search-github/getting-started-with-searching-on-github/understanding-the-search-syntax#query-for-dates).
