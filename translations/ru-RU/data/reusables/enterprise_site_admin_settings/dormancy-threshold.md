Порог состояния бездействия — это продолжительность времени, в течение которого пользователь должен быть неактивным, чтобы считаться бездействующим. Порог дорманности по умолчанию составляет 90 дней, однако можно настроить порог дорманности для {% данных variables.location.product_location %}.
