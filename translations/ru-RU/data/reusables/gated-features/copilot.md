{% data variables.product.prodname_copilot %} доступно клиентам {% data variables.product.company_short %} с личной учетной записью в {% data variables.product.prodname_dotcom_the_website %} и {% data variables.product.prodname_ghe_cloud %}, принадлежащих корпоративной учетной записи.<br><br>

{% data variables.product.prodname_copilot %} бесплатно используется для проверенных учащихся, преподавателей и ведения популярных проектов открытый код. Если вы не являетесь учащимся, преподавателем или не являетесь руководителем популярного проекта открытый код, вы можете попробовать {% data variables.product.prodname_copilot %} бесплатно с помощью однократной 60-дневной пробной версии. После пробного периода вы сможете приобрести платную подписку. Дополнительные сведения см. в статье [Сведения о выставлении счетов за {% data variables.product.prodname_copilot %}](/billing/managing-billing-for-github-copilot/about-billing-for-github-copilot).
