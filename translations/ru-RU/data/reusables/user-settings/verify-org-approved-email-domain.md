Если организация, членом которой вы являетесь, ограничивает уведомления по электронной почте утвержденным доменом электронной почты, вам нужно будет подтвердить адрес электронной почты в этом домене, чтобы получать уведомления по электронной почте о деятельности в организации. Дополнительные сведения см. в статье "[Ограничение уведомлений по электронной почте утвержденным доменом](/articles/restricting-email-notifications-to-an-approved-domain)".
