{% tip %}

  **Совет.** Если приложению требуется доступ на уровне организации, то в этом случае только владелец организации может приобрести, установить или отменить приложение, а также управлять выставлением счетов за приложение для организации. Если приложению не требуется доступ на уровне организации, администратор репозитория может устанавливать и удалять приложение.

{% endtip %}
