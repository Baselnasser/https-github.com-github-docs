{% ifversion ghec %} Если ваша лицензия включает {% data variables.visual_studio.prodname_vss_ghe %}, вы можете определить, успешно ли соответствует учетная запись пользователя в {% data variables.product.prodname_dotcom_the_website %} подписчику {% data variables.product.prodname_vs %}, скачав CSV-файл, содержащий дополнительные сведения о лицензии. Лицензия будет иметь одно из следующих состояний:
- "Соответствует": учетная запись пользователя на сайте {% data variables.product.prodname_dotcom_the_website %} связана с подписчиком {% data variables.product.prodname_vs %}.
- "Приглашение без ответа": подписчику {% data variables.product.prodname_vs %} было отправлено приглашение, но подписчик его не принял. 
- Пусто: связь {% data variables.product.prodname_vs %} с учетной записью пользователя на сайте {% data variables.product.prodname_dotcom_the_website %} отсутствует.
