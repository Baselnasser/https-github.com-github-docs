1. Перейдите к настройкам учетной записи.
   - Для приложения {% data variables.product.prodname_github_app %}, принадлежащего учетной записи пользователя, нажмите на фото своего профиля в правом верхнем углу любой страницы и выберите **Настройки**.
    ![Значок "Настройки" на панели пользователя](/assets/images/settings/userbar-account-settings_post2dot12.png)
   - Для приложения {% data variables.product.prodname_github_app %}, принадлежащего организации, нажмите на фото своего профиля в правом верхнем углу любой страницы и выберите **Ваши организации**. Затем нажмите **Настройки** справа от организации.
    ![Ваша организация в меню профиля](/assets/images/help/profile/your-organizations.png)
    ![Кнопка "Настройки"](/assets/images/help/organizations/settings-button.png)
