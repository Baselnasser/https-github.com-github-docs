Разрешение на запись для API проверки доступно только для приложений GitHub. Приложения OAuth и пользователи, прошедшие проверку подлинности, могут просматривать проверки и наборы проверок, но не могут их создавать. Если вы не создаете приложение GitHub, вам может пригодиться [API состояний](/rest/reference/commits#commit-statuses).
