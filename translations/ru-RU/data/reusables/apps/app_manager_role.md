По умолчанию управлять настройками приложений GitHub в организации могут только владельцы организации. Чтобы разрешить дополнительным пользователям управлять приложениями GitHub в организации, владелец может предоставить им разрешения диспетчера приложений GitHub. Сведения о добавлении и удалении диспетчеров приложений GitHub в организации см. в статье [Диспетчеры приложений GitHub](/articles/permission-levels-for-an-organization/#github-app-managers).
