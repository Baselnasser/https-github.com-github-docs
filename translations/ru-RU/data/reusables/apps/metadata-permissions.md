Если вы задали для метаданных разрешение `No access` и выбрали разрешение, требующее доступа к репозиторию, GitHub переопределит выбранный вами параметр и вернет разрешению метаданных значение `Read-only`. Чтобы задать для метаданных разрешение `No access`, необходимо сначала задать для всех разрешений, которые требуют доступа к репозиторию, значение `No access`.
