Для каждого пользователя поставщик удостоверений SAML и клиент SCIM должны использовать соответствующие значения `NameID` и `userName`. Это позволяет пользователю выполнять проверку подлинности с помощью SAML, чтобы установить связь с подготовленным удостоверением SCIM.
