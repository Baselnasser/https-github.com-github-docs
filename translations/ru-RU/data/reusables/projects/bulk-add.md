  ![Снимок экрана: раскрывающийся список репозиториев](/assets/images/help/projects-v2/add-bulk-select-repo.png)
1. Выберите проблемы и запросы на вытягивание, которые вы хотите добавить.
  ![Снимок экрана: выбор проблем и запросов на вытягивание для добавления](/assets/images/help/projects-v2/add-bulk-select-issues.png)
