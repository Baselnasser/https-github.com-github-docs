Команды — это группы участников организации, которым можно предоставить разрешения для определенных репозиториев в рамках отдельной группы. Вы можете создавать отдельные команды или несколько уровней вложенных команд в каждой из своих организаций. Дополнительные сведения см. в разделах [Создание команд](/organizations/organizing-members-into-teams/creating-a-team) и [Добавление людей в команды](/organizations/organizing-members-into-teams/adding-organization-members-to-a-team).
