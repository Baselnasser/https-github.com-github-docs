1. При необходимости выберите репозитории, представленные в вашем профиле {% data variables.product.prodname_sponsors %}.
    - Справа от заголовка "Избранная работа" нажмите кнопку **Изменить**.
      ![Кнопка "Изменить" для избранной работы](/assets/images/help/sponsors/featured-work-edit-button.png)
    - Выберите до 6 репозиториев и нажмите кнопку **Сохранить**.
      ![Флажки для выбора репозиториев](/assets/images/help/sponsors/featured-work-select.png)
