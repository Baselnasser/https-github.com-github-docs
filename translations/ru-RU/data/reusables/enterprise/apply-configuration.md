1. Чтобы применить конфигурацию, введите следующую команду.

   {% note %}
   
   **Примечание.** Во время выполнения конфигурации службы на {% данных variables.location.product_location %} могут перезапуститься, что может привести к краткому простою для пользователей.

   {% endnote %}

    ```shell
    ghe-config-apply
    ```
1. Подождите завершения запуска конфигурации.
