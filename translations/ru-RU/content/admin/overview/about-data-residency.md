{% данных reusables.github-ae.github-ae-enables-you %} Вы можете выбрать географию, в которой хранятся все данные клиента для {% данных variables.location.product_location %}, а члены предприятия могут получить доступ к {% данных variables.product.product_name %} из любой точки мира.

При создании предприятия можно указать {% data variables.contact.contact_enterprise_sales %}, где следует хранить данные клиентов. {% data variables.product.company_short %} не будет хранить данные вашего предприятия вне выбранного географического региона или перемещать их за его пределы.

Дополнительные сведения о доступных географических регионах можно получить в {% data variables.contact.contact_enterprise_sales %}.
