## Разрешение или запрет создания обсуждений пользователям с доступом на чтение

По умолчанию участники организации с доступом на чтение могут создавать обсуждения, если администратор репозитория или владелец организации включил обсуждения для репозитория, принадлежащего организации.

{% data reusables.profile.access_org %} {% data reusables.profile.org_settings %} {% data reusables.organizations.member-privileges %}
5. В разделе "Обсуждения репозитория" установите или снимите флажок **Разрешить пользователям с доступом на чтение создавать обсуждения**.
  ![Флажок, позволяющий пользователям с доступом на чтение создавать обсуждения](/assets/images/help/discussions/toggle-allow-users-with-read-access-checkbox.png)
6. Выберите команду **Сохранить**.
  ![Кнопка "Сохранить" для параметров обсуждений](/assets/images/help/discussions/click-save.png)
