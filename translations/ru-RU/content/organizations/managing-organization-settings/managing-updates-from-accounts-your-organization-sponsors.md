Разработчики и организации, которые спонсирует ваша организация, могут отправлять вам обновления о своей работе. Вы можете изменять адрес электронной почты для получения таких обновлений.

Кроме того, обновления из учетных записей, которые спонсирует ваша организация, можно отключить. Дополнительные сведения см. в разделе [Управление спонсорством](/sponsors/sponsoring-open-source-contributors/managing-your-sponsorship#managing-email-updates-for-your-sponsorship).

{% data reusables.profile.access_org %} {% data reusables.profile.org_settings %}
1. В разделе "Адрес электронной почты для уведомления спонсоров (частный)" введите адрес электронной почты, на который вы хотите получать обновления из учетных записей, которые спонсирует ваша организация.
    ![Текстовое поле для ввода адреса электронной почты для получения обновлений из спонсируемых учетных записей](/assets/images/help/sponsors/organization-update-email-textbox.png)
1. Нажмите **Обновить профиль**.
    ![Кнопка "Обновить профиль"](/assets/images/help/organizations/update-profile-button.png)
