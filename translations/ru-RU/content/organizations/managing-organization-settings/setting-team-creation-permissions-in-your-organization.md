Владельцы организации могут настраивать разрешения на создание команды.

Если вы не настроите разрешения на создание команды, все члены организации смогут создавать команды по умолчанию.

{% data reusables.profile.access_org %} {% data reusables.profile.org_settings %} {% data reusables.organizations.member-privileges %}
5. В разделе "Правила создания команд" установите или снимите флажок **Разрешить участникам создавать команды**.
![Флажок, позволяющий участникам создавать команды](/assets/images/help/organizations/allow-members-to-create-teams.png)
6. Выберите команду **Сохранить**.
