4. Выберите приложение GitHub, владение которым нужно передать.
![Выбор приложения](/assets/images/github-apps/github_apps_select-app.png) {% data reusables.user-settings.github_apps_advanced %}
6. Нажмите кнопку **Передать владения**.
![Кнопка для передачи права владения](/assets/images/github-apps/github_apps_transfer_ownership.png)
7. Введите имя приложения GitHub, которое требуется перенести.
![Поле для ввода имени приложения для переноса](/assets/images/github-apps/github_apps_transfer_app_name.png)
8. Введите имя пользователя или организации, в которую вы хотите перенести приложение GitHub.
![Поле для ввода имени пользователя или организации, в который выполняется перенос](/assets/images/github-apps/github_apps_transfer_new_owner.png)
9. Нажмите кнопку **Передать это приложение GitHub**.
![Кнопка для подтверждения переноса приложения GitHub](/assets/images/github-apps/github_apps_transfer_integration.png)
