Приложения GitHub по умолчанию не имеют разрешений. При создании приложения GitHub можно выбрать разрешения, необходимые для доступа к данным конечного пользователя. Разрешения также можно добавлять и удалять. Дополнительные сведения см. в разделе [Изменение разрешений приложения GitHub](/apps/managing-github-apps/editing-a-github-app-s-permissions/).
