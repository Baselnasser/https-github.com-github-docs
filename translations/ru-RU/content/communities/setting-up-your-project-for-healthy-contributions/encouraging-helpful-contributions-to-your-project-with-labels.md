Вы можете применить метку `good first issue` к проблемам в общедоступном репозитории, чтобы пользователи могли их найти при поиске по меткам. Дополнительные сведения о поиске по меткам см. в статье "[Поиск проблем и запросов на вытягивание](/search-github/searching-on-github/searching-issues-and-pull-requests/#search-by-label)".

{% data variables.product.prodname_dotcom %}использует алгоритм для определения наиболее доступных проблем в каждом репозитории и их размещения в различных местах на сайте {% data variables.product.prodname_dotcom_the_website %}. Добавление метки `good first issue` может увеличить вероятность доступности проблем.
