# GraphQL

В каталоге `/content/graphql` находится документация по API GraphQL для GitHub.

* В каталогах `/content/graphql/guides` и `/content/graphql/overview` содержатся статьи, доступные для редактирования человеком.
* Каталог `/content/graphql/reference` содержит по одной статье для каждого типа данных GraphQL, используемого в API GraphQL для GitHub. Большая часть содержимого этого каталога отображается с помощью тегов `include`.

  Содержимое, отображаемое с помощью тегов `include`, берется из каталога `/lib/graphql/static`, который автоматически создается из исходного кода API внутри GitHub и не должен изменяться человеком. Более подробную информацию см. в разделе [`/lib/graphql/README.md`](/lib/graphql/README.md).

  **Поэтому мы не можем принимать вклады в содержимое справки по API GraphQL в этом репозитории.**
