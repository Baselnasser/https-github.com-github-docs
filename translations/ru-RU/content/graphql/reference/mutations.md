## Сведения об изменениях

Каждая схема GraphQL имеет корневой тип как для запросов, так и для изменений. [Тип изменения](https://graphql.github.io/graphql-spec/June2018/#sec-Type-System) определяет операции GraphQL, изменяющие данные на сервере. Это аналогично выполнению HTTP-команд, таких как `POST`, `PATCH` и `DELETE`.

Дополнительные сведения см. в разделе [Сведения об изменениях](/graphql/guides/forming-calls-with-graphql#about-mutations).
