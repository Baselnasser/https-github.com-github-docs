## Сведения о скалярах

[Скаляры](https://graphql.github.io/graphql-spec/June2018/#sec-Scalars) — это примитивные значения: `Int`, `Float`, `String`, `Boolean` или `ID`.

При вызове API GraphQL необходимо указывать вложенные подполя, пока не будут возвращены только скаляры.

Дополнительные сведения см. в разделе [Общие сведения о GraphQL](/graphql/guides/introduction-to-graphql#field).
