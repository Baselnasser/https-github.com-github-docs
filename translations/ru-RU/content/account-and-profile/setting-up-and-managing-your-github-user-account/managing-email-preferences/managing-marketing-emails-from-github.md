## <a name="unsubscribing-from--data-variablesproductprodname_dotcom--marketing-emails"></a>Отмена подписки на маркетинговые сообщения электронной почты из {% data variables.product.prodname_dotcom %}

{% tip %}

**Совет.** Если вы отмените подписку на все маркетинговые сообщения электронной почты, а затем подпишитесь на информационный бюллетень Explore, вы будете получать только информационный бюллетень Explore и не будете получать другие маркетинговые сообщения.

{% endtip %}
