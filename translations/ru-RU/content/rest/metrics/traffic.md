## Сведения об API трафика репозитория

Для репозиториев, к которым у вас есть доступ на отправку, API трафика репозитория обеспечивает доступ к информации, предоставленной в графе репозитория. Дополнительные сведения см. в разделе "[Просмотр трафика в репозиторий](/repositories/viewing-activity-and-data-for-your-repository/viewing-traffic-to-a-repository)".
