## Сведения о API обсуждений в команде

Командные обсуждения предназначены для бесед, которые не относятся к репозиторию или проекту. Любой участник команды [организации](/rest/reference/orgs) может создавать и читать записи в общедоступном обсуждении. Дополнительные сведения см. в разделе [Сведения об обсуждениях в командах](//organizations/collaborating-with-your-team/about-team-discussions/). Дополнительные сведения о комментариях к записи в обсуждении см. в разделе [API комментариев к обсуждениям команды](/rest/reference/teams#discussion-comments).
