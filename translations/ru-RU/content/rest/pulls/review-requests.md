## Сведения об API запросов на проверку

Авторы запросов на вытягивание, владельцы репозиториев и участники совместной работы могут запрашивать проверку запросов на вытягивание у любого пользователя с доступом на запись в репозиторий. Каждый запрошенный рецензент получит уведомление с просьбой проверить запрос на вытягивание.
