## Сведения об API кодексов поведения

Для получения сведений о кодексе поведения репозитория можно использовать API кодексов поведения. Чтобы получить кодекс поведения репозитория, используйте конечную точку [Получить репозиторий](/rest/reference/repos#get-a-repository).
