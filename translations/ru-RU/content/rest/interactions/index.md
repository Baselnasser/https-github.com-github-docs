Пользователи взаимодействуют с репозиториями, комментируя, открывая проблемы и создавая запросы на вытягивание. API взаимодействия позволяют пользователям с доступом владельца или администратора временно ограничить взаимодействие с общедоступными репозиториями определенному типу пользователя.
