## Сведения об API миграции организаций

API миграции доступен только для владельцев организаций, прошедших проверку подлинности. Дополнительные сведения см. в разделах [Роли в организации](/organizations/managing-peoples-access-to-your-organization-with-roles/roles-in-an-organization#permission-levels-for-an-organization) и [Другие методы проверки подлинности](/rest/overview/other-authentication-methods).
