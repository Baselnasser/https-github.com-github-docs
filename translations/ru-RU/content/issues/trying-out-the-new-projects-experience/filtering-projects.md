## <a name="about-filtering-projects"></a>Сведения о фильтрации проектов

Для настройки представлений можно использовать фильтры метаданных элементов, таких как исполнители и метки, применяемые к проблемам, а также поля в проекте. Фильтры можно объединять и сохранять как представления. Дополнительные сведения см. в разделе [Настройка представлений проекта](/issues/trying-out-the-new-projects-experience/customizing-your-project-views).

Чтобы отфильтровать проект, нажмите на {% octicon "filter" aria-label="The Filter icon" %} и начните вводить поля и значения, которые вы хотите отфильтровать. По мере ввода будут отображаться возможные значения. Вы также можете открыть палитру команды и ввести "Фильтр по" для выбора из доступных фильтров.

Те же фильтры доступны для аналитических сведений, что позволяет фильтровать данные, которые используются для создания диаграмм. Дополнительные сведения см. в разделе [Использование аналитических сведений с проектами](/issues/trying-out-the-new-projects-experience/using-insights-with-projects).
