## Включение или отключение {% data variables.product.prodname_discussions %} для репозитория

{% data reusables.discussions.enabling-or-disabling-github-discussions-for-your-repository %}
1. Чтобы отключить обсуждения, в разделе «Компоненты» снимите флажок **Обсуждения**.

Можно также использовать обсуждения в организации для содействия беседам, которые охватывают несколько репозиториев в организации. Дополнительные сведения см. в разделе [Включение или отключение GitHub Discussions для организации](/organizations/managing-organization-settings/enabling-or-disabling-github-discussions-for-an-organization).
