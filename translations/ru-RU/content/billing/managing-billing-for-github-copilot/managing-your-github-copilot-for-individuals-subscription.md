## Настройка пробной версии {% data variables.product.prodname_copilot %} для личной учетной записи

Прежде чем приступить к использованию {% data variables.product.prodname_copilot %}, необходимо настроить бесплатную пробную версию или подписку.

{% data reusables.copilot.signup-procedure %}

## Изменение подписки {% data variables.product.prodname_copilot %}

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.billing_plans %}
1. В разделе {% data variables.product.prodname_copilot %} откройте раскрывающееся меню **Правка**.
   - Если вы получаете счета каждый месяц, выберите **Изменить на ежегодное выставление счетов**.
   - Если вы получаете счета каждый год, выберите **Изменить на ежемесячное выставление счетов**.

   ![Снимок экрана: раздел {% data variables.product.prodname_copilot %} на странице выставления счетов с выделенным раскрывающимся списком "Изменить"](/assets/images/help/copilot/copilot-settings-edit-dropdown.png)
   

## Отмена подписки {% data variables.product.prodname_copilot %}

Вы можете в любое время изменить свою подписку {% data variables.product.prodname_copilot %}. При отмене в течение 60-дневной пробной версии плата не будет взиматься. Если вы не отмените подписку на {% data variables.product.prodname_copilot %} до окончания 60-дневной пробной версии, плата будет взиматься в соответствии с периодом выставления счетов, выбранным при настройке пробной версии.

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.billing_plans %}
1. В разделе {% data variables.product.prodname_copilot %} откройте раскрывающееся меню **Правка**.
   - Если у вас есть платная подписка, нажмите кнопку **Отменить {% data variables.product.prodname_copilot %}** .
   
   ![Снимок экрана: раздел {% data variables.product.prodname_copilot %} на странице выставления счетов с выделенным параметром отмены {% data variables.product.prodname_copilot %}](/assets/images/help/copilot/copilot-billing-edit-dropdown.png)

   - Если вы используете 60-дневный пробный период, нажмите кнопку **Отменить пробную версию**.
   
   ![Снимок экрана: раздел {% data variables.product.prodname_copilot %} на странице выставления счетов с выделенным вариантом отмены пробной версии](/assets/images/help/copilot/copilot-cancel-trial.png)

2. В диалоговом окне "Отменить {% data variables.product.prodname_copilot %}" щелкните **Я понимаю, отменить {% data variables.product.prodname_copilot %}**

   ![Снимок экрана: {% data variables.product.prodname_copilot %} отмена модального действия с выделенным элементом "Я понимаю, отменить {% data variables.product.prodname_copilot %}](/assets/images/help/copilot/copilot-cancel-modal.png)

   - Если вы находитесь в течение 60-дневного пробного периода, щелкните **Я понимаю, отменить {% data variables.product.prodname_copilot %} пробную версию**.

   ![Снимок экрана: {% data variables.product.prodname_copilot %} отмена пробной версии с выделенным элементом "Я понимаю, кнопка отмены пробной версии"](/assets/images/help/copilot/copilot-trial-cancel-modal.png)
