## Обращение к поставщику приложения за поддержкой

Каждое приложение в {% data variables.product.prodname_marketplace %} содержит ссылку для обращения в службу поддержки приложения.

{% data reusables.marketplace.visit-marketplace %}
2. Перейдите на страницу приложения.
3. На левой боковой панели в разделе "Средства разработчика" нажмите кнопку **Поддержка**.
4. На странице поддержки приложения следуйте инструкциям, чтобы связаться со службой поддержки приложения.
