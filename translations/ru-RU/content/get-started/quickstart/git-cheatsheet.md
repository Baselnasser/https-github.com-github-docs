Освоить сразу все доступные команды GIT может быть непросто. Для быстрого получения справки по часто используемым командам можно обращаться к [памятке по GIT](https://training.github.com/). Памятка по использованию GIT доступна на нескольких языках. 

Кроме того, ознакомьтесь со страницей [обучающих материалов по GIT и GitHub](/articles/git-and-github-learning-resources/), которая содержит ссылки на руководства, видео и многое другое.
