![Заголовок сохраненного ответа](/assets/images/help/settings/saved-replies-title.png)

5. Чтобы просмотреть ответ, нажмите кнопку **Предварительный просмотр**.
![Добавление сохраненного ответа](/assets/images/help/settings/saved-replies-preview.png)
6. Щелкните **Добавить сохраненный ответ**.
![Кнопка "Добавить сохраненный ответ"](/assets/images/help/settings/saved-replies-add-button.png)
