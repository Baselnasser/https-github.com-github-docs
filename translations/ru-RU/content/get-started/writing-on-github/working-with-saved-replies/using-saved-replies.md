**Советы**
- Комментарий можно автоматически заполнить сохраненным ответом с помощью сочетания клавиш. Дополнительные сведения см. в разделе [Сочетания клавиш](/articles/keyboard-shortcuts/#comments).
- Чтобы отфильтровать список, введите заголовок сохраненного ответа.
