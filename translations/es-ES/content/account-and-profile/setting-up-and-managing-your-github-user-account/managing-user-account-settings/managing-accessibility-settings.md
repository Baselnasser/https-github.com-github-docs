## <a name="about-accessibility-settings"></a>Acerca de los ajustes de accesibilidad

{% data variables.product.product_name %} incluye varios atajaos de teclado para que puedas realizar acciones en todo el sitio sin utilizar tu mouse para navegar. Si bien los atajos son útiles para ahorrar tiempo, algunas veces estos pueden hacer que {% data variables.product.prodname_dotcom %} sea más difícil de utilizar y menos accesible.

Todos los atajos de teclado se habilitan predeterminadamente en {% data variables.product.product_name %}, pero puedes elegir inhabilitar los atajos con teclas de caracteres en tus ajustes de accesibilidad. Este valor no afecta a los métodos abreviados de teclado que proporciona el explorador ni a los de {% data variables.product.prodname_dotcom %} que utilizan una tecla modificadora, como <kbd>Control</kbd> o <kbd>Comando</kbd>.

## <a name="managing-character-key-shortcuts"></a>Administrar los atajos de tecla de carácter

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.accessibility_settings %}
1. Active o desactive la casilla **Habilitar métodos abreviados de teclado de caracteres**.
   ![Captura de pantalla de la casilla "Habilitar métodos abreviados de teclado de caracteres"](/assets/images/help/settings/disable-character-key-shortcuts.png)
2. Haga clic en **Save**(Guardar).
