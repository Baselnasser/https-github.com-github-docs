## <a name="-data-variablesproductprodname_education--for-educators"></a>{% data variables.product.prodname_education %} para educadores

Con los servicios y las herramientas de {% data variables.product.prodname_education %} para educadores de todos los niveles, puedes:
  - Use [{% data variables.product.prodname_classroom %}](https://classroom.github.com) para distribuir el código, proporcionar comentarios a los alumnos y recopilar asignaciones en {% data variables.product.prodname_dotcom %}.
  - Únase a nuestra [{% data variables.product.prodname_education_community %}](https://github.com/orgs/community/discussions/categories/github-education) para debatir tendencias actuales sobre educación tecnológica con colegas de todo el mundo.
  - [Solicite una bolsa de productos de {% data variables.product.prodname_dotcom %}](https://github.com/orgs/community/discussions/13) con materiales educativos y productos para los alumnos.
  {% data reusables.education.apply-for-team %}

## <a name="-data-variablesproductprodname_education--for-researchers"></a>{% data variables.product.prodname_education %} para investigadores

Con los servicios y las herramientas de {% data variables.product.prodname_education %} para investigadores, puedes:
  - Colaborar con otros en tu trabajo de investigación en todo el mundo en {% data variables.product.prodname_dotcom %}.
  - [Obtenga información](https://education.github.com/stories) sobre cómo las instituciones académicas de todo el mundo usan {% data variables.product.prodname_dotcom %} para su investigación.
  {% data reusables.education.apply-for-team %}
