## Acerca de el código de los alumnos y los IDE

Si configuras un ambiente de desarrollo integrado (IDE) para una tarea, puedes ejecutar el código dentro de éste. No necesitas clonar el repositorio de la tarea en tu computadora.

Para más información sobre los IDE, vea "[Integración de {% data variables.product.prodname_classroom %} con un IDE](/education/manage-coursework-with-github-classroom/integrate-github-classroom-with-an-ide/integrate-github-classroom-with-an-ide)".

## Ejecutar el código de los alumnos en el IDE 

{% data reusables.classroom.sign-into-github-classroom %} {% data reusables.classroom.click-classroom-in-list %} {% data reusables.classroom.click-assignment-in-list %}
1. A la derecha de un envío, haga clic en **Ver IDE**.
