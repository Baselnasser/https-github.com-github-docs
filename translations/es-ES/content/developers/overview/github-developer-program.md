¿Estás creando una aplicación que se integra con GitHub? ¡Regístrate para nuestro Programa de Desarrollador! Las posibilidades son infinitas, y disfrutarás del reconocimiento de la comunidad. [Regístrese ahora](https://github.com/developer/register)

## <a name="scratch-an-itch"></a>Quítate la tentación

Crea tus propias herramientas que se integren de manera continua con el lugar en donde cargas código todos los días.

## <a name="take-on-the-enterprise"></a>Házte cargo de la empresa

Envíe un correo electrónico a <a href="mailto:partnerships@github.com">partnerships@github.com</a> para solicitar licencias de desarrollador para compilar y probar la aplicación en {% data variables.product.prodname_ghe_server %}.

## <a name="have-an-integration-that-works-with-github"></a>¿Tienes una integración que opera con GitHub?

Genial. Nos gustaría que fueras parte del programa. Aquí te mostramos como puedes correr la voz:</p>
* [Infórmenos sobre su integración](https://support.github.com/contact?tags=rr-general-technical&form[subject]=New+GitHub+Integration)
* Use el [logotipo de Octocat o GitHub](https://github.com/logos) para identificar que el producto funciona con GitHub
* Publica un video o un blog en tu sitio web que explique esta integración

## <a name="ready-to-join-the-github-developer-program"></a>¿Estás listo para unirte al Programa de Desarrollador de GitHub?

La membrecía está disponible para los desarrolladores individuales y para las comañías que tienen:

* Una integración en producción o desarrollo que utilice la API de GitHub.
* Una dirección de correo electrónico en donde los usuarios de GitHub puedan contactarte para soporte.
