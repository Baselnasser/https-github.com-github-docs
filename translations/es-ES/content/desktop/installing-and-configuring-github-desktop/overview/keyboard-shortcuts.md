Atajos de teclado de GitHyb Desktop en macOS

## Atajos en todo el sitio

| Método abreviado de teclado | Descripción
|-----------|------------
|<kbd>Comando</kbd>+<kbd>,</kbd> | Ir a Preferences (Preferencias)
|<kbd>Comando</kbd>+<kbd>H</kbd> | Ocultar la aplicación {% data variables.product.prodname_desktop %}
|<kbd>Opción</kbd>+<kbd>Comando</kbd>+<kbd>H</kbd> | Ocultar todas las otras aplicaciones
|<kbd>Comando</kbd>+<kbd>Q</kbd> | Dejar {% data variables.product.prodname_desktop %}
|<kbd>Control</kbd>+<kbd>Comando</kbd>+<kbd>F</kbd> | Alternar vista de pantalla completa
|<kbd>Comando</kbd>+<kbd>0</kbd> | Restablecer zoom al tamaño de texto predeterminado
|<kbd>Comando</kbd>+<kbd>=</kbd> | Acercar para textos y gráficos más grandes
|<kbd>Comando</kbd>+<kbd>-</kbd> | Alejar para textos y gráficos más pequeños
|<kbd>Opción</kbd>+<kbd>Comando</kbd>+<kbd>I</kbd> | Alternar herramientas de desarrollo

## Repositorios

| Método abreviado de teclado | Descripción
|-----------|------------
|<kbd>Comando</kbd>+<kbd>N</kbd> | Agregar un repositorio nuevo
|<kbd>Comando</kbd>+<kbd>O</kbd> | Agregar un repositorio local
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>O</kbd> | Clonar un repositorio desde {% data variables.product.prodname_dotcom %}
|<kbd>Comando</kbd>+<kbd>T</kbd> | Mostrar una lista de tus repositorios
|<kbd>Comando</kbd>+<kbd>P</kbd> | Subir las últimas confirmaciones a {% data variables.product.prodname_dotcom %}
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>P</kbd> | Bajar los últimos cambios de {% data variables.product.prodname_dotcom %}
|<kbd>Comando</kbd>+<kbd>Eliminar</kbd> | Eliminar un repositorio existente
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>G</kbd> | Ver el repositorio en {% data variables.product.prodname_dotcom %}
|<kbd>Control</kbd>+<kbd>&grave;</kbd> | Abrir el repositorio en tu herramienta de terminal preferida
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>F</kbd> | Mostrar el repositorio en Finder
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>A</kbd> | Abrir el repositorio en tu herramienta de editor preferida
|<kbd>Comando</kbd>+<kbd>I</kbd> | Crear un informe de problemas en {% data variables.product.prodname_dotcom %}

## Ramas

| Método abreviado de teclado | Descripción
|-----------|------------
|<kbd>Comando</kbd>+<kbd>1</kbd> | Mostrar todos los cambios antes de confirmar
|<kbd>Comando</kbd>+<kbd>2</kbd> | Mostrar tu historial de confirmaciones
|<kbd>Comando</kbd>+<kbd>B</kbd> | Mostrar todas tus ramas
|<kbd>Comando</kbd>+<kbd>G</kbd> | Ir al campo de resumen de confirmaciones
|<kbd>Comando</kbd>+<kbd>Entrar</kbd> | Confirma los cambios cuando el campo de resumen o de descripción está activo
|<kbd>Espacio</kbd>| Selecciona o deselecciona todos los archivos resaltados
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>N</kbd> | Creación de una rama
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>R</kbd> | Renombrar la rama actual
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>D</kbd> | Eliminar la rama actual
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>U</kbd> | Actualizar desde la rama predeterminada
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>B</kbd> | Comparar con una rama existente
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>B</kbd> | Fusionar en una rama actual
|<kbd>Control</kbd>+<kbd>H</kbd> | Mostrar u ocultar cambios acumulados
|<kbd>Mayús</kbd>+<kbd>Comando</kbd>+<kbd>C</kbd> | Comparar ramas en {% data variables.product.prodname_dotcom %}
|<kbd>Comando</kbd>+<kbd>R</kbd> | Mostrar la solicitud de extracción actual en {% data variables.product.prodname_dotcom %}
