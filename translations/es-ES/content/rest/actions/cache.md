## Acerca de la API de caché

La API de caché de {% data variables.product.prodname_actions %} permite consultar y administrar la caché de {% data variables.product.prodname_actions %} para repositorios. {% ifversion actions-cache-management %}También puedes instalar una extensión de {% data variables.product.prodname_cli %} para administrar las memorias caché desde la línea de comandos. {% endif %}Para obtener más información, consulta "[Almacenamiento en caché de dependencias para acelerar los flujos de trabajo](/actions/advanced-guides/caching-dependencies-to-speed-up-workflows#managing-caches)".
