## Acerca de la API de instalaciones de {% data variables.product.prodname_github_app %}

Una _instalación_ hace referencia a cualquier cuenta de usuario u organización que haya instalado la aplicación. Para obtener información sobre cómo autenticarse como una instalación y limitar el acceso a repositorios específicos, vea "[Autenticación como una instalación](/apps/building-github-apps/authenticating-with-github-apps/#authenticating-as-an-installation)".

Para enumerar todas las instalaciones de aplicaciones de GitHub para una organización, vea "[Enumeración de instalaciones de aplicaciones para una organización](/rest/orgs/orgs#list-app-installations-for-an-organization)".
