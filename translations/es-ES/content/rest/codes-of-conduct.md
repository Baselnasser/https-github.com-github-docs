## Acerca de la API de códigos de conducta

Puedes usar la API de Códigos de Conducta para recuperar la información acerca del código de conducta de un repositorio. Para obtener un código de conducta de un repositorio, usa el punto de conexión "[Obtención de un repositorio](/rest/reference/repos#get-a-repository)".
