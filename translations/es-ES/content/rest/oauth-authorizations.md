## Acerca de la API de autorizaciones de OAuth

Puedes utilizar esta API para administrar el acceso que las aplicaciones de OAuth tienen en tu cuenta. Solo puede acceder a esta API mediante la [autenticación básica](/rest/overview/other-authentication-methods#basic-authentication) con el nombre de usuario y la contraseña, no con tokens.

Si usted o los usuarios tienen habilitada la autenticación en dos fases, asegúrese de comprender cómo [trabajar con la autenticación en dos fases](/rest/overview/other-authentication-methods#working-with-two-factor-authentication).
