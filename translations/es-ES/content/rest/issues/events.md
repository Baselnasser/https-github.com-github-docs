## About the Issue events API

The Issue events API can return different types of events triggered by activity in issues and pull requests. For more information about the specific events that you can receive from the Issue events API, see "[Issue event types](/developers/webhooks-and-events/issue-event-types)." Para obtener más información acerca de los eventos específicos que puedes recibir de la API de Eventos para Solicitudes de Extracción, consulta la sección "[Tipos de evento de las Solicitudes de Extracción](/developers/webhooks-and-events/issue-event-types)". Para obtener más información, consulta la "[API de Eventos](/developers/webhooks-and-events/github-event-types)".
