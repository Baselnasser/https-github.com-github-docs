## Acerca de las cuentas nuevas de {% data variables.product.prodname_dotcom_the_website %}

Puedes crear una cuenta personal, lo cual funciona como tu identidad en {% data variables.product.prodname_dotcom_the_website %} o una organización, la cual permite cuentas personales múltiples para colaborar en varios proyectos. Para más información sobre los tipos de cuenta, vea "[Tipos de cuentas de {% data variables.product.prodname_dotcom %}](/get-started/learning-about-github/types-of-github-accounts)".

Cuando creas una cuenta personal o de organización, debes seleccionar un plan de facturación para ella. Para más información, vea "[Productos de {% data variables.product.company_short %}](/get-started/learning-about-github/githubs-products)".

## Registrarse para una cuenta nueva

1. Si quieres crear una cuenta personal nueva, asegúrate de haber cerrado sesión en GitHub.
{% data reusables.accounts.create-account %}
1. Sigue las indicaciones para crear tu cuenta personal o de organización.
