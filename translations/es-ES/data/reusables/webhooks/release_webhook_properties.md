Clave | Tipo | Descripción
----|------|-------------
`action`|`string` | La acción que se ha realizado. Puede ser una de las siguientes:<ul><li>`published`: se publica una versión, una versión preliminar o un borrador de una versión</li><li>`unpublished`: se elimina una versión o una versión preliminar</li><li>`created`: se guarda un borrador, o se publica una versión o una versión preliminar sin que se haya guardado previamente como un borrador</li><li>`edited`: se edita una versión, una versión preliminar o una versión de borrador</li><li>`deleted`: se elimina una versión, una versión preliminar o una versión de borrador</li><li>`prereleased`: se crea una versión preliminar</li><li>`released`: se publica una versión o se cambia una versión preliminar a una versión</li>
