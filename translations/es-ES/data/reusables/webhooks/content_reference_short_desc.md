Una nueva referencia de contenido es `created`. Se crea una referenci de contenido nueva cuando el cuerpo o el comentario de un informe de problemas o solicitud de extracción incluye un URL que empte con un dominio de referencia de contenido configurado. Para más información sobre las referencias de contenido y los datos adjuntos, vea "[Uso de datos adjuntos de contenido](/apps/using-content-attachments/)".
