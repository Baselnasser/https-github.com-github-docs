1. Compruebe el campo `repository` en el archivo *package.json* del proyecto. El campo `repository` debe coincidir con la URL del repositorio {% data variables.product.prodname_dotcom %}. Por ejemplo, si la URL del repositorio es `github.com/my-org/test`, el campo del repositorio debe ser `https://github.com/my-org/test.git`.
