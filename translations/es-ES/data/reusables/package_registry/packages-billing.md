El uso de {% data variables.product.prodname_registry %} es gratuito para los paquetes públicos. En el caos de los paquetes privados, cada cuenta en {% data variables.product.product_location %} recibe cierta cantidad de almacenamiento gratuito y transferencia de datos, dependiendo de los productos que se utilicen con la cuenta. Cualquier uso que supere las cantidades incluidas se controla mediante límites de gasto.
