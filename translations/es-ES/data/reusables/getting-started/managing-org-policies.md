Puedes administrar permisos y políticas para varias acciones y características diferentes de tu organización.

Por ejemplo, para proteger los datos de tu organización, puedes restringir la creación de repositorios en la misma. También puedes elegir permitir o prevenir la bifurcación de los repositorios privados que pertenecen a tu organización. Para más información, vea "[Restricción de la creación de repositorios en la organización](/organizations/managing-organization-settings/restricting-repository-creation-in-your-organization)" y "[Administración de la directiva de bifurcación para la organización](/organizations/managing-organization-settings/managing-the-forking-policy-for-your-organization)".

Para obtener la lista completa de opciones que puede configurar para la organización, vea "[Administración de la configuración de la organización](/organizations/managing-organization-settings)".
