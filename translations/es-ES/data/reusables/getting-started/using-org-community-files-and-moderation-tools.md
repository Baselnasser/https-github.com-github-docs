Puedes crear archivos de salud comunitaria, tales como el archivo CONTRIBUTING.md, un archivo CODE_OF_CONDUCT.md o incluso plantillas de propuestas y de solicitudes de cambios para tu organziación. Estos archivos predeterminados pueden utilizarse para cualquier repositorio que le pertenezca a tu organización y no contenga su propio archivo de este tipo. Para más información, vea "[Creación de un archivo de estado de la comunidad predeterminado](/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file)".
