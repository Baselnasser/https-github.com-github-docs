1. Visite la página [Canjear cupón](https://github.com/redeem).
2. En el cuadro "Escribir el código del cupón", escriba el código de cupón y haga clic en **Canjear**.
   ![Cuadro para canjear el cupón](/assets/images/help/settings/redeem-coupon-box.png)
3. Si no has iniciado sesión, puedes iniciar sesión en esta página, o crear una nueva cuenta para utilizar tu cupón.
