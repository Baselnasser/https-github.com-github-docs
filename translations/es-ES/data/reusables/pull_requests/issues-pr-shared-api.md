Cada solicitud de extracción es un informe de problemas, pero no todos los informes de problemas son una solicitud de extracción. Por este motivo, las acciones "compartidas" de ambas características, como la administración de los usuarios asignados, las etiquetas y los hitos, se proporcionan en Issues API.
