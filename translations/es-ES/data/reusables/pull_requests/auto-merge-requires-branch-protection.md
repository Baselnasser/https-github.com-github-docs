**Nota**: La opción para habilitar la fusión automática solo se muestra en las solicitudes de incorporación de cambios que no se pueden fusionar de manera inmediata. Por ejemplo, cuando una regla de protección de rama tiene la opción de "Requerir revisiones de solicitudes de cambio antes de fusionar" o "Requerir verificaciones de estado para pasar antes de fusionar" y estas condiciones aún no se cumplen. Para más información, vea "[Administración de una regla de protección de rama](/github/administering-a-repository/defining-the-mergeability-of-pull-requests/managing-a-branch-protection-rule)".

{% endnote %}
