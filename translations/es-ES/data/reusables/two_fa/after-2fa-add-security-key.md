Después de que configures la 2FA, utilizando una app de contraseña de una sola ocasión limitada en tiempo (TOTP){% ifversion fpt or ghec %} o mediante mensaje de texto{% endif %}, puedes agregar una llave de seguridad, como una huella dactilar o Windows Hello. La tecnología que habilita la autenticación con una llave de seguridad se llama WebAuthn. WebAuthn es el sucesor de la autenticación bifactorial y funciona en todos los buscadores modernos. Para obtener más información, vea "[WebAuthn](https://webauthn.guide/)" y "[Puedo usar](https://caniuse.com/#search=webauthn)".
