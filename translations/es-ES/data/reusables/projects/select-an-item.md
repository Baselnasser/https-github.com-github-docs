1. Selecciona el elemento. Para seleccionar elementos múltiples, realiza alguna de las siguientes acciones:
     - Mantén presionada la tecla <kbd>Comando</kbd> (Mac) o <kbd>Ctrl</kbd> (Windows/Linux), y haz clic en cada elemento.
     - Selecciona un elemento y, después, presiona <kbd>Mayús</kbd>+<kbd>↑</kbd> o <kbd>Mayús</kbd>+<kbd>↓</kbd> para seleccionar elementos adicionales por encima o por debajo del seleccionado inicialmente.
     - Selecciona un elemento y luego presiona <kbd>Mayús</kbd> y haz clic en otro elemento para seleccionar todos los que estén entre estos dos.
     - Con una fila o un elemento ya seleccionado, presiona <kbd>Comando</kbd>+<kbd>A</kbd> (Mac) o <kbd>Ctrl</kbd>+<kbd>A</kbd> (Windows o Linux) para seleccionar todos los elementos de una columna en un diseño de panel o todos los elementos de un diseño de tabla.
