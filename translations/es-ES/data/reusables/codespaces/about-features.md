Las características son unidades independientes de código de instalación y configuración del contenedor de desarrollo diseñadas para funcionar en una amplia gama de imágenes de contenedor base. Puedes usar características para agregar rápidamente herramientas, runtimes o bibliotecas a la imagen de codespace. Para obtener más información, consulta las [características disponibles](https://containers.dev/features) y la [especificación de las características](https://containers.dev/implementors/features/) en containers.dev.

Puedes agregar características a un archivo `devcontainer.json` desde {% data variables.product.prodname_vscode_shortname %} o desde tu repositorio en {% data variables.product.prodname_dotcom_the_website %}.
