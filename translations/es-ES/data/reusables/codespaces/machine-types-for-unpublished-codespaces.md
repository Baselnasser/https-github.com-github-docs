Los codespaces no publicados (codespaces creados a partir de una plantilla que no estén vinculados a un repositorio de {% data variables.product.product_name %}) siempre se ejecutarán en una máquina virtual con las mismas especificaciones. No se puede cambiar el tipo de máquina de un codespace no publicado.
