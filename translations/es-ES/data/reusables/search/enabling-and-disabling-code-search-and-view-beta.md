Recibirás un correo electrónico cuando se te acepte en la lista de espera para unirte a la versión beta. Una vez concedido acceso, la nueva búsqueda de código y la vista de código beta se habilitarán automáticamente en tu cuenta.

Puedes deshabilitar o habilitar la versión beta en el {% data variables.product.prodname_dotcom_the_website %} en cualquier momento. Ten en cuenta que esta configuración se aplica tanto a la búsqueda de código como a la vista de código.

{% data reusables.feature-preview.feature-preview-setting  %}
1. A la derecha de "Nueva búsqueda de código y vista de código (beta)", haz clic en **Habilitar** o **deshabilitar**.
