Para mostrar una insignia de "Verificado", el sitio web y la información de correo electrónico que se muestran en el perfil de una organización deben coincidir con el o los dominios verificados. Si el sitio web y la dirección de correo electrónico que se muestran en el perfil de tu organización se alojan en un dominio diferente, debes verificar ambos dominios. Si el sitio web y dirección de correo electrónico utilizan variantes del mismo dominio, debes verificar ambas variantes. Por ejemplo, si el perfil muestra el sitio web `www.example.com` y la dirección de correo electrónico `info@example.com`, tendrá que comprobar tanto `www.example.com` como `example.com`.
