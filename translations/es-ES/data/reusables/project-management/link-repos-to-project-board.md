Puedes vincular hasta 25 repositorios a la instancia de {% data variables.projects.projects_v1_board %} propiedad de la organización o el usuario. La vinculación de repositorios facilita la adición de incidencias y solicitudes de incorporación de cambios desde esos repositorios a la instancia de {% data variables.projects.projects_v1_board %} mediante **{% octicon "plus" aria-label="The Plus icon" %} Agregar tarjetas**, o bien desde la barra lateral de incidencias o solicitudes de incorporación de cambios.
