1. Encima de la lista de archivos, haga clic en {% octicon "download" aria-label="The download icon" %} **Código**.
  ![Botón "Código"](/assets/images/help/repository/code-button.png)
1. Copia la dirección URL del repositorio.

   - Para clonar el repositorio con HTTPS, en «HTTPS» haz clic en {% octicon "clippy" aria-label="The clipboard icon" %}. 
   - Para clonar el repositorio mediante una clave SSH, incluido un certificado emitido por la entidad de certificación SSH de la organización, haga clic en **Usar SSH** y luego en {% octicon "clippy" aria-label="The clipboard icon" %}. 
   - Para clonar un repositorio mediante {% data variables.product.prodname_cli %}, haz clic en **{% data variables.product.prodname_cli %}** y, después, haz clic en {% octicon "clippy" aria-label="The clipboard icon" %}.
  ![El icono del portapapeles para copiar la URL para clonar un repositorio con el CLI de GitHub](/assets/images/help/repository/https-url-clone-cli.png)
