Puedes utilizar {% data variables.product.prodname_desktop %} para mover tus cambios a una rama nueva y confirmarlos. Para más información, vea "[Confirmación y revisión de los cambios en el proyecto](/desktop/contributing-to-projects/committing-and-reviewing-changes-to-your-project)".
