1. Para crear una solicitud de incorporación de cambios que esté lista para revisión, haga clic en **Crear solicitud de incorporación de cambios**.
Para crear un borrador de una solicitud de incorporación de cambios, use el menú desplegable y seleccione **Crear solicitud de incorporación de cambios de borrador**, y después haga clic en **Solicitud de incorporación de cambios de borrador**. Para más información sobre el borrador de solicitudes de incorporación de cambios, vea "[Acerca de las solicitudes de incorporación de cambios](/articles/about-pull-requests#draft-pull-requests)".![Botón para crear una solicitud de incorporación de cambios](/assets/images/help/pull_requests/pullrequest-send.png)
