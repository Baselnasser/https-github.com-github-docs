1. En la esquina superior derecha de la vista del archivo, haga clic en {% octicon "pencil" aria-label="The edit icon" %} para abrir el editor de archivos.

   ![Botón Edit file](/assets/images/help/repository/edit-file-edit-button.png)

   {% ifversion githubdev-editor -%} {% note %}

   **Nota:** En lugar de editar y confirmar el archivo con el editor de archivos predeterminado, puedes optar por usar el [editor de código github.dev](/codespaces/the-githubdev-web-based-editor); para ello, haz clic en el menú desplegable situado junto a {% octicon "pencil" aria-label="The edit icon" %} y selecciona **Abrir en github.dev**.

   ![Menú desplegable del botón Editar archivo](/assets/images/help/repository/edit-file-edit-dropdown.png)

   {% endnote %} {%- endif %}
