Al intentar insertar un secreto admitido en un repositorio u organización con {% data variables.product.prodname_secret_scanning %} como protección de inserción habilitada, {% data variables.product.prodname_dotcom %} bloqueará la inserción. Puedes quitar el secreto de la rama o seguir una dirección URL proporcionada para permitir la inserción.
