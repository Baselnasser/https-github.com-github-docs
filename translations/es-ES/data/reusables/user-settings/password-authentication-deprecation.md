Cuando Git solicite tu contraseña, escribe el token de acceso personal (PAT). También puedes usar un asistente de credenciales como el [Administrador de credenciales de Git](https://github.com/GitCredentialManager/git-credential-manager/blob/main/README.md).{% ifversion not ghae %} La autenticación basada en contraseña para Git se ha quitado en favor de métodos de autenticación más seguros.{% endif %} Para obtener más información, consulta «[Creación de un token de acceso personal](/github/authenticating-to-github/creating-a-personal-access-token)».
