Puedes habilitar o inhabilitar los {% data variables.product.prodname_discussions %} para un repositorio.

{% data reusables.repositories.navigate-to-repo %}
1. Debajo del nombre del repositorio, haga clic en {% octicon "gear" aria-label="The gear icon" %} **Configuración**.
![Botón de configuración del repositorio](/assets/images/help/discussions/public-repo-settings.png)
1. En "Características", seleccione **Debates**.
    ![Casilla debajo de "Características" para habilitar o deshabilitar {% data variables.product.prodname_discussions %} para un repositorio](/assets/images/help/discussions/select-discussions-checkbox.png)
