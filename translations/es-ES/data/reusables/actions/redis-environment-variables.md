El script *client.js* busca las variables de entorno `REDIS_HOST` y `REDIS_PORT` para crear el cliente. El flujo de trabajo configura esas dos variables de entorno como parte del paso "Connect to Redis" para que estén disponibles en el script *client.js*. Para más información sobre el script, vea "[Prueba del contenedor del servicio Redis](#testing-the-redis-service-container)".
