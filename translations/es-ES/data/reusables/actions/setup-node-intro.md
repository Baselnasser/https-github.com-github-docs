Puede usar la acción `setup-node` para crear un archivo *.npmrc* local en el ejecutor que configure el registro y el ámbito predeterminados. La acción `setup-node` también acepta un token de autenticación como entrada, que se usa para acceder a los registros privados o publicar paquetes de nodos. Para más información, vea [`setup-node`](https://github.com/actions/setup-node/).
