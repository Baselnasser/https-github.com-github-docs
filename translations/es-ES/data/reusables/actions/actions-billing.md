El uso de {% data variables.product.prodname_actions %} es gratuito para los ejecutores estándar de {% data variables.product.prodname_dotcom %} hospedados en repositorios públicos y para ejecutores autohospedados. Para los repositorios privados, cada cuenta {% data variables.product.prodname_dotcom %} recibe una cantidad determinada de minutos y almacenamiento gratuitos para usar con ejecutores hospedados {% data variables.product.prodname_dotcom %} dependiendo del producto que se utilice con la cuenta. Cualquier uso que supere las cantidades incluidas se controla mediante límites de gasto.
