Puedes establecer los permisos predeterminados concedidos al `GITHUB_TOKEN`. Para obtener más información sobre el `GITHUB_TOKEN`, consulta [Autenticación de token automática](/actions/security-guides/automatic-token-authentication). Puedes elegir un conjunto restringido de permisos como valor predeterminado o aplicar la configuración permisiva.
