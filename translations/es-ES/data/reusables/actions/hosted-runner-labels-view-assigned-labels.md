1. Localice el ejecutor que quiera actualizar y haga clic en {% octicon "triangle-down" aria-label="The downward triangle" %} para ver el menú de selección de etiquetas. Este menú muestra todas las etiquetas personalizadas disponibles para tu ejecutor. Las etiquetas que ya se hayan asignado a tu ejecutor tienen una {% octicon "check" aria-label="Check mark" %} junto a ellas.
