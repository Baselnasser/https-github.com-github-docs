El script *client.js* busca las variables de entorno `POSTGRES_HOST` y `POSTGRES_PORT` para crear el cliente. El flujo de trabajo configura esas dos variables de entorno como parte del paso "Conexión con PostgreSQL" para que estén disponibles en el script *client.js*. Para obtener más información sobre el script, consulta [Probar el contenedor de servicios de PostgreSQL](#testing-the-postgresql-service-container).
