El secreto de `GITHUB_TOKEN` se configuro para un token de acceso para el repositorio cada vez que comienza un job en un flujo de trabajo. Debes configurar los permisos para este token de acceso en el archivo de flujo de trabajo para otorgar acceso de lectura para el alcance `contents` y de escritura para el alcance `packages`. Para obtener más información, consulta "[Autenticar con el GITHUB_TOKEN](/actions/configuring-and-managing-workflows/authenticating-with-the-github_token)".
