Use `jobs.<job_id>.defaults` para crear un elemento `map` de configuración predeterminada que se aplicará a todos los pasos del trabajo. También puedes configurar ajustes predeterminados para todo el flujo de trabajo. Para obtener más información, vea [`defaults`](/actions/using-workflows/workflow-syntax-for-github-actions#defaults).
