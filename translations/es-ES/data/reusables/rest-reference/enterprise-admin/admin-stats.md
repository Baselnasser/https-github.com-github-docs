## <a name="admin-stats"></a>Estadísticas de los Administradores

La API de estadísticas de los administradores proporciona diversas métricas sobre tu instalación. *Solo está disponible para los administradores de sitios [autenticados](/rest/overview/resources-in-the-rest-api#authentication).* Los usuarios normales recibirán una respuesta `404` si intentan acceder a ella.
