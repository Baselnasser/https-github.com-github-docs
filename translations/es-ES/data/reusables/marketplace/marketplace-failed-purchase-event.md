En caso de que un cliente mejore su plan y el pago falle, GitHub revertirá su suscripción de {% data variables.product.prodname_marketplace %} a su estado previo. GitHub también enviará un correo electrónico al cliente para informarle de dicho fallo y para permitirles volver a intentar la compra. Recibirá un webhook con la acción `changed` en el que se solicita revertir al plan anterior.
