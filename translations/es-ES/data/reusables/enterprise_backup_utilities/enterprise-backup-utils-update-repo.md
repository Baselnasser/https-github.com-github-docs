1. Para actualizar a la versión de lanzamiento del proyecto más reciente, usa la rama `stable` ejecutando el comando `git checkout stable`.

  ```shell
  git checkout stable
  ```
  
  Como alternativa, para usar una versión de proyecto específica, ejecuta el siguiente comando, reemplazando `X.Y.Z` por la versión de lanzamiento deseada.

  ```shell
  $ git checkout vX.Y.Z
  ```
