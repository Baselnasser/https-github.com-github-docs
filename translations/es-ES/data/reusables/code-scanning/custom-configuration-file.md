El archivo de configuración se puede encontrar en el repositorio que va a analizar o en un repositorio externo. El uso de un repositorio externo permite especificar opciones de configuración para varios repositorios en un solo lugar. Al hacer referencia a un archivo de configuración ubicado en un repositorio externo, puede usar la sintaxis _OWNER/REPOSITORY/FILENAME@BRANCH_ . Por ejemplo, _octo-org/shared/codeql-config.yml@main_.
