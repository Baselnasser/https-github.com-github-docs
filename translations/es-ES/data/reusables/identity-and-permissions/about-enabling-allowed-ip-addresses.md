Después de crear una lista de direcciones IP permitidas, puedes habilitar las direcciones IP permitidas. Al habilitar las direcciones IP permitidas, {% data variables.product.company_short %} aplica inmediatamente las entradas habilitadas en la lista de direcciones IP permitidas.

Antes de habilitar la lista de direcciones IP permitidas, puedes comprobar si la lista de permitidos permitirá una conexión desde una dirección IP determinada. Para obtener más información, consulta "[Comprobación de permiso para una dirección IP](#checking-if-an-ip-address-is-permitted)".
