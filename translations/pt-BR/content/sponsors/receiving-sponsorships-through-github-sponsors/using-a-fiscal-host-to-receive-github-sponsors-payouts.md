## Como usar um host fiscal para receber pagamentos do {% data variables.product.prodname_sponsors %}

Ao se inscrever em um perfil do {% data variables.product.prodname_sponsors %} para ser patrocinado, você pode escolher se deseja receber os pagamentos em uma conta do Stripe Connect sob o seu controle ou por meio de um host fiscal com o qual já trabalha.

{% data reusables.sponsors.bank-info-fiscal-host-reminder %}

O {% data variables.product.prodname_sponsors %} dá suporte a estes hosts fiscais:

- [Open Source Collective](https://oscollective.org/)
- [NumFOCUS](http://www.numfocus.org/)
- [Open Collective Europe](https://opencollective.com/europe)
- [Software Underground](https://softwareunderground.org/)
- [Software Freedom Conservancy](https://sfconservancy.org/)

## Como escolher usar um host fiscal

Depois de navegar até a página de configuração do perfil "Ingressar na lista de espera" da conta a ser patrocinada, você verá a seção "Como receber pagamentos" ou "Conta bancária da organização", dependendo do tipo de conta que está sendo configurado. Para obter mais informações de como se inscrever em um perfil do {% data variables.product.prodname_sponsors %}, confira "[Como configurar o {% data variables.product.prodname_sponsors %} para sua conta pessoal](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account)", "[Como configurar o {% data variables.product.prodname_sponsors %} para a organização](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization)".

1. Em "Como receber pagamentos" ou "Conta bancária da organização", selecione a opção **Usar um host fiscal** ou **Esta organização está usando um host fiscal**.

   ![Captura de tela da opção para escolher entre uma conta bancária e um host fiscal](/assets/images/help/sponsors/user-waitlist-payout-options.png)
2. Use a lista suspensa **Selecionar um host fiscal** para escolher o host fiscal que deseja usar.
3. Na caixa de texto em "URL do perfil do projeto de host fiscal", digite a URL na página de perfil no site do host fiscal. Isso ajuda a verificar se você está realmente registrado no host fiscal.
  
  ![Captura de tela da caixa de texto da URL do perfil de projeto do host fiscal](/assets/images/help/sponsors/fiscal-host-project-profile-field.png)
