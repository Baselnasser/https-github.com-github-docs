## Sobre as novas contas em {% data variables.product.prodname_dotcom_the_website %}

É possível criar uma conta pessoal, que serve como sua identidade em {% data variables.product.prodname_dotcom_the_website %}, ou em uma organização, o que permite que várias contas pessoais colaborem em múltiplos projetos. Para obter mais informações sobre os tipos de contas, confira "[Tipos de contas do {% data variables.product.prodname_dotcom %}](/get-started/learning-about-github/types-of-github-accounts)".

Ao criar uma conta pessoal ou organização, você deve selecionar um plano de cobrança para a conta. Para obter mais informações, confira "[Produtos do {% data variables.product.company_short %}](/get-started/learning-about-github/githubs-products)".

## Inscrevendo-se em uma nova conta

1. Se você deseja criar uma nova conta pessoal, certifique-se de que você está desconectado do GitHub.
{% data reusables.accounts.create-account %}
1. Siga as instruções para criar a conta pessoa ou organização.
