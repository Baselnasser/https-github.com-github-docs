{% tip %}

**Dicas:**
- Você pode usar um atalho de teclado para preencher automaticamente o comentário com uma resposta salva. Para obter mais informações, confira "[Atalhos de teclado](/articles/keyboard-shortcuts/#comments)".
- Você pode filtrar a lista digitando o título da resposta salva.
