## Sobre o suporte do navegador da Web para o {% data variables.product.product_name %}

Projetamos o {% data variables.product.product_name %} com os navegadores da Web mais recentes em mente. Recomendamos que você use a última versão de um dos navegadores a seguir.

 - [Apple Safari](https://apple.com/safari)
 - [Google Chrome](https://google.com/chrome)
 - [Microsoft Edge](https://microsoft.com/windows/microsoft-edge)
 - [Mozilla Firefox](https://mozilla.org/firefox)
 
Se você não usar a última versão de um navegador recomendado ou se usar um navegador que não está listado acima, o {% data variables.product.product_name %} ou alguns recursos poderão não funcionar como você espera ou simplesmente não funcionar.

Para obter mais informações sobre como mantemos a compatibilidade do navegador nos produtos do {% data variables.product.company_short %}, confira o repositório [`github/browser-support`](https://github.com/github/browser-support).
## Suporte estendido para navegadores da Web recomendados

Alguns fornecedores de navegador fornecem versões de suporte estendido. Fazemos o possível para garantir que o {% data variables.product.product_name %} funcione corretamente na última versão de suporte estendido para:

- [Canal estável estendido](https://support.google.com/chrome/a/answer/9027636) do Chrome
- [Canal estável estendido](https://docs.microsoft.com/en-gb/deployedge/microsoft-edge-channels#extended-stable-channel) do Edge
- ESR ([versão de suporte estendido](https://www.mozilla.org/en-US/firefox/organizations/)) do Firefox

Nas versões anteriores de suporte estendido, talvez o {% data variables.product.product_name %} não funcione como esperado e alguns recursos não estejam disponíveis.


## Compilações Beta e de desenvolvedor

Podem ocorrer erros inesperados nas compilações Beta e de desenvolvedor dos navegadores compatíveis. Se você encontrar um erro no {% data variables.product.product_name %} em uma dessas compilações ainda não lançadas, verifique se ele também ocorre na versão estável do mesmo navegador. Caso o erro só exista na versão instável, considere a possibilidade de relatar o erro ao desenvolvedor do navegador.
