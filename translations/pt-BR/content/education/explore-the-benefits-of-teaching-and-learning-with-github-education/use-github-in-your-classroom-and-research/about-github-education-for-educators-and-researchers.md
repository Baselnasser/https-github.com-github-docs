## <a name="-data-variablesproductprodname_education--for-educators"></a>{% data variables.product.prodname_education %} para educadores

Com os serviços e ferramentas do {% data variables.product.prodname_education %} para educadores de todos os níveis, você pode:
  - Use o [{% data variables.product.prodname_classroom %}](https://classroom.github.com) para distribuir código, fornecer comentários aos alunos e recolher tarefas no {% data variables.product.prodname_dotcom %}.
  - Junte-se à nossa [{% data variables.product.prodname_education_community %}](https://github.com/orgs/community/discussions/categories/github-education) para discutir as tendências atuais na educação tecnológica com colegas de todo o mundo.
  - [Solicite um kit surpresa do {% data variables.product.prodname_dotcom %}](https://github.com/orgs/community/discussions/13) com materiais e brindes educacionais para os alunos.
  {% data reusables.education.apply-for-team %}

## <a name="-data-variablesproductprodname_education--for-researchers"></a>{% data variables.product.prodname_education %} para pesquisadores

Com os serviços e ferramentas do {% data variables.product.prodname_education %} para pesquisadores, você pode:
  - Colaborar com outras pessoas no trabalho de pesquisa ao redor do mundo no {% data variables.product.prodname_dotcom %}.
  - [Saiba](https://education.github.com/stories) como as instituições acadêmicas em todo o mundo estão usando o {% data variables.product.prodname_dotcom %} para pesquisas.
  {% data reusables.education.apply-for-team %}
