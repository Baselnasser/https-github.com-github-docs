## Sobre o {% data variables.product.prodname_community_exchange %}

{% data reusables.education.about-github-community-exchange-intro %}

![Captura de tela da página do Community Exchange](/assets/images/help/education/community-exchange-page.png)

Você pode ajudar seus colegas a aprender habilidades de código aberto, tornar-se um mantenedor de projeto e aumentar seu portfólio e rede de {% data variables.product.prodname_dotcom %} em uma plataforma de comunidade segura e verificada.

{% data variables.product.prodname_community_exchange %} permite:
- Descobrir repositórios criados por estudantes
- Marcar repositórios de interesse com estrela
- Carregar um repositório que precisa de colaboradores
- Carregar um repositório para ensinar novas habilidades
- Gerenciar seus repositório enviados

Para obter mais informações, confira "[Colaborar com o GitHub Community Exchange](/education/contribute-with-github-community-exchange)."
