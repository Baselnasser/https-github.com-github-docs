## Sobre seus envios

Durante o processo de envio de {% data variables.product.prodname_community_exchange %}, você escolherá uma finalidade, os tópicos e as ofertas para seu repositório. Depois que um repositório for enviado para {% data variables.product.prodname_community_exchange %}, ele será publicado com esses detalhes. Para obter mais informações, confira "[Como enviar seu repositório para o GitHub Community Exchange](/education/contribute-with-github-community-exchange/submitting-your-repository-to-github-community-exchange)".

Depois de enviar, você ainda pode editar os tópicos e ofertas associados ao seu repositório. Você também pode atualizar a finalidade do repositório alterando os tópicos correspondentes. Para obter mais informações, confira "[Como classificar seu repositório com tópicos](/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/classifying-your-repository-with-topics)".

A linguagem associada ao seu repositório é a linguagem primária usada e é determinada automaticamente por {% data variables.product.prodname_dotcom %}. Para obter mais informações, confira "[Sobre linguagens do repositório](/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-repository-languages)".

A comunidade {% data variables.product.prodname_community_exchange %} modera todos os envios de repositório.

## Como gerenciar seus envios

1. No seu painel {% data variables.product.prodname_global_campus %}, navegue até a página inicial do {% data variables.product.prodname_community_exchange %}.
1. Em cima da lista de repositórios, clique na guia **Envios**. ![Captura de tela da guia Envios](/assets/images/help/education/community-exchange-submissions-tab.png)
1. Opcionalmente, edite o repositório enviado. À direita do repositório qeu você deseja editar, clique em {% octicon "pencil" aria-label="The edit icon" %} para acessar diretamente a página inicial do seu repositório. Daí, você pode atualizar a finalidade, os tópicos e as ofertas atribuídos ao seu repositório.
1. Opcionalmente, exclua um repositório enviado da galeria. À direita do envio de repositório que você deseja remover, clique em {% octicon "trash" aria-label="The trash icon" %}.
