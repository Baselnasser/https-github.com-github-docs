## Sobre o código do aluno e os IDEs

Se você configurar um ambiente integrado de desenvolvimento (IDE) para uma atividade, você poderá executar o código dentro do IDE. Você não precisa clonar o repositório da atividade no seu computador.

Para obter mais informações sobre IDEs, confira "[Integrar o {% data variables.product.prodname_classroom %} com um IDE](/education/manage-coursework-with-github-classroom/integrate-github-classroom-with-an-ide/integrate-github-classroom-with-an-ide)".
