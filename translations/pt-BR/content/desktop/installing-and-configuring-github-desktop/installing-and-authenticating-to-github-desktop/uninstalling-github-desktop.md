{% mac %}

1. Em uma janela do Finder, navegue até a pasta Aplicativos.
  ![Pasta Aplicativos na janela do Localizador](/assets/images/help/desktop/applications-folder.png)
2. Enquanto pressiona o botão **CTRL** no teclado, clique em **{% data variables.product.prodname_desktop %}** .
3. Escolha **Mover para a Lixeira**.
  ![A opção Mover para a Lixeira](/assets/images/help/desktop/mac-move-to-trash.png)
4. Na barra de menus, use o menu suspenso **Localizador** e clique em **Esvaziar Lixeira**.
  ![A opção Esvaziar Lixeira na barra de menus](/assets/images/help/desktop/mac-empty-trash-menu.png)
5. Leia o aviso na caixa pop-up e clique em **Esvaziar Lixeira**.
  ![O botão Esvaziar Lixeira](/assets/images/help/desktop/mac-empty-trash-button.png)

{% endmac %}

{% windows %}

1. Abra o Painel de Controle. Para obter mais informações, confira [Onde está o Painel de Controle?](https://support.microsoft.com/en-us/help/13764/windows-where-is-control-panel) na Ajuda do Windows.
2. Em "Programas", clique em **Desinstalar um programa**.
  ![A opção Desinstalar um Programa no Painel de Controle](/assets/images/help/desktop/windows-uninstall-a-program.png)
3. Clique com o botão direito do mouse na entrada chamada **{% data variables.product.prodname_desktop %}** e clique em **Desinstalar**.
  ![A opção Desinstalar](/assets/images/help/desktop/windows-click-uninstall.png)

{% endwindows %}
