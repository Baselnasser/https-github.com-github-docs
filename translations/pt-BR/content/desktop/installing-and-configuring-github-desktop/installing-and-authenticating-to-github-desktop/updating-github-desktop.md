{% mac %}

1. No menu do **GitHub Desktop**, clique em **Sobre o GitHub Desktop**.
![Opção de menu Sobre o GitHub Desktop](/assets/images/help/desktop/desktop-menu-about-desktop-mac.png)
2. Clique em **Verificar se há Atualizações**.
![Botão Verificar se há Atualizações](/assets/images/help/desktop/check-for-updates.png)
3. Se houver alguma atualização disponível, saia e inicie o {% data variables.product.prodname_desktop %} novamente para fazer a instalação.

{% endmac %}

{% windows %}

1. No menu **Ajuda**, clique em **Sobre o GitHub Desktop**.
![Opção de menu Sobre o GitHub Desktop](/assets/images/help/desktop/help-about-desktop-win.png)
2. Clique em **Verificar se há Atualizações**.
![Botão Verificar se há Atualizações](/assets/images/help/desktop/check-for-updates.png)
3. Se houver alguma atualização disponível, saia e inicie o {% data variables.product.prodname_desktop %} novamente para fazer a instalação.

{% endwindows %}
