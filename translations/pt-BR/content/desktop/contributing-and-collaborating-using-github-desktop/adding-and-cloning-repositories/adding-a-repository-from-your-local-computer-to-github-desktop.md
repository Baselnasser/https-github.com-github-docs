{% tip %}

**Dica:** você pode adicionar um repositório Git do computador local ao GitHub Desktop arrastando a pasta para a janela do {% data variables.product.prodname_desktop %}. Ao arrastar várias pastas do Git para o {% data variables.product.prodname_desktop %} simultaneamente, cada pasta será acessada como um repositório Git individual.

{% endtip %}

{% mac %}

1. No menu **Arquivo**, clique em **Adicionar Repositório Local**.
  ![Opção de menu Adicionar Repositório Local](/assets/images/help/desktop/add-local-repository-mac.png)
2. Clique em **Escolher…** e, usando a janela do Localizador, procure o repositório local que deseja adicionar.
  ![Campo Caminho local no aplicativo do Mac](/assets/images/help/desktop/add-repo-choose-button-mac.png)
4. Clique em **Adicionar Repositório**.
  ![Botão Adicionar Repositório no aplicativo do Mac](/assets/images/help/desktop/add-repository-button-mac.png)

{% endmac %}

{% windows %}

1. No menu **Arquivo**, clique em **Adicionar repositório local**.
  ![Opção de menu Adicionar Repositório Local](/assets/images/help/desktop/add-local-repository-windows.png)
2. Clique em **Escolher…** e, usando o Windows Explorer, procure o repositório local que deseja adicionar.
  ![Campo Caminho local no aplicativo do Windows](/assets/images/help/desktop/add-repo-choose-button-win.png)
4. Clique em **Adicionar repositório**.
  ![Botão Adicionar repositório no aplicativo do Windows](/assets/images/help/desktop/add-repository-button-windows.png)

{% endwindows %}
