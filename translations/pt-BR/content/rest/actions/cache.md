## Sobre a API de Cache

A API de Cache do {% data variables.product.prodname_actions %} permite consultar e gerenciar o cache do {% data variables.product.prodname_actions %} em repositórios. {% ifversion actions-cache-management %}Você também pode instalar uma extensão do {% data variables.product.prodname_cli %} para gerenciar seus caches da linha de comando. {% endif %}Para obter mais informações, confira "[Armazenar dependências em cache para acelerar fluxos de trabalho](/actions/advanced-guides/caching-dependencies-to-speed-up-workflows#managing-caches)."
