## Sobre a API de Execuções de Fluxo de Trabalho

A API de Execução de Fluxo de Trabalho permite visualizar, executar novamente, cancelar e visualizar os logs de execuções de fluxo de trabalho. {% data reusables.actions.about-workflow-runs %} Para obter mais informações, confira "[Como gerenciar uma execução de fluxo de trabalho](/actions/automating-your-workflow-with-github-actions/managing-a-workflow-run)".
