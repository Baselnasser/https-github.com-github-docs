## Sobre o banco de dados Git

A API REST oferece acesso para ler e gravar objetos Git brutos em seu banco de dados Git em {% data variables.product.product_name %} e para listar e atualizar suas referências (branch heads e tags). Para obter mais informações sobre como usar a API REST para interagir com seu banco de dados Git, consulte "[Introdução à API de dados Git](/rest/guides/getting-started-with-the-git-database-api)".
