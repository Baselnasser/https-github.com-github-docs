A API do banco de dados do Git dá acesso para ler e gravar objetos do Git sem processamento no seu banco de dados do Git no {% data variables.product.product_name %} e para listar e atualizar suas referências (cabeçalhos de branch e etiquetas). Para obter mais informações sobre como usar a API do Banco de Dados do Git, veja "[Introdução à API de dados do Git](/rest/guides/getting-started-with-the-git-database-api)".
