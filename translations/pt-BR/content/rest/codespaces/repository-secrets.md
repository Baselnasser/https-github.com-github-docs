## Sobre a API de segredos do repositório de codespaces

A API de segredos do repositório codespaces permite que um usuário crie, liste e exclua segredos (como tokens de acesso para serviços de nuvem) de repositórios aos quais o usuário tem acesso. Estes segredos são disponibilizados para o codespace em tempo de execução. Para obter mais informações, confira "[Como gerenciar segredos criptografados para seus codespaces](/codespaces/managing-your-codespaces/managing-encrypted-secrets-for-your-codespaces)".
