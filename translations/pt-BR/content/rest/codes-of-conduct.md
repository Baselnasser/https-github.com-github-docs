## About the Codes of conduct API

Você pode usar a API de Códigos de Conduta para recuperar informações sobre o código de conduta de um repositório. Para obter um código de conduta para um repositório, use o ponto de extremidade "[Obter um repositório](/rest/reference/repos#get-a-repository)".
