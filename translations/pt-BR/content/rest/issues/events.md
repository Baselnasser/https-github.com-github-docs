## Sobre a API de eventos de problema

A API de Eventos de problema pode retornar diferentes tipos de eventos desencadeados por atividades em problemas e pull requests. Para obter mais informações sobre eventos específicos que você pode receber dos eventos da API de problemas, consulte "[Tipos de eventos de problemas](/developers/webhooks-and-events/issue-event-types)". For more information about the specific events that you can receive from the Issue Events API, see "[Issue event types](/developers/webhooks-and-events/issue-event-types)." Para obter mais informações, consulte a "[API de Eventos](/developers/webhooks-and-events/github-event-types)".
