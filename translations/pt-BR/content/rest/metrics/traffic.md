## Sobre a API de tráfego do repositório

Para repositórios aos quais você tem acesso por push, a API de tráfego do repositório fornece acesso às informações fornecidas no gráfico do repositório. Para obter mais informações, confira "[Como ver o tráfego de um repositório](/repositories/viewing-activity-and-data-for-your-repository/viewing-traffic-to-a-repository)".
