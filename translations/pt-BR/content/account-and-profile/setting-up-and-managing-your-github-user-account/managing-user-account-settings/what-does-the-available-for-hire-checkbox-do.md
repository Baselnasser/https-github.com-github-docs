Observação de depreciação: O GitHub Jobs agora está obsoleto. A última data para publicar um trabalho foi 19 de maio de 2021. O site GitHub Jobs foi totalmente encerrado em 19 de agosto de 2021 e agora redireciona o usuário para o aviso de [postagem no blog do GitHub](https://github.blog/changelog/2021-04-19-deprecation-notice-github-jobs-site/), que traz mais informações sobre a reprovação já concluída do GitHub Jobs.

{% endwarning %}

A plataforma de empregos do [GitHub Jobs](https://jobs.github.com/) é uma ótima forma de encontrar vagas na área de tecnologia. Você pode optar por ver trabalhos postados lá no seu painel do GitHub.

![Anúncios do GitHub Jobs no painel](/assets/images/help/settings/jobs-ads-on-dashboard.png)
