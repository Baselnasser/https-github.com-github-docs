O painel Atividade gera gráficos semanais, mensais e anuais informando o número de:
- Novas pull requests;
- Pull requests com merge;
- Novos problemas
- Problemas corrigidos
- Comentários de novos problemas;
- Novos repositórios;
- Novas contas de usuário;
- Novas organizações;
- Novas equipes.

![Painel Atividade](/assets/images/enterprise/activity/activity-dashboard-yearly.png)

## Acessar o painel Atividade

1. Na parte superior de qualquer página, clique em **Explorar**.
![Guia Explorar](/assets/images/enterprise/settings/ent-new-explore.png)
2. No canto superior direito, clique em **Atividade**.
![Botão Atividade](/assets/images/enterprise/activity/activity-button.png)
