{% tip %}

**Dica**: se você quiser manter o acesso a um {% data variables.projects.projects_v1_board %} concluído ou não necessário sem perder o acesso ao seu conteúdo, poderá [fechar o {% data variables.projects.projects_v1_board %}](/articles/closing-a-project-board) em vez de excluí-los.

{% endtip %}

1. Navegue até o {% data variables.projects.projects_v1_board %} que você deseja excluir.
{% data reusables.project-management.click-menu %} {% data reusables.project-management.click-edit-sidebar-menu-project-board %}
4. Clique em **Excluir projeto**.
![Botão Delete project](/assets/images/help/projects/delete-project-button.png) (Excluir projeto)
5. Para confirmar que você deseja excluir o quadro de projeto, clique em **OK**.
