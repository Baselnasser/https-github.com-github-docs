4. Opcionalmente, para alterar o nome do novo gráfico, clique em {% octicon "triangle-down" aria-label="The triangle icon" %}, digite um novo nome e pressione <kbd>Retornar</kbd>. ![Captura de tela que mostra como renomear um gráfico](/assets/images/help/projects-v2/insights-rename.png)
5. Acima do gráfico, digite os filtros para alterar os dados utilizados para a construção do gráfico. Para obter mais informações, consulte "[Filtrando projetos](/issues/planning-and-tracking-with-projects/customizing-views-in-your-project/filtering-projects)".
6. À direita da caixa de texto do filtro, clique em **Salvar alterações**. ![Captura de tela que mostra o botão salvar](/assets/images/help/projects-v2/insights-save-filter.png)
