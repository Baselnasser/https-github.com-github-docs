## Habilitar ou desabilitar {% data variables.product.prodname_discussions %} no seu repositório

{% data reusables.discussions.enabling-or-disabling-github-discussions-for-your-repository %}
1. Para desabilitar discussões, em "Recursos", desmarque **Discussões**.

Você também pode usar discussões de organização para facilitar conversas que abrangem vários repositórios em sua organização. Para obter mais informações, confira "[Habilitar ou desabilitar o GitHub Discussions em uma organização](/organizations/managing-organization-settings/enabling-or-disabling-github-discussions-for-an-organization)".
