# Glossários

Os seguintes arquivos compõem nosso [Glossário do Crowdin](https://support.crowdin.com/glossary/):

* `external.yml` contém entradas do glossário voltadas para o cliente.
  * As cadeias de caracteres de `external.yml` são suporte a condicionais do Liquid. Confira [contributing/liquid-helpers.md](/contributing/liquid-helpers.md).
* `internal.yml` contém entradas usadas apenas por tradutores. Estes termos são exibidos na interface do usuário Crowdin para dar aos tradutores um contexto adicional sobre o que eles estão traduzindo, além de uma string localizada sugerida para esse termo.
* `candidates.yml` contém termos que devem estar potencialmente no glossário interno ou externo, mas que ainda não foram definidos.
