1. Ao lado do projeto que você quer migrar, clique em {% octicon "kebab-horizontal" aria-label="The menu icon" %} para abrir o menu.
1. No menu, clique em **Migrar**.
  ![Captura de tela mostrando a opção Migrar](/assets/images/help/issues/migrate-menu-item.png)
1. No pop-up, selecione o que você quer que aconteça com o projeto depois que ele for migrado. É recomendável selecionar "Fechar o projeto antigo" para que fique claro qual projeto está em uso.
  ![Captura de tela mostrando a caixa de diálogo modal Migrar](/assets/images/help/issues/migrate-option.png)
1. Na parte inferior direita do pop-up, clique em **Iniciar migração**.
  ![Captura de tela mostrando o botão Iniciar migração](/assets/images/help/issues/migrate-start.png)
