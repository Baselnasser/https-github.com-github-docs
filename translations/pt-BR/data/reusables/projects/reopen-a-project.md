1. Clique na guia **Projetos**. ![Captura de tela mostrando o botão Fechar projeto](/assets/images/help/issues/projects-profile-tab.png)
1. Para ver os projetos fechados, clique em **Fechados**.
   ![Captura de tela mostrando o botão Fechar projeto](/assets/images/help/issues/closed-projects-tab.png)
1. Clique no projeto que quer reabrir.
1. No canto superior direito, clique em {% octicon "kebab-horizontal" aria-label="The menu icon" %} para abrir o menu.
  ![Captura de tela mostrando o ícone de menu](/assets/images/help/projects-v2/open-menu.png)
1. No menu, clique em {% octicon "gear" aria-label="The gear icon" %} **Configurações** para acessar as configurações do projeto.
  ![Captura de tela mostrando o item de menu 'Configurações'](/assets/images/help/projects-v2/settings-menu-item.png)
1. Na parte inferior da página, clique em **Reabrir o projeto**.
   ![Captura de tela mostrando o botão Reabrir projeto](/assets/images/help/issues/reopen-project-button.png)
