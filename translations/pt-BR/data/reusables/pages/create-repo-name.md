1. Digite um nome para o repositório e uma descrição opcional. Se você estiver criando um site de usuário ou de organização, seu repositório precisará ser chamado `<user>.github.io` ou `<organization>.github.io`. Se o nome do usuário ou da organização contiver letras maiúsculas, você precisará colocá-las em minúsculas.
Para obter mais informações, confira "[Sobre o {% data variables.product.prodname_pages %}](/articles/about-github-pages#types-of-github-pages-sites)".
