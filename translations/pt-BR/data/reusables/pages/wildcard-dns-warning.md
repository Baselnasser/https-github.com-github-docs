**Aviso:** é altamente recomendável não usar registros DNS curinga, como `*.example.com`. O registro DNS curinga permite que qualquer pessoa hospede um site do {% data variables.product.prodname_pages %} em um dos subdomínios que você tem, mesmo quando eles são verificados. Para obter mais informações, confira "[Como verificar seu domínio personalizado do {% data variables.product.prodname_pages %}](/pages/configuring-a-custom-domain-for-your-github-pages-site/verifying-your-custom-domain-for-github-pages)".

{% endwarning %}
