Caso contrário, se você tiver dados confidenciais no repositório do seu site, talvez queira remover os dados antes da publicação. Para obter mais informações, confira "[Sobre repositórios](/repositories/creating-and-managing-repositories/about-repositories#about-repository-visibility)" e "[Alterar a visibilidade de seu site {% data variables.product.prodname_pages %}](/pages/getting-started-with-github-pages/changing-the-visibility-of-your-github-pages-site)".

Para obter mais informações, confira "[Sobre os repositórios](/repositories/creating-and-managing-repositories/about-repositories#about-repository-visibility)".
{% elsif ghes %} **Aviso**: se o administrador do site tiver habilitado Páginas Públicas, sites {% data variables.product.prodname_pages %} estarão disponíveis publicamente na Internet, mesmo que o repositório do site seja privado ou interno. Para obter mais informações, confira "[Configurar {% data variables.product.prodname_pages %} para sua empresa](/admin/configuration/configuring-github-pages-for-your-enterprise#enabling-public-sites-for-github-pages)" e "[Sobre repositórios](/repositories/creating-and-managing-repositories/about-repositories#about-repository-visibility)".
