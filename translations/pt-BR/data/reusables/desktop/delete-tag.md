1. Clique com o botão direito no commit.

    - Se um commit tiver apenas uma tag, clique em **Excluir Tag TAG NAME**.
  ![Seleção do item de menu Excluir tag](/assets/images/help/desktop/select-delete-tag.png)
    - Se um commit tiver várias tags, posicione o cursor sobre **Excluir tag…** e clique na tag que deseja excluir.
  ![Cursor posicionado sobre o menu Excluir tag](/assets/images/help/desktop/delete-tag-multiple.png)
