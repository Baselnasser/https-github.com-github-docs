Uma equipe [sincronizada com um grupo LDAP](/enterprise/admin/authentication/using-ldap#enabling-ldap-sync) é indicada com um selo do LDAP especial. A lista de membros de uma equipe sincronizada com LDAP só pode ser gerenciada a partir do grupo LDAP que está mapeado.
