Quando o {% data variables.product.prodname_dependabot %} gera solicitações de pull, essas solicitações de pull podem ser referentes a atualizações de _segurança_ ou de _versão_:

- As _{% data variables.product.prodname_dependabot_security_updates %}_ são solicitações de pull automatizadas que ajudam você a atualizar as dependências com vulnerabilidades conhecidas. 
- As _{% data variables.product.prodname_dependabot_version_updates %}_ são solicitações de pull automatizadas que mantêm suas dependências atualizadas, mesmo quando elas não têm nenhuma vulnerabilidade. Para verificar o status das atualizações da versão, acesse a aba Insights do seu repositório e, em seguida, gráfico de dependência e {% data variables.product.prodname_dependabot %}.
