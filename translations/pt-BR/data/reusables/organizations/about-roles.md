Para executar qualquer ação no {% data variables.product.prodname_dotcom %}, como criar uma solicitação de pull em um repositório ou alterar as configurações de cobrança de uma organização, uma pessoa precisa ter acesso suficiente na conta ou no recurso relevante. Esse acesso é controlado por permissões. Uma permissão é a capacidade de executar uma ação específica. Por exemplo, a capacidade de excluir um problema é uma permissão. Uma função é um conjunto de permissões que você pode atribuir a pessoas ou equipes.
