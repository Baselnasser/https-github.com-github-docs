**Observação:** quando você remove um usuário da sua organização, os dados de associação dele são salvos por três meses. É possível restaurar os dados deles ou quaisquer bifurcações privadas que pertençam aos repositórios da organização, se você convidar o usuário para se juntar novamente à organização dentro desse período. Para obter mais informações, confira "[Como restabelecer um ex-membro da sua organização](/articles/reinstating-a-former-member-of-your-organization)".

{% endnote %}
