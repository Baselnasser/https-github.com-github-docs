Quando o Git solicitar sua senha, insira seu PAT (token de acesso pessoal). Como alternativa, você pode usar um auxiliar de credencial como o [Gerenciador de Credenciais do Git](https://github.com/GitCredentialManager/git-credential-manager/blob/main/README.md).{% ifversion not ghae %} A autenticação baseada em senha do Git foi removida para dar lugar a métodos de autenticação mais seguros. {% endif %} Para obter mais informações, veja "[Criando um token de acesso pessoal](/github/authenticating-to-github/creating-a-personal-access-token)".
