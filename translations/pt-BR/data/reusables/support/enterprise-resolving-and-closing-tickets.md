O {% data variables.contact.enterprise_support %} poderá considerar um tíquete resolvido após o fornecimento de uma explicação, uma recomendação, instruções de uso {% ifversion ghae %}ou {% endif %} instruções de solução alternativa{% ifversion ghes %} ou pela recomendação de uma versão disponível que resolva o problema{% endif %}.

{% ifversion ghes or ghae %} Se você instalou programas de software adicionais de terceiros ou fez modificações no {% data variables.product.product_name %} fora das recomendações do {% data variables.contact.enterprise_support %}, o {% data variables.contact.enterprise_support %} poderá solicitar que você remova os programas de software e/ou as modificações ao tentar resolver o problema. Se o problema for corrigido quando os programas de software sem suporte ou as modificações tiverem sido removidos, o {% data variables.contact.enterprise_support %} poderá considerar o tíquete como resolvido.
{% endif %}

{% data variables.contact.enterprise_support %} pode fechar um tíquete se este estiver fora do escopo do suporte ou se várias tentativas de entrar em contato com você ficaram sem resposta. Se o {% data variables.contact.enterprise_support %} fechar um tíquete devido à falta de resposta, você poderá solicitar que o {% data variables.contact.enterprise_support %} reabra o tíquete.
