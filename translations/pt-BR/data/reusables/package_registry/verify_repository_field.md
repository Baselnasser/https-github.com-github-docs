1. Verifique o campo `repository` em *package.json* do seu projeto. O campo `repository` deve corresponder à URL do repositório do {% data variables.product.prodname_dotcom %}. Por exemplo, se a URL do repositório for `github.com/my-org/test`, o campo de repositório deverá ser `https://github.com/my-org/test.git`.
