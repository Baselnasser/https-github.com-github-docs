No repositório, crie um arquivo chamado `.github/workflows/YOUR_WORKFLOW.yml`, substituindo `YOUR_WORKFLOW` por um nome de sua escolha. Este é um arquivo do fluxo de trabaho. Para obter mais informações sobre como criar arquivos no GitHub, confira "[Como criar arquivos](/github/managing-files-in-a-repository/creating-new-files)".
