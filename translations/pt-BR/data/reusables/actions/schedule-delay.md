Observação: o evento `schedule` pode ser atrasado durante períodos de cargas altas de execuções de fluxo de trabalho do {% data variables.product.prodname_actions %}. Os tempos de carregamento altos incluem o início de cada hora. Para diminuir a probabilidade de atraso, agende o fluxo de trabalho para ser executado em uma parte diferente da hora.
