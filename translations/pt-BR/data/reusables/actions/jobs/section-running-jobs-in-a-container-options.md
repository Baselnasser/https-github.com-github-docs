Use `jobs.<job_id>.container.options` para configurar opções adicionais de recurso de contêiner do Docker. Para obter uma lista de opções, confira "[Opções de `docker create`](https://docs.docker.com/engine/reference/commandline/create/#options)".

{% warning %}

**Aviso:** não há suporte para a opção `--network`.

{% endwarning %}
