Use `jobs.<job_id>.defaults` para criar um `map` das configurações padrão que será aplicado a todas as etapas do trabalho. Você também pode definir as configurações-padrão para todo o fluxo de trabalho. Para obter mais informações, confira [`defaults`](/actions/using-workflows/workflow-syntax-for-github-actions#defaults).
