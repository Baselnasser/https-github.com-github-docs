Você não pode usar essa implementação do SCIM com uma conta corporativa ou com um {% data variables.product.prodname_emu_org %}. Se a sua empresa estiver habilitada para {% data variables.product.prodname_emus %}, você deverá usar uma implementação diferente do SCIM. Caso contrário, o SCIM não estará disponível no nível empresarial. Para saber mais, confira "[Configurar o provisionamento scim para {% data variables.product.prodname_emus %}](/admin/identity-and-access-management/managing-iam-with-enterprise-managed-users/configuring-scim-provisioning-for-enterprise-managed-users).".
