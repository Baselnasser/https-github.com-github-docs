A permissão de escrita para a API de verificação só está disponível para aplicativos GitHub. Aplicativos OAuth e usuários autenticados podem ver as execuções de verificação e conjunto de verificações, mas eles não são capazes de criá-los. Se você não estiver criando um Aplicativo do GitHub, talvez você esteja interessado na [API de Status](/rest/reference/commits#commit-statuses).
