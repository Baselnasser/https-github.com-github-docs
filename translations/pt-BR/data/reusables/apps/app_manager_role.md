Por padrão, somente proprietários da organização podem gerenciar as configurações dos aplicativos GitHub em uma organização. Para permitir que usuários adicionais gerenciem os Aplicativos GitHub em uma organização, um proprietário pode conceder-lhes permissões de gerente de aplicativos do GitHub. Confira "[Gerentes de Aplicativos do GitHub](/articles/permission-levels-for-an-organization/#github-app-managers)" para saber como adicionar e remover gerentes de aplicativos do GitHub em sua organização.
