Quando as restrições de acesso do {% data variables.product.prodname_oauth_app %} são habilitadas, os membros da organização e os colaboradores externos não podem autorizar o acesso do {% data variables.product.prodname_oauth_app %} aos recursos da organização. Os integrantes da organização podem solicitar a aprovação do proprietário para {% data variables.product.prodname_oauth_apps %} que gostariam de usar, e os proprietários da organização receberão uma notificação de solicitações pendentes.
