1. Para atualizar para a versão mais recente do projeto, use o branch `stable` executando o comando `git checkout stable`.

  ```shell
  git checkout stable
  ```
  
  Como alternativa, para usar uma versão específica do projeto, execute o comando a seguir, substituindo `X.Y.Z` pela versão desejada.

  ```shell
  $ git checkout vX.Y.Z
  ```
