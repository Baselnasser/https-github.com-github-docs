Após o agrupamento de uma solicitação de pull com a última versão do branch de destino e as alterações à frente dela na fila, se houver verificações de status obrigatórias com falha ou conflitos com o branch base, o {% data variables.product.product_name %} removerá a solicitação de pull da fila. A linha do tempo da solicitação de pull exibirá o motivo pelo qual ela foi removida da fila.
