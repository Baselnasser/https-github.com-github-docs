Para uma pessoa que usa vários ambientes do {% data variables.product.prodname_enterprise %} para consumir apenas uma licença, você precisa sincronizar o uso da licença entre ambientes. Depois, o {% data variables.product.company_short %} eliminará a duplicação de usuários com base nos endereços de email associados às respectivas contas de usuário. Para obter mais informações, confira "[Solução de problemas no uso da licença do {% data variables.product.prodname_enterprise %}](/billing/managing-your-license-for-github-enterprise/troubleshooting-license-usage-for-github-enterprise#about-the-calculation-of-consumed-licenses)".
