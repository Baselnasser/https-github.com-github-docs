O {% data variables.product.company_short %} usa um modelo de licenciamento de usuário único. Para produtos corporativos que incluem várias opções de implantação, o {% data variables.product.company_short %} determina quantas estações você está consumindo com base no número de usuários únicos em todas as suas implantações.

Cada usuário consome apenas uma licença, não importa o número de instâncias do {% data variables.product.prodname_ghe_server %} usadas pelo usuário ou o número de organizações das quais o usuário é membro no {% data variables.product.prodname_ghe_cloud %}. Esse modelo permite que cada pessoa use várias implantações do {% data variables.product.prodname_enterprise %} sem gerar custos extras.
