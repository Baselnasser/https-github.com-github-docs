Se você usar um cupom para pagar uma assinatura, quando o cupom vencer, será cobrado o custo total da sua assinatura do seu método de pagamento. Se você não tem um método de pagamento salvo, sua conta será rebaixada para {% data variables.product.prodname_free_user %} para contas pessoais ou {% data variables.product.prodname_free_team %} para organizações.
