1. Acesse a página [Resgatar cupom](https://github.com/redeem).
2. Na caixa Inserir código de cupom, digite seu código de cupom e clique em **Resgatar**.
   ![Caixa Resgatar cupom](/assets/images/help/settings/redeem-coupon-box.png)
3. Se você ainda não tiver feito login, você pode fazê-lo nesta página ou criar uma conta nova para usar o cupom.
