1. Se solicitado, carregue o seu status acadêmico.
   - Clique em **Usar sua câmera para tirar uma foto**.
     ![Botão usado para usar sua câmera para tirar uma foto](/assets/images/help/education/upload-proof-status.png)
   - Opcionalmente, para alterar a câmera que você quer usar, use o menu suspenso da câmera.
    ![Menu suspenso Câmera](/assets/images/help/education/camera-drop-down.png)
   - Coloque sua ID acadêmica válida ou outra prova do status acadêmico atual no quadro e clique em **Tirar foto**.
    ![Botão Tirar foto](/assets/images/help/education/take-photo-button.png)
   - Se você não conseguir tirar uma foto da sua prova de status acadêmico, para carregar uma imagem do computador, clique em **Carregar imagem**.
     ![Link Carregar imagem](/assets/images/help/education/upload-image-link.png) {% note %}

    **Observação:** não há suporte para arquivos PDF, e não é possível carregá-los como prova de status acadêmico.

    {% endnote %}
