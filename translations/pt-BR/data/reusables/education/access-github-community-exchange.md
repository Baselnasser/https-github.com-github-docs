Para acessar o {% data variables.product.prodname_community_exchange %}, visite nosso painel do {% data variables.product.prodname_global_campus %} em https://education.github.com.

Se você for estudante ou membro do corpo docente de uma instituição educacional credenciada, poderá se inscrever para receber os benefícios de {% data variables.product.prodname_education %}, que incluem acesso a {% data variables.product.prodname_community_exchange %} dentro de {% data variables.product.prodname_global_campus %}.

- Se você é um estudante e ainda não se inscreveu em {% data variables.product.prodname_education %}, inscreva-se usando o [formulário de inscrição para estudantes]( https://education.github.com/discount_requests/student_application). Para obter mais informações, confira "[Sobre o {% data variables.product.prodname_global_campus %} para alunos](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/about-github-global-campus-for-students)".

- Se você é um educador e ainda não se inscreveu em {% data variables.product.prodname_education %}, inscreva-se usando o [formulário de inscrição ]( https://education.github.com/discount_requests/teacher_application). Para obter mais informações, confira "[Candidatar-se para o {% data variables.product.prodname_global_campus %} como professor](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-teachers/apply-to-github-global-campus-as-a-teacher)."
