O {% data variables.product.prodname_pages %} é um serviço de hospedagem de site estático que usa arquivos HTML, CSS e JavaScript diretamente de um repositório e publica um site. Você pode habilitar ou desabilitar o {% data variables.product.prodname_pages %} para os membros corporativos no nível da organização. Para obter mais informações, confira "[Como configurar o {% data variables.product.prodname_pages %} para sua empresa](/admin/configuration/configuring-your-enterprise/configuring-github-pages-for-your-enterprise)" e "[Sobre o {% data variables.product.prodname_pages %}](/pages/getting-started-with-github-pages/about-github-pages)".
