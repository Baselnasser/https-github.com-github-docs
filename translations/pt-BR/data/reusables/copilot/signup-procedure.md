Antes de começar a usar {% data variables.product.prodname_copilot %}, você precisará configurar uma avaliação ou assinatura gratuita.

{% data reusables.user-settings.access_settings %}
1. Na seção "Código, planejamento e automação" da barra lateral, clique em **GitHub Copilot {% octicon "copilot" aria-label="The copilot icon" %}** .
1. Na página de configurações de {% data variables.product.prodname_copilot %}, clique em **Habilitar {% data variables.product.prodname_copilot %}** .

   ![Captura de tela das configurações do GitHub Copilot com o botão Habilitar {% data variables.product.prodname_copilot %} enfatizado](/assets/images/help/copilot/copilot-settings-enable-button.png)
1. Escolha se deseja pagar mensal ou anualmente e clique em **Continuar para obter acesso ao Copilot**.
   - Se sua conta pessoal atender aos critérios para uma assinatura gratuita do {% data variables.product.prodname_copilot %} em vez de uma avaliação, você será direcionado automaticamente para a etapa 6.

   ![Captura de tela da configuração de teste com o botão Continuar para obter acesso ao Copilot enfatizado](/assets/images/help/copilot/copilot-billing-cycle.png)
1. Siga as etapas para confirmar seus detalhes de pagamento e clique em **Enviar**.
1. Selecione suas preferências e clique em **Salvar e começar**.

   ![Captura de tela das configurações do GitHub Copilot com o botão Salvar e começar](/assets/images/help/copilot/copilot-signup-preferences.png)

   Você pode alterar essas preferências posteriormente retornando às suas configurações do {% data variables.product.prodname_copilot %}. Para ver mais informações, confira "[Como configurar o GitHub Copilot no Visual Studio Code](/copilot/configuring-github-copilot/configuring-github-copilot-in-visual-studio-code#configuring-github-copilot-settings-on-githubcom)".
