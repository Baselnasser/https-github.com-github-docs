O arquivo de configuração pode ser localizado no repositório que você está analisando ou em um repositório externo. O uso de um repositório externo permite que você especifique opções de configuração para vários repositórios em um único local. Ao fazer referência a um arquivo de configuração localizado em um repositório externo, você poderá usar a sintaxe _OWNER/REPOSITORY/FILENAME@BRANCH_ . Por exemplo, _octo-org/shared/codeql-config.yml@main_.
