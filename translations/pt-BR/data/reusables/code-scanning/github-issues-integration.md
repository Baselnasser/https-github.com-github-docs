Os alertas da {% data variables.product.prodname_code_scanning_capc %} se integram às listas de tarefas do {% data variables.product.prodname_github_issues %} para facilitar a priorização e o acompanhamento de alertas com todo o seu trabalho de desenvolvimento. Para obter mais informações sobre problemas, confira "[Sobre os problemas](/issues/tracking-your-work-with-issues/about-issues)".

Para acompanhar um alerta da verificação de código em um problema, adicione a URL do alerta como um item da lista de tarefas no problema. Para obter mais informações sobre listas de tarefas, confira "[Sobre as listas de tarefas](/issues/tracking-your-work-with-issues/about-task-lists)".
