Quando você tentar efetuar push de um segredo compatível para um repositório ou uma organização com a {% data variables.product.prodname_secret_scanning %} como uma proteção por push habilitada, o {% data variables.product.prodname_dotcom %} bloqueará o push. Você pode remover o segredo do branch ou seguir uma URL fornecida para permitir o push.
