1. Opcionalmente, ao lado de "Uso deste mês", clique em **Obter relatório de uso** a fim de receber um email com um link para baixar um relatório CSV de uso de armazenamento de {% data variables.product.prodname_actions %}, {% data variables.product.prodname_registry %} e {% data variables.product.prodname_github_codespaces %}. O email é enviado para o endereço de email principal da sua conta. Você pode escolher se o relatório deve abranger os últimos 7, 30, 90 ou 180 dias.

   ![Baixar relatório em CSV](/assets/images/help/billing/actions-packages-report-download.png)

   Os dados utilizados para este relatório são atualizados diariamente. 

   Para ver os custos de uso e armazenamento de computação do {% data variables.product.prodname_github_codespaces %}, filtre o relatório para mostrar apenas as linhas que mencionam "Codespaces" na coluna `Product`.

   ![Captura de tela de um relatório de uso filtrado para {% data variables.product.prodname_codespaces %}](/assets/images/help/codespaces/CSV-usage-report.png)

   Para ver apenas os custos de criação, atualização e armazenamento de pré-compilações, filtre o relatório para mostrar apenas as linhas que mencionam "Criar pré-compilações de codespaces" na coluna `Actions Workflow`.

   ![Captura de tela de um relatório de uso filtrado para pré-compilações](/assets/images/help/codespaces/CSV-usage-report-prebuilds.png)
