## Acessar um codespace existente

1. {% data reusables.codespaces.you-can-see-all-your-codespaces %}
1. Clique no nome do codespace em que você deseja desenvolver.

   ![Nome do codespace](/assets/images/help/codespaces/click-name-codespace.png)

Como alternativa, é possível ver qualquer um dos codespaces de um repositório específico acessando esse repositório e clicando em **{% octicon "code" aria-label="The code icon" %} Código**. O menu suspenso exibirá todos os codespaces ativos de um repositório.
