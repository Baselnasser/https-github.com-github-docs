{% note %}

**Observação:** você precisa definir um limite de gastos para usar o {% data variables.product.prodname_codespaces %}.

{% endnote %}

Por padrão, sua organização ou empresa terá um limite de gastos igual a US$ 0 no {% data variables.product.prodname_github_codespaces %}, o que impedirá a criação de codespaces ou a abertura dos codespaces existentes. Para permitir que os usuários criem codespaces na sua organização, defina o limite com um valor maior que US$ 0.
