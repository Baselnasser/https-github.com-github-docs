1. Se o branch atual não for mostrado na barra de status, na parte inferior do seu codespace, clique com o botão direito do mouse na barra de status e selecione **Controle do código-fonte**.
1. Clique no nome do branch na barra de status.
![O branch na barra de status](/assets/images/help/codespaces/branch-in-status-bar.png)
1. No menu suspenso, clique no branch para o qual deseja alternar ou insira o nome de um novo branch e clique em **Criar branch**.
![Escolha no menu do branch](/assets/images/help/codespaces/create-new-branch.png)
