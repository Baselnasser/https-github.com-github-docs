Para configurar seu repositório e usar um contêiner de desenvolvimento personalizado, você precisará criar um ou mais arquivos `devcontainer.json`. Você pode adicioná-los usando um modelo de configuração predefinido, no {% data variables.product.prodname_vscode %} ou pode criar um. Para saber mais sobre contêineres de desenvolvimento, confira "[Introdução aos contêineres de desenvolvimento](/codespaces/setting-up-your-project-for-codespaces/introduction-to-dev-containers)".
