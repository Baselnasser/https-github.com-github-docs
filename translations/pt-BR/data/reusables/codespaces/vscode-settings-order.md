Quando você define as configurações do editor para o {% data variables.product.prodname_vscode_shortname %}, há três escopos disponíveis: _Workspace_, _Repositórios Remotos [Codespaces]_ e _Usuário_. Se uma configuração for definida em vários escopos, as configurações de _Workspace_ serão prioritárias e, em seguida, _Repositórios Remotos [Codespaces]_ e, por fim, _Usuário_.
