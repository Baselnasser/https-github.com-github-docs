Se a conta pessoal, organizacional ou corporativa atingir o limite de gastos, você não poderá mais criar nem retomar codespaces faturáveis para essa conta. Você ainda poderá exportar qualquer trabalho em andamento para um novo branch. Para obter mais informações, confira "[Como exportar alterações para um branch](/codespaces/troubleshooting/exporting-changes-to-a-branch)".
