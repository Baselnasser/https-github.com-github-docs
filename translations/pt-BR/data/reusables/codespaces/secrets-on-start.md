Após criar um segredo, ele estará disponível quando você criar um novo código ou reiniciar o codespace. Para usar um segredo que você acabou de criar em um espaço de código atual, você deverá parar o ritmo do código e retomá-lo. Para obter informações sobre como parar o codespace, confira "[Usando a Paleta de comandos no {% data variables.product.prodname_codespaces %}](/codespaces/codespaces-reference/using-the-command-palette-in-codespaces#suspending-or-stopping-a-codespace)".
