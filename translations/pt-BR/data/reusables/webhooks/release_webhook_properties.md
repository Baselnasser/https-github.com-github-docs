Chave | Tipo | Descrição
----|------|-------------
`action`|`string` | A ação que foi executada. Pode ser uma das ações a seguir:<ul><li>`published`: uma versão, pré-versão ou um rascunho de uma versão foi publicada</li><li>`unpublished`: uma versão ou pré-versão foi excluída</li><li>`created`: um rascunho foi salvo, ou uma versão ou pré-versão foi publicada sem ser previamente salva como rascunho</li><li>`edited`: uma versão, pré-versão ou um rascunho de versão de foi editada</li><li>`deleted`: uma versão, pré-versão ou um rascunho de versão de foi excluída</li><li>`prereleased`: uma pré-versão foi criada</li><li>`released`: uma versão foi publicada ou uma pré-versão foi alterada para uma versão</li>
