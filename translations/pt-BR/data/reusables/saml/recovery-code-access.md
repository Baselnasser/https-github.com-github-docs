1. Na parte inferior da caixa de diálogo de logon único, clique em **Usar um código de recuperação** para ignorar o logon único.

   ![Captura de tela do link usado para inserir o código de recuperação](/assets/images/help/saml/saml_use_recovery_code.png)

2. No campo "Recovery Code" (Código de Recuperação), digite seu código de recuperação.

   ![Captura de tela do campo usado para inserir o código de recuperação](/assets/images/help/saml/saml_recovery_code_entry.png)

3. Clique em **Verificar**.

   ![Captura de tela do botão usado para verificar o código de recuperação](/assets/images/help/saml/saml_verify_recovery_codes.png)

Depois de usar um código de recuperação, observe que o código não é mais válido. Você não poderá reutilizar o código de recuperação.
