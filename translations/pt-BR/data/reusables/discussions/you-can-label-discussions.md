Para organizar as discussões de forma mais granular, você pode aplicar etiquetas. Por exemplo, você pode usar etiquetas para indicar o status de uma discussão para tornar a triagem mais eficiente. Cada repositório tem um conjunto compartilhado de etiquetas para problemas, pull requests e discussões. Para obter mais informações, confira "[Como gerenciar rótulos](/github/managing-your-work-on-github/managing-labels)".
