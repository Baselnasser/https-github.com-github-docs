{% data reusables.repositories.navigate-to-repo %} {% data reusables.discussions.discussions-tab %}
1. Na lista de categorias, clique em **Votações**.
  ![Captura de tela mostrando a categoria "Votação"](/assets/images/help/discussions/poll-category.png)
1. À direita, clique em **Iniciar votação**.
  ![Captura de tela mostrando o botão "Iniciar votação"](/assets/images/help/discussions/start-poll-button.png)
1. Digite um título e um corpo opcional para sua votação.
  ![Captura de tela mostrando campos de texto para título e corpo](/assets/images/help/discussions/new-poll-title-and-body-fields.png)
1. Digite uma pergunta para sua votação.
  ![Captura de tela mostrando campos de texto para a pergunta da votação](/assets/images/help/discussions/new-poll-question.png)
1. Digite pelo menos duas opções para a votação.
  ![Captura de tela mostrando campos de texto para as opções da votação](/assets/images/help/discussions/new-poll-options.png)
1. Opcionalmente, para adicionar uma opção de votação extra, clique em **Adicionar uma opção**.
  ![Captura de tela mostrando o botão "Adicionar uma opção"](/assets/images/help/discussions/new-poll-add-option.png)
1. Clique em **Iniciar votação**.
  ![Captura de tela mostrando o botão "Iniciar votação"](/assets/images/help/discussions/new-poll-start-poll-button.png)
