Você pode habilitar ou desabilitar o {% data variables.product.prodname_discussions %} para um repositório.

{% data reusables.repositories.navigate-to-repo %}
1. Abaixo do nome do repositório, clique em {% octicon "gear" aria-label="The gear icon" %} **Configurações**.
![Botão Configurações do repositório](/assets/images/help/discussions/public-repo-settings.png)
1. Em "Recursos", selecione **Discussões**.
    ![Caixa de seleção em "Recursos" usada para habilitar ou desabilitar o {% data variables.product.prodname_discussions %} para um repositório](/assets/images/help/discussions/select-discussions-checkbox.png)
