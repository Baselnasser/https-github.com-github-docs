- **Gravação**, que dá às pessoas permissão para exibir um {% data variables.projects.projects_v1_board %}, vincular repositórios a um {% data variables.projects.projects_v1_board %} e interagir com um {% data variables.projects.projects_v1_board %}. - **Administrador**, que dá às pessoas permissão para exibir um {% data variables.projects.projects_v1_board %}, interagir com um {% data variables.projects.projects_v1_board %}, gerenciar configurações do {% data variables.projects.projects_v1_board %} e gerenciar o acesso de outras pessoas ao {% data variables.projects.projects_v1_board %}.
