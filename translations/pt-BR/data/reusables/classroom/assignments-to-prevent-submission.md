Para impedir a aceitação ou o envio de uma tarefa pelos alunos, altere o "Status da Tarefa" na exibição "Editar tarefa". Quando uma tarefa estiver Ativa, os alunos poderão aceitá-la usando o link de convite. Quando ela estiver Inativa, esse link não será mais válido.
