1. Opcionalmente, para desativar lembretes agendados para solicitações de pull que já foram revisadas e aprovadas, selecione **Ignorar solicitações de pull aprovadas**. Em seguida, clique em **Ignorar com uma ou mais aprovações** para escolher quantas aprovações uma solicitação de pull precisa ter para ser ignorada.
