1. Opcionalmente, para incluir somente as solicitações de pull das quais uma revisão é solicitada especificamente da equipe ou de um membro da equipe, selecione **Exigir solicitações de revisão**. Se você não selecionar esta opção, todos os pull requests serão incluídos no lembrete agendado.
