Os {% data variables.product.prodname_dependency_review_action %} verifica as solicitações de pull em busca de alterações de dependência e gera um erro quando novas dependências têm vulnerabilidades conhecidas. A ação tem o suporte de um ponto de extremidade de API que compara as dependências entre duas revisões e relata as diferenças.

Para obter mais informações sobre a ação e o ponto de extremidade da API, confira a documentação [`dependency-review-action`](https://github.com/actions/dependency-review-action) e a "[Análise de dependência](/rest/dependency-graph/dependency-review)" na documentação da API.
