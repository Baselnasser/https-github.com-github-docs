## À propos des profils de sponsor

Votre profil {% data variables.product.prodname_sponsors %} indique aux sponsors potentiels la raison pour laquelle ils doivent vous soutenir. Les gens voient votre profil GitHub Sponsors quand ils cliquent sur le bouton **Sponsoriser** de votre profil. Nous vous recommandons d’inclure les informations suivantes.

- Travail open source auquel vous contribuez
- Raison pour laquelle vous vous êtes engagé dans le développement open source
