Si vous avez des sponsors, vous devez contacter le support GitHub pour qu’il vous aide à annuler la publication de votre compte {% data variables.product.prodname_sponsors %}.

Une fois que l’annulation de la publication de votre compte {% data variables.product.prodname_sponsors %} est effective :
- Vous ne recevez plus de paiements de parrainage.
- Vous pouvez demander à republier votre profil.

{% data reusables.sponsors.navigate-to-sponsors-dashboard %} {% data reusables.sponsors.settings-tab %}
3. Sous « Annuler la publication de votre profil GitHub Sponsors », cliquez sur **Annuler la publication du profil**.
  ![Bouton « Annuler la publication du profil »](/assets/images/help/sponsors/unpublish-profile-button.png)
4. Pour confirmer, cliquez sur « Annuler la publication du compte GitHub Sponsors ».
  ![Bouton « Annuler la publication du profil GitHub Sponsors »](/assets/images/help/sponsors/unpublish-profile-dialog.png)
