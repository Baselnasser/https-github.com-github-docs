## À propos de l’utilisation d’un hôte fiscal pour recevoir des paiements {% data variables.product.prodname_sponsors %}

Quand vous vous inscrivez pour obtenir un profil {% data variables.product.prodname_sponsors %} afin d’être sponsorisé, vous pouvez choisir de recevoir vos paiements sur un compte Stripe Connect placé sous votre contrôle, ou via un hôte fiscal que vous utilisez déjà.

{% data reusables.sponsors.bank-info-fiscal-host-reminder %}

{% data variables.product.prodname_sponsors %} prend en charge les hôtes fiscaux suivants :

- [Open Source Collective](https://oscollective.org/)
- [NumFOCUS](http://www.numfocus.org/)
- [Open Collective Europe](https://opencollective.com/europe)
- [Software Underground](https://softwareunderground.org/)
- [Software Freedom Conservancy](https://sfconservancy.org/)

## Choix relatif à l’utilisation d’un hôte fiscal

Une fois que vous avez accédé à la page de configuration du profil « Rejoindre la liste d’attente » du compte à sponsoriser, vous voyez s’afficher la section « Comment recevoir les paiements » ou « Compte bancaire de l’organisation », selon le type de compte que vous configurez. Pour plus d’informations sur l’inscription d’un profil {% data variables.product.prodname_sponsors %}, consultez « [Configuration de {% data variables.product.prodname_sponsors %} pour votre compte personnel](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account) » ou « [Configuration de {% data variables.product.prodname_sponsors %} pour votre organisation](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization) ».

1. Sous « Comment recevoir les paiements » ou « Compte bancaire de l’organisation », sélectionnez l’option **Utiliser un hôte fiscal** ou **Cette organisation utilise un hôte fiscal**.

   ![Capture d’écran de l’option permettant de choisir entre un compte bancaire et un hôte fiscal](/assets/images/help/sponsors/user-waitlist-payout-options.png)
2. Utilisez la liste déroulante **Sélectionner un hôte fiscal** pour choisir l’hôte fiscal à utiliser.
3. Dans la zone de texte sous « URL de profil du projet d’hôte fiscal », tapez l’URL de votre page de profil sur le site de l’hôte fiscal. Cela nous permet de vérifier que vous êtes bien inscrit auprès de l’hôte fiscal.
  
  ![Capture d’écran de la zone de texte de l’URL de profil du projet de votre hôte fiscal](/assets/images/help/sponsors/fiscal-host-project-profile-field.png)
