Si vous recevez des paiements de parrainage, vous devez contacter le support technique GitHub pour désactiver votre compte {% data variables.product.prodname_sponsors %}.

En l’absence de commanditaire, vous pouvez désactiver votre compte {% data variables.product.prodname_sponsors %} sur GitHub.

Si vous décidez de recréer un compte {% data variables.product.prodname_sponsors %}, vous devez effectuer une nouvelle demande.

Vous pouvez également annuler la publication de votre profil de sponsors, puis republier le compte sans effectuer de nouvelle demande. Pour plus d’informations, consultez « [Annuler la publication de votre profil {% data variables.product.prodname_sponsors %}](/sponsors/receiving-sponsorships-through-github-sponsors/unpublishing-your-github-sponsors-profile) ».

{% data reusables.sponsors.navigate-to-sponsors-dashboard %} {% data reusables.sponsors.settings-tab %}
1. Sous « Désactiver le compte GitHub Sponsors », cliquez sur **Désactiver le compte**.  
  ![Bouton « Désactiver votre compte »](/assets/images/help/sponsors/disable-your-account-button.png)
2. Pour confirmer, cliquez sur **Désactiver le compte GitHub Sponsors**.
  ![Bouton « Désactiver le compte GitHub Sponsors »](/assets/images/help/sponsors/disable-github-sponsors-account-dialog.png)
