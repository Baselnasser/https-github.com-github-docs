## Participation à {% data variables.product.prodname_sponsors %}

{% data reusables.sponsors.you-can-be-a-sponsored-developer %} Pour plus d’informations, consultez « [Configuration de {% data variables.product.prodname_sponsors %} pour un compte personnel](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account) ».

{% data reusables.sponsors.you-can-be-a-sponsored-organization %} Pour plus d’informations, consultez « [Configuration de {% data variables.product.prodname_sponsors %} pour une organisation](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization) ».

Une fois que vous avez rejoint {% data variables.product.prodname_sponsors %}, vous pouvez ajouter un bouton de parrainage au référentiel open source auquel vous contribuez pour augmenter la visibilité de votre profil {% data variables.product.prodname_sponsors %} et d’autres plateformes de financement. Pour plus d’informations, consultez « [Affichage d’un bouton de sponsor dans votre dépôt](/articles/displaying-a-sponsor-button-in-your-repository) ».

Vous pouvez définir un objectif pour vos parrainages. Pour plus d’informations, consultez « [Gestion de votre objectif de parrainage](/sponsors/receiving-sponsorships-through-github-sponsors/managing-your-sponsorship-goal) ».

## Niveaux de parrainage

{% data reusables.sponsors.tier-details %} Pour plus d’informations, consultez « [Configuration de {% data variables.product.prodname_sponsors %} pour un compte personnel](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account) », « [Configuration de {% data variables.product.prodname_sponsors %} pour une organisation](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization) » et « [Gestion des niveaux de parrainage](/sponsors/receiving-sponsorships-through-github-sponsors/managing-your-sponsorship-tiers) ».

Il est préférable de configurer différentes options de parrainage (notamment un niveau mensuel et un niveau ponctuel) pour que les autres puissent vous aider facilement dans votre travail. En particulier, les paiements ponctuels permettent aux gens de récompenser vos efforts sans se soucier de savoir si leurs finances supportent un calendrier de paiement régulier.
