## Formulaires fiscaux W-9/W-8

Selon la loi, {% data variables.product.prodname_dotcom %} est tenu par l’IRS (Internal Revenue Service), une agence de l’administration américaine, de collecter les informations fiscales de tous les responsables américains et non américains. Ces formulaires sont conservés par {% data variables.product.prodname_dotcom %}. Ils n’ont pas être envoyés à l’IRS. 

### W-9 (États-Unis)

Les informations des formulaires fiscaux W-9 permettent à {% data variables.product.prodname_dotcom %} d’utiliser le numéro d’identification fiscal (TIN) approprié pour déclarer à l’IRS les revenus qui vous sont versés, via le formulaire 1099.

Si vous êtes un contribuable situé aux États-Unis, vous devez envoyer un formulaire [W-9](https://www.irs.gov/forms-pubs/about-form-w-9) pour pouvoir publier votre profil {% data variables.product.prodname_sponsors %}. 

### W-8 BEN/W-8 BEN-E (hors États-Unis)

Les formulaires fiscaux W-8 BEN et W-8 BEN-E permettent à {% data variables.product.prodname_dotcom %} d’identifier le bénéficiaire effectif d’un montant soumis à une retenue à la source.

Si vous êtes un contribuable situé dans une autre région que les États-Unis, vous devez envoyer un formulaire [W-8 BEN](https://www.irs.gov/pub/irs-pdf/fw8ben.pdf) (destiné aux particuliers) ou [W-8 BEN-E](https://www.irs.gov/forms-pubs/about-form-w-8-ben-e) (destiné aux entreprises) pour pouvoir publier votre profil {% data variables.product.prodname_sponsors %}. Pour plus d’informations, consultez « [Configuration de {% data variables.product.prodname_sponsors %} pour votre compte personnel](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account#submitting-your-tax-information) » et « [Configuration de {% data variables.product.prodname_sponsors %} pour votre organisation](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization#submitting-your-tax-information) ». {% data variables.product.prodname_dotcom %} vous enverra les formulaires appropriés, vous notifiera leur date limite de retour, et vous donnera un délai raisonnable pour les remplir et les envoyer.

Si un formulaire fiscal incorrect vous a été affecté, [contactez le support technique de {% data variables.product.prodname_dotcom %}](https://support.github.com/contact?form%5Bsubject%5D=GitHub%20Sponsors:%20tax%20form&tags=sponsors) pour que le formulaire approprié vous soit réaffecté en fonction de votre situation.

### Numéros d’identification fiscale

Les formulaires fiscaux W-8 BEN et W-8 BEN-E demandent un « numéro d’identification fiscal (SSN ou ITIN) américain » ou un « numéro d’identification fiscal étranger ». 

Si vous n’êtes pas citoyen américain ou si vous ne vivez pas aux États-Unis, notamment en tant que résident étranger, vous devez généralement fournir le « numéro d’identification fiscal étranger ». Il s’agit d’un numéro d’identification fiscale que votre pays de résidence vous a attribué.

Si votre pays de résidence ou de facturation est l’Inde, vous pouvez utiliser votre numéro de compte permanent (PAN) en tant que numéro d’identification fiscale.

{% note %}

**Remarque :** Si votre pays n’impose pas à ses citoyens d’avoir un numéro fiscal émis par l’administration, vous pouvez écrire à la place « not legally required » (non obligatoire légalement).  

Conformément aux directives de l’IRS, vous ne devez pas écrire « not applicable » (non applicable). Pour plus d’informations, consultez les « [instructions relatives au formulaire W-8BEN, ligne 6](https://www.irs.gov/instructions/iw8ben#idm139867098922656) » sur le site web de l’IRS.
{% endnote %}

L’IRS permet aux étrangers non-résidents et résidents de demander un [numéro d’identification fiscal individuel](https://www.irs.gov/individuals/international-taxpayers/taxpayer-identification-numbers-tin#itin) ou ITIN. Il est peu probable que cela soit nécessaire pour les responsables qui perçoivent des revenus via {% data variables.product.prodname_sponsors %}, mais lisez les [instructions](https://www.irs.gov/pub/irs-pdf/iw8ben.pdf) complètes afin d’évaluer au mieux votre situation personnelle.

## Formulaires 1099

Si vous êtes un contribuable situé aux États-Unis, et si vous gagnez plus de 600 $ US au cours d’une année fiscale, {% data variables.product.prodname_dotcom %} vous envoie un formulaire [1099-NEC](https://www.irs.gov/forms-pubs/about-form-1099-nec) avant le 31 janvier de l’année civile suivante. Nous ne fournissons pas de formulaires fiscaux pour les contribuables internationaux.

## Informations fiscales générales

{% data variables.product.prodname_dotcom %} n’effectue pas de retenue fiscale sur les paiements de {% data variables.product.prodname_sponsors %}. Les développeurs sponsorisés sont responsables de l’évaluation et du paiement de leurs propres impôts.

## Taxe de vente

GitHub fournit des informations pour vous aider à déterminer quelles sont vos obligations fiscales dans le cadre d’opérations de vente. Ces informations ne sont pas personnalisées en fonction de votre pays ou de votre situation fiscale. Nous vous recommandons donc de consulter un professionnel pour comprendre vos obligations spécifiques. Toutefois, nous souhaitons vous apporter des informations globales pour vous aider à comprendre les principes généraux liés à la fiscalité des opérations de vente dans le domaine du numérique.

Dans la plupart des pays du monde, la taxe sur les transactions numériques est basée sur la localisation du destinataire, et non sur celle du vendeur. Par exemple, si vous êtes un responsable situé aux États-Unis, et si vous fournissez un service taxable à un sponsor situé en Allemagne, la fiscalité allemande relative aux opérations de vente s’applique.

La fiscalité relative aux opérations de vente s’applique généralement uniquement quand un bien ou un service de valeur est fourni. En principe, la bonne volonté, l’aide en général ou la gratitude ne sont pas des éléments imposables.

Aux États-Unis, les ventes B2B (interentreprises) et B2C (entreprise-client) sont soumises à une taxe de vente.

Dans l’UE ainsi que dans la plupart des autres pays et régions, les ventes B2C sont soumises à une taxe de vente. Les ventes B2B ne sont pas soumises à une taxe. Les ventes C2C et C2B pour lesquelles le client n’est pas assujetti à la TVA ne sont pas taxables.
