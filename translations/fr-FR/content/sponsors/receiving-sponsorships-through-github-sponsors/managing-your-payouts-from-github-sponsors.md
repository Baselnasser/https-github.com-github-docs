## À propos des paiements dans {% data variables.product.prodname_sponsors %}

Vous pouvez uniquement gérer vos paiements dans {% data variables.product.prodname_sponsors %} si vous êtes payé via Stripe Connect. Si vous êtes payé par virement ACH ou par virement bancaire pour votre compte personnel sponsorisé, vous ne voyez pas l’onglet **Paiements** dans votre tableau de bord de contributeur open source sponsorisé. Les organisations sponsorisées peuvent uniquement utiliser Stripe Connect pour les paiements. Pour plus d’informations, consultez « [À propos de {% data variables.product.prodname_sponsors %} pour les contributeurs open source](/sponsors/receiving-sponsorships-through-github-sponsors/about-github-sponsors-for-open-source-contributors#sponsorship-payouts) ».
