{% data reusables.sponsors.beta-note-sponsors-for-companies %}

Les entreprises peuvent utiliser une méthode de facture pour participer au programme {% data variables.product.prodname_sponsors %}. Cela peut être particulièrement pratique si votre entreprise a une relation de facturation avec {% data variables.product.company_short %}. Les entreprises payant par facture ont accès à un tableau de bord {% data variables.product.product_name %} qui mesure les contributions et l’activité open source dans les projets publics sur {% data variables.product.prodname_dotcom_the_website %} qui les intéressent.

Pour rejoindre ce programme bêta, adressez-vous à votre représentant commercial ou [contactez-nous](https://support.github.com/contact/org-sponsors-waitlist).
