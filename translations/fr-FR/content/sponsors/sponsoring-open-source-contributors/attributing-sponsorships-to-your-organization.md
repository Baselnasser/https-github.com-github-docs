1. Accédez à l’organisation dont vous souhaitez attribuer les sponsorisations à une autre organisation.
2. Sous le nom de votre organisation, cliquez sur {% octicon "heart" aria-label="The heart icon" %} **Sponsorisation**.
  ![Onglet « Sponsorisation »](/assets/images/help/sponsors/sponsoring-tab.png)
1. Dans l’angle supérieur droit, cliquez sur {% octicon "gear" aria-label="The gear icon" %} **Paramètres**.
  ![Bouton « Paramètres »](/assets/images/help/sponsors/sponsoring-settings-button.png)
1. Sous « Lier les sponsorisations à un autre compte », utilisez le menu déroulant, puis cliquez sur l’organisation à laquelle vous souhaitez attribuer les sponsorisations de cette organisation.
  ![Menu déroulant pour sélectionner un compte](/assets/images/help/sponsors/select-an-account-drop-down.png)
1. Cliquez sur **Lier le compte**.
  ![Bouton « Lier le compte »](/assets/images/help/sponsors/link-account-button.png)
