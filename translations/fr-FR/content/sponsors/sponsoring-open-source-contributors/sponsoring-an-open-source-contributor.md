## À propos des parrainages

{% data reusables.sponsors.sponsorship-details %}

Vous pouvez parrainer un compte au nom de votre compte personnel pour investir dans des projets dont vous bénéficiez personnellement. Vous pouvez parrainer un compte pour le compte de votre organisation pour de nombreuses raisons.
- Gestion de bibliothèques spécifiques dont dépend le travail de votre organisation
- Investissement dans l’écosystème sur lequel vous vous appuyez en tant qu’organisation (telle qu’une blockchain)
- Développement de la sensibilisation de la marque en tant qu’organisation qui valorise l’open source
- Merci aux développeurs open source pour la génération de bibliothèques qui complètent le produit qu’offre votre organisation

Vous pouvez utiliser une carte de crédit pour parrainer un compte sur {% data variables.product.product_name %}. Si votre organisation souhaite payer par facture, vous pouvez en savoir plus sur [Le paiement de {% data variables.product.prodname_sponsors %} par facture](/sponsors/sponsoring-open-source-contributors/paying-for-github-sponsors-via-invoice).

{% data reusables.sponsors.no-fees %} Pour plus d’informations, consultez « [À propos de la facturation pour {% data variables.product.prodname_sponsors %}](/articles/about-billing-for-github-sponsors) ».

Lorsque vous parrainez un compte à l’aide d’une carte de crédit, la modification prend effet immédiatement. {% data reusables.sponsors.prorated-sponsorship %}

Nous pouvons partager certaines informations fiscales limitées avec des comptes parrainés. Pour plus d’informations, consultez « [Informations fiscales](#tax-information) ».

{% data reusables.sponsors.manage-updates-for-orgs %}

Vous pouvez choisir d’afficher votre parrainage publiquement. Les parrainages ponctuels restent visibles pendant un mois. 

Si le compte parrainé met hors service votre niveau, le niveau reste en place jusqu’à ce que vous choisissiez un autre niveau ou annuliez votre abonnement. Pour plus d’informations, consultez « [Mise à niveau d’un parrainage](/articles/upgrading-a-sponsorship) » et « [Déclassement d’un parrainage](/articles/downgrading-a-sponsorship) ».

Si le compte que vous souhaitez parrainer n’a pas de profil sur {% data variables.product.prodname_sponsors %}, vous pouvez encourager le compte à adhérer. Pour plus d’informations, consultez « [Configuration de {% data variables.product.prodname_sponsors %} pour votre compte personnel](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account) » et « [Configuration de {% data variables.product.prodname_sponsors %} pour votre organisation](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization) ».

{% data reusables.sponsors.sponsorships-not-tax-deductible %}

{% note %}

**Remarque :** {% data variables.product.prodname_dotcom %} n’est pas responsable de la façon dont les développeurs se représentent eux-mêmes et {% data variables.product.prodname_dotcom %} n’endosse aucun projet open source parrainé. Les revendications sont la seule responsabilité du développeur qui reçoit les fonds. Veillez à bien faire confiance à une personne avant d’offrir un parrainage. Pour plus d’informations, consultez les [conditions supplémentaires {% data variables.product.prodname_sponsors %}](/free-pro-team@latest/github/site-policy/github-sponsors-additional-terms).

{% endnote %}

## Informations fiscales

En tant que commanditaire, vous reconnaissez que nous pouvons divulguer au propriétaire de chaque compte que vous parrainez les informations limitées suivantes sur vos paiements de parrainage au compte, depuis le lancement du Programme des commanditaires :

- Date de la transaction
- Montant payé
- Le pays, l’État et la province à partir duquel le paiement a été effectué
- Indique si le paiement a été effectué par une entreprise ou une personne

Ces renseignements sont nécessaires pour permettre le paiement et la déclaration des impôts découlant de ces paiements de parrainage.

## Parrainage d’un compte

Avant de pouvoir parrainer un compte, vous devez disposer d’une adresse e-mail vérifiée. Pour plus d’informations, consultez « [Vérification de votre adresse e-mail](/github/getting-started-with-github/verifying-your-email-address) ».

1. Sur {% data variables.product.product_name %}, accédez au profil de l’utilisateur ou de l’organisation que vous souhaitez parrainer.
1. Accédez à votre tableau de bord de parrainage pour le compte.
   - Pour parrainer un développeur, sous le nom du développeur, cliquez sur **Commanditaire**.
     ![Bouton Commanditaire](/assets/images/help/profile/sponsor-button.png)
   - Pour parrainer une organisation, à droite du nom de l’organisation, cliquez sur **Commanditaire**.
     ![Bouton Commanditaire](/assets/images/help/sponsors/sponsor-org-button.png)
1. Si vous le souhaitez, pour parrainer le compte au nom de votre organisation, utilisez le menu déroulant **Commanditaire** à droite de la page, puis cliquez sur l’organisation.
