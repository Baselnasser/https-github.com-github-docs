## Explorer les projets dont vous dépendez

Quand vous cherchez des contributeurs open source à parrainer, envisagez de commencer par https://github.com/sponsors/community.

Si vous êtes connecté, les résultats des commanditaires commencent par les dépendances publiques dont dépendent vos référentiels.

![Le projet Babel open source avec du texte mis en surbrillance qui indique « 4 de vos référentiels dépendent de ceci »](/assets/images/help/sponsors/example-of-open-source-project-dependency.png)

Vous pouvez également choisir quel compte utiliser pour afficher les résultats et filtrer ou rechercher davantage.

![Options de recherche sur la page de recherche de commanditaires](/assets/images/help/sponsors/sponsors-search-options.png)

## Découvrir qui les autres parrainent

Vous pouvez voir qui un utilisateur ou une organisation parraine sur sa page de profil {% data variables.product.prodname_dotcom %}.

## Tenez compte de la façon dont vous bénéficiez de différentes contributions

Déterminez si vous bénéficiez du travail de contributions open source, comme :
 - Documentation
 - Conception
 - Traduction
 - Bulletins d’informations
 - Examens du code
 - Tutoriels
 - Mentorat
 - Triage des problèmes et aide à la gestion des projets
 - Réponses aux questions
 - Planification d’événements

Pour plus d’informations sur les différents types de contributions open source, consultez le [Guide open source](https://opensource.guide/how-to-contribute/#you-dont-have-to-contribute-code).

Vous pouvez vérifier si les contributeurs dont vous avez bénéficié ont des profils {% data variables.product.prodname_sponsors %} à l’aide de la recherche sur https://github.com/sponsors/community.

## Rechercher un projet ou un contributeur spécifique

Sur https://github.com/sponsors/community, vous pouvez effectuer une recherche par écosystème, l’utilisation la plus récente, la plus récente mise à jour et plus encore. Vous pouvez filtrer les résultats de recherche pour votre compte personnel ou pour les organisations dont vous êtes membre.

## Étapes suivantes

Pour plus d’informations sur le démarrage de votre parrainage, consultez « [Parrainer un contributeur open source](/sponsors/sponsoring-open-source-contributors/sponsoring-an-open-source-contributor) ».
