## À propos de {% data variables.product.prodname_matching_fund %}

{% note %}

**Remarque :** {% data reusables.sponsors.matching-fund-eligible %}

{% endnote %}

{% data variables.product.prodname_matching_fund %} vise à bénéficier aux membres de la communauté {% data variables.product.prodname_dotcom %} qui développent des logiciels open source favorisant les [Recommandations de la communauté {% data variables.product.prodname_dotcom %}](/free-pro-team@latest/github/site-policy/github-community-guidelines). Les paiements aux organisations parrainées et les paiements provenant des organisations ne sont pas éligibles au programme {% data variables.product.prodname_matching_fund %}.

Pour être éligible au programme {% data variables.product.prodname_matching_fund %}, vous devez créer un profil susceptible d’attirer une communauté qui vous aidera à long terme. Pour plus d’informations, consultez « [Modification des informations de profil pour {% data variables.product.prodname_sponsors %}](/sponsors/receiving-sponsorships-through-github-sponsors/editing-your-profile-details-for-github-sponsors) ».

Les dons entre les développeurs parrainés ne sont pas doublés.
