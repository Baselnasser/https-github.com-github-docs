Vous pouvez découvrir, parcourir et installer des outils gratuits et payants, y compris des {% data variables.product.prodname_github_apps %}, des {% data variables.product.prodname_oauth_apps %} et des {% data variables.product.prodname_actions %}, dans [{% data variables.product.prodname_marketplace %}](https://github.com/marketplace).

Si vous achetez un outil payant, vous payez votre abonnement à l’outil avec les mêmes informations de facturation que celles que vous utilisez pour votre abonnement {% data variables.product.product_name %} et vous recevez une seule facture le jour habituel de votre facturation. Pour plus d’informations, consultez « [À propos de la facturation pour {% data variables.product.prodname_marketplace %}](/articles/about-billing-for-github-marketplace) ».

Vous pouvez également avoir la possibilité de sélectionner un essai gratuit de 14 jours pour certains outils. Vous pouvez annuler à tout moment votre essai gratuit sans être facturé, mais vous perdez alors automatiquement l’accès à l’outil. Votre abonnement payant commence à la fin de l’essai gratuit de 14 jours. Pour plus d’informations, consultez « [À propos de la facturation pour {% data variables.product.prodname_marketplace %}](/articles/about-billing-for-github-marketplace) ».

## Recherche d’outils sur {% data variables.product.prodname_marketplace %}

Vous pouvez découvrir, parcourir et installer des applications et des actions créées par d’autres utilisateurs sur {% data variables.product.prodname_marketplace %}, consultez « [Recherche dans {% data variables.product.prodname_marketplace %}](/search-github/searching-on-github/searching-github-marketplace) ».

{% data reusables.actions.actions-not-verified %}

Tout le monde peut ajouter une {% data variables.product.prodname_github_app %} ou une {% data variables.product.prodname_oauth_app %} gratuite sur {% data variables.product.prodname_marketplace %}. Les éditeurs d’applications payantes sont vérifiés par {% data variables.product.company_short %} et les listings de ces applications sont indiqués par un badge du marketplace {% octicon "verified" aria-label="Verified creator badge" %}. Vous voyez également des badges pour les applications non vérifiées et vérifiées. Ces applications ont été publiées avec la méthode précédente de vérification des applications individuelles. Pour plus d’informations sur le processus actuel, consultez « [À propos de GitHub Marketplace](/developers/github-marketplace/about-github-marketplace) » et « [Exigences de listing pour une application](/developers/github-marketplace/requirements-for-listing-an-app) ».
