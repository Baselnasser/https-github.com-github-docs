Apprendre toutes les commandes Git disponibles en une seule fois peut s’avérer une tâche intimidante. N’hésitez pas à utiliser les « [Aide-mémoire Git](https://training.github.com/) » pour obtenir rapidement des informations de référence sur les commandes les plus fréquemment utilisées. L’aide-mémoire « Utilisation de Git » est disponible en plusieurs langues. 

De plus, examinez notre page [Ressources pédagogiques Git et GitHub](/articles/git-and-github-learning-resources/) qui contient des liens vers des guides, des vidéos et bien plus encore.
