Pour en savoir plus sur l’accès à votre tableau de bord personnel, consultez « [À propos de votre tableau de bord personnel](/articles/about-your-personal-dashboard) ».

## Suivre des personnes

Lorsque vous suivez une personne sur {% data variables.product.prodname_dotcom %}, vous recevez des notifications à propos de son activité publiques sur votre tableau de bord personnel. Si une personne que vous suivez crée un dépôt, attribue des étoiles à un dépôt ou suit un autre utilisateur, l’activité s’affiche dans votre tableau de bord. 

Cliquez sur **Suivre** dans la page de profil d’une personne pour la suivre.

![Bouton S’abonner à l’utilisateur](/assets/images/help/profile/follow-user-button.png)

Pour plus d’informations, consultez « [Suivi de personnes](/get-started/exploring-projects-on-github/following-people) ».

## Surveillance d’un dépôt

Vous pouvez vous abonner à un dépôt pour le surveiller et recevoir des notifications concernant l’activité dans celui-ci. Quand le propriétaire met à jour le dépôt, les modifications sont visibles dans votre tableau de bord personnel. Pour en savoir plus, consultez « [Affichage de vos abonnements](/github/managing-subscriptions-and-notifications-on-github/viewing-your-subscriptions) ».

Cliquez sur **Surveiller** en haut d’un dépôt pour le surveiller.

![Bouton Surveiller un dépôt](/assets/images/help/repository/repo-actions-watch.png)

## Communication sur {% data variables.product.product_name %}

{% data variables.product.product_name %} fournit des outils de communication collaboratifs intégrés qui vous permettent de d’interagir étroitement avec votre communauté lors de la création de logiciels remarquables. Par exemple, vous pouvez contribuer au projet d’un autre utilisateur en créant une demande de tirage (pull request) ou utiliser des problèmes pour suivre les bogues ou suggérer des idées dans un dépôt. Vous pouvez également réfléchir à de nouvelles idées avec votre équipe par le biais de discussions. 

Pour obtenir une vue d’ensemble de ces outils, consultez « [Démarrage rapide pour la communication sur {% data variables.product.prodname_dotcom %}](/github/collaborating-with-issues-and-pull-requests/quickstart-for-communicating-on-github) ».

## Participation à des organisations

Les organisations sont des comptes partagés dans lesquels des entreprises et des projets open source peuvent collaborer sur de nombreux projets à la fois. Les propriétaires et les administrateurs peuvent constituer des équipes dotées d’autorisations spéciales, avoir un profil d’organisation publique et suivre l’activité au sein de l’organisation. Pour plus d’informations, consultez « [À propos des organisations](/articles/about-organizations/) ».

Dans votre tableau de bord, cliquez sur le menu déroulant de votre nom d’utilisateur situé sur le côté gauche. Ici, vous pouvez voir les organisations dont vous faites partie et basculer facilement entre elles.

![Liste déroulante Changer de contexte de compte](/assets/images/help/overview/dashboard-contextswitcher.png)

## Exploration d’autres projets sur {% data variables.product.prodname_dotcom %}

Vous pouvez découvrir de nouveaux projets intéressants dans la page Explorer de {% data variables.product.prodname_dotcom %}. Vous pouvez attribuer des étoiles aux projets intéressants pour qu’ils soient faciles à retrouver plus tard. Visitez votre page d’étoiles pour voir tous vos projets marqués d’une étoile. Pour plus d’informations sur les étoiles, consultez « [Enregistrement de dépôts avec étoiles](/get-started/exploring-projects-on-github/saving-repositories-with-stars) ».

Le flux de votre tableau de bord affiche des projets en fonction de vos intérêts, notamment les dépôts auxquels vous attribuez des étoiles, les personnes que vous suivez et vos contributions aux dépôts publics. Dans votre tableau de bord, vous pouvez voir les projets tendance, afficher les rubriques et consulter les collections. 

Pour explorer d’autres projets, consultez « {% data variables.explore.explore_github %} ».
