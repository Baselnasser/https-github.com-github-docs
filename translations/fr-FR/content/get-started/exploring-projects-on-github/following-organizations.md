## À propos des abonnés sur {% data variables.product.product_name %}

{% data reusables.organizations.about-following-organizations %} Pour plus d’informations, consultez « [À propos de votre tableau de bord personnel](/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-personal-account-settings/about-your-personal-dashboard#staying-updated-with-activity-from-the-community) ».

Vous pouvez arrêter de suivre une organisation si vous n’avez pas besoin d’examiner son activité {% ifversion fpt or ghec %}public{% endif %} sur {% data variables.product.product_name %}.

## Suivi d’une organisation sur {% data variables.product.product_name %}

1. Accédez à la page de l’organisation que vous voulez suivre
2. Cliquez sur **S’abonner** en haut à droite de l’en-tête

  ![Capture d’écran de l’en-tête de l’organisation, avec le bouton S’abonner sélectionné](/assets/images/help/profile/organization-profile-following.png)

## Arrêter de suivre une organisation sur {% data variables.product.product_name %}

1. Accédez à la page de l’organisation que vous ne voulez plus suivre
2. Cliquez sur **Se désabonner** en haut à droite de l’en-tête.

  ![Capture d’écran de l’en-tête de l’organisation, avec le bouton Se désabonner sélectionné](/assets/images/help/profile/organization-profile-unfollowing.png)

## Consultation des personnes et des organisations qu’un utilisateur suit sur {% data variables.product.product_name %}

1. Cliquez sur une image d’utilisateur pour afficher le profil de cet utilisateur.

  ![Image d’utilisateur](/assets/images/help/profile/user-profile-image.png)

2. Cliquez sur **abonnements** sous son image de profil.

  ![Abonnements de l’utilisateur](/assets/images/help/profile/user-profile-following.png)
