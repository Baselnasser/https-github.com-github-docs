## À propos des abonnés sur {% data variables.product.product_name %}

Quand vous suivez des personnes, vous voyez leur activité publique sur votre tableau de bord personnel. {% ifversion fpt or ghec %} Si une personne que vous suivez marque d’une étoile un dépôt public, {% data variables.product.product_name %} peut vous recommander le dépôt. {% endif %} Pour plus d’informations, consultez « [À propos de votre tableau de bord personnel](/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-personal-account-settings/about-your-personal-dashboard#staying-updated-with-activity-from-the-community) ».

Vous pouvez arrêter de suivre une personne si vous ne voulez pas voir son activité publique sur {% data variables.product.product_name %}.

## Suivi d’un utilisateur sur {% data variables.product.product_name %}

1. Cliquez sur une image d’utilisateur pour afficher le profil de cet utilisateur.

  ![Image d’utilisateur](/assets/images/help/profile/user-profile-image.png)

2. Cliquez sur **S’abonner** sous son image de profil.

  ![Bouton S’abonner à l’utilisateur](/assets/images/help/profile/follow-user-button.png)

## Arrêter de suivre un utilisateur sur {% data variables.product.product_name %}

1.  Cliquez sur une image d’utilisateur pour afficher le profil de cet utilisateur.

  ![Image d’utilisateur](/assets/images/help/profile/user-profile-image.png)

2.  Cliquez sur **Se désabonner** sous son image de profil.

  ![Bouton Profil utilisateur](/assets/images/help/profile/user-profile-unfollow-button.png)

## Consultation des utilisateurs suivis sur {% data variables.product.product_name %}

1. Cliquez sur une image d’utilisateur pour afficher le profil de cet utilisateur.

  ![Image d’utilisateur](/assets/images/help/profile/user-profile-image.png)

2. Cliquez sur **abonnements** sous son image de profil.

  ![Abonnements de l’utilisateur](/assets/images/help/profile/user-profile-following.png)

## Consultation des abonnés sur {% data variables.product.product_name %}

1. Cliquez sur une image d’utilisateur pour afficher le profil de cet utilisateur.

  ![Image d’utilisateur](/assets/images/help/profile/user-profile-image.png)

2. Cliquez sur **abonnés** sous son image de profil.

  ![Abonnements de l’utilisateur](/assets/images/help/profile/user-profile-followers.png)
