{% tip %}

**Conseils :**
- Vous pouvez utiliser un raccourci clavier pour remplir automatiquement le commentaire avec une réponse enregistrée. Pour plus d’informations, consultez « [Raccourcis clavier](/articles/keyboard-shortcuts/#comments) ».
- Vous pouvez filtrer la liste en tapant le titre de la réponse enregistrée.
