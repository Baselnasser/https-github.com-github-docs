## À propos de la prise en charge du navigateur web pour {% data variables.product.product_name %}

Nous concevons {% data variables.product.product_name %} en tenant compte des derniers navigateurs web. Nous vous recommandons d’utiliser la dernière version de l’un des navigateurs suivants.

 - [Apple Safari](https://apple.com/safari)
 - [Google Chrome](https://google.com/chrome)
 - [Microsoft Edge](https://microsoft.com/windows/microsoft-edge)
 - [Mozilla Firefox](https://mozilla.org/firefox)
 
Si vous n’utilisez pas la dernière version d’un navigateur recommandé, ou si vous utilisez un navigateur qui ne figure pas dans la liste ci-dessus, {% data variables.product.product_name %} ou certaines fonctionnalités peuvent ne pas fonctionner comme prévu, voire pas du tout.

Pour plus d’informations sur la façon dont nous maintenons la compatibilité des navigateurs pour les produits de {% data variables.product.company_short %}, consultez le référentiel [`github/browser-support`](https://github.com/github/browser-support).
## Support étendu des navigateurs web recommandés

Certains fournisseurs de navigateurs proposent des versions de support étendu. Nous mettons tout en œuvre pour assurer le bon fonctionnement des fonctions {% data variables.product.product_name %} dans la dernière version de support étendu pour :

- [Canal stable étendu](https://support.google.com/chrome/a/answer/9027636) de Chrome
- [Canal stable étendu](https://docs.microsoft.com/en-gb/deployedge/microsoft-edge-channels#extended-stable-channel) de Edge
- [Version de support étendu](https://www.mozilla.org/en-US/firefox/organizations/) de Firefox (ESR)

Dans les versions antérieures de support étendu, {% data variables.product.product_name %} peut ne pas fonctionner comme prévu, et certaines fonctionnalités peuvent ne pas être disponibles.


## Builds bêta et développeur

Vous pouvez rencontrer des bogues inattendus dans les builds bêta et développeur des navigateurs pris en charge. Si vous rencontrez un bogue lié à {% data variables.product.product_name %} dans l’une de ces builds non publiées, vérifiez qu’il existe également dans la version stable du même navigateur. Si le bogue est uniquement présent dans la version instable, vous pouvez le signaler au développeur du navigateur.
