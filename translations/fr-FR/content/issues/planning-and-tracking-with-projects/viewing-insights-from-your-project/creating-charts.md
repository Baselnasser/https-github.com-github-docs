3. Dans le menu de gauche, cliquez sur **Nouveau graphique**.
   ![Capture d’écran montrant le bouton Nouveau graphique](/assets/images/help/projects-v2/insights-new-chart.png)
4. Si vous le souhaitez, pour modifier le nom du nouveau graphique, cliquez sur {% octicon "triangle-down" aria-label="The triangle icon" %}, tapez un nouveau nom, puis appuyez sur <kbd>Retour</kbd>.
   ![Capture d’écran montrant comment renommer un graphique](/assets/images/help/projects-v2/insights-rename.png)
5. Au-dessus du graphique, tapez des filtres pour modifier les données utilisées pour générer le graphique. Pour plus d’informations, consultez « [Filtrage des projets](/issues/planning-and-tracking-with-projects/customizing-views-in-your-project/filtering-projects) ».
6. À droite de la zone de texte de filtre, cliquez sur **Enregistrer les modifications**.
   ![Capture d’écran montrant le bouton Enregistrer](/assets/images/help/projects-v2/insights-save-filter.png)
