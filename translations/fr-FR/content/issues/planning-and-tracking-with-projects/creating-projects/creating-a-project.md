Les {% data variables.product.prodname_projects_v2 %} sont une collection adaptable d’éléments qui sont toujours à jour avec les données {% data variables.product.company_short %}. Vos projets peuvent suivre les problèmes, les demandes de tirage et les idées que vous notez. Vous pouvez ajouter des champs personnalisés et créer des vues à des fins spécifiques.

## Création d’un projet

### Création d’un projet d’organisation

Les projets d’organisation peuvent suivre les problèmes et les demandes de tirage des dépôts de l’organisation.

{% data reusables.projects.create-project %}

### Création d’un projet d’utilisateur

Les projets d’utilisateur peuvent suivre les problèmes et les demandes de tirage des dépôts appartenant à votre compte personnel.

{% data reusables.projects.create-user-project %}
