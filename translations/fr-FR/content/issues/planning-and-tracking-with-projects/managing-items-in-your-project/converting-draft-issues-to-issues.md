## Conversion de brouillons de problèmes dans une disposition de table

1. Cliquez sur {% octicon "triangle-down" aria-label="the item menu" %} dans le brouillon de problème que vous souhaitez convertir.
  ![Capture d’écran montrant le bouton du menu d’éléments](/assets/images/help/projects-v2/item-context-menu-button-table.png)
2. Sélectionnez **Convertir en problème**.
  ![Capture d’écran montrant l’option « Convertir en problème »](/assets/images/help/projects-v2/item-convert-to-issue.png)
3. Sélectionnez le référentiel auquel vous souhaitez ajouter le problème.
  ![Capture d’écran montrant la sélection du dépôt](/assets/images/help/projects-v2/convert-to-issue-select-repo.png)

## Conversion de brouillons de problèmes dans une disposition de panneau

1. Cliquez sur {% octicon "kebab-horizontal" aria-label="the item menu" %} dans le brouillon de problème que vous souhaitez convertir.
  ![Capture d’écran montrant le bouton du menu d’éléments](/assets/images/help/projects-v2/item-context-menu-button-board.png)
2. Sélectionnez **Convertir en problème**.
  ![Capture d’écran montrant l’option « Convertir en problème »](/assets/images/help/projects-v2/item-convert-to-issue.png)
3. Sélectionnez le référentiel auquel vous souhaitez ajouter le problème.
  ![Capture d’écran montrant la sélection du dépôt](/assets/images/help/projects-v2/convert-to-issue-select-repo.png)
