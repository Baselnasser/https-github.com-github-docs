Les applications GitHub n’ont aucune autorisation par défaut. Quand vous créez une application GitHub, vous pouvez sélectionner les autorisations dont elle a besoin pour accéder aux données de l’utilisateur final. Vous pouvez également ajouter et supprimer des autorisations. Pour plus d’informations, consultez « [Modification des autorisations d’une application GitHub](/apps/managing-github-apps/editing-a-github-app-s-permissions/) ».
