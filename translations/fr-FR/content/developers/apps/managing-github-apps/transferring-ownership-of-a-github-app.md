![Sélection de l’application](/assets/images/github-apps/github_apps_select-app.png) {% data reusables.user-settings.github_apps_advanced %}
6. Cliquez sur **Transférer la propriété**.
![Bouton pour transférer la propriété](/assets/images/github-apps/github_apps_transfer_ownership.png)
7. Tapez le nom de l’application GitHub à transférer.
![Champ pour entrer le nom de l’application à transférer](/assets/images/github-apps/github_apps_transfer_app_name.png)
8. Tapez le nom de l’utilisateur ou de l’organisation auquel transférer l’application GitHub.
![Champ pour entrer l’utilisateur ou l’organisation vers lequel ou laquelle effectuer le transfert](/assets/images/github-apps/github_apps_transfer_new_owner.png)
9. Cliquez sur **Transférer cette application GitHub**.
![Bouton pour confirmer le transfert d’une application GitHub](/assets/images/github-apps/github_apps_transfer_integration.png)
