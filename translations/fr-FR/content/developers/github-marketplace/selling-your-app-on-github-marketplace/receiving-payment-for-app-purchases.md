Une fois que votre référencement sur {% data variables.product.prodname_marketplace %} est créé et approuvé pour une application avec un plan payant, vous fournissez des informations de paiement à {% data variables.product.product_name %} dans le cadre du processus d’intégration financière.

Une fois que votre chiffre d’affaires a atteint un minimum de 500 dollars US pour le mois, vous recevez un paiement électronique de la part de {% data variables.product.company_short %}. Ce paiement correspond au revenu issu des transactions sur la Place de marché auquel est soustrait le montant facturé par {% data variables.product.company_short %} pour couvrir les coûts d’exécution.

Pour les transactions effectuées avant le 1er janvier 2021, {% data variables.product.company_short %} prélève 25 % du revenu des transactions. Pour les transactions effectuées après cette date, seulement 5 % sont prélevés par {% data variables.product.company_short %}. Ce changement est répercuté sur les paiements reçus à compter de la fin janvier 2021.

{% note %}

**Remarque :** Pour plus d’informations sur les tarifs et les conditions de paiement en vigueur, consultez « [Contrat de développeur {% data variables.product.prodname_marketplace %}](/free-pro-team@latest/github/site-policy/github-marketplace-developer-agreement) ».

{% endnote %}
