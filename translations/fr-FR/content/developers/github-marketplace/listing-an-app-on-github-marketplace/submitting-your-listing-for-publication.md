Une fois que le référencement de votre application répond aux exigences de {% data variables.product.prodname_marketplace %}, vous pouvez soumettre votre application. Pour une vue d’ensemble des exigences de référencement d’application, consultez « [Conditions requises pour le référencement d’une application](/developers/github-marketplace/requirements-for-listing-an-app) ».

1. En haut à gauche de la page de référencement d’application, cliquez sur **Vue d’ensemble**. Pour plus d’informations sur la recherche de la page de référencement de votre application, consultez « [Rédaction d’un référencement pour votre application](/developers/github-marketplace/drafting-a-listing-for-your-app) ».
  
  ![Option Vue d’ensemble pour le brouillon de référencement sur la Place de marché](/assets/images/marketplace/edit-marketplace-listing-overview.png)

2. Pour soumettre le référencement de votre application terminé, cliquez sur **Demander la publication**.

  ![Liste de contrôle « Publier votre application sur la Place de marché » avec le bouton de soumission en bas](/assets/images/marketplace/publish-your-app-checklist-and-submission.png)

Un expert en intégration vous contactera avec des informations supplémentaires.
