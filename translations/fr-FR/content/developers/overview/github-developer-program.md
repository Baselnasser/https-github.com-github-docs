Vous créez une application intégrée à GitHub ? Inscrivez-vous à notre programme de développement ! Les possibilités sont infinies et vous profitez des kudos. [S’inscrire maintenant](https://github.com/developer/register)

## Faites-vous plaisir

Créez vos propres outils qui s’intègrent parfaitement là où vous poussez du code tous les jours.

## Prenez en compte l’entreprise

Envoyez un e-mail à <a href="mailto:partnerships@github.com">partnerships@github.com</a> pour demander des licences de développeur afin de créer et tester votre application sur {% data variables.product.prodname_ghe_server %}.

## Vous avez une intégration qui fonctionne avec GitHub ?

Magnifique. Participez à notre programme ! Voici comment passer l’information :</p>
* [Parlez-nous de votre intégration](https://support.github.com/contact?tags=rr-general-technical&form[subject]=New+GitHub+Integration).
* Utilisez le [logo Octocat ou GitHub](https://github.com/logos) pour montrer que votre produit fonctionne avec GitHub.
* Publiez une vidéo ou un blog sur votre site web pour présenter votre intégration.

## Vous êtes prêt à rejoindre le programme de développement GitHub ?

L’adhésion est ouverte aux développeurs individuels et aux entreprises qui ont :

* Une intégration en production ou en développement avec l’API GitHub.
* Une adresse e-mail que les utilisateurs GitHub peuvent utiliser pour obtenir un support.
