Les propriétaires de l’organisation peuvent définir des autorisations de création d’équipe.

Si vous ne définissez pas d’autorisations de création d’équipe, tous les membres de l’organisation peuvent créer des équipes par défaut.

{% data reusables.profile.access_org %} {% data reusables.profile.org_settings %} {% data reusables.organizations.member-privileges %}
5. Sous « Règles de création d’équipe », cochez ou décochez **Autoriser les membres à créer des équipes**.
![Case à cocher pour autoriser les membres à créer des équipes](/assets/images/help/organizations/allow-members-to-create-teams.png)
6. Cliquez sur **Enregistrer**.
