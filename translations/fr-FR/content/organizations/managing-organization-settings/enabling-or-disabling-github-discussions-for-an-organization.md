## À propos des discussions d’organisation

{% data reusables.discussions.about-organization-discussions %}

Vous pouvez également gérer des discussions pour un dépôt. Pour plus d’informations, consultez « [Activation ou désactivation de GitHub Discussions pour un dépôt](/repositories/managing-your-repositorys-settings-and-features/enabling-features-for-your-repository/enabling-or-disabling-github-discussions-for-a-repository) » et « [Gestion de la création de discussion pour les dépôts dans votre organisation](/organizations/managing-organization-settings/managing-discussion-creation-for-repositories-in-your-organization) ».

## Activation de désactivation de {% data variables.product.prodname_discussions %} pour votre organisation

{% data reusables.discussions.enabling-or-disabling-github-discussions-for-your-organization %}
1. Pour désactiver les discussions, sous « Discussions », désélectionnez **Activer les discussions pour cette organisation**.
