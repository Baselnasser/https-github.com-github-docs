Les propriétaires d’organisation peuvent définir des limitations pour l’affichage d’insights sur les dépendances de l’organisation. Par défaut, tous les membres d’une organisation peuvent afficher des insights sur les dépendances de l’organisation.

{% ifversion ghec %} Des propriétaires d’entreprise peuvent définir des limitations pour l’affichage d’insights sur les dépendances d’organisation pour toutes les organisations dans votre compte d’entreprise. Pour plus d’informations, consultez « [Application de stratégies pour les insights sur les dépendances dans votre entreprise](/admin/policies/enforcing-policies-for-your-enterprise/enforcing-policies-for-dependency-insights-in-your-enterprise) ».
{% endif %}
