Une fois que vous avez désactivé les insights des projets de votre organisation, il n’est plus possible d’accéder aux insights d’aucun projet appartenant à l’organisation.


{% data reusables.profile.access_org %} {% data reusables.profile.org_settings %}
1. Dans la barre latérale, cliquez sur **{% octicon "sliders" aria-label="The sliders icon" %} Fonctionnalités**.
   ![Capture d’écran montrant l’élément de menu Fonctionnalités](/assets/images/help/projects-v2/features-org-menu.png)
1. Sous « Insights », désélectionnez **Activer les insights pour l’organisation**.
   ![Capture d’écran montrant la case à cocher Activer les insights pour l’organisation](/assets/images/help/projects-v2/disable-insights-checkbox.png)
1. Cliquez sur **Enregistrer**.
   ![Capture d’écran montrant le bouton Enregistrer](/assets/images/help/projects-v2/disable-insights-save.png)
