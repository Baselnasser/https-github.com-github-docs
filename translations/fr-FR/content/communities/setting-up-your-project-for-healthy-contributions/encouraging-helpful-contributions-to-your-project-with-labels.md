Vous pouvez appliquer l’étiquette `good first issue` aux problèmes de votre dépôt public pour permettre aux utilisateurs de les trouver durant une recherche par étiquette. Pour plus d’informations sur la recherche par étiquette, consultez « [Recherche de problèmes et de demandes de tirage](/search-github/searching-on-github/searching-issues-and-pull-requests/#search-by-label) ».

{% data variables.product.prodname_dotcom %} utilise un algorithme pour identifier les problèmes les plus accessibles dans chaque dépôt, et les exposer à divers emplacements sur {% data variables.product.prodname_dotcom_the_website %}. L’ajout de l’étiquette `good first issue` peut augmenter la probabilité d’exposition de vos problèmes.
