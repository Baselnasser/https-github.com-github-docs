Quand vous débloquez un utilisateur, il peut vous inviter en tant que collaborateur dans ses dépôts. S’il vous mentionne via [@mention ](/articles/basic-writing-and-formatting-syntax/#mentioning-people-and-teams) n’importe où sur GitHub, vous recevez les notifications.

Dans les dépôts dont vous êtes propriétaire, l’utilisateur peut collaborer normalement.

Vous pouvez débloquer un utilisateur dans les paramètres de votre compte ou à partir de la page de profil de l’utilisateur.

## Déblocage d’un utilisateur dans les paramètres de votre compte

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.blocked_users %}
3. Sous « Utilisateurs bloqués », à côté de l’utilisateur à débloquer, cliquez sur **Débloquer**.
![Bouton Débloquer l’utilisateur](/assets/images/help/organizations/org-unblock-user-button.png)

## Déblocage d’un utilisateur à partir de sa page de profil

{% data reusables.profile.user_profile_page_navigation %}
2. Dans la barre latérale gauche, sous la photo de profil de l’utilisateur, cliquez sur {% octicon "kebab-horizontal" aria-label="The horizontal kebab icon" %}, puis sur **Débloquer ou signaler l’utilisateur**.
![Lien Débloquer ou signaler l’utilisateur](/assets/images/help/profile/profile-unblock-or-report-user.png)
3. Cliquez sur **Débloquer l’utilisateur**.
  ![Boîte modale avec les options permettant de débloquer un utilisateur ou de signaler un abus](/assets/images/help/profile/profile-unblockuser.png)

{% tip %}

**Conseil** : Les paramètres supprimés au moment où vous avez bloqué l’utilisateur, par exemple l’état de collaboration, les étoiles et les suivis, ne sont pas restaurés quand vous débloquez l’utilisateur.

{% endtip %}
