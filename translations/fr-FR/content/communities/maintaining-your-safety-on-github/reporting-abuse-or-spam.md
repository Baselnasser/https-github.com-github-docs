Les propriétaires, les collaborateurs, les contributeurs précédents et les personnes disposant d’un accès en écriture peuvent signaler les problèmes et les demandes de tirage ainsi que les commentaires relatifs aux problèmes, aux demandes de tirage et aux commits. Tout le monde peut signaler des applications dans {% data variables.product.prodname_marketplace %}.

## À propos du signalement d’abus ou de courrier indésirable

{% data reusables.policies.github-community-guidelines-and-terms %}

Vous pouvez signaler les utilisateurs qui ont enfreint les recommandations relatives à la communauté ou les conditions d’utilisation de {% data variables.product.prodname_dotcom %} via {% data variables.contact.report_abuse %} ou {% data variables.contact.report_content %}. Vous pouvez également signaler les problèmes et les demandes de tirage (pull requests) ainsi que les commentaires relatifs aux problèmes, aux demandes de tirage et aux commits.

Si le contenu signalé est activé pour un dépôt public, vous pouvez également signaler le contenu directement aux responsables de la gestion du dépôt.

Les utilisateurs situés en Inde peuvent contacter le responsable des réclamations de GitHub pour l’Inde en cliquant [ici](https://support.github.com/contact/india-grievance-officer).

## Signalement d’un problème ou d’une demande de tirage

1. Accédez au problème ou à la demande de tirage à signaler.
2. Dans le coin supérieur droit du problème ou de la demande de tirage, cliquez sur {% octicon "kebab-horizontal" aria-label="The horizontal kebab octicon" %}, puis sur **Signaler un contenu**.
  ![Bouton permettant de signaler un commentaire](/assets/images/help/repository/menu-report-issue-or-pr.png) {% data reusables.community.report-content %}

## Signalement d’un commentaire

1. Accédez au commentaire à signaler.
2. Dans le coin supérieur droit du commentaire, cliquez sur {% octicon "kebab-horizontal" aria-label="The horizontal kebab octicon" %}, puis sur **Signaler un contenu**.
![Menu Kebab avec l’option permettant de signaler un commentaire](/assets/images/help/repository/menu-report-comment.png) {% data reusables.community.report-content %}

## Signalement d’une application dans {% data variables.product.prodname_marketplace %}

  ![Bouton permettant de signaler une application dans {% data variables.product.prodname_marketplace %}](/assets/images/help/marketplace/marketplace-report-app.png)

## Signalement d’un abus relatif à un lien de contact dans le sélecteur de modèles

1. Accédez au dépôt qui contient le lien de contact à signaler.
2. Sous le nom du dépôt, cliquez sur {% octicon "issue-opened" aria-label="The issues icon" %} **Problèmes**.
3. Dans le coin inférieur droit du sélecteur de modèles, cliquez sur **Signaler un abus**.
  ![Lien permettant de signaler un abus](/assets/images/help/repository/template-chooser-report-abuse.png)
4. Remplissez le formulaire de contact pour informer le {% data variables.contact.contact_support %} du comportement du lien de contact, puis cliquez sur **Envoyer une demande**.
