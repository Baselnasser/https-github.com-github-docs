Toute personne disposant d’autorisations d’administrateur sur un dépôt peut gérer la façon dont les contributeurs signalent les abus dans le dépôt.

Vous pouvez activer ou désactiver le contenu signalé pour les dépôts publics appartenant à une organisation.

{% data reusables.repositories.navigate-to-repo %} {% data reusables.repositories.sidebar-settings %} {% data reusables.repositories.sidebar-moderation-reported-content %}
4. Sous « Contenu signalé », sélectionnez ou désélectionnez **Accepter les rapports de contenu des collaborateurs et des contributeurs antérieurs**.
  ![Case à cocher permettant d’accepter ou de refuser le signalement de contenu pour un dépôt](/assets/images/help/repository/reported-content-opt-in-checkbox.png)
