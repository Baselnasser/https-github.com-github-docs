Avec les API, vous pouvez automatiser de nombreuses tâches d’administration. Voici quelques exemples :

{% ifversion ghes %}
- Apporter des modifications à la {% data variables.enterprise.management_console %}. Pour plus d’informations, consultez « [{% data variables.enterprise.management_console %}](/enterprise/user/rest/reference/enterprise-admin#management-console) ».
- Configurer la synchronisation LDAP. Pour plus d’informations, consultez « [LDAP](/enterprise/user/rest/reference/enterprise-admin#ldap) ».{% endif %}
- Collecter des statistiques sur votre entreprise. Pour plus d’informations, consultez « [Statistiques d’administration](/rest/reference/enterprise-admin#admin-stats) ».
- Gérer votre compte d’entreprise. Pour plus d’informations, consultez « [Comptes d’entreprise](/graphql/guides/managing-enterprise-accounts) ».

Pour obtenir la documentation complète sur l’{% data variables.product.prodname_enterprise_api %}, consultez [API REST {% data variables.product.prodname_dotcom %}](/rest) et [API GraphQL {% data variables.product.prodname_dotcom%}](/graphql).
