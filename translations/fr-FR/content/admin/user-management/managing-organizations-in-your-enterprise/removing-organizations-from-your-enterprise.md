{% warning %}

**Avertissement** : Quand vous supprimez une organisation de votre entreprise :
- La facturation, la gestion des identités, les exigences d’authentification à 2 facteurs et d’autres stratégies pour l’organisation ne sont plus régies par votre entreprise.
- L’organisation est rétrogradée au plan gratuit.
- L’organisation est régie par nos conditions d’utilisation du service standard.
- Tous les dépôts internes à l’organisation sont convertis en dépôts privés.

{% endwarning %}

## Suppression d’une organisation de votre entreprise

{% data reusables.enterprise-accounts.access-enterprise %}
2. Sous « Organisations », dans la barre de recherche, commencez à taper le nom de l’organisation jusqu’à ce que l’organisation apparaisse dans les résultats de la recherche.
![Capture d’écran du champ de recherche pour les organisations](/assets/images/help/enterprises/organization-search.png)
3. À droite du nom de l’organisation, sélectionnez le menu déroulant sous {% octicon "gear" aria-label="The gear icon" %}, puis cliquez sur **Supprimer l’organisation**.
![Capture d’écran montrant une organisation dans les résultats de la recherche](/assets/images/help/enterprises/remove-organization.png)
4. Passez en revue les avertissements, puis cliquez sur **Supprimer l’organisation**.
![Capture d’écran montrant un message d’avertissement et le bouton pour supprimer l’organisation](/assets/images/help/enterprises/remove-organization-warning.png)
