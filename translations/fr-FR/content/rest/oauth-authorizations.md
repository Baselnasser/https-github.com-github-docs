## À propos de l’API OAuth authorizations

Vous pouvez utiliser cette API pour gérer l’accès que les applications OAuth ont sur votre compte. Vous pouvez uniquement accéder à cette API via l’[authentification de base](/rest/overview/other-authentication-methods#basic-authentication) en utilisant votre nom d’utilisateur et votre mot de passe, et non avec des jetons.

Si vous ou vos utilisateurs avez activé l’authentification à deux facteurs, veillez à comprendre comment [utiliser l’authentification à deux facteurs](/rest/overview/other-authentication-methods#working-with-two-factor-authentication).
