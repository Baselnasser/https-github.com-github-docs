## À propos de l’API Installations {% data variables.product.prodname_github_app %}

Une _installation_ fait référence à tout compte d’utilisateur ou d’organisation ayant installé l’application. Pour plus d’informations sur la manière d’authentifier en tant qu’installation, et de limiter l’accès à des dépôts spécifiques, consultez « [Authentification en tant qu’installation](/apps/building-github-apps/authenticating-with-github-apps/#authenticating-as-an-installation) ».

Pour répertorier toutes les installations d’application GitHub pour une organisation, consultez « [Répertorier les installations d’application pour une organisation](/rest/orgs/orgs#list-app-installations-for-an-organization) ».
