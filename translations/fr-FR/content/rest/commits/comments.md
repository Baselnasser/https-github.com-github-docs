## À propos de l’API Commit comments

L’API Commit comments vous permet de créer et de modifier des commentaires liés à des commits spécifiques.

### Types de médias personnalisés pour les commentaires de commit

Il s’agit des types de médias pris en charge pour les commentaires de commit. Plus d’informations sur l’utilisation des types de médias dans l’API sont disponibles [ici](/rest/overview/media-types).

    application/vnd.github-commitcomment.raw+json
    application/vnd.github-commitcomment.text+json
    application/vnd.github-commitcomment.html+json
    application/vnd.github-commitcomment.full+json

Pour plus d’informations, consultez « [Types de médias personnalisés](/rest/overview/media-types) ».
