## À propos de l’API Cache

L’API Cache {% data variables.product.prodname_actions %} vous permet d’interroger et de gérer le cache {% data variables.product.prodname_actions %} pour les référentiels. {% ifversion actions-cache-management %}Vous pouvez également installer une extension {% data variables.product.prodname_cli %} pour gérer vos caches à partir de la ligne de commande. {% endif %}Pour plus d’informations, consultez « [Mise en cache des dépendances pour accélérer les workflows](/actions/advanced-guides/caching-dependencies-to-speed-up-workflows#managing-caches) ».
