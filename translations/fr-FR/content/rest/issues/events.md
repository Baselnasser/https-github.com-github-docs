## À propos de l’API Événements de problème

L’API Événements de problème peut retourner différents types d’événements déclenchés par l’activité dans les problèmes et les demandes d’extraction. Pour plus d’informations sur les événements spécifiques que vous pouvez recevoir à partir de l’API Événements de problème, consultez « [Types d’événements de problème](/developers/webhooks-and-events/issue-event-types) ». Une API d’événements pour une activité GitHub en dehors des problèmes et demandes de tirage est également disponible. Pour plus d’informations, consultez [API d’événements](/developers/webhooks-and-events/github-event-types).
