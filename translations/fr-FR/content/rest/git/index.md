## À propos de la base de données Git

L’API REST vous permet de lire et d’écrire des objets Git bruts dans votre base de données Git sur {% data variables.product.product_name %} et de lister et de mettre à jour vos références (têtes et étiquettes de branche). Pour plus d’informations sur l’utilisation de l’API REST pour interagir avec votre base de données Git, consultez « [Bien démarrer avec l’API de données Git](/rest/guides/getting-started-with-the-git-database-api) ».
