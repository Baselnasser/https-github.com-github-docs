## À propos de l’API Arborescences Git

Un objet d’arborescence Git crée la hiérarchie entre les fichiers d’un référentiel Git. Vous pouvez utiliser l’objet d’arborescence Git pour créer la relation entre les répertoires et les fichiers qu’ils contiennent. Ces points de terminaison vous permettent de lire et d’écrire des [objets d’arborescence](https://git-scm.com/book/en/v1/Git-Internals-Git-Objects#Tree-Objects) dans votre base de données Git sur {% data variables.product.product_name %}.
