Les utilisateurs interagissent avec les référentiels en commentant, en ouvrant des problèmes et en créant des demandes de tirage. Les API Interactions permettent aux personnes disposant d’un accès propriétaire ou administrateur de restreindre temporairement l’interaction avec les référentiels publics à un certain type d’utilisateur.
