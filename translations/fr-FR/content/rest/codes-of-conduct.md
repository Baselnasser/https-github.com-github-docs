## À propos de l’API Codes of conduct

Vous pouvez utiliser l’API Codes of conduct pour récupérer des informations sur le code de conduite d’un dépôt. Pour obtenir le code de conduite d’un dépôt, utilisez le point de terminaison « [Obtenir un dépôt](/rest/reference/repos#get-a-repository) ».
