## À propos des objets

Les [Objets](https://graphql.github.io/graphql-spec/June2018/#sec-Objects) dans GraphQL représentent les ressources auxquelles vous pouvez accéder. Un objet peut contenir une liste de champs qui sont spécifiquement typés.

Par exemple, l’objet [`Repository`](/graphql/reference/objects#repository) a un champ appelé `name`, qui est une `String`.

Pour plus d’informations, consultez « [Présentation de GraphQL](/graphql/guides/introduction-to-graphql) ».
