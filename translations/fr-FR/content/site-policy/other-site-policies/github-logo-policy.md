Vous pouvez, dans certains cas, ajouter des logos {% data variables.product.prodname_dotcom %} à votre site Web ou à une application tierce. Pour de plus amples informations et des directives spécifiques sur l’utilisation des logos, consultez la [page Utilisation des logos de {% data variables.product.prodname_dotcom %}](https://github.com/logos).

Vous pouvez également utiliser un Octocat comme avatar personnel ou sur votre site Web pour établir un lien avec votre compte {% data variables.product.prodname_dotcom %}, mais pas pour votre entreprise ou un produit que vous concevez. {% data variables.product.prodname_dotcom %} possède une vaste collection d’Octocats dans l’[Octodex](https://octodex.github.com/). Pour plus d’informations sur l’utilisation des Octocats de l’Octodex, consultez la [FAQ de l’Octodex](https://octodex.github.com/faq/).
