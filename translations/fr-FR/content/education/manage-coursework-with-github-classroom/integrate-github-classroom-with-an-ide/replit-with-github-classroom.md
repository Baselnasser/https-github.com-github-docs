## Ce qui change avec l’intégration Repl.it

Depuis le 13 juillet 2021, l’intégration Repl.it n’est plus disponible. Voici ce qui a changé :

- Les nouveaux REPL privés et publics ne peuvent plus être créés à partir de GitHub Classroom.
- Les avantages Replit Hacker, comme Always On et Boosts, ne sont plus disponibles pour les nouveaux utilisateurs GitHub Classroom.

## Quel est l’impact sur les devoirs existants ?
Vos REPL GitHub Classroom existants restent privés et vous pouvez envoyer des travaux à GitHub Classroom via ces REPL jusqu’au 30 septembre 2021. Après septembre 2021, l’icône Replit sur le devoir README ne fonctionnera plus. Nous vous suggérons de supprimer les instructions des README utilisant l’intégration et d’informer les étudiants que l’icône ne fonctionnera plus.

Bien que vous ne puissiez plus créer de REPL privés à partir de Classroom, vous pouvez continuer à en créer dans Replit jusqu’au 30 septembre.

Nous savons que les intégrations d’IDE cloud sont importantes pour votre classe et nous nous efforçons de trouver d’autres options à vous proposer.
