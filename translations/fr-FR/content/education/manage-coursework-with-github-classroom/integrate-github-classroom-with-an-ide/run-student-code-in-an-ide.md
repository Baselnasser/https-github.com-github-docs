## À propos du code d’étudiant et des IDE

Si vous configurez un environnement de développement intégré (IDE) pour un devoir, vous pouvez exécuter le code dans l’IDE. Vous n’avez pas besoin de cloner le dépôt de devoir sur votre ordinateur.

Pour plus d’informations sur les IDE, consultez « [Intégrer {% data variables.product.prodname_classroom %} à un IDE](/education/manage-coursework-with-github-classroom/integrate-github-classroom-with-an-ide/integrate-github-classroom-with-an-ide) ».

## Exécution de code d’étudiant dans l’IDE 

{% data reusables.classroom.sign-into-github-classroom %} {% data reusables.classroom.click-classroom-in-list %} {% data reusables.classroom.click-assignment-in-list %}
1. À droite de l’envoi, cliquez sur **Voir l’IDE**.
