Vous pouvez utiliser un modèle de dépôt sur {% data variables.product.product_name %} comme code de démarrage d’un devoir sur {% data variables.product.prodname_classroom %}. Votre modèle de dépôt peut contenir du code réutilisable, de la documentation et d’autres ressources pour vos étudiants. Pour plus d’informations, consultez « [Création d’un modèle de dépôt](/github/creating-cloning-and-archiving-repositories/creating-a-template-repository) ».

Si vous voulez utiliser le modèle de dépôt pour votre devoir, il doit être détenu par votre organisation ou sa visibilité doit être publique.

Vous pouvez réutiliser un devoir existant, même s’il utilise un dépôt de modèles, dans toute autre classe à laquelle vous avez accès en tant qu’administrateur, y compris les classes d’une autre organisation. Pour plus d’informations, consultez « [Réutiliser un devoir](/education/manage-coursework-with-github-classroom/teach-with-github-classroom/reuse-an-assignment) ».
