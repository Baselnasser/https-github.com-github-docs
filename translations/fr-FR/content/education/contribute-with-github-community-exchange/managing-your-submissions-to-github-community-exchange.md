## À propos de vos soumissions

Pendant le processus de soumission à {% data variables.product.prodname_community_exchange %}, vous choisissez un objectif, des thèmes et des offres pour votre dépôt. Une fois qu’un dépôt a été soumis à {% data variables.product.prodname_community_exchange %}, il est publié avec ces détails. Pour plus d’informations, consultez « [Soumission de votre dépôt à GitHub Community Exchange](/education/contribute-with-github-community-exchange/submitting-your-repository-to-github-community-exchange) ».

Une fois que vous avez soumis votre dépôt, vous pouvez toujours modifier les thèmes et les offres qui lui sont associés. Vous pouvez également mettre à jour l’objectif de votre dépôt en modifiant le ou les thèmes correspondants. Pour plus d’informations, consultez « [Classification de votre référentiel avec des rubriques](/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/classifying-your-repository-with-topics) ».

Le langage associé à votre dépôt est le langage principal utilisé et est automatiquement déterminé par {% data variables.product.prodname_dotcom %}. Pour plus d’informations, consultez « [À propos des langages de dépôt](/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-repository-languages) ».

La communauté {% data variables.product.prodname_community_exchange %} modère toutes les soumissions de dépôt.

## Gestion de vos soumissions

1. Depuis votre tableau de bord {% data variables.product.prodname_global_campus %}, accédez à la page d’accueil de {% data variables.product.prodname_community_exchange %}.
1. Au-dessus de la liste des dépôts, cliquez sur l’onglet **Soumissions**. ![Capture d’écran de l’onglet Soumissions](/assets/images/help/education/community-exchange-submissions-tab.png)
1. Si vous le souhaitez, modifiez votre dépôt soumis. À droite du dépôt que vous souhaitez modifier, cliquez sur {% octicon "pencil" aria-label="The edit icon" %} pour accéder directement à la page d’accueil de votre dépôt. À partir de là, vous pouvez mettre à jour l’objectif, les thèmes et les offres affectés à votre dépôt.
1. Si vous le souhaitez, supprimez un dépôt soumis de la galerie. À droite de la soumission du dépôt que vous souhaitez supprimer, cliquez sur {% octicon "trash" aria-label="The trash icon" %}.
