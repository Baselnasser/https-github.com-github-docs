## Recherche de dépôts intéressants

Vous pouvez parcourir les dépôts soumis à {% data variables.product.prodname_community_exchange %} dans la page de la galerie.

Alors que vous vous demandez quels dépôts open source pourraient être intéressants, réfléchissez à ce que vous voulez faire, apprendre ou collaborer, ou à des thèmes ou des langages en particulier qui pourraient constituer un bon point de départ.

Lorsque vous explorez les dépôts de la galerie {% data variables.product.prodname_community_exchange %}, vous pouvez filtrer les dépôts disponibles par objectif, thèmes ou langages, et rechercher des dépôts par nom et description. Vous pouvez trier la liste des dépôts de la galerie par date de soumission ou par nombre d’étoiles, de duplications ou de problèmes que présente un dépôt.

![Capture d’écran de la zone de recherche Community Exchange et des filtres déroulants](/assets/images/help/education/community-exchange-search-and-filter.png)

## Ajout d’étoiles aux dépôts

Vous pouvez ajouter des étoiles aux dépôts listés dans la galerie {% data variables.product.prodname_community_exchange %}. L’ajout d’étoiles permet de retrouver facilement un dépôt dans la galerie. Quand vous ajoutez une étoile à un dépôt, vous montrez également au chargé de maintenance du dépôt que vous appréciez son travail.

Les listes de dépôts de la galerie {% data variables.product.prodname_community_exchange %} peuvent être triées sur le nombre d’étoiles qu’a un dépôt.

Pour ajouter des étoiles à un dépôt : accédez à votre page d’accueil {% data variables.product.prodname_community_exchange %}, recherchez le dépôt auquel vous voulez ajouter des étoiles et cliquez sur le bouton {% octicon "star" aria-label="The star icon" %} **Étoile** à côté de son nom.

## Signalement d’abus

La communauté {% data variables.product.prodname_community_exchange %} modère les soumissions de dépôt. Vous pouvez signaler des dépôts abusifs ou du contenu indésirable ou dérangeant à tout moment.

Pour signaler un dépôt abusif : accédez à votre page d’accueil {% data variables.product.prodname_community_exchange %}, recherchez le dépôt que vous souhaitez signaler, cliquez sur la liste déroulante {% octicon "kebab-horizontal" aria-label="The edit icon" %} à côté de son nom, puis cliquez sur {% octicon "report" aria-label="The report symbol" %} **Signaler un abus**.
