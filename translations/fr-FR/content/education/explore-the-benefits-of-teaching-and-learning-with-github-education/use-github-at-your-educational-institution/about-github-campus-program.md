{% data variables.product.prodname_campus_program %} est un package d’accès Premium à {% data variables.product.prodname_dotcom %} pour les établissements d’enseignement qui remettent des diplômes ou des certificats. {% data variables.product.prodname_campus_program %} comprend :

- Accès gratuit à {% data variables.product.prodname_ghe_cloud %} et {% data variables.product.prodname_ghe_server %} pour tous vos services techniques et universitaires
- 50 000 minutes de {% data variables.product.prodname_actions %} et 50 Go de stockage {% data variables.product.prodname_registry %}
- Formation des enseignants pour maîtriser Git et {% data variables.product.prodname_dotcom %}
- Accès exclusif aux nouvelles fonctionnalités, récompense propre à GitHub Education et outils de développement gratuits des partenaires {% data variables.product.prodname_dotcom %}
- Accès automatique aux fonctionnalités {% data variables.product.prodname_education %} Premium, comme le {% data variables.product.prodname_student_pack %}

Pour en savoir plus sur l’utilisation de GitHub par les enseignants, consultez [Témoignages sur GitHub Education.](https://education.github.com/stories)

## Conditions générales de {% data variables.product.prodname_campus_program %}

- La licence est gratuite pendant un an et automatiquement renouvelée gratuitement tous les 2 ans. Vous pouvez continuer à utiliser la licence gratuite à condition de respecter les conditions du contrat. Tout établissement scolaire pouvant accepter les [conditions du programme](https://education.github.com/schools/terms) est le bienvenu.

- Notez que les licences sont utilisées par l’ensemble de l’établissement. Les services informatiques internes, les groupes de recherche universitaires, les collaborateurs, les étudiants et d’autres services non universitaires peuvent utiliser les licences à condition de ne pas faire commerce de son utilisation. Les groupes de recherche financés par des sources externes qui sont hébergés à l’université ne peuvent pas utiliser les licences gratuites.

- Vous devez proposer {% data variables.product.prodname_dotcom %} à tous vos services techniques et universitaires, et le logo de votre établissement scolaire est partagé sur le site web GitHub Education en tant que partenaire {% data variables.product.prodname_campus_program %}.

- Les nouvelles organisations de votre entreprise sont automatiquement ajoutées à votre compte d’entreprise. Pour ajouter des organisations qui existaient avant que votre établissement ne rejoigne {% data variables.product.prodname_campus_program %}, contactez le [support GitHub Education](https://support.github.com/contact/education). Pour plus d’informations sur l’administration de votre entreprise, consultez la [documentation des administrateurs d’entreprise](/admin). Les nouvelles organisations de votre entreprise sont automatiquement ajoutées à votre compte d’entreprise. Pour ajouter des organisations qui existaient avant que votre établissement ne rejoigne {% data variables.product.prodname_campus_program %}, contactez le support GitHub Education. 


Pour en savoir plus sur les pratiques de confidentialité de {% data variables.product.prodname_dotcom %}, consultez « [Pratiques de confidentialité générales](/github/site-policy/global-privacy-practices) ».

## Éligibilité à {% data variables.product.prodname_campus_program %}

- Souvent, dans un campus, c’est le directeur technique/directeur de l’information, doyen, directeur de service ou directeur technologique qui signe les conditions du programme pour le compte du campus.

- Si votre établissement scolaire n’a pas d’adresses e-mail, {% data variables.product.prodname_dotcom %} contacte vos administrateurs de compte avec une option alternative pour vous permettre de distribuer le Student Developer Pack à vos étudiants.

Pour plus d’informations, consultez la page [{% data variables.product.prodname_campus_program %}](https://education.github.com/schools) officielle.

Si vous êtes étudiant ou enseignant et que votre établissement n’est pas partenaire de {% data variables.product.prodname_dotcom %} en tant qu’établissement scolaire {% data variables.product.prodname_campus_program %}, vous pouvez toujours faire la demande individuellement pour obtenir des réductions sur l’utilisation de {% data variables.product.prodname_dotcom %}. Pour demander le Student Developer Pack, [consultez le formulaire de demande](https://education.github.com/pack/join).
