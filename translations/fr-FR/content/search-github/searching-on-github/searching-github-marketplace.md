## À propos de la recherche sur la {% data variables.product.prodname_marketplace %}

Vous pouvez trouver des applications et des actions sur la {% data variables.product.prodname_marketplace %} de deux manières :

- Recherchez sur la {% data variables.product.prodname_marketplace %}.
- Recherchez sur tout le {% data variables.product.prodname_dotcom_the_website %}, puis filtrez les résultats.

## Recherche sur la {% data variables.product.prodname_marketplace %}

1. En haut d’une page, cliquez sur **Place de marché**.
  ![Lien Place de marché](/assets/images/help/search/marketplace-link.png)
2. Tapez des mots clés et appuyez sur **Entrée**.
  ![Recherche de linter sur la {% data variables.product.prodname_marketplace %}](/assets/images/help/search/marketplace-apps-and-actions-search-field.png)
3. Filtrez éventuellement vos résultats en cliquant sur une ou plusieurs options dans la barre latérale située à gauche.

## Recherche sur tout le {% data variables.product.prodname_dotcom_the_website %}

Chaque fois que vous effectuez une recherche sur tout le {% data variables.product.prodname_dotcom_the_website %}, vous pouvez filtrer les résultats pour voir uniquement les applications et actions correspondantes de la {% data variables.product.prodname_marketplace %}.

1. Accédez à https://github.com/search.
2. Tapez des mots clés et appuyez sur **Entrée**.
  ![champ de recherche](/assets/images/help/search/search-field.png)
3. Dans la barre latérale située à gauche, cliquez sur **Place de marché**.
  ![Résultats de la recherche de linter avec l’option du menu latéral Place de marché mise en évidence](/assets/images/help/search/marketplace-left-side-navigation.png)
