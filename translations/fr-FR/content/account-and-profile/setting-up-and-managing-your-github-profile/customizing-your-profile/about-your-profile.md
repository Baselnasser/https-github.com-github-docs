Vous pouvez ajouter des informations personnelles sur vous-même dans votre bio, comme les lieux où vous avez travaillé, les projets auxquels vous avez contribué ou vos domaines intérêts que d’autres personnes pourraient aimer connaître. Pour plus d’informations, consultez « [Ajout d’une bio à votre profil](/articles/personalizing-your-profile/#adding-a-bio-to-your-profile) ».

{% ifversion fpt or ghes or ghec %}

{% data reusables.profile.profile-readme %}

![Fichier README de profil affiché sur le profil](/assets/images/help/repository/profile-with-readme.png)

{% endif %}

Les personnes qui visitent votre profil voient une chronologie de votre activité de contribution, comme les problèmes et les demandes de tirage (pull requests) que vous avez ouverts, les commits que vous avez effectués et les demandes de tirage que vous avez révisées. Vous pouvez choisir d’afficher uniquement les contributions publiques, ou d’inclure également les contributions privées et anonymisées. Pour plus d’informations, consultez « [Affichage des contributions sur votre page de profil](/articles/viewing-contributions-on-your-profile-page) » ou « [Publicisation ou masquage de vos contributions privées sur votre profil](/articles/publicizing-or-hiding-your-private-contributions-on-your-profile) ».

Les personnes qui visitent votre profil peuvent également voir les informations suivantes.

- Dépôts et gists dont vous êtes propriétaire ou auxquels vous contribuez. {% ifversion fpt or ghes or ghec %}Vous pouvez mettre en avant votre meilleur travail en épinglant des dépôts et des gists à votre profil. Pour plus d’informations, consultez « [Épinglage d’éléments à votre profil](/github/setting-up-and-managing-your-github-profile/pinning-items-to-your-profile) ».{% endif %}
- Référentiels que vous avez mis en favoris{% ifversion fpt or ghec %} et organisés en listes{% endif %}. Pour plus d’informations, consultez « [Enregistrement de dépôts avec des étoiles](/articles/saving-repositories-with-stars/) ».
- Vue d’ensemble de votre activité dans les organisations, les dépôts et les équipes où vous êtes le plus actif. Pour plus d’informations, consultez « [Affichage d’une vue d’ensemble de votre activité sur votre profil](/articles/showing-an-overview-of-your-activity-on-your-profile) ».{% ifversion fpt or ghec %}
- Badges et succès digitaux qui soulignent votre activité qui montrent si vous utilisez {% data variables.product.prodname_pro %} ou participez à des programmes tels que {% data variables.product.prodname_arctic_vault %}, {% data variables.product.prodname_sponsors %} ou le {% data variables.product.company_short %} Developer Program. Pour plus d’informations, consultez « [Personnalisation de votre profil](/github/setting-up-and-managing-your-github-profile/personalizing-your-profile#displaying-badges-on-your-profile) ».{% endif %}

Vous pouvez également définir un état sur votre profil afin de fournir des informations sur votre disponibilité. Pour plus d’informations, consultez « [Définition d’un état](/articles/personalizing-your-profile/#setting-a-status) ».
