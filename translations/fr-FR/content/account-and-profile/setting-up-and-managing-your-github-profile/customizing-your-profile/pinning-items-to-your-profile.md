Vous pouvez épingler un dépôt public si vous en êtes le propriétaire ou si vous y avez apporté des contributions. Les commits sur les duplications (forks) ne comptent pas comme contributions ; vous ne pouvez donc pas épingler une duplication dont vous n’êtes pas propriétaire. Pour plus d’informations, consultez « [Pourquoi mes contributions ne s’affichent-elles pas sur mon profil ?](/articles/why-are-my-contributions-not-showing-up-on-my-profile) ».

Vous pouvez épingler n’importe quel gist public dont vous êtes propriétaire.

Les éléments épinglés incluent des informations importantes sur l’élément, comme le nombre d’étoiles qu’un dépôt a reçu ou les premières lignes d’un gist. Une fois que vous avez épinglé des éléments à votre profil, la section « Épinglés » remplace la section « Dépôts populaires » sur votre profil.

Vous pouvez réorganiser les éléments dans la section « Épinglés ». Dans le coin supérieur droit d’une épingle, cliquez sur {% octicon "grabber" aria-label="The grabber symbol" %} et faites glisser l’épingle vers un nouvel emplacement.

{% data reusables.profile.access_profile %}
2. Dans la section « Dépôts populaires » ou « Épinglés », cliquez sur **Personnaliser vos épingles**.
    ![Bouton Personnaliser vos épingles](/assets/images/help/profile/customize-pinned-repositories.png)
3. Pour afficher une liste d’éléments à épingler pouvant faire l’objet d’une recherche, sélectionnez « Dépôts », « Gists » ou les deux.
    ![Cases à cocher pour sélectionner les types d’éléments à afficher](/assets/images/help/profile/pinned-repo-picker.png)
4. Si vous le souhaitez, pour faciliter la recherche d’un élément spécifique, dans le champ de filtre, tapez le nom d’un utilisateur, d’une organisation, d’un dépôt ou d’un gist.
    ![Filtrer des éléments](/assets/images/help/profile/pinned-repo-search.png)
5. Sélectionnez une combinaison de six dépôts et/ou gists maximum à afficher.
    ![Sélectionner des éléments](/assets/images/help/profile/select-items-to-pin.png)
6. Cliquez sur **Enregistrer les épingles**.
    ![Bouton Enregistrer les épingles](/assets/images/help/profile/save-pinned-repositories.png)
