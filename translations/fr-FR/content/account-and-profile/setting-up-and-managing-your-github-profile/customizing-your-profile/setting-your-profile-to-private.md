## À propos des profils privés

Pour masquer des parties de votre page de profil, vous pouvez rendre votre profil privé. Cela masque également votre activité dans différentes fonctionnalités sociales sur {% data variables.product.prodname_dotcom_the_website %}. Un profil privé masque les informations auprès de tous les utilisateurs, et il n’existe actuellement aucune option permettant aux utilisateurs spécifiés de voir votre activité.

Après avoir rendu votre profil privé, vous pouvez toujours afficher toutes vos informations lorsque vous visitez votre propre profil.

Les profils privés ne peuvent pas recevoir de parrainages sous [{% data variables.product.prodname_sponsors %}](/sponsors/getting-started-with-github-sponsors/about-github-sponsors). Pour être éligible à {% data variables.product.prodname_sponsors %}, votre profil ne peut pas être privé.

## Différences entre les profils privés et publics

Lorsque votre profil est privé, le contenu suivant est masqué dans votre page de profil :

- Réalisations et points forts.
- Vue d’ensemble de l’activité et flux d’activité.
- Graphe de contribution.
- Nombre d’abonnements/d’abonnés.
- Boutons S’abonner et Parrainer.
- Appartenances à l’organisation.
- Onglets liés aux étoiles, aux projets, aux packages et au parrainage.

{% note %}

**Remarque** : Lorsque votre profil est privé, certains champs facultatifs sont toujours visibles publiquement, comme README, la biographie et la photo de profil.

{% endnote %}

## Modifications apportées aux rapports sur vos activités

En rendant votre profil privé, vous ne supprimerez ni ne masquerez l’activité passée ; ce paramètre s’applique uniquement à votre activité pendant que le paramètre privé est activé.

Lorsque votre profil est privé, votre activité {% data variables.product.prodname_dotcom_the_website %} n’apparaît pas dans les emplacements suivants :

- Flux d’activité pour les autres utilisateurs.
- Leaderboards de discussions.
- La page [Tendances](https://github.com/trending).

{% note %}

**Remarque** : votre activité sur les référentiels publics sera toujours visible publiquement par toute personne affichant ces référentiels, et certaines données d’activité peuvent toujours être disponibles via l’API {% data variables.product.prodname_dotcom %}.

{% endnote %}
