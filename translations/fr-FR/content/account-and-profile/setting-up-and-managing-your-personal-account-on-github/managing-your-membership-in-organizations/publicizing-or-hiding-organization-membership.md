## Modification de la visibilité de votre appartenance à une organisation

{% data reusables.profile.access_org %} {% data reusables.user-settings.access_org %} {% data reusables.organizations.people %}
4. Recherchez votre nom d’utilisateur dans la liste des membres. Si la liste est volumineuse, vous pouvez rechercher votre nom d’utilisateur dans la zone de recherche.
![Zone de recherche des membres de l’organisation](/assets/images/help/organizations/member-search-box.png)
5. Dans le menu à droite de votre nom d’utilisateur, choisissez une nouvelle option de visibilité :
    - Pour publiciser votre appartenance, choisissez **Public**.
    - Pour masquer votre appartenance, choisissez **Privé**.
  ![Lien de visibilité des membres de l’organisation](/assets/images/help/organizations/member-visibility-link.png)
