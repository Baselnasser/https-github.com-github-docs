## À propos de la gestion du nom de branche par défaut

Lorsque vous créez un dépôt sur {% data variables.product.product_location %}, il contient une seule branche, qui est la branche par défaut. Vous pouvez modifier le nom que {% data variables.product.product_name %} utilise pour la branche par défaut dans les nouveaux dépôts que vous créez. Pour plus d’informations sur la branche par défaut, consultez « [À propos des branches](/github/collaborating-with-issues-and-pull-requests/about-branches#about-the-default-branch) ».

## Définition du nom de branche par défaut

3. Sous « Branche par défaut du dépôt », cliquez sur **Changer la branche par défaut du dépôt**.
    ![Bouton de remplacement](/assets/images/help/settings/repo-default-name-button.png)
4. Tapez le nom par défaut que vous souhaitez utiliser pour les nouvelles branches.

5. Cliquez sur **Update**.
    ![Bouton Mettre à jour](/assets/images/help/settings/repo-default-name-update.png)
