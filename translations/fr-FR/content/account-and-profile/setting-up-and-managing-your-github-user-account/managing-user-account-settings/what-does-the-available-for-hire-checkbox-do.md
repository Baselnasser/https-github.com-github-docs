Remarque de dépréciation : GitHub Jobs est désormais déprécié. La dernière date de publication d’un emploi était le 19 mai 2021. Le site GitHub Jobs a été mis complètement hors service le 19 août 2021, et redirige maintenant vers un [billet de blog GitHub](https://github.blog/changelog/2021-04-19-deprecation-notice-github-jobs-site/) qui contient plus d’informations sur la dépréciation de GitHub Jobs.

{% endwarning %}

Le tableau [GitHub Jobs](https://jobs.github.com/) est un excellent moyen de trouver un emploi dans le secteur de la technologie. Vous pouvez choisir d’afficher les emplois publiés ici sur votre tableau de bord GitHub.

![Annonces GitHub Jobs sur le tableau de bord](/assets/images/help/settings/jobs-ads-on-dashboard.png)
