Quand vous annulez une application, votre abonnement reste actif jusqu’à la fin de votre période de facturation actuelle. L’annulation prend effet à votre prochaine date de facturation. Pour plus d’informations, consultez « [À propos de la facturation pour la {% data variables.product.prodname_marketplace %}](/articles/about-billing-for-github-marketplace) ».

Quand vous annulez un essai gratuit sur un plan payant, votre abonnement est immédiatement annulé et vous perdez l’accès à l’application. Si vous n’annulez pas votre essai gratuit pendant la période d’évaluation, le mode de paiement associé à votre compte est facturé pour le plan que vous avez choisi à la fin de la période d’évaluation. Pour plus d’informations, consultez « [À propos de la facturation pour la {% data variables.product.prodname_marketplace %}](/articles/about-billing-for-github-marketplace) ».
