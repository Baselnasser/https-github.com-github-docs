Quand vous faites passer une application au niveau inférieur, votre abonnement reste actif jusqu’à la fin de votre période de facturation actuelle. Le passage au niveau inférieur prend effet à votre prochaine date de facturation. Pour plus d’informations, consultez « [À propos de la facturation pour la {% data variables.product.prodname_marketplace %}](/articles/about-billing-for-github-marketplace) ».
