## Achat d’un stockage et d’une bande passante supplémentaires

Le supplément de stockage et de bande passante est proposé dans un même pack de données. Un pack de données coûte {% data variables.large_files.pack_monthly_price %} et offre un quota mensuel de {% data variables.large_files.pack_monthly_bandwidth %} pour la bande passante et de {% data variables.large_files.pack_monthly_storage %} pour le stockage. Vous pouvez acheter autant de packs de données que nécessaire. Par exemple, si vous avez besoin de 150 Go de stockage, vous achetez trois packs de données.

L’achat de packs de données pour {% data variables.large_files.product_name_short %} est indépendant de toute autre fonctionnalité ou produit payant sur {% data variables.product.product_name %}.

## Réduction du stockage et de la bande passante

Si vous réduisez le nombre de packs de données supplémentaires, vos modifications prennent effet à votre prochaine date de facturation.
