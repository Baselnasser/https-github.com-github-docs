Nous ne fournissons pas de facturation ou de prise en charge des commandes d’achat pour les comptes personnels. Nous envoyons par e-mail des reçus mensuels ou annuels à la date de facturation de votre compte. Si votre entreprise, votre pays ou votre comptable exige que vos reçus fournissent plus de détails, vous pouvez également [ajouter des informations](/articles/adding-information-to-your-personal-account-s-receipts) à vos reçus.

## Mise à jour du mode de paiement de votre compte personnel

{% data reusables.user-settings.billing_plans %} {% data reusables.dotcom_billing.update_payment_method %}
1. Si votre compte comporte des informations de facturation que vous souhaitez mettre à jour, cliquez sur **Modifier**.
![Facturation - Bouton Nouvelle carte](/assets/images/help/billing/billing-information-edit-button.png) {% data reusables.dotcom_billing.enter-billing-info %}
1. Si votre compte dispose d’un mode de paiement que vous souhaitez mettre à jour, cliquez sur **Modifier**.
![Facturation - Bouton Nouvelle carte](/assets/images/help/billing/billing-payment-method-edit-button.png) {% data reusables.dotcom_billing.enter-payment-info %}

## Mise à jour du mode de paiement de votre organisation

{% data reusables.dotcom_billing.org-billing-perms %}

Si votre organisation est basée en dehors des États-Unis ou que vous utilisez un compte courant d’entreprise pour payer pour {% data variables.product.product_name %}, PayPal peut être un mode de paiement utile.

{% data reusables.organizations.billing-settings %} {% data reusables.dotcom_billing.update_payment_method %}
1. Si votre compte dispose d’une carte de crédit que vous souhaitez mettre à jour, cliquez sur **Nouvelle carte**.
![Facturation - Bouton Nouvelle carte](/assets/images/help/billing/billing-new-card-button.png) {% data reusables.dotcom_billing.enter-payment-info %}
