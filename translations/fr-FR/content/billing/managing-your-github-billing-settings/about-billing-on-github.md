## Fonctionnalités et produits payants

Vous pouvez mettre à niveau votre compte personnel ou votre organisation vers un abonnement payant à tout moment. Vous pouvez également choisir d’acheter des applications à partir de la {% data variables.product.prodname_marketplace %} ou mettre à niveau le stockage et la bande passante pour {% data variables.large_files.product_name_long %}. Pour plus d'informations, consultez les pages suivantes :
- « [À propos de la facturation pour les comptes {% data variables.product.prodname_dotcom %}](/articles/about-billing-for-github-accounts) »
- « [À propos de la facturation pour la {% data variables.product.prodname_marketplace %}](/articles/about-billing-for-github-marketplace) »
- « [À propos de la facturation pour {% data variables.large_files.product_name_long %}](/articles/about-billing-for-git-large-file-storage) »
- « [À propos de la facturation pour {% data variables.product.prodname_sponsors %}](/articles/about-billing-for-github-sponsors) »

## Gestion des paramètres de facturation

Vous devez gérer les paramètres de facturation, le mode de paiement et les fonctionnalités et produits payants pour chacun de vos comptes séparément. Vous pouvez choisir de payer mensuellement ou annuellement pour les produits et fonctionnalités payants de chaque compte. Chaque fonctionnalité ou produit payant associé à un compte partage une date de facturation, un mode de paiement et un reçu.

{% data reusables.dotcom_billing.payment-methods %} {% data reusables.dotcom_billing.same-payment-method %}

Pour plus d’informations, consultez « [Gestion de vos paramètres de facturation {% data variables.product.prodname_dotcom %}](/articles/managing-your-github-billing-settings) ».

## Basculement entre les paramètres de vos différents comptes

Si vous êtes propriétaire d’une organisation ou d’une entreprise, vous pouvez basculer entre les paramètres de vos différents comptes en utilisant le sélecteur de contexte dans vos paramètres. 

{% data reusables.user-settings.access_settings %}
1. En haut de la page, à droite de votre nom d’utilisateur, cliquez sur **Basculer vers un autre compte**.
![Bouton du sélecteur de contexte](/assets/images/help/settings/context-switcher-button.png)
1. Commencez à taper le nom du compte vers lequel vous souhaitez basculer, puis cliquez sur le nom du compte.
![Menu du sélecteur de contexte](/assets/images/help/settings/context-switcher-menu.png)
1. Dans la barre latérale gauche, cliquez sur **{% octicon "credit-card" aria-label="The credit card icon" %} Facturation et plans**.
