Si votre carte est refusée, nous vous envoyons un e-mail expliquant la raison pour laquelle le paiement a été refusé. Vous disposez de quelques jours pour résoudre le problème avant que nous essayions de vous refacturer.

## Vérifier la date d’expiration de votre carte

Si votre carte a expiré, vous devez mettre à jour les informations de paiement de votre compte. Pour plus d’informations, consultez « [Ajout ou modification d’un mode de paiement](/articles/adding-or-editing-a-payment-method) ».

## Vérifier la politique de votre banque sur les restrictions de carte

Certaines banques internationales imposent des restrictions sur les transactions internationales, e-commerce et périodiques automatiques. Si vous rencontrez des difficultés à effectuer un paiement avec votre carte de crédit internationale, appelez votre banque pour voir s’il existe des restrictions sur votre carte.

Nous prenons également en charge les paiements par le biais de PayPal. Pour plus d’informations, consultez « [Ajout ou modification d’un mode de paiement](/articles/adding-or-editing-a-payment-method) ».

## Contacter votre banque pour obtenir plus d’informations sur la transaction

Votre banque peut fournir des informations supplémentaires sur les paiements refusés si vous l’interrogez spécifiquement sur la tentative de transaction. S’il existe des restrictions sur votre carte et que vous devez appeler votre banque, fournissez-lui les informations suivantes :

- **Montant dont vous êtes facturé.** Le montant de votre abonnement apparaît sur les reçus de votre compte. Pour plus d’informations, consultez « [Affichage de votre historique de paiement et des reçus](/articles/viewing-your-payment-history-and-receipts) ».
- **Date à laquelle {% data variables.product.product_name %} vous facture.** La date de facturation de votre compte apparaît sur vos reçus.
- **Numéro d’identification de la transaction.** L’ID de transaction de votre compte apparaît sur vos reçus.
- **Nom du marchand.** Le nom du marchand est {% data variables.product.prodname_dotcom %}.
- **Message d’erreur envoyé par votre banque quand elle a refusé le débit.** Vous trouverez le message d’erreur de votre banque sur l’e-mail que nous vous envoyons quand un débit est refusé.
