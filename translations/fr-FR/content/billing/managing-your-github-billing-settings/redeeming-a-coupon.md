{% data variables.product.product_name %} ne peut pas émettre de remboursement si vous payez pour un compte avant d’appliquer un coupon. En outre, nous ne pouvons pas transférer un coupon échangé ou vous donner un nouveau coupon si vous l’appliquez au compte incorrect. Vérifiez que vous appliquez le coupon au compte approprié avant d’échanger un coupon.

{% data reusables.dotcom_billing.coupon-expires %}

Vous ne pouvez pas appliquer de coupons aux plans payants pour des applications de la {% data variables.product.prodname_marketplace %}.

## Échange d’un coupon pour votre compte personnel

{% data reusables.dotcom_billing.enter_coupon_code_on_redeem_page %}
4. Sous « Échanger votre coupon », cliquez sur **Choisir** en regard du nom d’utilisateur de votre compte *personnel*.
   ![Bouton Choisir](/assets/images/help/settings/redeem-coupon-choose-button-for-personal-accounts.png) {% data reusables.dotcom_billing.redeem_coupon %}

## Échange d’un coupon pour votre organisation

{% data reusables.dotcom_billing.org-billing-perms %}

{% data reusables.dotcom_billing.enter_coupon_code_on_redeem_page %}
4. Sous « Échanger votre coupon », cliquez sur **Choisir** en regard de l’*organisation* à laquelle vous souhaitez appliquer le coupon. Si vous souhaitez appliquer votre coupon à une nouvelle organisation qui n’existe pas encore, cliquez sur **Créer une organisation**.
   ![Bouton Choisir](/assets/images/help/settings/redeem-coupon-choose-button.png) {% data reusables.dotcom_billing.redeem_coupon %}
