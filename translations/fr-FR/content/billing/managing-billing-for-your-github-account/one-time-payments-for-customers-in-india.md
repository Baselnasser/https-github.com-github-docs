## À propos de la réglementation sur les paiements récurrents de la Banque de réserve de l’Inde

Une nouvelle réglementation sur les paiements de la Banque de réserve de l’Inde (RBI) a récemment été appliquée. Cette réglementation impose des exigences supplémentaires sur les transactions en ligne récurrentes et a empêché certains clients {% data variables.product.company_short %} en Inde d’effectuer des paiements récurrents. Les clients utilisant des modes de paiement émis en Inde pour toutes les transactions récurrentes sur {% data variables.product.product_name %} peuvent constater que leurs paiements sont refusés par leurs banques ou émetteurs de cartes. Pour plus d’informations, lisez le [communiqué de presse de la RBI](https://www.rbi.org.in/Scripts/BS_PressReleaseDisplay.aspx?prid=51353).

La réglementation s’applique à toutes les transactions récurrentes, notamment :
- Abonnements dans le cadre d’un plan {% data variables.product.prodname_dotcom %} (Pro, Team, Enterprise)
- Achats sur la {% data variables.product.prodname_marketplace %}
- Transactions {% data variables.product.prodname_sponsors %}
- Achats Git Large File Storage
- Consommation de {% data variables.product.prodname_actions %}, {% data variables.product.prodname_registry %} et {% data variables.product.prodname_github_codespaces %}

Pour réduire les perturbations, les paiements récurrents pour nos clients affectés ont été suspendus le 29 octobre 2021. Les fonctionnalités et services payants sont restés disponibles pour les clients affectés par la réglementation de la RBI.

## À propos des paiements ponctuels sur {% data variables.product.company_short %}

Tout en collaborant avec notre fournisseur de passerelle de paiement pour répondre aux nouvelles exigences, nous fournissons une option de paiement ponctuel temporaire pour les clients affectés en Inde. Depuis le 15 février 2022, les clients {% data variables.product.company_short %} en Inde qui ont été affectés par la nouvelle réglementation de la RBI peuvent effectuer des paiements ponctuels selon la cadence habituelle de leur période de facturation.

### Pour les clients avec facturation mensuelle

Les clients abonnés à des plans de facturation mensuelle peuvent effectuer un paiement ponctuel le même jour où leur période de facturation est généralement renouvelée. Par exemple, si vous êtes généralement facturé le 7 de chaque mois, vous pouvez désormais effectuer un paiement ponctuel depuis votre compte à partir du 7 de chaque mois. Votre premier paiement ponctuel inclut également toute utilisation accumulée à partir d’octobre 2021.

Si vous êtes facturé mensuellement et que vous souhaitez passer à la facturation annuelle, vous pouvez réduire la fréquence de vos paiements ponctuels. Pour plus d’informations, consultez « [Modification de la durée de votre période de facturation](/en/billing/managing-your-github-billing-settings/changing-the-duration-of-your-billing-cycle) ».

### Pour les clients avec facturation annuelle

Si vous êtes facturé annuellement et que votre date de renouvellement était comprise entre le 1er octobre 2021 et le 14 février 2022, vous pouvez effectuer un paiement ponctuel pour vos abonnements annuels à partir du 15 février. Ce paiement initial inclut le coût dû au prorata de votre abonnement pour la période écoulée depuis la fin de votre période de facturation précédente.

Si votre période de facturation est censée se renouveler après le 15 février, nous tentons d’accepter le paiement récurrent. Si la tentative de paiement est refusée, vous pouvez effectuer un paiement ponctuel par le biais de la page de facturation de votre compte.

En attendant, nous travaillons activement avec nos partenaires de paiement afin de restaurer les paiements récurrents pour les clients impactés. Pour obtenir plus d’informations ou poser des questions, vous pouvez contacter le [support GitHub](https://support.github.com/contact).

### Impact sur {% data variables.product.prodname_sponsors %}

Les parrainages existants restent en place pendant cette période et les chargés de maintenance continuent d’être payés comme prévu. Les paiements pour les montants de parrainage cumulés du compte de financement sont collectés en même temps que les autres frais accumulés.
