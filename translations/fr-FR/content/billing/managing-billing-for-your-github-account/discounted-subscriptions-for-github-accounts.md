{% tip %}

**Conseil** : Les remises pour {% data variables.product.prodname_dotcom %} ne s’appliquent pas aux abonnements pour d’autres produits et fonctionnalités payants.

{% endtip %}

## Remises pour les comptes personnels

En plus des dépôts publics et privés illimités pour les étudiants et les enseignants disposant de {% data variables.product.prodname_free_user %}, les étudiants vérifiés peuvent demander le {% data variables.product.prodname_student_pack %} pour bénéficier d’avantages supplémentaires de la part des partenaires {% data variables.product.prodname_dotcom %}. Pour plus d’informations, consultez « [Demander à rejoindre {% data variables.product.prodname_global_campus %} en tant qu’étudiant](/free-pro-team@latest/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/apply-to-github-global-campus-as-a-student) ».

## Remises pour les écoles et les universités

Les enseignants universitaires vérifiés peuvent demander {% data variables.product.prodname_team %} pour l’enseignement ou la recherche universitaire. Pour plus d’informations, consultez « [{% data variables.product.prodname_global_campus %} pour les enseignants](/free-pro-team@latest/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-teachers) ». Vous pouvez également demander des matériaux pédagogiques pour vos étudiants. Pour plus d’informations, visitez [{% data variables.product.prodname_education %}](https://education.github.com/).

## Remises pour les organisations à but non lucratif et les bibliothèques

{% data variables.product.product_name %} fournit {% data variables.product.prodname_team %} gratuitement pour les organisations avec un nombre illimité de dépôts privés, un nombre illimité de collaborateurs et un ensemble de fonctionnalités complet pour les organisations et bibliothèques 501(c)3 (ou équivalentes). Vous pouvez demander une remise pour votre organisation sur [notre page dédiée aux organisations à but non lucratif](https://github.com/nonprofit).

Si votre organisation dispose déjà d’un abonnement payant, la dernière transaction de votre organisation est remboursée une fois que votre remise pour organisation à but non lucratif a été appliquée.
