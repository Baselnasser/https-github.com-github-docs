Pour plus d’informations sur les produits disponibles pour votre compte, consultez « [Produits de {% data variables.product.prodname_dotcom %}](/articles/github-s-products) ». Vous pouvez voir les tarifs et une liste complète des fonctionnalités de chaque produit à l’adresse <{% data variables.product.pricing_url %}>. {% data variables.product.product_name %} n’offre pas de produits ou d’abonnements personnalisés.  

Vous pouvez choisir une facturation mensuelle ou annuelle, et vous pouvez faire passer votre abonnement au niveau supérieur ou inférieur à tout moment. Pour plus d’informations, consultez « [Gestion de la facturation pour votre compte {% data variables.product.prodname_dotcom %}](/articles/managing-billing-for-your-github-account) ».

Vous pouvez acheter d’autres fonctionnalités et produits avec vos informations de paiement {% data variables.product.product_name %} existantes. Pour plus d’informations, consultez « [À propos de la facturation sur {% data variables.product.prodname_dotcom %}](/articles/about-billing-on-github) ».
