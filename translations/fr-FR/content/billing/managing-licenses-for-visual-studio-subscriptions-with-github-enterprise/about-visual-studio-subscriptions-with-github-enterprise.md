## À propos de {% data variables.visual_studio.prodname_vss_ghe %}

{% data reusables.enterprise-accounts.vss-ghe-description %} {% data variables.visual_studio.prodname_vss_ghe %} est disponible auprès de Microsoft en vertu des conditions du Contrat Entreprise Microsoft. Pour plus d’informations, consultez [{% data variables.visual_studio.prodname_vss_ghe %}](https://visualstudio.microsoft.com/subscriptions/visual-studio-github/) sur le site web {% data variables.product.prodname_vs %}.

Pour utiliser la partie {% data variables.product.prodname_enterprise %} de la licence, le compte personnel de chaque abonné sur {% data variables.product.prodname_dotcom_the_website %} doit être ou devenir membre d’une organisation détenue par votre entreprise sur {% data variables.product.prodname_dotcom_the_website %}. Pour ce faire, les propriétaires d’organisation peuvent inviter de nouveaux membres à une organisation par adresse e-mail. L’abonné peut accepter l’invitation avec un compte personnel existant sur {% data variables.product.prodname_dotcom_the_website %} ou créer un compte.

Pour plus d’informations sur la configuration de {% data variables.visual_studio.prodname_vss_ghe %}, consultez « [Configuration de {% data variables.visual_studio.prodname_vss_ghe %}](/billing/managing-licenses-for-visual-studio-subscriptions-with-github-enterprise/setting-up-visual-studio-subscriptions-with-github-enterprise) ».

## À propos des licences pour {% data variables.visual_studio.prodname_vss_ghe %}

Une fois que vous avez attribué une licence pour {% data variables.visual_studio.prodname_vss_ghe %} à un abonné, celui-ci utilise la partie {% data variables.product.prodname_enterprise %} de la licence en rejoignant une organisation de votre entreprise avec un compte personnel sur {% data variables.product.prodname_dotcom_the_website %}. Si l’adresse e-mail vérifiée du compte personnel d’un membre d’entreprise sur {% data variables.product.prodname_dotcom_the_website %} correspond au nom principal d’utilisateur (UPN) d’un abonné à votre compte {% data variables.product.prodname_vs %}, l’abonné {% data variables.product.prodname_vs %} consomme automatiquement une licence pour {% data variables.visual_studio.prodname_vss_ghe %}.

La quantité totale de vos licences pour votre entreprise sur {% data variables.product.prodname_dotcom %} correspond à la somme des licences {% data variables.product.prodname_enterprise %} standard et du nombre de licences d’abonnement {% data variables.product.prodname_vs %} qui incluent l’accès à {% data variables.product.prodname_dotcom %}. Si le compte personnel d’un membre d’entreprise ne correspond pas à l’adresse e-mail d’un abonné {% data variables.product.prodname_vs %}, la licence consommée par le compte personnel n’est pas disponible pour un abonné {% data variables.product.prodname_vs %}.

Pour plus d’informations sur {% data variables.product.prodname_enterprise %}, consultez « [Produits de {% data variables.product.company_short %}](/github/getting-started-with-github/githubs-products#github-enterprise) ». Pour plus d’informations sur les comptes sur {% data variables.product.prodname_dotcom_the_website %}, consultez « [Types de comptes {% data variables.product.prodname_dotcom %}](/github/getting-started-with-github/types-of-github-accounts) ».

Vous pouvez afficher le nombre de licences {% data variables.product.prodname_enterprise %} disponibles pour votre entreprise sur {% data variables.location.product_location %}. La liste des invitations en attente inclut les abonnés qui ne sont pas encore membres d’au moins une organisation de votre entreprise. Pour plus d’informations, consultez « [Affichage de l’abonnement et de l’utilisation de votre compte d’entreprise](/billing/managing-billing-for-your-github-account/viewing-the-subscription-and-usage-for-your-enterprise-account) » et « [Affichage des personnes de votre entreprise](/admin/user-management/managing-users-in-your-enterprise/viewing-people-in-your-enterprise#viewing-members-and-outside-collaborators) ».

{% tip %}

**Conseil** : Si vous téléchargez un fichier CSV indiquant l’utilisation de la licence de votre entreprise à l’étape 6 de « [Affichage de l’abonnement et de l’utilisation de votre compte d’entreprise](/billing/managing-billing-for-your-github-account/viewing-the-subscription-and-usage-for-your-enterprise-account#viewing-the-subscription-and-usage-for-your-enterprise-account) », tous les membres pour lesquels les colonnes « Nom » ou « Profil » sont dépourvues de valeur n’ont pas encore accepté d’invitation à rejoindre une organisation au sein de l’entreprise.

{% endtip %}

Vous pouvez également voir les invitations {% data variables.product.prodname_enterprise %} en attente adressées à des abonnés dans {% data variables.visual_studio.prodname_vss_admin_portal_with_url %}.
