Vous pouvez sélectionner manuellement la région dans laquelle vos espaces de code seront créés, ce qui vous permet de répondre à des exigences strictes en matière de sécurité et de conformité. Par défaut, votre région est définie automatiquement en fonction de votre emplacement.

## <a name="setting-your-default-region"></a>Définition de votre région par défaut

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.codespaces-tab %}
1. Sous « Région », sélectionnez le paramètre souhaité.
2. Si vous avez choisi « Définir manuellement », sélectionnez votre région dans la liste déroulante.
   ![Sélection de votre région](/assets/images/help/codespaces/select-default-region.png)
