Chaque commit montre les éléments suivants :

 - Message de commit
 - Heure de création du commit
 - Nom d’utilisateur et photo du profil de l’auteur du commit (le cas échéant)
 - Code de hachage SHA-1 du commit (ID unique)

{% data reusables.desktop.history-tab %}
2. Sous l’onglet **Historique**, cliquez sur le commit à passer en revue.
![Commit sous l’onglet Historique](/assets/images/help/desktop/branch-history-commit.png)
3. S’il existe plusieurs fichiers dans le commit, cliquez sur un seul d’entre eux pour voir les changements apportés à ce fichier dans le commit.
![Fichier dans le commit](/assets/images/help/desktop/branch-history-file.png)
