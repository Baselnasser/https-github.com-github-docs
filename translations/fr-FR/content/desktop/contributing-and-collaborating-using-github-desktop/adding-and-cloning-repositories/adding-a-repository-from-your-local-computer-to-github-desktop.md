{% tip %}

**Conseil :** Vous pouvez ajouter un dépôt Git de votre ordinateur local vers GitHub Desktop en faisant glisser le dossier vers la fenêtre {% data variables.product.prodname_desktop %}. Si vous faites glisser plusieurs dossiers Git vers {% data variables.product.prodname_desktop %} en même temps, chaque dossier est ajouté en tant que dépôt Git distinct.

{% endtip %}

{% mac %}

1. Dans le menu **Fichier**, cliquez sur **Ajouter un dépôt local**.
  ![Option de menu Ajouter un dépôt local](/assets/images/help/desktop/add-local-repository-mac.png)
2. Cliquez sur **Choisir**, puis, à l’aide de la fenêtre du Finder, accédez au dépôt local à ajouter.
  ![Champ Chemin local dans l’application Mac](/assets/images/help/desktop/add-repo-choose-button-mac.png)
4. Cliquez sur **Ajouter un dépôt**.
  ![Bouton Ajouter un dépôt dans l’application Mac](/assets/images/help/desktop/add-repository-button-mac.png)

{% endmac %}

{% windows %}

1. Dans le menu **Fichier**, cliquez sur **Ajouter un dépôt local**.
  ![Option de menu Ajouter un dépôt local](/assets/images/help/desktop/add-local-repository-windows.png)
2. Cliquez sur **Choisir**, puis, à l’aide de la fenêtre de l’Explorateur Windows, accédez au dépôt local à ajouter.
  ![Champ Chemin local dans l’application Windows](/assets/images/help/desktop/add-repo-choose-button-win.png)
4. Cliquez sur **Ajouter un dépôt**.
  ![Bouton Ajouter un dépôt dans l’application Windows](/assets/images/help/desktop/add-repository-button-windows.png)

{% endwindows %}
