{% mac %}

{% data reusables.git.remove-git-remote %}
2. [Ajoutez le dépôt à GitHub Desktop](/desktop/guides/contributing-to-projects/adding-a-repository-from-your-local-computer-to-github-desktop/).
{% data reusables.desktop.publish-repository %}
4. Tapez le nom souhaité du dépôt dans le champ **Nom**, ou utilisez le nom du dépôt local actuel par défaut.
  ![Champ Nom](/assets/images/help/desktop/publish-repository-name-mac.png)
5. Pour publier un dépôt public, décochez la case **Garder ce code privé**.
  ![Case à cocher Garder ce code privé](/assets/images/help/desktop/publish-repository-private-checkbox-mac.png)
6. Dans la liste déroulante **Organisation**, choisissez l’organisation où vous souhaitez publier le dépôt, ou sélectionnez **Aucun** pour publier le dépôt sur votre compte personnel.
  ![Liste déroulante Organisation](/assets/images/help/desktop/publish-repository-org-dropdown-mac.png)
7. Cliquez sur le bouton **Publier le dépôt**.
  ![Bouton Publier le dépôt dans la boîte de dialogue Publier le dépôt](/assets/images/help/desktop/publish-repository-dialog-button-mac.png)

{% endmac %}
