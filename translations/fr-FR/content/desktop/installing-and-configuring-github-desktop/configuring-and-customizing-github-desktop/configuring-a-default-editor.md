## Configuration d’un éditeur par défaut

{% mac %}

{% data reusables.desktop.mac-select-desktop-menu %}
3. Dans la fenêtre Préférences, sélectionnez **Intégrations**.
  ![Volet Intégrations de la fenêtre Préférences](/assets/images/help/desktop/mac-select-integrations-pane.png)
4. Utilisez le menu déroulant « Éditeur externe », puis choisissez l’éditeur à définir par défaut.
  ![Menu relatif à l’éditeur externe dans la barre de menus Préférences](/assets/images/help/desktop/mac-editor-menu.png)
5. Cliquez sur **Enregistrer**.  

{% endmac %}

{% windows %}

{% data reusables.desktop.windows-choose-options %}
3. Dans la fenêtre Options, sélectionnez **Intégrations**.
  ![Volet Intégrations de la fenêtre Options](/assets/images/help/desktop/windows-select-integrations-pane.png)
4. Utilisez le menu déroulant « Éditeur externe », puis choisissez l’éditeur à définir par défaut.
  ![Menu relatif à l’éditeur externe dans la barre de menus Options](/assets/images/help/desktop/windows-editor-menu.png)
5. Cliquez sur **Enregistrer**.  

{% endwindows %}
