{% mac %}

1. Dans une fenêtre du Finder, accédez au dossier Applications.
  ![Dossier Applications dans la fenêtre du Finder](/assets/images/help/desktop/applications-folder.png)
2. Tout en appuyant sur le bouton **Contrôle** de votre clavier, cliquez sur **{% data variables.product.prodname_desktop %}** .
3. Choisissez **Déplacer vers la corbeille**.
  ![Option Déplacer vers la corbeille](/assets/images/help/desktop/mac-move-to-trash.png)
4. Dans la barre de menus, utilisez le menu déroulant **Finder**, puis cliquez sur **Vider la corbeille**.
  ![Option Vider la corbeille dans la barre de menus](/assets/images/help/desktop/mac-empty-trash-menu.png)
5. Lisez l’avertissement dans la fenêtre indépendante, puis cliquez sur **Vider la corbeille**.
  ![Bouton Vider la corbeille](/assets/images/help/desktop/mac-empty-trash-button.png)

{% endmac %}

{% windows %}

1. Ouvrez le Panneau de configuration. Pour plus d’informations, consultez [Où se trouve le Panneau de configuration ?](https://support.microsoft.com/en-us/help/13764/windows-where-is-control-panel) dans l’Aide de Windows.
2. Sous « Programmes », cliquez sur **Désinstaller un programme**.
  ![Option Désinstaller un programme dans le Panneau de configuration](/assets/images/help/desktop/windows-uninstall-a-program.png)
3. Cliquez avec le bouton droit sur l’entrée nommée **{% data variables.product.prodname_desktop %}** , puis cliquez sur **Désinstaller**.
  ![Option Désinstaller](/assets/images/help/desktop/windows-click-uninstall.png)

{% endwindows %}
