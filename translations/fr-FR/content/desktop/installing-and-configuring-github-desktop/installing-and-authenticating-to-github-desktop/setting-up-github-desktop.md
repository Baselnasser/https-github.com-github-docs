## Partie 1 : Installation de {% data variables.product.prodname_desktop %}

Vous pouvez installer {% data variables.product.prodname_desktop %} sur n’importe quel système d’exploitation pris en charge. Pour plus d’informations, consultez « [Systèmes d’exploitation pris en charge](/desktop/getting-started-with-github-desktop/supported-operating-systems) ».

Pour installer {% data variables.product.prodname_desktop %}, accédez à [https://desktop.github.com/](https://desktop.github.com/), puis téléchargez la version appropriée de {% data variables.product.prodname_desktop %} pour votre système d’exploitation. Suivez les instructions pour terminer l’installation. Pour plus d’informations, consultez « [Installation de {% data variables.product.prodname_desktop %}](/desktop/getting-started-with-github-desktop/installing-github-desktop) ».

## Partie 2 : Configuration de votre compte

Si vous avez un compte sur {% data variables.product.prodname_dotcom %} ou {% data variables.product.prodname_enterprise %}, vous pouvez utiliser {% data variables.product.prodname_desktop %} pour échanger des données entre vos dépôts locaux et distants.

### Création d’un compte
Si vous n’avez pas encore de compte sur {% data variables.product.prodname_dotcom %}, consultez « [Inscription pour l’obtention d’un nouveau compte {% data variables.product.prodname_dotcom %} »](/articles/signing-up-for-a-new-github-account/).

Si vous faites partie d’une organisation qui utilise {% data variables.product.prodname_enterprise %}, et si vous n’avez pas de compte, contactez l’administrateur de site {% data variables.product.prodname_enterprise %}.

### Authentification auprès de {% data variables.product.prodname_dotcom %}
Pour vous connecter à {% data variables.product.prodname_desktop %} avec {% data variables.product.prodname_dotcom %}, vous devez authentifier votre compte. Pour plus d’informations, consultez « [Authentification auprès de {% data variables.product.prodname_desktop %}](/desktop/getting-started-with-github-desktop/authenticating-to-github) ».

Une fois votre compte authentifié, vous êtes prêt à gérer les projets et à y contribuer avec {% data variables.product.prodname_desktop %}.

## Partie 3 : Configuration de Git
Vous devez avoir installé Git avant d’utiliser {% data variables.product.prodname_desktop %}. Si vous n’avez pas encore installé Git, vous pouvez télécharger et installer la dernière version de Git à partir de [https://git-scm.com/downloads](https://git-scm.com/downloads).

Une fois Git installé, vous devez le configurer pour {% data variables.product.prodname_desktop %}. Pour plus d’informations, consultez « [Configuration de Git pour {% data variables.product.prodname_desktop %}](/desktop/getting-started-with-github-desktop/configuring-git-for-github-desktop) ».

## Partie 4 : Personnalisation de {% data variables.product.prodname_desktop %}
Vous pouvez changer les valeurs par défaut et les paramètres pour adapter {% data variables.product.prodname_desktop %} à vos besoins.

### Choix d’un éditeur de texte par défaut
Vous pouvez ouvrir un éditeur de texte à partir de {% data variables.product.prodname_desktop %} pour manipuler les fichiers et les dépôts. {% data variables.product.prodname_desktop %} prend en charge un grand nombre d’éditeurs de texte et d’environnements de développement intégrés (IDE) pour Windows et macOS. Vous pouvez choisir un éditeur par défaut dans les paramètres de {% data variables.product.prodname_desktop %}. Pour plus d’informations, consultez « [Configuration d’un éditeur par défaut](/desktop/getting-started-with-github-desktop/configuring-a-default-editor) ».

### Choix d’un thème
{% data variables.product.prodname_desktop %} dispose de plusieurs thèmes pour la personnalisation de l’apparence de l’application. Vous pouvez choisir un thème dans les paramètres de {% data variables.product.prodname_desktop %}. Pour plus d’informations, consultez « [Définition d’un thème pour {% data variables.product.prodname_desktop %}](/desktop/getting-started-with-github-desktop/setting-a-theme-for-github-desktop) ».
