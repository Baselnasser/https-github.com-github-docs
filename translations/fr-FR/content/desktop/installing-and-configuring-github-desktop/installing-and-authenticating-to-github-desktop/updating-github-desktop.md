{% mac %}

1. Dans le menu de **GitHub Desktop**, cliquez sur **À propos de GitHub Desktop**.
![Option de menu À propos de GitHub Desktop](/assets/images/help/desktop/desktop-menu-about-desktop-mac.png)
2. Cliquez sur **Rechercher les mises à jour**.
![Bouton Rechercher les mises à jour](/assets/images/help/desktop/check-for-updates.png)
3. Si une mise à jour est disponible, quittez et redémarrez {% data variables.product.prodname_desktop %} pour installer la mise à jour.

{% endmac %}

{% windows %}

1. Dans le menu **Aide**, cliquez sur **À propos de GitHub Desktop**.
![Option de menu À propos de GitHub Desktop](/assets/images/help/desktop/help-about-desktop-win.png)
2. Cliquez sur **Rechercher les mises à jour**.
![Bouton Rechercher les mises à jour](/assets/images/help/desktop/check-for-updates.png)
3. Si une mise à jour est disponible, quittez et redémarrez {% data variables.product.prodname_desktop %} pour installer la mise à jour.

{% endwindows %}
