{% mac %}

Raccourcis clavier GitHub Desktop sur macOS

## Raccourcis à l’échelle du site

| Raccourci clavier | Description
|-----------|------------
|<kbd>Commande</kbd>+<kbd>,</kbd> | Accéder aux préférences
|<kbd>Commande</kbd>+<kbd>H</kbd> | Masquer l’application {% data variables.product.prodname_desktop %}
|<kbd>Option</kbd>+<kbd>Commande</kbd>+<kbd>H</kbd> | Masquer toutes les autres applications
|<kbd>Commande</kbd>+<kbd>Q</kbd> | Quitter {% data variables.product.prodname_desktop %}
|<kbd>Contrôle</kbd>+<kbd>Commande</kbd>+<kbd>F</kbd> | Activer/Désactiver l’affichage plein écran
|<kbd>Commande</kbd>+<kbd>0</kbd> | Réinitialiser le zoom à la taille de texte par défaut
|<kbd>Commande</kbd>+<kbd>=</kbd> | Effectuer un zoom avant pour agrandir l’affichage du texte et des graphismes
|<kbd>Commande</kbd>+<kbd>-</kbd> | Effectuer un zoom arrière pour réduire l’affichage du texte et des graphismes
|<kbd>Option</kbd>+<kbd>Commande</kbd>+<kbd>I</kbd> | Activer/désactiver des outils de développement

## Référentiels

| Raccourci clavier | Description
|-----------|------------
|<kbd>Commande</kbd>+<kbd>N</kbd> | Ajouter un nouveau dépôt
|<kbd>Commande</kbd>+<kbd>O</kbd> | Ajouter un dépôt local
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>O</kbd> | Cloner un dépôt à partir de {% data variables.product.prodname_dotcom %}
|<kbd>Commande</kbd>+<kbd>T</kbd> | Afficher une liste de vos dépôts
|<kbd>Commande</kbd>+<kbd>P</kbd> | Pousser les derniers commits vers {% data variables.product.prodname_dotcom %}
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>P</kbd> | Tirer les derniers changements à partir de {% data variables.product.prodname_dotcom %}
|<kbd>Commande</kbd>+<kbd>Supprimer</kbd> | Supprimer un dépôt existant
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>G</kbd> | Voir le dépôt sur {% data variables.product.prodname_dotcom %}
|<kbd>Contrôle</kbd>+<kbd>&grave;</kbd> | Ouvrir le dépôt dans votre outil de terminal préféré
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>F</kbd> | Afficher le dépôt dans le Finder
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>A</kbd> | Ouvrir le dépôt dans votre outil d’édition préféré
|<kbd>Commande</kbd>+<kbd>I</kbd> | Créer un problème sur {% data variables.product.prodname_dotcom %}

## Branches

| Raccourci clavier | Description
|-----------|------------
|<kbd>Commande</kbd>+<kbd>1</kbd> | Afficher tous vos changements avant le commit
|<kbd>Commande</kbd>+<kbd>2</kbd> | Afficher l’historique de vos commits
|<kbd>Commande</kbd>+<kbd>B</kbd> | Afficher toutes vos branches
|<kbd>Commande</kbd>+<kbd>G</kbd> | Accéder au champ récapitulatif du commit
|<kbd>Commande</kbd>+<kbd>Entrée</kbd> | Commiter les changements quand le champ récapitulatif ou le champ de description est actif
|<kbd>Espace</kbd>| Sélectionner ou désélectionner tous les fichiers en surbrillance
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>N</kbd> | Créer une branche
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>R</kbd> | Renommer la branche actuelle
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>D</kbd> | Supprimer la branche actuelle
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>U</kbd> | Mettre à jour à partir de la branche par défaut
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>B</kbd> | Comparer à une branche existante
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>M</kbd> | Fusionner dans la branche actuelle
|<kbd>Contrôle</kbd>+<kbd>H</kbd> | Afficher ou masquer les changements ayant fait l’objet d’un stash
|<kbd>Maj</kbd>+<kbd>Commande</kbd>+<kbd>C</kbd> | Comparer les branches sur {% data variables.product.prodname_dotcom %}
|<kbd>Commande</kbd>+<kbd>R</kbd> | Afficher la demande de tirage (pull request) actuelle sur {% data variables.product.prodname_dotcom %}

{% endmac %}
