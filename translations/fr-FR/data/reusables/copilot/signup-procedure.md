{% data reusables.user-settings.access_settings %}
1. Dans la section « Code, planification et automatisation » de la barre latérale, cliquez sur **{% octicon "copilot" aria-label="The copilot icon" %} {% data variables.product.prodname_copilot %}** .
1. Sur la page des paramètres de {% data variables.product.prodname_copilot %}, cliquez sur **Activer {% data variables.product.prodname_copilot %}** .

   ![Capture d’écran des paramètres de GitHub Copilot avec le bouton Activer {% data variables.product.prodname_copilot %} mis en évidence](/assets/images/help/copilot/copilot-settings-enable-button.png)
1. Choisissez si vous voulez payer par mois ou par an, et cliquez sur **Continuer pour obtenir l’accès à Copilot**.
   - Si votre compte personnel répond aux critères d’un abonnement gratuit {% data variables.product.prodname_copilot %} plutôt qu’un essai, vous passerez automatiquement à l’étape 6.

   ![Capture d’écran de la configuration de l’essai avec le bouton Continuer pour obtenir l’accès à Copilot mis en évidence](/assets/images/help/copilot/copilot-billing-cycle.png)
1. Suivez les étapes pour confirmer les détails de votre paiement, puis cliquez sur **Envoyer**.
1. Sélectionnez vos préférences, puis cliquez sur **Enregistrer et commencer**.

   ![Capture d’écran des paramètres de GitHub Copilot avec le bouton Enregistrer et prise en main](/assets/images/help/copilot/copilot-signup-preferences.png)

   Vous pouvez modifier ces préférences ultérieurement en retournant à vos paramètres {% data variables.product.prodname_copilot %}. Pour plus d’informations, consultez « [Configuration de GitHub Copilot dans Visual Studio Code](/copilot/configuring-github-copilot/configuring-github-copilot-in-visual-studio-code#configuring-github-copilot-settings-on-githubcom) ».
