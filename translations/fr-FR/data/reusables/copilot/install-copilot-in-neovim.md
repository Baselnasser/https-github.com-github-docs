1. Pour utiliser {% data variables.product.prodname_copilot %} dans Neovim, installez le plug-in {% data variables.product.prodname_copilot %}. Vous pouvez installer le plug-in à partir d’un gestionnaire de plug-ins ou l’installer directement.
   - Si vous utilisez un gestionnaire de plug-ins comme vim-plug ou packer.nvim, utilisez le gestionnaire de plug-ins pour installer `github/copilot.vim`. Pour plus d’informations, consultez la documentation du gestionnaire de plug-ins. Par exemple, vous pouvez consulter la documentation de [vim-plug](https://github.com/junegunn/vim-plug) ou de [packer.nvim](https://github.com/wbthomason/packer.nvim).
