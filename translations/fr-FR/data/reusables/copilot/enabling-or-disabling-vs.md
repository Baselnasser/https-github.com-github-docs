## Activation ou désactivation de {% data variables.product.prodname_copilot %}

L’icône d’état de {% data variables.product.prodname_copilot %} dans le panneau inférieur de la fenêtre {% data variables.product.prodname_vs %} indique si {% data variables.product.prodname_copilot %} est activé ou désactivé. Quand il est activé, la couleur d’arrière-plan de l’icône correspond à la couleur de la barre d’état. Quand il est désactivé, elle comporte une ligne en diagonal.

1. Pour activer ou désactiver {% data variables.product.prodname_copilot %}, cliquez sur l’icône {% data variables.product.prodname_copilot %} dans le panneau inférieur de la fenêtre {% data variables.product.prodname_vs %}.
  ![Capture d’écran de la marge de l’éditeur dans Visual Studio avec l’icône GitHub Copilot mise en évidence](/assets/images/help/copilot/editor-margin-visual-studio.png)
2. Si vous désactivez {% data variables.product.prodname_copilot %}, il vous sera demandé si vous souhaitez désactiver les suggestions de façon globale ou pour le langage du fichier que vous êtes en train de modifier.

   - Pour désactiver les suggestions de {% data variables.product.prodname_copilot %} globalement, cliquez sur **Activer globalement**.
   - Pour désactiver les suggestions de {% data variables.product.prodname_copilot %} pour le langage spécifié, cliquez sur **Désactiver pour _LANGAGE_**.
