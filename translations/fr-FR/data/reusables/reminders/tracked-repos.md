1. Sous « Dépôts suivis », choisissez les dépôts pour lesquels vous voulez que l’équipe reçoive des rappels de révision de demande de tirage.  
   - Pour recevoir des rappels pour tous les dépôts auxquels l’équipe choisie a accès, cliquez sur **Tous les dépôts**.  
   - Pour recevoir des rappels pour un sous-ensemble de dépôts, cliquez sur **Seulement certains dépôts**, puis sélectionnez un ou plusieurs dépôts auxquels l’équipe choisie a accès.
