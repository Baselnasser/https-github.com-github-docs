1. Si vous voulez désactiver les rappels planifiés pour les demandes de tirage déjà révisées et approuvées, sélectionnez **Ignorer les demandes de tirage approuvées**. Cliquez ensuite sur **Ignorer avec 1 ou plusieurs approbations** pour choisir le nombre d’approbations que doit avoir une demande de tirage pour être ignorée.
