1. Si vous voulez ajouter uniquement les demandes de tirage pour lesquelles une révision est spécifiquement demandée par l’équipe ou un membre d’équipe, sélectionnez **Exiger des demandes de révision**. Si vous ne sélectionnez pas cette option, toutes les demandes de tirage sont ajoutées au rappel planifié.
