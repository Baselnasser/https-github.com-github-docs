Si vous utilisez un coupon pour payer un abonnement, à l’expiration du coupon, le coût total de votre abonnement est facturé sur votre mode de paiement. Si vous n’avez pas de mode de paiement enregistré, votre compte passe au plan {% data variables.product.prodname_free_user %} pour les comptes personnels ou {% data variables.product.prodname_free_team %} pour les organisations.
