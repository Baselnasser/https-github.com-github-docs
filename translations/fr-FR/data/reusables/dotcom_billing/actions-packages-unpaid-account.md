Si votre compte a des frais impayés exceptionnels :

* Le stockage ou les minutes inclus dans votre compte pour {% data variables.product.prodname_actions %} et {% data variables.product.prodname_registry %} ne seront pas réinitialisés tant que le paiement n’a pas été correctement traité.
* Pour les comptes avec un stockage ou des minutes restants pour la période de facturation actuelle, {% data variables.product.prodname_actions %} et {% data variables.product.prodname_registry %} continueront d’être disponibles jusqu’à ce que toute utilisation incluse ait été atteinte.
* Pour les comptes qui ont atteint l’utilisation incluse pour la période de facturation actuelle pour {% data variables.product.prodname_actions %} ou {% data variables.product.prodname_registry %}, {% data variables.product.prodname_actions %} et {% data variables.product.prodname_registry %} seront désactivés pour empêcher tout dépassement supplémentaire.
