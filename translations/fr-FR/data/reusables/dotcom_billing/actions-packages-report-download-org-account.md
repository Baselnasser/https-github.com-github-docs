1. En regard de « Utilisation ce mois », vous pouvez aussi cliquer sur **Obtenir le rapport d’utilisation** afin de recevoir un e-mail contenant un lien pour télécharger un rapport CSV de l’utilisation du stockage pour {% data variables.product.prodname_actions %}, {% data variables.product.prodname_registry %} et {% data variables.product.prodname_codespaces %}. L’e-mail est envoyé à l’adresse e-mail principale de votre compte. Vous pouvez choisir la période couverte par le rapport : les 7, 30, 90 ou 180 derniers jours.
   ![Télécharger un rapport CSV](/assets/images/help/billing/actions-packages-report-download.png)
