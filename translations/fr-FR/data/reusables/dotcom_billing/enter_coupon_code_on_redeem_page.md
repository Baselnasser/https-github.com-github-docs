1. Visitez la page [Utiliser le bon](https://github.com/redeem).
2. Dans la zone Entrer le code du bon, tapez le code de votre bon, puis cliquez sur **Utiliser**.
   ![Zone Utiliser le bon](/assets/images/help/settings/redeem-coupon-box.png)
3. Si vous n’êtes pas déjà connecté, vous pouvez vous connecter à cette page ou créer un compte auquel appliquer votre bon.
