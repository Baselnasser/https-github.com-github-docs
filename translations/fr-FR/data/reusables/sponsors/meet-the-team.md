1. Si vous le souhaitez, choisissez les membres de l’organisation qui sont mis en évidence dans votre profil {% data variables.product.prodname_sponsors %}.
    - À droite de « Rencontrer l’équipe », cliquez sur **Modifier**.  
        ![Bouton Modifier les membres de l’organisation dans le profil](/assets/images/help/sponsors/edit-org-members-profile-button.png)
    - Sélectionnez jusqu’à six membres de l’organisation, puis cliquez sur **Enregistrer**.
        ![Sélectionner les membres de l’organisation mis en évidence](/assets/images/help/sponsors/select-highlighted-org-members.png)
    - Si vous le souhaitez, sous le nom de chaque membre mis en évidence, tapez une description du membre.
        ![Description des membres de l’organisation mis en évidence](/assets/images/help/sponsors/description-highlighted-org-members.png)
