1. Sous « Sur quoi est basé votre objectif ? », sélectionnez **Nombre de sponsors** ou **Parrainages par mois**.
![Types d’objectifs disponibles ](/assets/images/help/sponsors/select-goal-type.png)
2. Tapez le nombre de sponsors que vous aimeriez avoir, ou le montant mensuel que vous souhaitez obtenir.
3. Sous « Décrire votre objectif », tapez une description pour votre objectif.
![Paramètres de l’objectif](/assets/images/help/sponsors/goal-settings.png)
5. Si vous avez choisi un objectif basé sur les parrainages par mois, sélectionnez « Je comprends que tout le monde pourra voir le montant des fonds de parrainage que je recevrai chaque mois ».
![Case à cocher indiquant que vous comprenez que vos fonds de parrainage seront publiés](/assets/images/help/sponsors/sponsorships-published-checkbox.png)
