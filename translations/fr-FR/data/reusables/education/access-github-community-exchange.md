Pour accéder à {% data variables.product.prodname_community_exchange %}, visitez votre tableau de bord {% data variables.product.prodname_global_campus %} sur https://education.github.com.

Si vous êtes étudiant ou membre du corps enseignant d’un établissement d’enseignement agréé, vous pouvez faire une demande pour bénéficier des avantages {% data variables.product.prodname_education %}, qui comprennent l’accès à {% data variables.product.prodname_community_exchange %} dans {% data variables.product.prodname_global_campus %}.

- Si vous êtes étudiant et que vous n’avez pas encore rejoint {% data variables.product.prodname_education %}, faites-en la demande en utilisant le [formulaire de demande pour étudiants]( https://education.github.com/discount_requests/student_application). Pour plus d’informations, consultez « [À propos de {% data variables.product.prodname_global_campus %} pour les étudiants](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/about-github-global-campus-for-students) ».

- Si vous êtes enseignant et que vous n’avez pas encore rejoint {% data variables.product.prodname_education %}, faites-en la demande en utilisant le [formulaire de demande pour enseignants]( https://education.github.com/discount_requests/teacher_application). Pour plus d’informations, consultez « [Demander à rejoindre {% data variables.product.prodname_global_campus %} en tant qu’enseignant](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-teachers/apply-to-github-global-campus-as-a-teacher) ».
