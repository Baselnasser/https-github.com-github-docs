8. Vérifiez les détails de votre demande, puis cliquez sur **Traiter ma demande**.
 ![Bouton Traiter ma demande](/assets/images/help/education/submit-your-information-button.png)

    {% note %}

    **Remarque :** Si, après avoir cliqué sur le bouton **Traiter ma demande**, vous voyez une bannière vous demandant de corriger quelque chose dans votre demande, vous devez apporter les corrections nécessaires, puis cliquer sur **Retraiter ma demande**.

    {% endnote %}

 Si votre application est approuvée, vous recevrez un e-mail de confirmation. Les applications sont généralement traitées en quelques jours, mais cela peut prendre plus de temps pendant les heures de pointe, par exemple, au début d’un nouveau semestre.
