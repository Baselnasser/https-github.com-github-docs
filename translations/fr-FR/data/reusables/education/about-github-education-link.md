En tant qu’étudiant ou membre du corps enseignant d’un établissement d’enseignement accrédité, vous pouvez demander à rejoindre {% data variables.product.prodname_global_campus %}. {% data variables.product.prodname_global_campus %} est un portail qui permet aux membres de la communauté GitHub Éducation d’accéder à leurs avantages « enseignement » à un seul endroit ! Le portail {% data variables.product.prodname_global_campus %} inclut l’accès à {% data variables.product.prodname_education_community_with_url %}, à des outils du secteur utilisés par les développeurs professionnels, à des événements, au contenu de [Campus TV](https://www.twitch.tv/githubeducation), à {% data variables.product.prodname_classroom_with_url %}, à [{% data variables.product.prodname_community_exchange %}](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/about-github-community-exchange), à {% data variables.product.prodname_student_pack %} et à d’autres fonctionnalités exclusives pour aider les étudiants et les enseignants à façonner la prochaine génération du développement logiciel.

Avant de demander une remise individuelle, vérifiez si votre communauté d’apprentissage figure déjà parmi nos partenaires en tant qu’école {% data variables.product.prodname_campus_program %}. Pour plus d’informations, consultez « [À propos de {% data variables.product.prodname_campus_program %}](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/about-github-campus-program) ».
