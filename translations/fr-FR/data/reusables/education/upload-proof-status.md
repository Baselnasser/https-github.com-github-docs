1. Si vous y êtes invité, chargez la preuve de votre statut académique.
   - Cliquez sur **Utiliser votre appareil photo pour prendre une photo**.
     ![Bouton pour utiliser votre appareil photo pour prendre une photo](/assets/images/help/education/upload-proof-status.png)
   - Pour changer d’appareil photo, vous pouvez utiliser le menu déroulant d’appareil photo.
    ![Menu déroulant d’appareil photo](/assets/images/help/education/camera-drop-down.png)
   - Placez votre identifiant académique valide ou une autre preuve de votre statut académique actuel dans le cadre, puis cliquez sur **Prendre une photo**.
    ![Bouton Prendre une photo](/assets/images/help/education/take-photo-button.png)
   - Si vous ne pouvez pas prendre de photo de la preuve de votre statut académique, pour charger une image à partir de votre ordinateur à la place, cliquez sur **Charger une image**.
     ![Lien Charger une image](/assets/images/help/education/upload-image-link.png) {% note %}

    **Remarque :** Les fichiers PDF ne sont pas pris en charge et vous ne pouvez pas les charger comme preuve de statut académique.

    {% endnote %}
