1. Tapez un nom pour votre référentiel et une description facultative. Si vous créez un utilisateur ou un site d’organisation, votre référentiel doit être nommé `<user>.github.io` ou `<organization>.github.io`. Si votre nom d’utilisateur ou d’organisation contient des lettres majuscules, vous devez mettre les lettres en minuscules.
Pour plus d’informations, consultez « [À propos de {% data variables.product.prodname_pages %}](/articles/about-github-pages#types-of-github-pages-sites) ».
