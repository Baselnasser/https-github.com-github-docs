Nous vous recommandons d’utiliser [Bundler](http://bundler.io/) pour installer et exécuter Jekyll. Bundler gère les dépendances de gemme Ruby, réduit les erreurs de build Jekyll et empêche les bogues liés à l’environnement. Pour installer Bundler :

 1. Installer Ruby Pour plus d’informations, consultez « [Installation de Ruby](https://www.ruby-lang.org/en/documentation/installation/) » dans la documentation de Ruby.
 2. Installez Bundler. Pour plus d’informations, consultez « [Bundler](https://bundler.io/) ».
