{% warning %}

**Avertissement :** nous vous recommandons vivement de ne pas utiliser d’enregistrements DNS génériques, tels que `*.example.com`. Un enregistrement DNS générique permettra à toute personne d’héberger un site {% data variables.product.prodname_pages %} sur l’un de vos sous-domaines, même s’ils sont vérifiés. Pour plus d’informations, consultez « [Vérification de votre domaine personnalisé pour {% data variables.product.prodname_pages %}](/pages/configuring-a-custom-domain-for-your-github-pages-site/verifying-your-custom-domain-for-github-pages) ».

{% endwarning %}
