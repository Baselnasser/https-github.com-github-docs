Vous pouvez activer ou désactiver {% data variables.product.prodname_discussions %} pour un dépôt.

{% data reusables.repositories.navigate-to-repo %}
1. Sous le nom de votre dépôt, cliquez sur {% octicon "gear" aria-label="The gear icon" %} **Paramètres**.
![Bouton Paramètres du référentiel](/assets/images/help/discussions/public-repo-settings.png)
1. Sous « Fonctionnalités », sélectionnez **Discussions**.
    ![Case à cocher sous « Fonctionnalités » pour l’activation ou la désactivation de {% data variables.product.prodname_discussions %} pour un dépôt](/assets/images/help/discussions/select-discussions-checkbox.png) ».
