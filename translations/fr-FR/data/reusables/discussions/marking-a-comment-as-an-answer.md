Vous pouvez marquer un commentaire dans la discussion comme réponse à la discussion si une discussion fait partie d’une catégorie qui accepte les réponses. Pour plus d’informations, consultez « [À propos des discussions](/discussions/collaborating-with-your-community-using-discussions/about-discussions#about-categories-and-formats-for-discussions) ».

Quand vous marquez une question comme réponse, {% data variables.product.product_name %} met en surbrillance le commentaire et les réponses au commentaire pour aider les visiteurs à trouver rapidement la réponse.

![Commentaire marqué comme réponse à une discussion](/assets/images/help/discussions/comment-marked-as-answer.png)

Vous ne pouvez pas marquer un commentaire en thread en réponse à un commentaire comme réponse à une discussion. Il n’est également pas possible de marquer un commentaire réduit comme réponse à une discussion.

{% data reusables.discussions.navigate-to-repo-or-org %} {% data reusables.discussions.discussions-tab %}
1. Dans la liste des discussions, cliquez sur la discussion sans réponse que vous souhaitez marquer comme réponse.
  ![Discussion sans réponse](/assets/images/help/discussions/unanswered-discussion.png)
1. Dans la discussion, recherchez le commentaire que vous souhaitez marquer comme réponse.
1. Dans le coin supérieur droit du commentaire, cliquez sur {% octicon "check-circle" aria-label="The circular check icon" %}.
  ![Icône de coche circulaire « Marquer comme réponse » pour définir le commentaire comme réponse dans une discussion](/assets/images/help/discussions/comment-mark-as-answer-button.png)
1. Si vous le souhaitez, pour désélectionner un commentaire en tant que réponse {% octicon "check-circle-fill" aria-label="The filled circular check icon" %}.
