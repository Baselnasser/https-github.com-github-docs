Les chargés de maintenance peuvent utiliser des catégories avec le format d’annonce pour partager des informations, des versions ou des événements. Pour conserver les discussions dans ces catégories axées sur les mises à jour importantes, seules les personnes disposant d’autorisations d’administration ou de maintenance peuvent créer de nouvelles discussions, mais tout le monde peut commenter et répondre.
