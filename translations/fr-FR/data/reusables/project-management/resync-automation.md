Lorsque vous fermez un {% data variables.projects.projects_v1_board %}, toute automatisation de workflow configurée pour le {% data variables.projects.projects_v1_board %} est interrompue. Si vous rouvrez un {% data variables.projects.projects_v1_board %}, vous avez la possibilité de synchroniser l’automatisation, ce qui met à jour la position des cartes dans le panneau en fonction des paramètres d’automatisation configurés pour le projet. Pour plus d’informations, consultez « [Réouverture d’un {% data variables.product.prodname_project_v1 %} fermé](/articles/reopening-a-closed-project-board) » ou « [Fermeture d’un {% data variables.product.prodname_project_v1 %}](/articles/closing-a-project-board) ».
