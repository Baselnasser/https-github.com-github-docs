Dans votre dépôt, créez un fichier nommé `.github/workflows/YOUR_WORKFLOW.yml`, en `YOUR_WORKFLOW` remplaçant par un nom de votre choix. Il s’agit d’un fichier de workflow. Pour plus d’informations sur la création de fichiers sur GitHub, consultez « [Création de fichiers](/github/managing-files-in-a-repository/creating-new-files) ».
