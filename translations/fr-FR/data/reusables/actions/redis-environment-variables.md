Le script *client.js* recherche les variables d’environnement `REDIS_HOST` et `REDIS_PORT` pour créer le client. Le workflow définit ces deux variables d’environnement dans le cadre de l’étape « Se connecter à Redis » pour les rendre disponibles pour le script *client.js*. Pour plus d’informations sur le script, consultez « [Test du conteneur de service Redis](#testing-the-redis-service-container) ».
