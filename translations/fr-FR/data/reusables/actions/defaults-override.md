Quand plusieurs paramètres par défaut sont définis avec le même nom, {% data variables.product.prodname_dotcom %} utilise le paramètre par défaut le plus spécifique. Par exemple, un paramètre par défaut défini dans un travail remplace un paramètre par défaut portant le même nom défini dans un workflow.
