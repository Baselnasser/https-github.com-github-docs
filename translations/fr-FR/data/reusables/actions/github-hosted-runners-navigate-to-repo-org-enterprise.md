{% ifversion fpt %}
1. Accédez à la page principale de l’organisation ou du dépôt.
1. Cliquez sur {% octicon "gear" aria-label="The Settings gear" %} **Paramètres**.
1. Dans la barre latérale gauche, cliquez sur **Actions**, puis sur **Exécuteurs**.
{% elsif ghec %}
1. Accédez aux paramètres des exécuteurs :
   * **Dans une organisation ou un dépôt** : accédez à la page principale, puis cliquez sur {% octicon "gear" aria-label="The Settings gear" %} **Paramètres**.
   * **Si vous utilisez un compte d’entreprise** : accédez à votre compte d’entreprise en cliquant sur votre photo de profil dans le coin supérieur droit de {% data variables.product.prodname_dotcom_the_website %}, en cliquant sur **Vos entreprises**, puis en cliquant sur l’entreprise approprié.
1. Accédez aux paramètres de {% data variables.product.prodname_actions %} :
   * **Dans une organisation ou un dépôt** : cliquez sur **Actions** dans la barre latérale gauche, puis cliquez sur **Exécuteurs**.
   * **Si vous utilisez un compte d’entreprise** : dans la barre latérale gauche, cliquez sur **« {% octicon "law" aria-label="The law icon" %} Stratégies »** , sur **Actions**, puis sur l’onglet **Exécuteurs**. {% endif %}
