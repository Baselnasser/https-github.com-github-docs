### Utilisation d’exécuteurs auto-hébergés sur {% data variables.product.prodname_ghe_server %}

Quand vous utilisez des actions de configuration (comme `actions/setup-LANGUAGE`) sur {% data variables.product.prodname_ghe_server %} avec des exécuteurs auto-hébergés, vous pouvez être amené à configurer le cache des outils sur les exécuteurs qui n’ont pas accès à Internet. Pour plus d’informations, consultez « [Configuration du cache d’outils sur les exécuteurs auto-hébergés sans accès à Internet](/enterprise/admin/github-actions/setting-up-the-tool-cache-on-self-hosted-runners-without-internet-access) ».
