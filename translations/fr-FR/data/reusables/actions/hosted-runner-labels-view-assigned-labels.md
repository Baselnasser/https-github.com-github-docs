1. Recherchez l’exécuteur que vous souhaitez mettre à jour, puis cliquez sur {% octicon "triangle-down" aria-label="The downward triangle" %} pour afficher le menu de sélection des étiquettes. Ce menu affiche toutes les étiquettes personnalisées disponibles pour votre exécuteur. Une {% octicon "check" aria-label="Check mark" %} apparaît en regard des étiquettes déjà attribuées à votre exécuteur.
