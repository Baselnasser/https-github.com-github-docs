Remarque : L’événement `schedule` peut être retardé pendant les périodes où les charges des exécutions de workflows {% data variables.product.prodname_actions %} sont élevées. Les périodes de charges élevées incluent le début de chaque heure. Pour réduire le risque de retard, planifiez l’exécution de votre workflow à un autre moment dans l’heure.
