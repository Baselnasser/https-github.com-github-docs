Si vous avez besoin d’aide concernant la configuration d’un workflow, notamment la syntaxe, les exécuteurs hébergés {% data variables.product.prodname_dotcom %} ou les actions de génération, recherchez les sujets existants ou démarrez-en un nouveau dans la [catégorie {% data variables.product.prodname_github_community %} {% data variables.product.prodname_actions %} et {% data variables.product.prodname_registry %}](https://github.com/orgs/github-community/discussions/categories/actions-and-packages).

Si vous voulez émettre des commentaires ou demander des fonctionnalités {% data variables.product.prodname_actions %}, partagez-les dans les {% data variables.contact.contact_feedback_actions %}.

Contactez le {% data variables.contact.contact_support %} pour tous les cas suivants, que votre utilisation ou votre utilisation prévue s’inscrivent ou non dans les catégories des limites d’utilisation :

* Si vous pensez que votre compte a été limité de manière incorrecte
* Si vous rencontrez une erreur inattendue lors de l’exécution de l’une de vos actions
* Si vous êtes dans une situation où le comportement existant n’est pas conforme au comportement attendu, qui n’est pas toujours décrit
