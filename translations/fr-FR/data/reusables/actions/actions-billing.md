L’utilisation de {% data variables.product.prodname_actions %} est gratuite pour les exécuteurs standard hébergés par {% data variables.product.prodname_dotcom %} dans des dépôts publics et pour les exécuteurs auto-hébergés. Pour les dépôts privés, chaque compte {% data variables.product.prodname_dotcom %} reçoit une certaine quantité de minutes et un certain volume de stockage gratuits, utilisables avec des exécuteurs hébergés par {% data variables.product.prodname_dotcom %}, en fonction du produit utilisé avec le compte. Toute utilisation au-delà des montants inclus est contrôlée par des limites de dépense.
