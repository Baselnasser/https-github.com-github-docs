Si vous avez configuré un environnement de déploiement, remplacez la valeur `environment` par le nom de votre environnement. Si vous n’avez pas configuré d’environnement {% ifversion fpt or ghec %} ou si votre workflow se trouve dans un référentiel privé et que vous n’utilisez pas {% data variables.product.prodname_ghe_cloud %}{% endif %}, supprimez la clé `environment`.
