Utilisez `jobs.<job_id>.container.options` pour configurer des options supplémentaires de ressources de conteneur Docker. Pour obtenir la liste des options, consultez « [Options `docker create`](https://docs.docker.com/engine/reference/commandline/create/#options) ».

{% warning %}

**Avertissement :** L’option `--network` n’est pas prise en charge.

{% endwarning %}
