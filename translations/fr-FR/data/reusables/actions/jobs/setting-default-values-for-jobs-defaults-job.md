Utilisez `jobs.<job_id>.defaults` pour créer un `map` des paramètres par défaut applicable à toutes les étapes du travail. Vous pouvez également définir des paramètres par défaut pour l’ensemble du workflow. Pour plus d’informations, consultez [`defaults`](/actions/using-workflows/workflow-syntax-for-github-actions#defaults).
