Vous pouvez utiliser l’action `setup-node` pour créer un fichier *.npmrc* local sur l’exécuteur, qui configure le registre et l’étendue par défaut. L’action `setup-node` accepte également un jeton d’authentification en tant qu’entrée. Celui-ci est utilisé pour accéder à des registres privés ou pour publier des packages de nœuds. Pour plus d’informations, consultez [`setup-node`](https://github.com/actions/setup-node/).
