1. Recherchez l’exécuteur à mettre à jour, puis cliquez sur {% octicon "triangle-down" aria-label="The downward triangle" %} pour voir le menu de sélection d’étiquettes. Ce menu affiche toutes les étiquettes personnalisées disponibles pour votre exécuteur auto-hébergé. Les étiquettes déjà affectées à votre exécuteur autohébergé ont une {% octicon "check" aria-label="Check mark" %} en regard.
