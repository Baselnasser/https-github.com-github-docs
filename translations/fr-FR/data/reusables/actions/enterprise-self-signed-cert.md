Nous vous recommandons vivement de configurer TLS sur {% data variables.product.prodname_ghe_server %} avec un certificat signé par une autorité de confiance. Bien qu’un certificat autosigné puisse fonctionner, une configuration supplémentaire est nécessaire pour vos exécuteurs autohébergés. Elle n’est pas recommandée pour les environnements de production.
