Si vous vous appuyez sur l’utilisation de duplications de vos référentiels privés, vous pouvez configurer des stratégies qui contrôlent la façon dont des utilisateurs peuvent exécuter des workflow sur des événements `pull_request`. Disponible uniquement pour les dépôts {% ifversion ghec or ghes or ghae %}and internal{% endif %} uniquement, vous pouvez configurer ces paramètres de stratégie pour {% ifversion ghec %}des entreprises, {% elsif ghes or ghae %}votre entreprise, {% endif %}des organisations ou des dépôts.
