1. La première fois que vous utilisez Git pour cloner un dépôt ou accéder à des données dans un dépôt que vous avez déjà cloné, Git demande des informations d’identification. Fournissez le PAT du compte qui a accès au dépôt.

   Git va mettre en cache le PAT pour le répertoire dans lequel vous vous trouvez. Vous pourrez alors accéder aux données du dépôt et en écrire sur {% data variables.product.product_location %} à l’aide du compte approprié.
