1. Pour appliquer la configuration, entrez la commande suivante.

   {% note %}
   
   **Remarque** : Durant une exécution de configuration, les services sur {% data variables.product.product_location %} peuvent redémarrer, ce qui peut entraîner un bref temps d’arrêt pour les utilisateurs.

   {% endnote %}

    ```shell
    ghe-config-apply
    ```
1. Attendez la fin de l’exécution de la configuration.
