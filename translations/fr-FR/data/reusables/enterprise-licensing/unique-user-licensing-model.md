{% data variables.product.company_short %} utilisent un modèle de licence utilisateur unique. Pour les produits d’entreprise qui incluent plusieurs options de déploiement, {% data variables.product.company_short %} détermine le nombre de sièges avec licence que vous consommez en fonction du nombre d’utilisateurs uniques sur tous vos déploiements.

Chaque utilisateur ne consomme qu’une seule licence, quel que soit le nombre d’instances {% data variables.product.prodname_ghe_server %} dont il se sert ou le nombre d’organisations dont il est membre sur {% data variables.product.prodname_ghe_cloud %}. Ce modèle permet à chaque personne d’utiliser plusieurs déploiements de {% data variables.product.prodname_enterprise %} sans entraîner de coûts supplémentaires.
