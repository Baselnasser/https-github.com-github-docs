**Remarque :** si vous synchronisez l’utilisation de la licence et si votre compte d’entreprise sur {% data variables.product.prodname_dotcom_the_website %} n’utilise pas {% data variables.product.prodname_emus %}, nous vous recommandons vivement d’activer les domaines vérifiés pour votre compte d’entreprise sur {% data variables.product.prodname_dotcom_the_website %}. Pour des raisons de confidentialité, votre rapport de licence consommée n’inclut l’adresse e-mail associée à un compte d’utilisateur sur {% data variables.product.prodname_dotcom_the_website %} que si l’adresse est hébergée par un domaine vérifié. Si une personne consomme par erreur plusieurs licences, l’accès à l’adresse e-mail utilisée pour la déduplication facilite grandement la résolution des problèmes. Pour plus d’informations, consultez {% ifversion ghec or ghes %}« [Vérification ou approbation d’un domaine pour votre entreprise](/enterprise-cloud@latest/admin/configuration/configuring-your-enterprise/verifying-or-approving-a-domain-for-your-enterprise) » et {% endif %}« [À propos de {% data variables.product.prodname_emus %}](/enterprise-cloud@latest/admin/identity-and-access-management/managing-iam-with-enterprise-managed-users/about-enterprise-managed-users){% ifversion not ghec %} » dans la documentation de {% data variables.product.prodname_ghe_cloud %}.{% else %}.{% endif %}
