Pour une personne utilisant plusieurs environnements {% data variables.product.prodname_enterprise %} pour consommer une seule licence, vous devez synchroniser l’utilisation des licences entre les environnements. Ensuite, {% data variables.product.company_short %} va dédupliquer les utilisateurs en fonction des adresses e-mail associées à leurs comptes d’utilisateur. Pour plus d’informations, consultez « [Résolution des problèmes d’utilisation des licences de {% data variables.product.prodname_enterprise %}](/billing/managing-your-license-for-github-enterprise/troubleshooting-license-usage-for-github-enterprise#about-the-calculation-of-consumed-licenses) ».
