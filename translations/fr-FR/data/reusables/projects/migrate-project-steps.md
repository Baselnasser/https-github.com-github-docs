1. En regard du projet que vous voulez migrer, cliquez sur {% octicon "kebab-horizontal" aria-label="The menu icon" %} pour ouvrir le menu.
1. Dans le menu, cliquez sur **Migrer**.
  ![Capture d’écran montrant l’option Migrer](/assets/images/help/issues/migrate-menu-item.png)
1. Dans la fenêtre contextuelle, sélectionnez ce que vous voulez faire pour le projet après sa migration. Nous vous recommandons de sélectionner « Fermer l’ancien projet » afin que le projet en cours d’utilisation apparaisse clairement.
  ![Capture d’écran montrant la boîte de dialogue Migrer](/assets/images/help/issues/migrate-option.png)
1. En bas à droite de la fenêtre contextuelle, cliquez sur **Démarrer la migration**.
  ![Capture d’écran montrant le bouton Démarrer la migration](/assets/images/help/issues/migrate-start.png)
