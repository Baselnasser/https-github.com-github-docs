1. Cliquez sur l’onglet **Projets**. ![Capture d’écran montrant le bouton Fermer du projet](/assets/images/help/issues/projects-profile-tab.png)
1. Pour afficher les projets fermés, cliquez sur **Fermé**.
   ![Capture d’écran montrant le bouton Fermer du projet](/assets/images/help/issues/closed-projects-tab.png)
1. Cliquez sur le projet que vous voulez rouvrir.
1. En haut à droite, cliquez sur {% octicon "kebab-horizontal" aria-label="The menu icon" %} pour ouvrir le menu.
  ![Capture d’écran montrant l’icône du menu.](/assets/images/help/projects-v2/open-menu.png)
1. Dans le menu, cliquez sur {% octicon "gear" aria-label="The gear icon" %} **Paramètres** pour accéder aux paramètres du projet.
  ![Capture d’écran montrant l’élément de menu « Paramètres »](/assets/images/help/projects-v2/settings-menu-item.png)
1. Dans le bas de la page, cliquez sur **Rouvrir le projet**.
   ![Capture d’écran montrant le bouton Rouvrir le projet](/assets/images/help/issues/reopen-project-button.png)
