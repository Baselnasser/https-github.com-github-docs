1. Sous le nom de votre organisation, cliquez sur {% octicon "table" aria-label="The project icon" %} **Projets**.
  ![Capture d’écran montrant l’onglet « Projets ».](/assets/images/help/projects-v2/tab-projects.png)
1. Cliquez sur **Nouveau projet**.
   ![Capture d’écran montrant le bouton Nouveau projet.](/assets/images/help/projects-v2/new-project-button.png)
1. Quand vous êtes invité à sélectionner un modèle, cliquez sur un modèle ou, pour commencer avec un projet vide, cliquez sur **Tableau** ou sur **Panneau**. Ensuite, cliquez sur **Créer**.

   ![Capture d’écran montrant la fenêtre modale de sélection du modèle](/assets/images/help/issues/projects-select-template.png)
