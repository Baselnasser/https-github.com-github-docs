1. Sélectionnez l’élément. Pour sélectionner plusieurs éléments, effectuez l’une des opérations suivantes :
     - Maintenez la touche <kbd>Commande</kbd> (Mac) ou <kbd>Ctrl</kbd> (Windows/Linux) enfoncée, puis cliquez sur chaque élément.
     - Sélectionnez un élément, puis appuyez sur <kbd>Maj</kbd>+<kbd>↑</kbd> ou <kbd>Maj</kbd>+<kbd>↓</kbd> pour sélectionner des éléments supplémentaires au-dessus ou en dessous de l’élément sélectionné.
     - Sélectionnez un élément, puis appuyez sur <kbd>Maj</kbd> et cliquez sur un autre élément pour sélectionner tous les éléments entre les deux éléments.
     - Avec une ligne ou un élément déjà sélectionné(e), appuyez sur <kbd>Commande</kbd>+<kbd>A</kbd> (Mac) ou <kbd>Ctrl</kbd>+<kbd>A</kbd> (Windows/Linux) pour sélectionner tous les éléments d’une colonne d’une disposition de panneau ou tous les éléments d’une disposition de tableau.
