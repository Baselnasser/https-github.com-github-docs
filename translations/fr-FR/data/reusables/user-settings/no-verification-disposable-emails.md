Vous ne pouvez pas vérifier les adresses e-mail de services d’adresses e-mail jetables (services qui vous permettent de recevoir des e-mails à une adresse temporaire qui expire après un certain temps). Si vous souhaitez garder votre adresse e-mail privée, vous pouvez utiliser une adresse e-mail `noreply` fournie par {% data variables.product.prodname_dotcom %}. Pour plus d’informations, consultez « [Définition de votre adresse e-mail de validation sur {% data variables.product.prodname_dotcom %}](/articles/setting-your-commit-email-address#setting-your-commit-email-address-on-github) ».
