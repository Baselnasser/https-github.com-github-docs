1. À partir d’un compte d’administration sur {% data variables.product.product_name %}, cliquez sur {% octicon "rocket" aria-label="The rocket ship" %} dans le coin supérieur droit de n’importe quelle page.

   ![Capture d’écran de l’icône représentant une fusée qui donne accès aux paramètres d’administration du site](/assets/images/enterprise/site-admin-settings/access-new-settings.png)

   ![Capture d’écran du lien « Administrateur du site »](/assets/images/enterprise/site-admin-settings/site-admin-link.png)
