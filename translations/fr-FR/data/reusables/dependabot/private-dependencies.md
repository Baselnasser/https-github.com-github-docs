Actuellement, {% data variables.product.prodname_dependabot_version_updates %} ne prend pas en charge un manifeste ou des fichiers de verrouillage contenant des dépendances git privées ou des registres git privés. Cela est dû au fait que, lors de l’exécution de mises à jour de version, {% data variables.product.prodname_dependabot %} doit être en mesure de résoudre toutes les dépendances de leur source pour vérifier que les mises à jour de version ont abouti.
