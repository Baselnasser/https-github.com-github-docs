Quand {% data variables.product.prodname_dependabot %} déclenche des demandes de tirage, celles-ci peuvent concerner des mises à jour de _sécurité_ ou de _version_ :

- Les _{% data variables.product.prodname_dependabot_security_updates %}_ sont des demandes de tirage automatisées qui vous aident à mettre à jour les dépendances qui ont des vulnérabilités connues. 
- Les _{% data variables.product.prodname_dependabot_version_updates %}_ sont des demandes de tirage automatisées qui tiennent à jour les dépendances, même si elles ne présentent aucune vulnérabilité. Pour vérifier l’état des mises à jour de version, accédez à l’onglet Insights de votre dépôt, puis sélectionnez Dependency Graph et {% data variables.product.prodname_dependabot %}.
