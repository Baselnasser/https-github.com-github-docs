Le {% data variables.contact.enterprise_support %} peut considérer un ticket résolu après avoir fourni une explication, une recommandation, des instructions d’utilisation, {% ifversion ghae %}ou {% endif %} des instructions pour contourner le problème{% ifversion ghes %}, ou en vous conseillant une version disponible qui résout le problème{% endif %}.

{% ifversion ghes or ghae %} Si vous avez installé d’autres logiciels tiers ou apporté des modifications dans {% data variables.product.product_name %} en dehors des recommandations du {% data variables.contact.enterprise_support %}, le {% data variables.contact.enterprise_support %} peut vous demander de supprimer les logiciels et/ou les modifications lors d’une tentative de résolution de problème. Si le problème est résolu une fois que les logiciels ou les modifications non pris en charge ont été supprimés, le {% data variables.contact.enterprise_support %} peut considérer le ticket résolu.
{% endif %}

Le {% data variables.contact.enterprise_support %} peut fermer un ticket si le ticket sort du cadre du support ou si plusieurs tentatives de vous contacter sont restées sans réponse. Si le {% data variables.contact.enterprise_support %} ferme un ticket en l’absence de réponse, vous pouvez demander que le {% data variables.contact.enterprise_support %} rouvre le ticket.
