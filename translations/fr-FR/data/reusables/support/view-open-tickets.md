{% data reusables.support.navigate-to-my-tickets %}
1. Pour afficher les tickets associés à un compte d’entreprise ou d’organisation, vous pouvez sélectionner le menu déroulant **Mes tickets**, puis cliquer sur le nom du compte d’organisation ou d’entreprise. 

{% indented_data_reference reusables.support.entitlements-note spaces=3 %}

   ![Capture d’écran du menu déroulant « Mes tickets ».](/assets/images/help/support/ticket-context.png)
1. Dans la liste des tickets, cliquez sur l’objet du ticket que vous souhaitez afficher.
![Capture d’écran montrant la liste des tickets de support avec l’objet mis en surbrillance.](/assets/images/help/support/my-tickets-list.png)
