1. Pour effectuer une mise à jour vers la dernière mise en production du projet, utilisez la branche `stable` en exécutant la commande `git checkout stable`.

  ```shell
  git checkout stable
  ```
  
  Vous pouvez également utiliser une version de projet spécifique en exécutant la commande suivante et en remplaçant `X.Y.Z` par la mise en production souhaitée.

  ```shell
  $ git checkout vX.Y.Z
  ```
