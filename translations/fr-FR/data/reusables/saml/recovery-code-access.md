1. En bas de la boîte de dialogue de l’authentification unique, cliquez sur **Utiliser un code de récupération** pour contourner l’authentification unique.

   ![Capture d’écran du lien pour entrer votre code de récupération](/assets/images/help/saml/saml_use_recovery_code.png)

2. Dans le champ « Code de récupération », tapez votre code de récupération.

   ![Capture d’écran du champ pour entrer votre code de récupération](/assets/images/help/saml/saml_recovery_code_entry.png)

3. Cliquez sur **Vérifier**.

   ![Capture d’écran du bouton pour vérifier votre code de récupération](/assets/images/help/saml/saml_verify_recovery_codes.png)

Une fois que vous avez utilisé un code de récupération, veillez à noter que le code n’est plus valide. Vous ne pourrez pas réutiliser le code de récupération.
