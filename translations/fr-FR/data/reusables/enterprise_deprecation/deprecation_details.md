Pour de meilleures performances, une sécurité améliorée et de nouvelles fonctionnalités, <a href="/enterprise/admin/guides/installation/upgrading-github-enterprise/">effectuez une mise à niveau vers la dernière version de GitHub Enterprise</a>.
