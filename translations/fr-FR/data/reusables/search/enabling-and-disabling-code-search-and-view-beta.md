Vous recevez un e-mail lorsque vous êtes accepté à partir de la liste d’attente pour rejoindre la version bêta. Une fois l’accès donné, la version bêta de la nouvelle recherche de code et de la nouvelle vue de code est automatiquement activée sur votre compte.

Vous pouvez désactiver ou activer la version bêta sur {% data variables.product.prodname_dotcom_the_website %} à tout moment. Notez que ce paramètre s’applique à la recherche de code et à la vue code.

{% data reusables.feature-preview.feature-preview-setting  %}
1. À droite de « Nouvelle recherche de code et nouvelle vue de code (bêta) », cliquez sur **Activer** ou **Désactiver**.
