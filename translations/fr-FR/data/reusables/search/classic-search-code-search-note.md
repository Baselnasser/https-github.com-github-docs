Cet article fait référence à l’expérience de recherche GitHub classique. Si vous participez à la version bêta de la nouvelle recherche de code et de la nouvelle vue de code et que vous avez activé la nouvelle fonctionnalité, la syntaxe de recherche de code et l’interface de recherche GitHub.com sont différentes. Toutefois, la syntaxe de recherche pour les types de recherche de contenu autre que du code reste la même.

Pour plus d’informations sur la nouvelle recherche de code et la nouvelle vue de code, actuellement en version bêta publique limitée, consultez « [À propos de GitHub Code Search (bêta)](/search-github/github-code-search/about-github-code-search) ». Pour essayer la version bêta de la nouvelle recherche de code et de la nouvelle vue de code, rejoignez la [liste d’attente](https://github.com/features/code-search-code-view/signup).
