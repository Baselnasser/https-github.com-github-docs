{% tip %}

**Remarque :** le verrouillage d’un dépôt empêche tout accès en écriture à celui-ci. Vous ne pouvez pas associer de nouvelles équipes ou de nouveaux collaborateurs à un dépôt verrouillé.

Si vous effectuez une évaluation, vous n’avez pas besoin de verrouiller le dépôt. Lorsque vous migrez des données à partir d’un dépôt en cours d’utilisation, {% data variables.product.company_short %} recommande vivement de verrouiller le dépôt. Pour plus d’informations, consultez « [À propos des migrations](/enterprise/admin/migrations/about-migrations#types-of-migrations) ».
{% endtip %}
