Quand vous bloquez un utilisateur, vous pouvez choisir de le bloquer indéfiniment ou pour une durée spécifique. Si vous bloquez une personne pour une durée spécifique, elle est automatiquement débloquée à l’expiration du délai choisi. Si vous bloquez une personne indéfiniment, vous pouvez la débloquer manuellement à tout moment. Pour plus d’informations, consultez « [ Déblocage de l’accès d’un utilisateur à votre organisation](/communities/maintaining-your-safety-on-github/unblocking-a-user-from-your-organization) ».
