1. À droite de « Autorités de certification SSH », cliquez sur **Nouvelle autorité de certification**.
  ![Bouton Nouvelle autorité de certification](/assets/images/help/organizations/new-ca-button.png)
2. Sous « Clé », collez votre clé SSH publique.
    ![Champ de clé pour ajouter une autorité de certification](/assets/images/help/organizations/ca-key-field.png)
3. Cliquez sur **Ajouter une autorité de certification**.
