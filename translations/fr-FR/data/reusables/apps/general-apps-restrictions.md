{% ifversion fpt or ghec %} Si vous utilisez votre application avec GitHub Actions et souhaitez modifier des fichiers de workflow, vous devez vous authentifier pour le compte de l’utilisateur avec un jeton OAuth qui inclut l’étendue `workflow`. L’utilisateur doit disposer de l’autorisation d’administration ou d’écriture sur le dépôt contenant le fichier de workflow. Pour plus d’informations, consultez « [Compréhension des étendues pour les applications OAuth](/apps/building-oauth-apps/understanding-scopes-for-oauth-apps/#available-scopes) ».{% endif %}
