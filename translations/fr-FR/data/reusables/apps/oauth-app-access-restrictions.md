Quand les restrictions d’accès d’une {% data variables.product.prodname_oauth_app %} sont activées, les membres de l’organisation et les collaborateurs externes ne peuvent pas autoriser l’accès de l’{% data variables.product.prodname_oauth_app %} aux ressources de l’organisation. Les membres de l’organisation peuvent demander l’approbation du propriétaire pour les {% data variables.product.prodname_oauth_apps %} qu’ils souhaitent utiliser, et les propriétaires de l’organisation reçoivent une notification des demandes en attente.
