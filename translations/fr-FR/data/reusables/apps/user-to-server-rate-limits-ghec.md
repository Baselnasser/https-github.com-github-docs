Les requêtes utilisateur à serveur sont soumises à une limite plus élevée de 15 000 requêtes par heure et par utilisateur authentifié dans les scénarios suivants.

- La requête provient d’une {% data variables.product.prodname_github_app %} qui appartient à une organisation {% data variables.product.prodname_ghe_cloud %}.
- La requête provient d’une {% data variables.product.prodname_oauth_app %} qui appartient à ou est approuvée par une organisation {% data variables.product.prodname_ghe_cloud %}.
