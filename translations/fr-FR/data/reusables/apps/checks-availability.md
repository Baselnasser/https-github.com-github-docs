L’autorisation d’accès en écriture à l’API Checks est uniquement disponible pour les applications GitHub. Les applications OAuth et les utilisateurs authentifiés peuvent afficher les exécutions de vérification et les suites de vérifications, mais ils ne peuvent pas les créer. Si vous ne créez pas une application GitHub, l’[API Statuses](/rest/reference/commits#commit-statuses) peut vous intéresser.
