1. Accédez aux paramètres de votre compte.
   - Pour une {% data variables.product.prodname_github_app %} appartenant à un compte personnel, dans le coin supérieur droit d’une page, cliquez sur votre photo de profil, puis sur **Paramètres**.
    ![Icône Paramètres sur la barre de l’utilisateur](/assets/images/settings/userbar-account-settings_post2dot12.png)
   - Pour une {% data variables.product.prodname_github_app %} appartenant à une organisation, dans le coin supérieur droit d’une page, cliquez sur votre photo de profil, puis sur **Vos organisations**. Puis, à droite de l’organisation, cliquez sur **Paramètres**.
    ![Vos organisations dans le menu du profil](/assets/images/help/profile/your-organizations.png)
    ![Le bouton Paramètres](/assets/images/help/organizations/settings-button.png)
