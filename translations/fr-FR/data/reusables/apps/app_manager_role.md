Par défaut, seuls les propriétaires d’organisation peuvent gérer les paramètres des applications GitHub dans une organisation. Pour permettre à des utilisateurs supplémentaires de gérer les applications GitHub dans une organisation, un propriétaire peut leur octroyer des autorisations de gestionnaire d’applications GitHub. Consultez « [Gestionnaires d’applications GitHub](/articles/permission-levels-for-an-organization/#github-app-managers) » pour savoir comment ajouter et supprimer des gestionnaires d’applications GitHub dans votre organisation.
