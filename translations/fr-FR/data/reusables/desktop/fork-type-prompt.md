1. Si vous envisagez d’utiliser cette duplication pour contribuer au dépôt en amont d’origine, cliquez sur **Pour contribuer au projet parent**.

  {% mac %}

  ![Option de menu Contribuer au projet parent](/assets/images/help/desktop/mac-fork-options-prompt.png)

  {% endmac %}

  {% windows %}

  ![Option de menu Contribuer au projet parent](/assets/images/help/desktop/windows-fork-options-prompt.png)

  {% endwindows %}

2. Si vous envisagez d’utiliser cette duplication pour un projet qui n’est pas connecté à l’amont, cliquez **Pour mon propre usage**.

  {% mac %}

  ![Contribuer à une option de menu de projet indépendant](/assets/images/help/desktop/mac-fork-own-purposes.png)

  {% endmac %}

  {% windows %}

  ![Contribuer à une option de menu de projet indépendant](/assets/images/help/desktop/windows-fork-own-purposes.png)

  {% endwindows %}

3. Cliquez sur **Continuer**.

  {% mac %}

  ![Bouton Continuer](/assets/images/help/desktop/mac-forking-continue.png)

  {% endmac %}

  {% windows %}

  ![Bouton Continuer](/assets/images/help/desktop/windows-forking-continue.png)

  {% endwindows %}
