Vous pouvez utiliser la notation automatique afin de vérifier automatiquement un devoir effectué par un étudiant sur {% data variables.product.prodname_classroom %}. Vous configurez des tests pour un devoir, et les tests s’exécutent immédiatement chaque fois qu’un étudiant procède à un envoi (push) vers un référentiel de devoirs sur {% data variables.product.product_location %}. L’étudiant peut consulter les résultats des tests, apporter des modifications et procéder à des envois (push) pour consulter de nouveaux résultats.
