Pour chaque affectation de groupe, {% data variables.product.prodname_classroom %} crée automatiquement un dépôt partagé unique auquel l’équipe peut accéder. Le dépôt peut être vide, ou vous pouvez le créer à partir d’un modèle de dépôt avec du code de démarrage, de la documentation, des tests et d’autres ressources. Le dépôt appartient au compte de votre organisation sur {% data variables.product.product_name %}, et {% data variables.product.prodname_classroom %} accorde l’accès aux équipes que les étudiants créent ou rejoignent lors de l’acceptation de l’affectation.
