<br/>

Définissez les paramètres de votre cas de test, comme le nom, les commandes, les entrées, les sorties, le délai d’expiration et les points. Quand vous avez terminé, cliquez sur **Enregistrer le cas de test**.

<div class="procedural-image-wrapper">
  <img alt="'Save test case' button for an autograding test" class="procedural-image-wrapper" src="/assets/images/help/classroom/assignments-click-save-test-case-button.png">
</div>

<br/>

Vous pouvez ajouter d’autres tests avec le menu déroulant **Ajouter un test**, puis modifier ou supprimer des tests existants avec {% octicon "pencil" aria-label="The pencil icon" %} ou {% octicon "trash" aria-label="The trash icon" %}.

<div class="procedural-image-wrapper">
