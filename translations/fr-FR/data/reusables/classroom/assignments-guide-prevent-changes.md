Vous pouvez empêcher les étudiants de changer les fichiers qui contiennent les tests pour la notation automatique, ou d’autres fichiers importants dans le dépôt de devoirs.

Sous « Chemins de fichiers protégés », tapez les chemins relatifs à protéger dans le dépôt.
