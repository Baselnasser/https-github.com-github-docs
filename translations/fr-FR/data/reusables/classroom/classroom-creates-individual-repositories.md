Quand un étudiant accepte un devoir, {% data variables.product.prodname_classroom %} crée automatiquement un dépôt pour celui-ci. Vous pouvez partir d’un dépôt vide, ou vous pouvez créer un dépôt à partir d’un modèle de dépôt comportant du code de démarrage, de la documentation, des tests et d’autres ressources. Chaque dépôt de devoirs appartient à votre compte d’organisation sur {% data variables.product.product_name %}.
