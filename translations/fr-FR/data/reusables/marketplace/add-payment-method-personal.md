1. Si vous avez choisi un plan payant, vérifiez votre mode de paiement.
    - Pour changer le mode de paiement existant sur le fichier de votre compte personnel, cliquez sur **Modifier...** et remplissez le formulaire pour ajouter un nouveau mode de paiement. Une fois que vous avez ajouté un nouveau mode de paiement, vous revenez automatiquement à la page de commande.
      ![Lien pour ajouter un mode de paiement à votre compte personnel](/assets/images/help/marketplace/marketplace-edit-payment-method-personal.png)
    - S’il n’existe pas de mode de paiement sur le fichier de votre compte personnel, cliquez sur **Ajouter un mode de paiement** et remplissez le formulaire pour ajouter une carte de crédit ou un compte PayPal.
      ![Lien pour ajouter un mode de paiement à votre compte personnel](/assets/images/help/marketplace/marketplace-add-payment-method-personal.png)
