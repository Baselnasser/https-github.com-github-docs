L’annulation d’une application ou la rétrogradation d’une application au niveau Gratuit n’affecte pas vos [autres abonnements payants](/articles/about-billing-on-github) dans {% data variables.product.prodname_dotcom %}. Si vous souhaitez mettre fin à tous vos abonnements payants dans {% data variables.product.prodname_dotcom %}, vous devez rétrograder chaque abonnement payant séparément.
