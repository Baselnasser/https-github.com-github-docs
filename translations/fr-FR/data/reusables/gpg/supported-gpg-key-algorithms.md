### Algorithmes de clé GPG pris en charge

{% data variables.product.product_name %} prend en charge plusieurs algorithmes de clé GPG. Si vous essayez d’ajouter une clé générée avec un algorithme non pris en charge, vous pouvez rencontrer une erreur.

- RSA
- ElGamal
- DSA
- ECDH
- ECDSA
- EdDSA
