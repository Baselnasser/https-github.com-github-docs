Chaque demande de tirage est un problème, mais chaque problème n’est pas une demande de tirage. Pour cette raison, les actions « partagées » pour les deux fonctionnalités, comme la gestion des attributions, des étiquettes et des jalons, sont fournies dans l’API Problèmes.
