Vous pouvez redemander une révision, par exemple, après avoir fait des changements substantiels sur votre demande de tirage.
Pour demander une nouvelle révision auprès d’un réviseur, dans la barre latérale de l’onglet **Conversation**, cliquez sur l’icône {% octicon "sync" aria-label="The sync icon" %}.
