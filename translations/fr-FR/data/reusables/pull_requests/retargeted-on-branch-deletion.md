Si vous supprimez une branche de tête (head) une fois que sa demande de tirage a été fusionnée, {% data variables.product.prodname_dotcom %} recherche les demandes de tirage ouvertes du même dépôt qui spécifient la branche supprimée comme branche de base. {% data variables.product.prodname_dotcom %} met automatiquement à jour ces demandes de tirage, en remplaçant leur branche de base par la branche de base de la demande de tirage fusionnée.
