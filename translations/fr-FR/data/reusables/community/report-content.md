1. Vous pouvez éventuellement voir des options permettant de **faire un signalement aux administrateurs du dépôt** ou **signaler un abus au support {% data variables.product.prodname_dotcom %}** . Dans le cas contraire, passez à l’étape suivante.
    - Pour signaler le contenu au support {% data variables.product.prodname_dotcom %}, cliquez sur **Signaler un abus au support {% data variables.product.prodname_dotcom %}** .
    - Pour signaler le contenu à des chargés de maintenance du dépôt, utilisez la liste déroulante **Choisir une raison** pour sélectionner une raison, puis cliquez sur **Signaler aux administrateurs du dépôt**. Votre rapport a été soumis.
  ![Modal « Signaler le contenu »](/assets/images/help/repository/report-content-modal.png)
2. Remplissez le formulaire « Signaler le contenu », puis cliquez sur **Envoyer la demande**.
  ![Formulaire « Signaler le contenu »](/assets/images/help/repository/report-content-to-support-form.png)
