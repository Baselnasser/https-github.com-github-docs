1. Vérifiez le champ `repository` dans le fichier *package.json* de votre projet. Le champ `repository` doit correspondre à l’URL de votre référentiel {% data variables.product.prodname_dotcom %}. Par exemple, si l’URL de votre référentiel est `github.com/my-org/test`, le champ du référentiel doit être `https://github.com/my-org/test.git`.
