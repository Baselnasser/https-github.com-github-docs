1. Au-dessus de la liste des fichiers, cliquez sur {% octicon "download" aria-label="The download icon" %} **Code**.
  ![Bouton Code](/assets/images/help/repository/code-button.png)
1. Copiez l’URL du dépôt.

   - Pour cloner le dépôt avec le protocole HTTPS, sous « HTTPS », cliquez sur {% octicon "clippy" aria-label="The clipboard icon" %}. 
   - Pour cloner le dépôt avec une clé SSH, en incluant un certificat émis par l’autorité de certification SSH de votre organisation, cliquez sur **SSH** et sur {% octicon "clippy" aria-label="The clipboard icon" %}. 
   - Pour cloner un dépôt avec {% data variables.product.prodname_cli %}, cliquez sur **{% data variables.product.prodname_cli %}** et sur {% octicon "clippy" aria-label="The clipboard icon" %}.
  ![Icône du Presse-papiers pour copier l’URL et cloner un dépôt avec GitHub CLI](/assets/images/help/repository/https-url-clone-cli.png)
