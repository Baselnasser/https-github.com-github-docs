1. Sous « Zone danger », cliquez sur **Transférer**.
 ![Bouton de transfert](/assets/images/help/repository/repo-transfer.png)
1. Lisez les informations sur le transfert d’un référentiel, puis tapez le nom de l’utilisateur ou de l’organisation auxquels vous souhaitez transférer la propriété du dépôt.
  ![Informations sur le transfert de dépôt et champ pour taper le nom d’utilisateur du nouveau propriétaire](/assets/images/help/repository/transfer-repo-new-owner-name.png)
1. Lisez les avertissements concernant la perte potentielle de fonctionnalités en fonction de l’abonnement du nouveau propriétaire.
  ![Avertissements concernant le transfert d’un dépôt à une personne utilisant un produit gratuit](/assets/images/help/repository/repo-transfer-free-plan-warnings.png)
1. Tapez le nom du dépôt que vous souhaitez transférer, puis cliquez sur **Je comprends, transférer ce dépôt**.
 ![Bouton Transférer](/assets/images/help/repository/repo-transfer-complete.png) {% else %}
1. Cliquez sur **Transférer**.
 ![Bouton de transfert](/assets/images/help/repository/repo-transfer.png)
1. Lisez les avertissements, puis entrez le nom du dépôt pour confirmer que vous l’avez fait.
1. Tapez le nom du nouveau propriétaire, puis cliquez sur **Je comprends, transférer ce dépôt**.
 ![Bouton Transférer](/assets/images/help/repository/repo-transfer-complete.png) {% endif %}
