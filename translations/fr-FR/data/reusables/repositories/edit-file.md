1. Dans le coin supérieur droit de l’affichage du fichier, cliquez sur {% octicon "pencil" aria-label="The edit icon" %} pour ouvrir l’éditeur de fichier.

   ![Bouton Modifier le fichier](/assets/images/help/repository/edit-file-edit-button.png)

   {% ifversion githubdev-editor -%} {% note %}

   **Remarque :** Au lieu de modifier et de commiter le fichier en utilisant l’éditeur de fichier par défaut, si vous le souhaitez, vous pouvez choisir d’utiliser l’[éditeur de code github.dev](/codespaces/the-githubdev-web-based-editor) en cliquant sur le menu déroulant en regard de {% octicon "pencil" aria-label="The edit icon" %} et en sélectionnant **Ouvrir dans github.dev**.

   ![Liste déroulante du bouton Modifier le fichier](/assets/images/help/repository/edit-file-edit-dropdown.png)

   {% endnote %} {%- endif %}
