1. Choisissez de sélectionner une seule ligne ou une plage :
    - Pour sélectionner une seule ligne de code, cliquez sur le numéro de la ligne pour mettre celle-ci en surbrillance.
    ![Fichier avec une seule ligne de code sélectionnée](/assets/images/help/repository/highlight-line-of-code.png)
      - Pour sélectionner une plage de code, cliquez sur le numéro de la première ligne de code de la plage pour mettre celle-ci en surbrillance. Pointez ensuite sur la dernière ligne de la plage de code, appuyez sur <kbd>Maj</kbd> et cliquez sur le numéro de la ligne pour mettre la plage en surbrillance.
    ![Fichier avec une plage de code sélectionnée](/assets/images/help/repository/highlight-range-of-code.png)
