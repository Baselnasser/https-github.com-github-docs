**Avertissement :** Quand vous chargez une image ou une vidéo dans une demande de tirage ou chargez un fichier dans un ticket dans le {% data variables.contact.support_portal %}, tout le monde peut voir l’URL anonymisée sans authentification, même si la demande de tirage ou le problème se trouve dans un dépôt privé{% ifversion ghes %} ou si le mode privé est activé{% endif %}. Pour préserver la confidentialité des fichiers multimédias sensibles, servez-les à partir d’un réseau privé ou d’un serveur qui nécessite une authentification. {% ifversion fpt or ghec %}Pour plus d’informations sur les URL anonymisées, consultez « [À propos des URL anonymisées](/github/authenticating-to-github/about-anonymized-urls) ».{% endif %}

{% endwarning %}
