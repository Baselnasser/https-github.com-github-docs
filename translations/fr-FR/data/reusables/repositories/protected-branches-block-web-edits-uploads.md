Si un référentiel possède des branches protégées, vous ne pouvez pas modifier ou charger des fichiers dans la branche protégée à l’aide de {% data variables.product.prodname_dotcom %}. Pour plus d’informations, consultez « [À propos des branches protégées](/articles/about-protected-branches) ».

Vous pouvez utiliser {% data variables.product.prodname_desktop %} pour déplacer vos modifications vers une nouvelle branche et les valider. Pour plus d’informations, consultez « [Commit et revue des changements apportés à votre projet](/desktop/contributing-to-projects/committing-and-reviewing-changes-to-your-project) ».
