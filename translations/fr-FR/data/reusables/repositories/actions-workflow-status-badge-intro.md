Un badge d’état indique si un workflow est en train d’échouer ou de réussir. En règle générale, vous ajoutez un badge d’état dans le fichier `README.md` de votre dépôt, mais vous pouvez l’ajouter dans n’importe quelle page web de votre choix. Par défaut, les badges affichent l’état de votre branche par défaut. Vous pouvez également afficher l’état d’une exécution de workflow pour une branche ou un événement spécifique en utilisant les paramètres de requête `branch` et `event` dans l’URL.
