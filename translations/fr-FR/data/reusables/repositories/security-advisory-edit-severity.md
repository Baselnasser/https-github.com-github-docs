1. Sélectionnez la gravité de la faille de sécurité. Pour attribuer un score CVSS, sélectionnez « Évaluer la gravité à l’aide de CVSS », puis cliquez sur les valeurs appropriées dans le calculateur. {% data variables.product.product_name %} calcule le score à l’aide du [calculateur CVSS (Common Vulnerability Scoring System)](https://www.first.org/cvss/calculator).
