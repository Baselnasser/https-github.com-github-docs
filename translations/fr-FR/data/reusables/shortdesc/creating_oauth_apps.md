Vous pouvez créer et inscrire une application OAuth sous votre compte personnel ou sous toute organisation à laquelle vous avez un accès administratif. Lorsque vous créez votre application OAuth, n’oubliez pas de protéger votre confidentialité en utilisant uniquement les informations que vous considérez comme publiques.
