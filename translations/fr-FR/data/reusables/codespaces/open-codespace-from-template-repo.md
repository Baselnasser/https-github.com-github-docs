1. Sélectionnez un propriétaire pour le nouveau dépôt, entrez un nom de dépôt, sélectionnez votre paramètre de confidentialité préféré, puis cliquez sur **Créer un dépôt à partir du modèle**.
1. Dans la page principale du dépôt nouvellement créé, cliquez sur le bouton **{% octicon "code" aria-label="The code icon" %} Code**, puis sélectionnez l’onglet **Codespaces**.

   ![Bouton Nouveau codespace](/assets/images/help/codespaces/new-codespace-button.png)

1. Sous l’onglet **Codespaces**, cliquez sur **Créer un codespace sur le principal**.
