1. Pour indexer vos modifications, cliquez sur **+** à côté du fichier que vous avez modifié, ou à côté des **Modifications** si vous avez modifié plusieurs fichiers et que vous souhaitez les indexer.
![Barre latérale de contrôle de code source avec le bouton de préproduction mis en évidence](/assets/images/help/codespaces/codespaces-commit-stage.png)
1. Tapez un message de validation (commit) décrivant la modification que vous avez apportée.
![Barre latérale du contrôle de code source avec un message de commit](/assets/images/help/codespaces/codespaces-commit-commit-message.png)  
1. Pour commiter les changements indexés, cliquez sur la coche en haut de la barre latérale du contrôle de code source.
![Cliquez sur l’icône en forme de coche](/assets/images/help/codespaces/codespaces-commit-checkmark-icon.png)
