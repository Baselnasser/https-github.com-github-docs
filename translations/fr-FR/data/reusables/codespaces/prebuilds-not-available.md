Il n’y a pas de pré-builds disponibles si vous choisissez d’utiliser un fichier `devcontainer.json` à partir d’un emplacement `.devcontainer/SUBDIRECTORY` lorsque vous créez un codespace. Pour plus d’informations sur le choix d’un fichier `devcontainer.json`, consultez « [Création d’un codespace](/codespaces/developing-in-codespaces/creating-a-codespace#creating-a-codespace) ».
