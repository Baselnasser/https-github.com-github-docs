Votre choix de types d’ordinateurs disponibles peut être limité par un certain nombre de facteurs. Il peut s’agir par exemple d’une stratégie configurée pour votre organisation, ou une spécification de type d’ordinateur minimal pour votre dépôt. Pour plus d’informations, consultez « [Restriction de l’accès aux types de machine](/codespaces/managing-codespaces-for-your-organization/restricting-access-to-machine-types) » et « [Définition d’une spécification minimale pour les machines de codespaces](/codespaces/setting-up-your-project-for-codespaces/setting-a-minimum-specification-for-codespace-machines) ».
