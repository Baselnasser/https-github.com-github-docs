Une fois que vous avez créé un secret, il est disponible quand vous créez un codespace ou redémarrez ce codespace. Pour utiliser un secret que vous venez de créer dans un codespace actuel, vous avez besoin d’arrêter ce codespace, puis de le reprendre. Pour plus d’informations sur l’arrêt du codespace, consultez « [Utilisation de la palette de commandes dans {% data variables.product.prodname_codespaces %}](/codespaces/codespaces-reference/using-the-command-palette-in-codespaces#suspending-or-stopping-a-codespace) ».
