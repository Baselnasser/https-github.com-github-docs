## Navigation vers un espace de code existant

1. {% data reusables.codespaces.you-can-see-all-your-codespaces %}
1. Cliquez sur le nom de l’espace de code dans lequel vous souhaitez développer.

   ![Nom de l’espace de code](/assets/images/help/codespaces/click-name-codespace.png)

Vous pouvez également voir tous vos codespaces d’un dépôt spécifique en accédant à ce dépôt et en sélectionnant **{% octicon "code" aria-label="The code icon" %} Code**. Le menu déroulant affiche tous les codespaces actifs d’un dépôt.
