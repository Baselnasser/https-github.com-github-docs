Vous pouvez supprimer des codespaces depuis {% data variables.product.prodname_vscode_shortname %} à condition de ne pas travailler dans un codespace.

{% data reusables.codespaces.click-remote-explorer-icon-vscode %}
1. Sous « CODESPACES GITHUB », cliquez avec le bouton droit sur le codespace que vous souhaitez supprimer.
1. Cliquez sur **Supprimer le codespace**.
