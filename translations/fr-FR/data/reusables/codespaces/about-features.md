Les composants sont des unités autonomes de code d’installation et de configuration de conteneur de développement, conçus pour fonctionner sur un large éventail d’images conteneur de base. Vous pouvez utiliser des composants pour ajouter rapidement des outils, des runtimes ou des bibliothèques à votre image codespace. Pour plus d’informations, consultez les [composants disponibles](https://containers.dev/features) et la [spécification des composants](https://containers.dev/implementors/features/) sur containers.dev.

Vous pouvez ajouter des composants à un fichier `devcontainer.json` à partir de {% data variables.product.prodname_vscode_shortname %} ou de votre dépôt sur {% data variables.product.prodname_dotcom_the_website %}.
