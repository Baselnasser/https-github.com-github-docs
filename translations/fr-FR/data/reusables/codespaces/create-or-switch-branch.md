1. Si la branche actuelle n’est pas affichée dans la barre d’état, au bas de votre codespace, cliquez avec le bouton droit sur la barre d’état, puis sélectionnez **Contrôle de code source**.
1. Cliquez sur le nom de la branche dans la barre d’état.
![Branche dans la barre d’état](/assets/images/help/codespaces/branch-in-status-bar.png)
1. Dans la liste déroulante, cliquez sur la branche sur laquelle vous souhaitez basculer, ou entrez le nom d’une nouvelle branche, puis cliquez sur **Créer une branche**.
![Effectuer un choix à partir du menu Branche](/assets/images/help/codespaces/create-new-branch.png)
