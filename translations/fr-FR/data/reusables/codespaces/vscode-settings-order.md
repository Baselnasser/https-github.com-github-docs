Quand vous configurez les paramètres de l’éditeur pour {% data variables.product.prodname_vscode_shortname %}, trois étendues sont disponibles : _Espace de travail_, _[Codespaces] distants_ et _Utilisateur_. Si un paramètre est défini dans plusieurs étendues, les paramètres de l’étendue _Espace de travail_ sont prioritaires, suivis de ceux de l’étendue _[Codespaces] distants_ et de ceux de l’étendue _Utilisateur_.
