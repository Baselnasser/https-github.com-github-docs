Les codespaces non publiés (codespaces créés à partir d’un modèle qui ne sont pas liés à un dépôt sur {% data variables.product.product_name %}) s’exécutent toujours sur une machine virtuelle avec les mêmes spécifications. Vous ne pouvez pas changer le type d’ordinateur d’un codespace non publié.
