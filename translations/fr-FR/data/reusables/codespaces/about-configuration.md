Vous pouvez créer une configuration de codespace par défaut pour votre dépôt. Cela vous permet ainsi de déterminer l’environnement de chaque codespace créé pour ce dépôt. La configuration définit un conteneur de développement qui peut inclure des frameworks, des outils, des extensions et un réacheminement de port.
