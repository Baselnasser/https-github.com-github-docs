Le domaine {% data variables.product.prodname_dotcom %} utilisé pour le transfert de port peut changer de temps à autre. Si vous référencez un port transféré dans votre code, par exemple dans un test, nous vous recommandons d’utiliser une variable d’environnement au lieu de coder en dur l’URL. Pour plus d’informations, consultez « [Variables d’environnement par défaut pour votre codespace](/codespaces/developing-in-codespaces/default-environment-variables-for-your-codespace#list-of-default-environment-variables) ».
