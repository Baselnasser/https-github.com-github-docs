Nous vous recommandons de donner à un nombre limité de membres dans chaque organisation un rôle de propriétaire d’organisation, qui fournit un accès administratif complet pour cette organisation. Pour plus d’informations, consultez « [Rôles dans une organisation](/organizations/managing-peoples-access-to-your-organization-with-roles/roles-in-an-organization) ».

Pour les organisations où vous disposez d’autorisations d’administrateur, vous pouvez également personnaliser l’accès à chaque dépôt avec des niveaux d’autorisation précis. Pour plus d’informations, consultez « [Niveaux d’autorisations de dépôt pour une organisation](/organizations/managing-access-to-your-organizations-repositories/repository-permission-levels-for-an-organization) ».
