{% data variables.product.prodname_pages %} est un service d’hébergement de site statique qui prend des fichiers HTML, CSS et JavaScript directement à partir d’un dépôt et publie un site web. Vous pouvez activer ou désactiver {% data variables.product.prodname_pages %} pour les membres de votre entreprise au niveau de l’organisation. Pour plus d’informations, consultez « [Configuration de {% data variables.product.prodname_pages %} pour votre entreprise](/admin/configuration/configuring-your-enterprise/configuring-github-pages-for-your-enterprise) » et « [À propos de {% data variables.product.prodname_pages %}](/pages/getting-started-with-github-pages/about-github-pages) ».
