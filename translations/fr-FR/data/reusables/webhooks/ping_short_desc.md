Lorsque vous créez un nouveau webhook, nous vous envoyons un simple événement `ping` pour vous informer que vous avez correctement configuré le webhook. Cet événement n’est pas stocké et ne peut donc pas être récupéré via le point de terminaison de l’[API Événements](/rest/reference/activity#ping-a-repository-webhook).
