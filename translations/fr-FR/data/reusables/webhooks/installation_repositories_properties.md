Clé | Type | Description
----|------|------------
`action` | `string` | Action qui a été effectuée. La valeur peut être `added` ou `removed`.
`repository_selection` | `string` | Choix des dépôts sur lesquels est l’installation. La valeur peut être `selected` ou `all`.
`repositories_added` | `array` | Tableau d’objets de dépôt qui ont été ajoutés à l’installation.
`repositories_removed` | `array` | Tableau d’objets de dépôt qui ont été supprimés de l’installation.
