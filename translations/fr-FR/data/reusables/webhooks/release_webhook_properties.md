Clé | Type | Description
----|------|-------------
`action`|`string` | Action qui a été effectuée. Peut être :<ul><li>`published` : une version, une préversion ou un brouillon de version est publié</li><li>`unpublished` : une version ou une préversion est supprimée</li><li>`created` : un brouillon est enregistré, ou une version ou une préversion est publiée sans avoir été enregistrée en tant que brouillon</li><li>`edited` : une version, une préversion ou un brouillon de version est modifié</li><li>`deleted` : une version, une préversion ou un brouillon de version est supprimé</li><li>`prereleased` : une préversion est créée</li><li>`released` : une version est publiée ou une préversion est remplacée par une version</li>
