La prochaine fois que vous clonez une URL HTTPS qui nécessite une authentification, Git vous invite à vous connecter dans une fenêtre de navigateur. Vous pouvez d’abord être invité à autoriser une application OAuth.{% ifversion not ghae %} Si votre compte ou organisation nécessite l’[authentification à deux facteurs](/github/authenticating-to-github/securing-your-account-with-two-factor-authentication-2fa), vous devez également effectuer l’étape 2FA.{% endif %}
