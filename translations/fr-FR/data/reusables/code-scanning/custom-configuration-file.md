Le fichier de configuration peut être situé dans le référentiel que vous analysez ou dans un référentiel externe. L’utilisation d’un référentiel externe vous permet de spécifier des options de configuration pour plusieurs référentiels à un même emplacement. Quand vous référencez un fichier de configuration situé dans un dépôt externe, vous pouvez utiliser la syntaxe _OWNER/REPOSITORY/FILENAME@BRANCH_ . Par exemple : _octo-org/shared/codeql-config.yml@main_.
