- **읽기** - 사용자에게 {% data variables.projects.projects_v1_board %}를 볼 수 있는 권한을 부여합니다.
- **쓰기** - 사용자에게 {% data variables.projects.projects_v1_board %}를 보고, 리포지토리를 {% data variables.projects.projects_v1_board %}에 연결하고, {% data variables.projects.projects_v1_board %}와 상호 작용할 수 있는 권한을 부여합니다. 자세한 내용은 “[{% data variables.product.prodname_project_v1 %}에 리포지토리 연결](/articles/linking-a-repository-to-a-project-board)”을 참조하세요.
- **관리자** - 사용자에게 {% data variables.projects.projects_v1_board %}를 보고, {% data variables.projects.projects_v1_board %}와 상호 작용하고, {% data variables.projects.projects_v1_board %} 설정을 관리하고, 다른 사용자의 {% data variables.projects.projects_v1_board %}에 대한 액세스를 관리할 수 있는 권한을 부여합니다.
