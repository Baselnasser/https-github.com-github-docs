구성 파일은 분석 중인 리포지토리 또는 외부 리포지토리 내에 있을 수 있습니다. 외부 리포지토리를 사용하면 한 곳에서 여러 리포지토리에 대한 구성 옵션을 지정할 수 있습니다. 외부 리포지토리에 있는 구성 파일을 참조하는 경우 _OWNER/REPOSITORY/FILENAME@BRANCH_ 구문을 사용할 수 있습니다. 예를 들어 _octo-org/shared/codeql-config.yml@main_ 와 같습니다.
