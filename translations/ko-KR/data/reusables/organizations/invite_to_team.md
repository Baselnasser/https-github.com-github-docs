1. 팀에 조직 구성원을 추가하거나 비구성원을 팀에 참가하도록 초대합니다.
   - 추가하려는 사람이 이미 조직의 구성원인 경우 해당 구성원의 사용자 이름을 입력하고 **Enter** 키를 클릭합니다. 사용자가 즉시 팀에 추가됩니다.
   - 추가하려는 사람이 조직의 구성원이 아닌 경우 해당 사람의 사용자 이름 또는 메일 주소를 입력하고 **Enter** 키를 클릭합니다. 사용자가 조직에 가입하기 위한 메일 초대를 받고 초대를 수락하면 팀에 추가됩니다.{% else %}
