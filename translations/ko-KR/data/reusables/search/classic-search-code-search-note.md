이 문서에서는 클래식 GitHub 검색 환경을 참조합니다. 새 코드 검색 및 코드 뷰 베타에 참여하고 새 기능을 사용하도록 설정한 경우 코드 검색 구문과 GitHub.com 검색 인터페이스가 다릅니다. 그러나 코드가 아닌 검색 형식에 대한 검색 구문은 동일하게 유지됩니다.

현재 제한된 퍼블릭 베타에 있는 새 코드 검색 및 코드 보기에 대한 자세한 내용은 "[GitHub 코드 검색 정보(베타)](/search-github/github-code-search/about-github-code-search)"를 참조하세요. 새 코드 검색 및 코드 뷰 베타를 사용해 보려면 [대기 목록에](https://github.com/features/code-search-code-view/signup) 조인합니다.
