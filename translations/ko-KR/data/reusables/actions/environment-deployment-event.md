환경을 참조하는 워크플로 작업이 실행되면 환경 이름으로 설정된 `environment` 속성이 있는 배포 개체를 만듭니다. 워크플로가 진행됨에 따라 환경 이름으로 설정된 `environment` 속성, 환경의 URL로 설정된 `environment_url` 속성(워크플로에 지정된 경우), 작업의 상태로 설정된 `state` 속성이 있는 배포 상태 개체도 만듭니다.
