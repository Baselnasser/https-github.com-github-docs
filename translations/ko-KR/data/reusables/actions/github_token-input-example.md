이 예제 워크플로는 `repo-token` 입력 매개 변수의 값으로 `GITHUB_TOKEN`이 필요한 [레이블 지정자 작업](https://github.com/actions/labeler)을 사용합니다.

```yaml{:copy}
name: Pull request labeler
on: [ pull_request_target ]

permissions:
  contents: read
  pull-requests: write

jobs:
  triage:
    runs-on: ubuntu-latest
    steps:
      - uses: {% data reusables.actions.action-labeler %}
        with:
          repo-token: {% raw %}${{ secrets.GITHUB_TOKEN }}{% endraw %}
```
