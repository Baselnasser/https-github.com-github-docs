{% data variables.product.product_name %}는 Azure Web App과 같은 인기 있는 여러 서비스에 대한 배포 시작 워크플로를 제공합니다. 시작 워크플로 사용을 시작하는 방법을 알아보려면 “[시작 워크플로 사용](/actions/learn-github-actions/using-starter-workflows)”을 참조하거나 [배포 시작 워크플로의 전체 목록을 찾아보세요](https://github.com/actions/starter-workflows/tree/main/deployments). “[Azure App Service 배포](/actions/deployment/deploying-to-azure-app-service)”와 같은 특정 배포 워크플로에 대한 자세한 가이드를 확인할 수도 있습니다.

또한 많은 서비스 공급자는 자사 서비스에 배포하기 위한 {% data variables.product.prodname_marketplace %}에 대한 작업을 제공합니다. 전체 목록은 [{% data variables.product.prodname_marketplace %}](https://github.com/marketplace?category=deployment&type=actions)를 참조하세요.
