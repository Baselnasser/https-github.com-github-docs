*client.js* 스크립트는 클라이언트를 만들 `REDIS_HOST` 환경 변수 및 `REDIS_PORT` 환경 변수를 찾습니다. 워크플로는 두 환경 변수를 “Redis에 커넥트” 단계의 일부로 설정하여 *client.js* 스크립트에 사용할 수 있도록 합니다. 스크립트에 대한 자세한 내용은 “[Redis 서비스 컨테이너 테스트](#testing-the-redis-service-container)”를 참조하세요.
