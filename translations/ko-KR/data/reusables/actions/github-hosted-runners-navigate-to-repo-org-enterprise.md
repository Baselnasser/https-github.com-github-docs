{% ifversion fpt %}
1. 조직 또는 리포지토리의 기본 페이지로 이동합니다.
1. {% octicon "gear" aria-label="The Settings gear" %} **Settings**(설정)를 클릭합니다.
1. 왼쪽 사이드바에서 **Actions**(작업)를 클릭한 다음 **Runners**(실행기)를 클릭합니다.
{% elsif ghec %}
1. 실행기 설정으로 이동합니다.
   * **조직 또는 리포지토리에서**: 기본 페이지로 이동한 다음 {% octicon "gear" aria-label="The Settings gear" %} **Settings**(설정)를 클릭합니다.
   * **엔터프라이즈 계정을 사용하는 경우**: {% data variables.product.prodname_dotcom_the_website %}의 오른쪽 상단에 있는 프로필 사진을 클릭한 다음 **내 엔터프라이즈** 를 클릭하고 엔터프라이즈를 클릭하여 엔터프라이즈 계정으로 이동합니다.
1. {% data variables.product.prodname_actions %} 설정으로 이동합니다.
   * **조직 또는 리포지토리에서**: 왼쪽 사이드바에서 **Actions**(작업)를 클릭한 다음 **Runners**(실행기)를 클릭합니다.
   * **엔터프라이즈 계정을 사용하는 경우**: 왼쪽 사이드바에서 **“{% octicon "law" aria-label="The law icon" %} Policies”(정책)** 를 클릭한 다음, **Actions**(작업)를 클릭하고 **Runners**(실행기) 탭을 클릭합니다.{% endif %}
