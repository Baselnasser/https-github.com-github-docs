- **배포(CD)** . 자세한 내용은 “[지속적인 배포 정보](/actions/deployment/about-deployments/about-continuous-deployment)”를 참조하세요. {% ifversion fpt or ghec %}- **보안**. 자세한 내용은 “[시작 워크플로를 사용하여 {% data variables.product.prodname_code_scanning %} 설정](/code-security/code-scanning/automatically-scanning-your-code-for-vulnerabilities-and-errors/setting-up-code-scanning-for-a-repository#setting-up-code-scanning-using-starter-workflows)”을 참조하세요.{% endif %}

- **자동화**. 자동화 시작 워크플로는 끌어오기 요청을 심사하고 끌어오기 요청에서 수정된 경로를 기반으로 레이블을 적용하거나 리포지토리에 처음 기여하는 사용자에게 인사말을 하는 등 워크플로를 자동화하기 위한 솔루션을 제공합니다.
