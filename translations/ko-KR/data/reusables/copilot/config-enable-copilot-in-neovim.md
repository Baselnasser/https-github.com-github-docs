1. {% data variables.product.prodname_copilot %}을 구성하려면 Neovim을 열고 다음 명령을 입력합니다.
  
   ```
   :Copilot setup
   ```
  
1. Neovim 구성 또는 Neovim 명령을 사용하여 {% 데이터 variables.product.prodname_copilot %}을(를) 사용하도록 설정합니다.
  
   ```
   :Copilot enable
   ```
