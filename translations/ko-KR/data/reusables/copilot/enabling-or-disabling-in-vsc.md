## {% data variables.product.prodname_copilot %} 사용 또는 사용 안 함

{% data variables.product.prodname_vscode %}에서 {% data variables.product.prodname_copilot %}을 사용하거나 사용하지 않도록 설정할 수 있습니다. {% data variables.product.prodname_vscode %} 창의 아래쪽 패널에 있는 {% data variables.product.prodname_copilot %} 상태 아이콘은 {% data variables.product.prodname_copilot %}이 사용하도록 설정되었는지 여부를 나타냅니다. 사용하도록 설정하면 아이콘의 배경색이 상태 표시줄의 색과 일치합니다. 사용하지 않도록 설정하면 아이콘의 배경색이 상태 표시줄의 색과 대비됩니다.

1. {% data variables.product.prodname_copilot %}을 사용하거나 사용하지 않도록 설정하려면 {% data variables.product.prodname_vscode %} 창의 아래쪽 패널에서 상태 아이콘을 클릭합니다.
   ![{% data variables.product.prodname_vscode %}의 상태 아이콘 스크린샷](/assets/images/help/copilot/status-icon-visual-studio-code.png)
2. {% data variables.product.prodname_copilot %}을 사용하지 않도록 설정하는 경우 전역적으로 또는 현재 편집 중인 파일의 언어에 대해 제안을 사용하지 않도록 설정할지 묻는 메시지가 표시됩니다.

   - {% data variables.product.prodname_copilot %}에서 제안을 전역적으로 사용하지 않도록 설정하려면 **전역적으로 사용하지 않음** 을 클릭합니다.
   - 지정된 언어에 대한 {% data variables.product.prodname_copilot %}에서 제안을 사용하지 않으려면 **_언어_ 에 대해 사용하지 않음** 을 클릭합니다.
   ![전역적으로 또는 현재 언어에 대해 {% data variables.product.prodname_copilot %}을 사용하지 않도록 설정하는 옵션의 스크린샷](/assets/images/help/copilot/disable-copilot-global-or-langugage.png)
