또한 에코시스템에서 매니페스트 또는 잠금 파일 분석을 위한 종속성 그래프가 지원되지 않더라도 종속성 제출 API(베타)를 사용하여 선택한 패키지 관리자 또는 에코시스템에서 종속성을 제출할 수 있습니다. 종속성 그래프는 에코시스템별로 그룹화된 제출된 종속성을 표시하지만 매니페스트 또는 잠금 파일에서 구문 분석된 종속성과는 별도로 표시합니다. 종속성 제출 API에 대한 자세한 내용은 "[종속성 제출 API 사용](/code-security/supply-chain-security/understanding-your-software-supply-chain/using-the-dependency-submission-api)"을 참조하세요.
