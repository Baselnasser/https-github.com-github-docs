1. 프로젝트의 *package.json* 에 있는 `repository` 필드를 확인합니다. `repository` 필드는 {% data variables.product.prodname_dotcom %} 리포지토리의 URL과 일치해야 합니다. 예를 들어 리포지토리 URL이 `github.com/my-org/test`인 경우 리포지토리 필드는 `https://github.com/my-org/test.git`이어야 합니다.
