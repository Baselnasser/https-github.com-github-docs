다양한 콘텐츠 형식을 사용하여 웹후크를 전달할 수 있습니다.

- `application/json` 콘텐츠 형식은 JSON 페이로드를 `POST` 요청의 본문으로 직접 전달합니다.
- `application/x-www-form-urlencoded` 콘텐츠 형식은 JSON 페이로드를 `payload`라는 양식 매개 변수로 전송합니다.

요구에 가장 적합한 설정을 선택합니다.
