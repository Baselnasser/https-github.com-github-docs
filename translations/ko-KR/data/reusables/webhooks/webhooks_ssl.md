“페이로드 URL”이 보안 사이트(HTTPS)인 경우 SSL 확인 설정을 구성하는 옵션이 있습니다. “페이로드 URL”이 안전하지 않은 경우(HTTP) GitHub는 이 옵션을 표시하지 않습니다. 기본적으로 GitHub는 웹후크 페이로드를 제공할 때 웹 사이트의 SSL 인증서를 확인합니다. SSL 확인은 후크 페이로드가 URL 엔드포인트에 안전하게 전달되도록 하는 데 도움이 됩니다. SSL을 사용하지 않도록 설정하는 옵션이 있지만 **SSL 확인 사용** 을 선택된 상태로 유지하는 것이 좋습니다.
