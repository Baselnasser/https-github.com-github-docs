2. 비밀을 푸시할 수 있어야 하는 이유를 가장 잘 설명하는 옵션을 선택합니다.
    - 비밀이 테스트에만 사용되고 위협이 되지 않는 경우 **테스트에 사용됨** 을 클릭합니다.
    - 검색된 문자열이 비밀이 아니면 **가양성임** 을 클릭합니다.
    - 비밀이 진짜이지만 나중에 수정하려는 경우 **나중에 수정** 을 클릭합니다.
