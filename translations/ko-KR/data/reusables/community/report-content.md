1. **리포지토리 관리자에게 보고** 하거나 **{% data variables.product.prodname_dotcom %} 지원에 남용을 보고** 하는 옵션이 표시될 수 있습니다. 그렇지 않은 경우 다음 단계로 건너뜁니다.
    - 콘텐츠를 {% data variables.product.prodname_dotcom %} 지원에 보고하려면 **{% data variables.product.prodname_dotcom %} 지원에 남용 보고** 를 클릭합니다.
    - 리포지토리 유지 관리자에 콘텐츠를 보고하려면 **이유 선택** 드롭다운을 사용하여 이유를 선택한 다음 **리포지토리 관리자에게 보고** 를 클릭합니다. 보고서가 제출되었습니다.
  ![“보고서 콘텐츠” 모달](/assets/images/help/repository/report-content-modal.png)
2. “보고서 콘텐츠” 양식을 완료한 다음 **요청 보내기** 를 클릭합니다.
  ![“보고서 콘텐츠” 양식](/assets/images/help/repository/report-content-to-support-form.png)
