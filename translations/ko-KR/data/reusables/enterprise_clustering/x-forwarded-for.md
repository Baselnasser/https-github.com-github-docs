PROXY 프로토콜을 사용할 수 없는 **경우에만** X-Forwarded-For 프로토콜을 사용합니다. `X-Forwarded-For` 헤더는 HTTP 및 HTTPS에서만 작동합니다. SSH를 통한 Git 연결에 대해 보고된 IP 주소는 부하 분산 장치 IP를 표시합니다.
