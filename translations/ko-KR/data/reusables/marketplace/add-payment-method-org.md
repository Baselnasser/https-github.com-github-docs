1. 유료 플랜을 선택한 경우 결제 방법을 검토합니다.
   - 조직에 대한 파일에서 기존 결제 방법을 변경하려면 **편집...** 을 클릭한 다음 양식을 작성하여 새 결제 방법을 추가합니다. 새 결제 방법을 성공적으로 추가하면 주문 페이지로 자동으로 돌아갑니다.
     ![조직에 결제 방법을 추가하는 링크](/assets/images/help/marketplace/marketplace-edit-payment-method-org.png)
   - 조직에 대한 파일에 결제 방법이 없는 경우 **결제 방법 추가** 를 클릭한 다음 양식을 작성하여 신용 카드 또는 PayPal 계정을 추가합니다.
     ![조직에 결제 방법을 추가하는 링크](/assets/images/help/marketplace/marketplace-add-payment-method-org.png)
