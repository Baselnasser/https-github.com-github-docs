1. 필요에 따라 기본적으로 모든 커밋에 서명하도록 Git을 구성하려면 다음 명령을 입력합니다.

   ```Shell
   $ git config --global commit.gpgsign true
   ```
   
   자세한 내용은 "[커밋 서명"을 참조하세요.](/authentication/managing-commit-signature-verification/signing-commits)
