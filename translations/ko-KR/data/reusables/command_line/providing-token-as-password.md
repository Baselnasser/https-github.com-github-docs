토큰이 있으면 HTTPS를 통해 Git 작업을 수행할 때 암호 대신 토큰을 입력할 수 있습니다.

예를 들어 명령줄에서 다음을 입력합니다.

```shell
$ git clone https://{% data variables.command_line.codeblock %}/USERNAME/REPO.git
Username: YOUR_USERNAME
Password: YOUR_TOKEN
```
