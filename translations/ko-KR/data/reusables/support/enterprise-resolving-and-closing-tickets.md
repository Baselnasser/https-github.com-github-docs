{% data variables.contact.enterprise_support %}는 설명, 권장 사항, 사용 지침 {% ifversion ghae %}또는 {% endif %} 해결 방법 지침{% ifversion ghes %}을 제공하거나 문제를 해결하는 사용 가능한 릴리스를 알려준 후에 티켓이 해결된 것으로 간주할 수 있습니다.{% endif %}

{% ifversion ghes or ghae %} {% data variables.contact.enterprise_support %}의 권장 사항 외에 {% data variables.product.product_name %}에 추가 타사 소프트웨어를 설치했거나 수정한 경우 {% data variables.contact.enterprise_support %}에서 문제를 해결하는 동안 소프트웨어 및/또는 수정 사항을 제거하도록 요청할 수 있습니다. 지원되지 않는 소프트웨어 또는 수정 사항이 제거되었을 때 문제가 해결되면 {% data variables.contact.enterprise_support %}에서 티켓이 해결된 것으로 간주할 수 있습니다.
{% endif %}

{% data variables.contact.enterprise_support %}는 티켓이 지원 범위를 벗어나거나 여러 번 연락을 시도했지만 응답하지 않은 경우 티켓을 종료할 수 있습니다. {% data variables.contact.enterprise_support %}에서 응답 부족으로 티켓을 종료한 경우 {% data variables.contact.enterprise_support %}에서 티켓을 다시 열도록 요청할 수 있습니다.
