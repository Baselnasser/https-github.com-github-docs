편집된 파일을 저장하는 것과 마찬가지로 커밋은 분기에서 하나 이상의 파일에 변경 내용을 기록합니다. Git은 각 커밋에 다음을 식별하는 SHA 또는 해시라는 고유 ID를 할당합니다.

- 특정 변경 내용
- 변경된 시점
- 변경 내용을 만든 사람

커밋할 때 변경 내용을 간략하게 설명하는 커밋 메시지를 포함해야 합니다.
