   Git은 현재 있는 디렉터리에 대한 {% 데이터 variables.product.pat_generic %}을(를) 캐시하고 올바른 계정을 사용하여 {% 데이터 variables.location.product_location %}에서 리포지토리 데이터에 액세스하고 쓸 수 있습니다.
