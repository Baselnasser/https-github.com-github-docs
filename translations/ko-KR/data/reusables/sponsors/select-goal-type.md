1. “목표는 무엇을 기반으로 하나요?”에서 **스폰서 수** 또는 **월별 스폰서십** 을 선택합니다.
![사용 가능한 목표 유형 ](/assets/images/help/sponsors/select-goal-type.png)
2. 원하는 스폰서 수 또는 목표로 하는 월별 금액을 입력합니다.
3. “목표 설명”에서 목표에 대한 설명을 입력합니다.
![목표 설정](/assets/images/help/sponsors/goal-settings.png)
5. 월별 스폰서쉽을 기반으로 목표를 선택한 경우 “이로 인해 매월 받는 스폰서쉽 자금이 공개적으로 표시됨을 이해합니다”를 선택합니다.
![스폰서쉽 자금이 게시될 것임을 이해하고 있다는 확인란](/assets/images/help/sponsors/sponsorships-published-checkbox.png)
