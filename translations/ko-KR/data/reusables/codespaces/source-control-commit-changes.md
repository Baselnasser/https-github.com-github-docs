1. 변경 내용을 스테이징하려면 변경한 파일 옆 또는 여러 파일을 변경하고 모두 스테이징하려는 경우 **Changes**(변경 내용) 옆의 **+** 아이콘을 클릭합니다.
![스테이징 단추가 강조 표시된 소스 제어 사이드바](/assets/images/help/codespaces/codespaces-commit-stage.png)
1. 변경 내용을 설명하는 커밋 메시지를 입력합니다.
![커밋 메시지가 있는 소스 제어 사이드바](/assets/images/help/codespaces/codespaces-commit-commit-message.png)  
1. 스테이징된 변경 내용을 커밋하려면 소스 제어 사이드바 위쪽의 확인 표시를 클릭합니다.
![확인 표시 아이콘 클릭](/assets/images/help/codespaces/codespaces-commit-checkmark-icon.png)
