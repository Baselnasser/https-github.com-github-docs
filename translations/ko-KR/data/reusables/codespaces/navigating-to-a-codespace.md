## 기존 codespace로 이동

1. {% data reusables.codespaces.you-can-see-all-your-codespaces %}
1. 개발하려는 codespace의 이름을 클릭합니다.

   ![의 이름](/assets/images/help/codespaces/click-name-codespace.png)

또는 해당 리포지토리로 이동하고 **{% octicon "code" aria-label="The code icon" %} 코드를** 선택하여 특정 리포지토리에 대한 codespace를 볼 수 있습니다. 드롭다운 메뉴에는 리포지토리에 대한 모든 활성 codespace가 표시됩니다.
