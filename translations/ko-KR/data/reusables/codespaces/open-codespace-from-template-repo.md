1. 새 리포지토리의 소유자를 선택하고, 리포지토리 이름을 입력하고, 원하는 개인 정보 설정을 선택한 다음, **템플릿에서 리포지토리 만들기** 를 클릭합니다.
1. 새로 만든 리포지토리의 기본 페이지에서 **{% octicon "code" aria-label="The code icon" %} 코드** 단추를 클릭하고 **Codespaces** 탭을 선택합니다.

   ![새 codespace 단추](/assets/images/help/codespaces/new-codespace-button.png)

1. **Codespaces** 탭에서 **기본 프로그램에서 codespace 만들기** 를 클릭합니다.
