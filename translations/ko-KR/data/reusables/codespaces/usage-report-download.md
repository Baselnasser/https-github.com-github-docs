1. 필요에 따라 "이번 달 사용량 현황" 옆에 있는 **사용 현황 보고서 가져오기** 를 클릭하여 {% data variables.product.prodname_actions %}, {% data variables.product.prodname_registry %} 및 {% data variables.product.prodname_github_codespaces %}에 대한 스토리지 사용의 CSV 보고서를 다운로드하기 위한 링크가 포함된 이메일을 가져옵니다. 메일은 계정의 기본 메일 주소로 전송됩니다. 보고서에서 지난 7일, 30일, 90일 또는 180일을 포함할지 여부를 선택할 수 있습니다.

   ![CSV 보고서 다운로드](/assets/images/help/billing/actions-packages-report-download.png)

   이 보고서에 사용되는 데이터는 매일 업데이트됩니다. 

   {% data variables.product.prodname_github_codespaces %} 컴퓨팅 사용량 및 스토리지 비용을 보려면 보고서를 필터링하여 열에 `Product` "Codespaces"를 언급하는 행만 표시합니다.

   ![{% data variables.product.prodname_codespaces %}에 대해 필터링된 사용 현황 보고서의 스크린샷](/assets/images/help/codespaces/CSV-usage-report.png)

   사전 빌드를 만들고, 업데이트하고, 저장하는 데 드는 비용만 보려면 보고서를 필터링하여 열에 `Actions Workflow` "Codespaces 사전 빌드 만들기"를 언급하는 행만 표시합니다.

   ![사전 빌드에 대해 필터링된 사용량 보고서 스크린샷](/assets/images/help/codespaces/CSV-usage-report-prebuilds.png)
