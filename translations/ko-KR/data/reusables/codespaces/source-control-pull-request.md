1. 리포지토리의 로컬 복사본에 대한 변경 내용을 커밋한 후 **Create Pull Request**(끌어오기 요청 만들기) 아이콘을 클릭합니다. 
![스테이징 단추가 강조 표시된 소스 제어 사이드바](/assets/images/help/codespaces/codespaces-commit-pr-button.png)  
1. 병합하려는 로컬 분기 및 리포지토리와 병합하려는 원격 분기 및 리포지토리가 올바른지 확인합니다. 그런 다음 끌어오기 요청에 제목과 설명을 입력합니다. 
![스테이징 단추가 강조 표시된 소스 제어 사이드바](/assets/images/help/codespaces/codespaces-commit-pr.png)
1. **만들기** 를 클릭합니다.
