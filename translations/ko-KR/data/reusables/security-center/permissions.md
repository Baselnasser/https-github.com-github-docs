조직 소유자 및 보안 관리자는 조직의 보안 개요에 액세스하고{% ifversion ghec or ghes > 3.4 or ghae-issue-6199 %} 엔터프라이즈 수준 보안 개요를 통해 조직의 리포지토리를 볼 수 있습니다. 엔터프라이즈 소유자는 엔터프라이즈 수준 보안 개요를 사용하여 본인이 속한 엔터프라이즈 조직의 모든 리포지토리를 볼 수 있습니다{% endif %}. 팀 구성원은 팀이 관리자 권한을 보유한 리포지토리의 보안 개요를 볼 수 있습니다.
