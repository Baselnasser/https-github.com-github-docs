**경고:** 원격 리포지토리에 절대 중요한 정보를 `git add`, `commit` 또는 `push`하지 마세요. 중요한 정보에는 다음이 포함될 수 있지만 이에 국한되지 않습니다.

- 암호
- SSH 키
- [AWS 액세스 키](http://docs.aws.amazon.com/AWSSimpleQueueService/latest/SQSGettingStartedGuide/AWSCredentials.html)
- API 키
- 신용 카드 번호
- PIN 번호

자세한 내용은 “[리포지토리에서 중요한 데이터 제거](/articles/removing-sensitive-data-from-a-repository)”를 참조하세요.

{% endwarning %}
