1. 파일 목록 위에서 {% octicon "download" aria-label="The download icon" %} **코드** 를 클릭합니다.
  ![“Code”(코드) 단추](/assets/images/help/repository/code-button.png)
1. 리포지토리의 URL을 복사합니다.

   - HTTPS를 사용하여 리포지토리를 복제하려면 "HTTPS"에서 {% octicon "clippy" aria-label="The clipboard icon" %}을 클릭합니다. 
   - 조직의 SSH 인증 기관에서 발급한 인증서를 포함하여 SSH 키를 사용하여 리포지토리를 복제하려면 **SSH** 를 클릭한 다음, {% octicon "clippy" aria-label="The clipboard icon" %}을 클릭합니다. 
   - {% data variables.product.prodname_cli %}를 사용하여 리포지토리를 복제하려면 **{% data variables.product.prodname_cli %}** 를 클릭한 다음, {% octicon "clippy" aria-label="The clipboard icon" %}을 클릭합니다.
  ![URL을 복사하여 GitHub CLI를 사용하여 리포지토리를 복제하기 위한 클립보드 아이콘](/assets/images/help/repository/https-url-clone-cli.png)
