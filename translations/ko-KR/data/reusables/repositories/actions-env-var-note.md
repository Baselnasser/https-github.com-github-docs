둘 이상의 환경 변수가 동일한 이름으로 정의되면 {% data variables.product.prodname_dotcom %}는 가장 구체적인 환경 변수를 사용합니다. 예를 들어 단계에 정의된 환경 변수는 단계가 실행되는 동안 동일한 이름의 작업 및 워크플로 변수를 재정의합니다. 작업에 대해 정의된 변수는 작업이 실행되는 동안 동일한 이름의 워크플로 변수를 재정의합니다.
