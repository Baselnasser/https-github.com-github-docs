특정 분기, 모든 분기 또는 `fnmatch` 구문으로 지정한 이름 패턴과 일치하는 모든 분기에 대한 분기 보호 규칙을 리포지토리에 만들 수 있습니다. 예를 들어 `release` 단어를 포함하는 모든 분기를 보호하려면 `*release*`에 대한 분기 규칙을 만들 수 있습니다.
