Markdown 파일을 볼 때 파일 위쪽의 {% octicon "code" aria-label="The code icon" %}을 클릭하여 Markdown 렌더링을 사용하지 않도록 설정하고 대신 파일의 원본을 볼 수 있습니다.

![Markdown을 원본으로 표시](/assets/images/help/writing/display-markdown-as-source.png)

Markdown 렌더링을 사용하지 않도록 설정하면 렌더링된 Markdown 파일을 볼 때 사용할 수 없는 줄 연결과 같은 원본 보기 기능을 사용할 수 있습니다.
