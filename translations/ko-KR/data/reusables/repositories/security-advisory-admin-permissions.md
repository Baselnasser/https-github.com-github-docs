리포지토리에 대한 관리자 권한이 있는 모든 사용자가 보안 권고를 만들 수 있습니다.

리포지토리에 대한 관리자 권한이 있는 모든 사용자에게는 해당 리포지토리의 모든 보안 권고에 대한 관리자 권한도 있습니다. 보안 권고에 대한 관리자 권한이 있는 사용자는 협력자를 추가할 수 있으며 협력자는 보안 권고에 대한 쓰기 권한을 갖습니다.
