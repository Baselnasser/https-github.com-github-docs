{% data reusables.projects.access-workflows %}
1. **기본 워크플로** 에서 편집할 워크플로를 클릭합니다.
   
   ![기본 워크플로를 보여 주는 스크린샷](/assets/images/help/projects-v2/default-workflows.png)

2. 워크플로를 이슈 및 끌어오기 요청 둘 다에 적용할 수 있는 경우 **일시** 옆에 있는 작업할 항목 유형을 선택합니다.
   
   ![워크플로에 대한 “시기” 구성을 보여 주는 스크린샷](/assets/images/help/projects-v2/workflow-when.png)

3. **설정** 옆에서 상태를 설정할 값을 선택합니다.
   
   ![워크플로에 대한 “설정” 구성을 보여 주는 스크린샷](/assets/images/help/projects-v2/workflow-set.png)

4. 워크플로를 사용하지 않도록 설정한 경우 **끄** 기 옆의 토글을 클릭하여 워크플로를 사용하도록 설정합니다.
   
   ![워크플로에 대한 "켜기/끄기" 컨트롤을 보여 주는 스크린샷](/assets/images/help/projects-v2/workflow-enable.png)
