1. 항목을 선택합니다. 여러 항목을 선택하려면 다음 중 하나를 수행합니다.
     - <kbd>Command</kbd>(Mac) 또는 <kbd>Ctrl</kbd>(Windows/Linux)을 누른 채 각 항목을 클릭합니다.
     - 항목을 선택한 다음 <kbd>Shift</kbd>+<kbd>↑</kbd> 또는 <kbd>Shift</kbd>+<kbd>↓</kbd>를 눌러 선택한 항목 위 또는 아래에 있는 추가 항목을 선택합니다.
     - 항목을 선택한 다음 <kbd>Shift</kbd>를 누르고 다른 항목을 클릭하여 두 항목 사이의 모든 항목을 선택합니다.
     - 행이나 항목이 이미 선택된 경우 <kbd>Command</kbd>+<kbd>A</kbd>(Mac) 또는 <kbd>Ctrl</kbd>+<kbd>A</kbd>(Windows/Linux)를 눌러 보드 레이아웃의 열에 있는 모든 항목 또는 테이블 레이아웃의 모든 항목을 선택합니다.
