{% data reusables.profile.access_profile %}
1. 프로파일에서 {% octicon "table" aria-label="The project icon" %} **프로젝트** 를 클릭합니다.
  ![‘프로젝트’ 탭을 보여 주는 스크린샷](/assets/images/help/projects-v2/tab-projects.png)
1. **새 프로젝트** 를 클릭합니다.
   ![새 프로젝트](/assets/images/help/projects-v2/new-project-button.png)
1. 템플릿을 선택하라는 메시지가 표시되면 템플릿을 클릭하고, 빈 프로젝트로 시작하려면 **테이블** 또는 **보드** 를 클릭합니다. 그런 다음 **만들기** 를 클릭합니다.

   ![템플릿 선택 모달을 보여 주는 스크린샷](/assets/images/help/issues/projects-select-template.png)
