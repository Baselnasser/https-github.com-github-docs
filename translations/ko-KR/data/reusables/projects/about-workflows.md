{% data variables.product.prodname_projects_v2 %}에는 특정 이벤트에 따라 항목의 **상태** 를 업데이트하는 데 사용할 수 있는 기본 제공 워크플로가 포함되어 있습니다. 예를 들어 항목이 프로젝트에 추가되면 자동으로 상태를 **할 일** 로 설정하거나 이슈가 닫히면 상태를 **완료** 로 설정할 수 있습니다.

프로젝트가 초기화되면 기본적으로 워크플로 두 개가 활성화됩니다. 프로젝트의 이슈 또는 끌어오기 요청이 닫혀 있으면 상태는 **완료** 로 설정되고 프로젝트의 끌어오기 요청이 병합되면 상태는 **완료** 로 설정됩니다.
