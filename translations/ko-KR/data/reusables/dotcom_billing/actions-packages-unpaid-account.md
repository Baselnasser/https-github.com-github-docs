계정에 처리 중인 미납 요금이 있는 경우:

* {% data variables.product.prodname_actions %} 및 {% data variables.product.prodname_registry %}에 대한 계정에 포함된 스토리지 또는 분은 결제가 성공적으로 처리될 때까지 재설정되지 않습니다.
* 현재 청구 기간 동안 스토리지 또는 분이 남은 계정의 경우 {% data variables.product.prodname_actions %} 및 {% data variables.product.prodname_registry %}는 포함된 사용량에 도달할 때까지 계속 사용할 수 있습니다.
* {% data variables.product.prodname_actions %} 또는 {% data variables.product.prodname_registry %}에 대한 현재 청구 기간 동안 포함된 사용량에 도달한 계정의 경우 추가 초과분을 방지하기 위해 {% data variables.product.prodname_actions %} 및 {% data variables.product.prodname_registry %}가 모두 사용하지 않도록 설정됩니다.
