{%- ifversion custom-pattern-dry-run-ga %}
1. 시험 실행을 수행할 리포지토리를 선택합니다.
   * 전체 조직에서 시험 실행을 수행하려면 **조직의 모든 리포지토리** 를 선택합니다.
   ![시험 실행에 대해 선택된 리포지토리를 보여 주는 스크린샷](/assets/images/help/repository/secret-scanning-dry-run-custom-pattern-all-repos.png)
   * 시험 실행을 수행할 리포지토리를 지정하려면 **선택한 리포지토리** 를 선택한 다음 최대 10개의 리포지토리를 검색하여 선택합니다.
   ![시험 실행에 대해 선택된 리포지토리를 보여 주는 스크린샷](/assets/images/help/repository/secret-scanning-dry-run-custom-pattern-select-repos-option.png)
1. 새 사용자 지정 패턴을 테스트할 준비가 되면 **실행** 을 클릭합니다.
{%- else %}
1. 시험 실행을 수행할 리포지토리를 최대 10개까지 검색하여 선택합니다.
   ![시험 실행에 대해 선택된 리포지토리를 보여 주는 스크린샷](/assets/images/help/repository/secret-scanning-dry-run-custom-pattern-select-repo.png)
1. 새 사용자 지정 패턴을 테스트할 준비가 되면 **시험 실행** 을 클릭합니다.
{%- endif %}
