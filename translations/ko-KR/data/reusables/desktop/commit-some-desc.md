**하나 이상의 파일에 있는 모든 변경 내용** 을 단일 커밋에 추가하려면 포함하지 않을 파일 옆에 있는 확인란의 선택을 취소하고 커밋에 원하는 파일만 남겨둡니다. 파일을 선택한 후 확인란을 `Spacebar` 또는 `Enter` 키로 전환할 수 있습니다.
