1. 커밋을 마우스 오른쪽 단추로 클릭합니다.

    - 커밋에 태그가 하나만 있는 경우 **태그 이름 삭제** 를 클릭합니다.
  ![태그 삭제 메뉴 항목 선택](/assets/images/help/desktop/select-delete-tag.png)
    - 커밋에 여러 태그가 있는 경우 **태그 삭제** 를 마우스 위로 마우스를 가져온 다음 삭제할 태그를 클릭합니다.
  ![태그 삭제 메뉴 위로 마우스를 가져옴](/assets/images/help/desktop/delete-tag-multiple.png)
