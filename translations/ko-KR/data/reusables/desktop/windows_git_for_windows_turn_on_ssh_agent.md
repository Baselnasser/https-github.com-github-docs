[Git for Windows](https://git-for-windows.github.io/) 등의 **다른 터미널 프롬프트를 사용하는 경우** ssh-agent를 켭니다.
```shell
# start the ssh-agent in the background
$ eval $(ssh-agent -s)
> Agent pid 59566
```
