1. 원래 업스트림 리포지토리에 기여하기 위해 이 포크를 사용하려는 경우 **부모 프로젝트에 기여하려면** 을 클릭합니다.

  {% mac %}

  ![부모 프로젝트 메뉴 옵션에 기여](/assets/images/help/desktop/mac-fork-options-prompt.png)

  {% endmac %}

  {% windows %}

  ![부모 프로젝트 메뉴 옵션에 기여](/assets/images/help/desktop/windows-fork-options-prompt.png)

  {% endwindows %}

2. 업스트림에 연결되지 않은 프로젝트에 이 포크를 사용하려는 경우 **내 용도** 를 클릭합니다.

  {% mac %}

  ![독립 프로젝트 메뉴 옵션에 기여](/assets/images/help/desktop/mac-fork-own-purposes.png)

  {% endmac %}

  {% windows %}

  ![독립 프로젝트 메뉴 옵션에 기여](/assets/images/help/desktop/windows-fork-own-purposes.png)

  {% endwindows %}

3. **계속** 을 클릭합니다.

  {% mac %}

  ![계속 단추](/assets/images/help/desktop/mac-forking-continue.png)

  {% endmac %}

  {% windows %}

  ![계속 단추](/assets/images/help/desktop/windows-forking-continue.png)

  {% endwindows %}
