IP 허용 목록을 만든 후 허용된 IP 주소를 사용하도록 설정할 수 있습니다. 허용된 IP 주소를 사용하도록 설정하면 {% data variables.product.company_short %}가 IP 허용 목록에 활성화된 항목을 즉시 적용합니다.

IP 허용 목록을 사용하도록 설정하기 전에 허용 목록에서 특정 IP 주소의 연결을 허용할지 여부를 확인할 수 있습니다. 자세한 내용은 “[IP 주소가 허용되는지 확인](#checking-if-an-ip-address-is-permitted)”을 참조하세요.
