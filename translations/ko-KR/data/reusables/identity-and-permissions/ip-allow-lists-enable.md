IP 허용 목록을 적용하려면 먼저 목록에 IP 주소를 추가한 다음 IP 허용 목록을 사용하도록 설정해야 합니다. 목록을 완료한 후 목록의 활성화된 항목에서 특정 IP 주소를 허용할지 여부를 확인할 수 있습니다.

IP 허용 목록을 사용하도록 설정하기 전에 현재 IP 주소 또는 일치 범위를 추가해야 합니다.
