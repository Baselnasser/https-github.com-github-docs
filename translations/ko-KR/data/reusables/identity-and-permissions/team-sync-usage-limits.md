### 사용 제한

팀 동기화 기능에는 사용 제한이 있습니다.  제한을 초과하면 성능이 저하되고 동기화 실패가 발생할 수 있습니다.

- GitHub 팀의 최대 멤버 수: 5,000명
- GitHub 조직의 최대 멤버 수: 10,000명
- GitHub 조직의 최대 팀 수: 1,500개
