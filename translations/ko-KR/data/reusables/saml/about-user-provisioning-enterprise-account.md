Okta를 IdP로 사용하고 엔터프라이즈 계정의 프라이빗 베타에 참여하는 경우 SCIM을 사용하여 엔터프라이즈 계정 조직의 멤버 자격을 관리할 수 있습니다. SCIM은 사용자가 IdP의 각 조직에 해당하는 그룹의 구성원인지 여부에 따라 엔터프라이즈 계정의 조직에서 사용자를 자동으로 초대하거나 제거합니다.
