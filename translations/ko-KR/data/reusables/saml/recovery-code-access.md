1. Single Sign-On 대화 아래에서 **복구 코드 사용** 을 클릭하여 Single Sign-On을 우회합니다.

   ![복구 코드를 입력할 링크의 스크린샷](/assets/images/help/saml/saml_use_recovery_code.png)

2. “복구 코드” 필드에 복구 코드를 입력합니다.

   ![복구 코드를 입력할 필드의 스크린샷](/assets/images/help/saml/saml_recovery_code_entry.png)

3. **확인** 을 클릭합니다.

   ![복구 코드를 확인하기 위한 단추의 스크린샷](/assets/images/help/saml/saml_verify_recovery_codes.png)

복구 코드를 사용한 후에는 코드가 더 이상 유효하지 않음을 기억하세요. 복구 코드는 다시 사용할 수 없습니다.
