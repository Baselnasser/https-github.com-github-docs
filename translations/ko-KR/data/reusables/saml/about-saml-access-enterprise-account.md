{% data variables.product.product_name %}에서 각 조직의 리소스에 액세스하려면 구성원이 브라우저에 활성 SAML 세션이 있어야 합니다. API 및 Git을 사용하여 각 조직의 보호된 리소스에 액세스하려면 구성원이 조직과 함께 사용하도록 권한이 부여된 {% 데이터 variables.product.pat_generic %} 또는 SSH 키를 사용해야 합니다. 엔터프라이즈 소유자는 언제든지 멤버의 연결된 ID, 활성 세션 또는 권한 있는 자격 증명을 보고 해지할 수 있습니다.
