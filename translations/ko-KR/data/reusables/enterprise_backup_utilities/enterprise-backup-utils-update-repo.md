1. 최신 프로젝트 릴리스 버전으로 업데이트하려면 `git checkout stable` 명령을 실행하여 `stable` 분기를 사용합니다.

  ```shell
  git checkout stable
  ```
  
  또는 특정 프로젝트 버전을 사용하려면 다음 명령을 실행하여 `X.Y.Z`를 원하는 릴리스 버전으로 대체합니다.

  ```shell
  $ git checkout vX.Y.Z
  ```
