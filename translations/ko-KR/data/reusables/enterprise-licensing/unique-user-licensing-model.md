{% data variables.product.company_short %}는 고유한 사용자 라이선스 모델을 사용합니다. 여러 배포 옵션을 포함하는 엔터프라이즈 제품의 경우 {% data variables.product.company_short %}는 모든 배포의 고유한 사용자 수에 따라 사용이 허가된 사용자 수를 결정합니다.

각 사용자는 사용자가 사용하는 {% data variables.product.prodname_ghe_server %} 인스턴스 수 또는 {% data variables.product.prodname_ghe_cloud %}에서 사용자가 속하는 조직 수와 관계없이 라이선스를 하나만 사용합니다. 이 모델을 사용하면 각 사용자가 추가 비용 없이 여러 {% data variables.product.prodname_enterprise %} 배포를 사용할 수 있습니다.
