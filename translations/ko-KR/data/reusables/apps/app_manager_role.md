기본적으로 조직의 소유자만 조직에서 GitHub 앱의 설정을 관리할 수 있습니다. 추가 사용자가 조직에서 GitHub 앱을 관리할 수 있도록 소유자가 GitHub 앱 관리자 권한을 부여할 수 있습니다. 조직에서 GitHub 앱 관리자를 추가하고 제거하는 방법을 알아보려면 "[GitHub 앱 관리자](/articles/permission-levels-for-an-organization/#github-app-managers)"를 참조하세요.
