1. {% 데이터 variables.location.product_location %}에서 조직의 기본 페이지로 이동합니다.
1. 조직 이름에서 {% octicon "gear" aria-label="The gear icon" %} **설정** 을 클릭합니다.
![조직 설정 단추](/assets/images/help/discussions/org-settings.png)
1. “토론”에서 **이 조직에 대한 토론 사용** 을 선택합니다.
1. 조직 토론의 원본 리포지토리로 사용할 리포지토리를 선택합니다.
  ![조직에 대한 토론을 사용하도록 설정하는 설정](/assets/images/help/discussions/enable-org-discussions.png)
1. **저장** 을 클릭합니다.
