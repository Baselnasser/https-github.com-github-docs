토론이 답변을 허용하는 범주 내에 있는 경우 토론의 댓글을 토론에 대한 답변으로 표시할 수 있습니다. 자세한 내용은 “[토론 정보](/discussions/collaborating-with-your-community-using-discussions/about-discussions#about-categories-and-formats-for-discussions)”를 참조하세요.

질문을 답변으로 표시하면 {% data variables.product.product_name %}에서 댓글을 강조 표시하고 댓글에 회신하여 방문자가 신속하게 답변을 찾을 수 있도록 합니다.

![토론에 대한 답변으로 표시된 의견](/assets/images/help/discussions/comment-marked-as-answer.png)

토론에 대한 응답으로 스레드형 댓글을 토론에 대한 답변으로 표시할 수 없습니다. 최소화된 댓글을 토론에 대한 답변으로 표시할 수도 없습니다.

{% data reusables.discussions.navigate-to-repo-or-org %} {% data reusables.discussions.discussions-tab %}
1. 토론 목록에서 답변으로 표시할 미응답 토론을 클릭합니다.
  ![미응답 토론](/assets/images/help/discussions/unanswered-discussion.png)
1. 토론에서 답변으로 표시할 댓글을 찾습니다.
1. 댓글의 오른쪽 위 모서리에서 {% octicon "check-circle" aria-label="The circular check icon" %} 아이콘을 클릭합니다.
  ![토론에서 댓글을 답변으로 표시할 수 있는 “Mark as answer”(답변으로 표시) 원형 확인 아이콘](/assets/images/help/discussions/comment-mark-as-answer-button.png)
1. 필요에 따라 답변으로 표시한 댓글을 해제하려면 {% octicon "check-circle-fill" aria-label="The filled circular check icon" %} 아이콘을 클릭합니다.
