5. 앱 설정 방문 페이지에서 Marketplace 섹션까지 아래로 스크롤하고 **Marketplace의 목록** 을 클릭합니다. Marketplace 초안 목록이 이미 있는 경우 **Marketplace 목록 편집** 을 클릭합니다. Marketplace 섹션은 앱을 등록할 때 사용자 또는 조직에서 앱을 설치하도록 허용한 경우에만 표시됩니다.  자세한 내용은 [Marketplace 요구 사항](/apps/marketplace/creating-and-submitting-your-app-for-approval/requirements-for-listing-an-app-on-github-marketplace/) 목록을 참조하세요.
