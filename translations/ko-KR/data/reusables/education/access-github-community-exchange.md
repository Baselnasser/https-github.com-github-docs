{% data variables.product.prodname_community_exchange %}에 액세스하려면 https://education.github.com 의 {% data variables.product.prodname_global_campus %} 대시보드를 방문하세요.

공인 교육 기관의 학생 또는 교직원인 경우 {% data variables.product.prodname_education %} 혜택을 신청할 수 있습니다. 여기에는 {% data variables.product.prodname_global_campus %}에서 {% data variables.product.prodname_community_exchange %}에 액세스할 수 있는 권한이 포함됩니다.

- 학생이고 {% data variables.product.prodname_education %}에 아직 가입하지 않은 경우 [학생 신청서]( https://education.github.com/discount_requests/student_application)를 사용하여 신청합니다. 자세한 내용은 “[{% data variables.product.prodname_global_campus %} 정보](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/about-github-global-campus-for-students)”를 참조하세요.

- 교사이고 {% data variables.product.prodname_education %}에 아직 가입하지 않은 경우 [교사 신청서]( https://education.github.com/discount_requests/teacher_application)를 사용하여 신청합니다. 자세한 내용은 “[교사로 {% data variables.product.prodname_global_campus %}에 신청](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-teachers/apply-to-github-global-campus-as-a-teacher)”을 참조하세요.
