엄격한 보안 및 규정 준수 요구 사항을 충족하도록 Codespaces를 만들 지역을 수동으로 선택할 수 있습니다. 기본적으로 지역은 사용자의 위치에 따라 자동으로 설정됩니다.

## <a name="setting-your-default-region"></a>기본 지역 설정

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.codespaces-tab %}
1. “지역”에서 원하는 설정을 선택합니다.
2. “수동으로 설정”을 선택한 경우 드롭다운 목록에서 지역을 선택합니다.
   ![지역 선택](/assets/images/help/codespaces/select-default-region.png)
