## {% data variables.product.prodname_github_codespaces %}에 대한 보안 로그 정보

개인 계정이 소유한 리포지토리에서 {% data variables.product.prodname_github_codespaces %}와 관련된 작업을 수행하는 경우 보안 로그에서 작업을 검토할 수 있습니다. 로그에 액세스하는 방법에 대한 자세한 내용은 “[보안 로그 검토](/github/authenticating-to-github/reviewing-your-security-log#accessing-your-security-log)”를 참조하세요.

![Codespaces 정보가 포함된 보안 로그](/assets/images/help/settings/codespaces-audit-log.png)

보안 로그에는 발생한 작업 및 수행 시기에 대한 세부 정보가 포함됩니다. {% data variables.product.prodname_github_codespaces %} 작업에 대한 자세한 내용은 "[{% data variables.product.prodname_codespaces %} 범주 작업](/github/authenticating-to-github/reviewing-your-security-log#codespaces-category-actions)"을 참조하세요.
