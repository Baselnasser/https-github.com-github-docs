## 변형 정보

모든 GraphQL 스키마에는 쿼리 및 변형 모두에 대한 루트 형식이 있습니다. [변형 형식](https://graphql.github.io/graphql-spec/June2018/#sec-Type-System)은 서버의 데이터를 변경하는 GraphQL 작업을 정의합니다. HTTP 동사(예: `POST`, `PATCH`, `DELETE`)를 수행하는 것과 유사합니다.

자세한 내용은 “[변형 정보](/graphql/guides/forming-calls-with-graphql#about-mutations)”를 참조하세요.
