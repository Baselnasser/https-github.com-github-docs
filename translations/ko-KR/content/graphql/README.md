# GraphQL

`/content/graphql` 디렉터리에는 GitHub GraphQL API 문서가 있습니다.

* `/content/graphql/guides` 및 `/content/graphql/overview` 디렉터리에는 사람이 편집할 수 있는 문서가 포함되어 있습니다.
* `/content/graphql/reference` 디렉터리에는 GitHub GraphQL API에서 사용되는 각 GraphQL 데이터 형식에 대한 문서가 포함되어 있습니다. 이 디렉터리에 있는 콘텐츠의 대부분은 `include` 태그를 사용하여 렌더링됩니다.

  `include` 태그로 렌더링되는 콘텐츠는 `/lib/graphql/static` 디렉터리에서 오는 것으로, GitHub 내부의 API 소스 코드에서 자동으로 생성되며 사람이 편집해서는 안 됩니다. 자세한 내용은 [`/lib/graphql/README.md`](/lib/graphql/README.md)를 참조하세요.

  **따라서 이 리포지토리의 GraphQL API 참조 콘텐츠에 대한 기여를 허용할 수 없습니다.**
