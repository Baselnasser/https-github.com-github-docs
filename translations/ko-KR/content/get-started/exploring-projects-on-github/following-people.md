## {% data variables.product.product_name %}의 팔로워 정보

개인을 팔로우하면 개인 대시보드에 그 사람의 퍼블릭 활동이 표시됩니다.{% ifversion fpt or ghec %} 팔로우하는 사람이 퍼블릭 리포지토리에 별을 지정하는 경우 {% data variables.product.product_name %}에서 해당 리포지토리를 추천할 수 있습니다.{% endif %} 자세한 내용은 “[개인 대시보드 정보](/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-personal-account-settings/about-your-personal-dashboard#staying-updated-with-activity-from-the-community)”를 참조하세요.

{% data variables.product.product_name %}에서 퍼블릭 활동을 보지 않으려면 개인 팔로우를 취소할 수 있습니다.

## {% data variables.product.product_name %}에서 사용자 팔로우

1. 사용자 이미지를 클릭하여 해당 사용자의 프로필을 표시합니다.

  ![사용자 이미지](/assets/images/help/profile/user-profile-image.png)

2. 프로필 이미지 아래에 있는 **팔로우** 를 클릭합니다.

  ![사용자 팔로우 단추](/assets/images/help/profile/follow-user-button.png)

## {% data variables.product.product_name %}에서 사용자 팔로우 취소

1.  사용자 이미지를 클릭하여 해당 사용자의 프로필을 표시합니다.

  ![사용자 이미지](/assets/images/help/profile/user-profile-image.png)

2.  프로필 이미지 아래에 있는 **팔로우 취소** 를 클릭합니다.

  ![사용자 프로필 단추](/assets/images/help/profile/user-profile-unfollow-button.png)

## {% data variables.product.product_name %}에서 팔로우한 사용자 보기

1. 사용자 이미지를 클릭하여 해당 사용자의 프로필을 표시합니다.

  ![사용자 이미지](/assets/images/help/profile/user-profile-image.png)

2. 프로필 이미지 아래에 있는 **팔로우** 를 클릭합니다.

  ![사용자 팔로우](/assets/images/help/profile/user-profile-following.png)

## {% data variables.product.product_name %}에서 팔로워 보기

1. 사용자 이미지를 클릭하여 해당 사용자의 프로필을 표시합니다.

  ![사용자 이미지](/assets/images/help/profile/user-profile-image.png)

2. 프로필 이미지 아래에 있는 **팔로워** 를 클릭합니다.

  ![사용자 팔로우](/assets/images/help/profile/user-profile-followers.png)
