## {% data variables.product.product_name %}의 팔로워 정보

{% data reusables.organizations.about-following-organizations %} 자세한 내용은 “[개인 대시보드 정보](/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-personal-account-settings/about-your-personal-dashboard#staying-updated-with-activity-from-the-community)”.

{% data variables.product.product_name %}에서 {% ifversion fpt or ghec %}퍼블릭{% endif %} 활동을 보지 않으려면 조직 팔로우를 취소할 수 있습니다.

## {% data variables.product.product_name %}에서 조직 팔로우

1. 팔로우하려는 조직 페이지로 이동합니다.
2. 머리글 오른쪽 위에 있는 **팔로우** 를 클릭합니다.

  ![팔로우 단추가 강조 표시된 조직 머리글 스크린샷](/assets/images/help/profile/organization-profile-following.png)

## {% data variables.product.product_name %}에서 조직 팔로우 취소

1. 팔로우를 취소하려는 조직 페이지로 이동합니다.
2. 머리글 오른쪽 위에 있는 **팔로우 취소** 를 클릭합니다.

  ![팔로우 취소 단추가 강조 표시된 조직 머리글 스크린샷](/assets/images/help/profile/organization-profile-unfollowing.png)

## 사용자가 {% data variables.product.product_name %}에서 팔로우하는 개인 및 조직 보기

1. 사용자 이미지를 클릭하여 해당 사용자의 프로필을 표시합니다.

  ![사용자 이미지](/assets/images/help/profile/user-profile-image.png)

2. 프로필 이미지 아래에 있는 **팔로우** 를 클릭합니다.

  ![사용자 팔로우](/assets/images/help/profile/user-profile-following.png)
