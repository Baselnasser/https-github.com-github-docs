## GitHub에서 4월 14일에 발표한 플랜 및 가격 변경 내용은 무엇인가요?

GitHub는 다음과 같은 플랜 및 가격 변경 내용을 발표했습니다.

- 조직용 GitHub Free는 즉시 사용할 수 있으며 무제한 사용자를 위한 프라이빗 리포지토리를 포함합니다.
- 이전에 Team for Open Source를 사용하던 모든 조직에 이제 GitHub Free가 제공됩니다.
- 개별 개발자용 GitHub Free에 이제 무제한 협력자가 포함됩니다.
- GitHub Free를 사용하는 조직 및 개인은 GitHub Community 지원을 받게 됩니다.
- 이제 GitHub Pro에 2GB 패키지 스토리지와 10GB 데이터 전송이 포함됩니다.
- 이제 GitHub Pro의 월별 요금이 $4 인하되었습니다.
- 이제 GitHub Team의 월별 요금이 사용자당 $4 인하되었습니다.
- GitHub Team에 더 이상 최소 5명 사용자 요구 사항이 없습니다.
- GitHub Team에는 프라이빗 리포지토리에 대한 작업 시간(분)으로 월별 3,000분이 포함됩니다.

## 플랜을 업그레이드하려면 어떻게 해야 하나요?

개인 계정의 플랜을 업그레이드하려면 “[개인 계정의 구독 업그레이드](/billing/managing-billing-for-your-github-account/upgrading-your-github-subscription#upgrading-your-personal-accounts-subscription)”를 참조하세요. 조직의 플랜을 업그레이드하려면 “[조직의 구독 업그레이드](/billing/managing-billing-for-your-github-account/upgrading-your-github-subscription#upgrading-your-organizations-subscription)”를 참조하세요.

## 플랜을 다운그레이드하려면 어떻게 해야 하나요?

개인 계정의 플랜을 다운그레이드하려면 “[개인 계정의 구독 다운그레이드](/billing/managing-billing-for-your-github-account/downgrading-your-github-subscription#downgrading-your-personal-accounts-subscription)”를 참조하세요. 조직의 플랜을 다운그레이드하려면 “[조직의 구독 다운그레이드](/billing/managing-billing-for-your-github-account/downgrading-your-github-subscription#downgrading-your-organizations-subscription)”를 참조하세요.

## GitHub Team 또는 레거시 플랜에서 GitHub Free로 다운그레이드하는 경우 어떤 기능이 손실되나요?
- 프라이빗 리포지토리의 필수 검토자
- 프라이빗 리포지토리의 보호된 분기
- 프라이빗 리포지토리의 코드 소유자
- 프라이빗 리포지토리의 GitHub 페이지
- 프라이빗 리포지토리의 Wiki
- 프라이빗 리포지토리의 이슈 및 끌어오기 요청에 대한 여러 담당자
- 프라이빗 리포지토리의 여러 끌어오기 요청 검토자
- 프라이빗 리포지토리의 초안 끌어오기 요청
- 프라이빗 리포지토리의 자동 연결된 참조
- 표준 지원

## GitHub Free 또는 GitHub Team 조직을 만들려면 어떻게 해야 하나요?

GitHub 계정이 있는 경우 [팀 플랜 페이지](https://github.com/organizations/plan)에서 새 조직을 만듭니다. GitHub 계정이 없는 경우 [홈페이지에서 계정을 만들](https://github.com/) 수 있으며 등록 프로세스 중에 조직을 만들라는 메시지가 표시됩니다.

## GitHub Free가 무제한 협력자를 허용하는 기간은 어떻게 되나요?

계속 허용됩니다. 이 변경 내용은 영구적입니다.

## 이미 연간 Pro 또는 Team 플랜 요금을 지불했는데 차액을 환불 받을 수 있나요?

현재 Pro 및 Team 플랜 요금을 매년 지불하고 있으며 4월 14일 이전에 등록한 고객은 연간 청구 기간이 끝날 때 비례 배분 금액을 환불 받게 됩니다. 예를 들어 매년 청구되는 구독을 2개월 동안 사용한 경우 나머지 10개월의 차액을 환불 받게 됩니다. 환불은 약 30일 후에 처리됩니다. 환불이 처리되면 청구 메일로 알림이 전송됩니다.

## 개별 개발자용 GitHub Free와 조직용 GitHub Free 간 차이점은 무엇인가요?

각 GitHub Free 제품의 차이점에 대한 자세한 내용은 “[{% data variables.product.prodname_dotcom %} 제품](/articles/github-s-products)”을 참조하세요.

## GitHub Team 플랜에서 할당되는 것보다 더 많은 작업 시간(분)이 필요한 경우 어떻게 얻을 수 있나요?

대부분의 소규모 팀은 매월 3,000분 미만의 작업 시간(분)을 사용하지만 일부 팀은 더 많은 작업 시간을 사용할 수도 있습니다. 팀에 프라이빗 리포지토리에 대한 추가 시간(분)이 필요한 경우 GitHub Actions에 대한 지출 한도를 늘릴 수 있습니다. 퍼블릭 리포지토리에 대한 작업 시간(분) 및 패키지 스토리지는 여전히 무료입니다. 자세한 내용은 “[GitHub Actions에 대한 지출 한도 관리](/billing/managing-billing-for-github-actions/managing-your-spending-limit-for-github-actions)”를 참조하세요.

## 사용자 한도 변경 내용은 내게 어떤 영향을 주나요?

이제 {% data variables.product.prodname_team %}에 최소 사용자 요구 사항이 없으므로 조직에서 5명 미만의 사용자를 사용하는 경우 유료 사용자를 제거하여 요금을 줄일 수 있습니다. 자세한 내용은 “[{% data variables.product.prodname_dotcom %} 구독 다운그레이드](/billing/managing-billing-for-your-github-account/downgrading-your-github-subscription#removing-paid-seats-from-your-organization)”를 참조하세요.

## GitHub Team for Open Source 플랜을 사용하는 팀에 대한 변경 내용이 있나요?

조직 계정이 현재 GitHub Team for Open Source 플랜을 사용하는 경우 자동으로 GitHub Free로 이동되었습니다. 여기에는 무제한 협력자가 있는 프라이빗 리포지토리, 프라이빗 리포지토리에 대한 2,000분 작업 시간, GitHub Community 지원이 포함됩니다. 퍼블릭 리포지토리에 대한 작업 시간(분) 및 패키지 스토리지는 여전히 무료입니다.

## GitHub Community 지원이란 무엇인가요?

GitHub Community 지원에는 GitHub 커뮤니티에서 솔루션을 찾고, 새로운 질문을 하고, 아이디어를 공유할 수 있는 [{% data variables.product.prodname_github_community %} 토론](https://github.com/orgs/community/discussions)을 통한 지원이 포함됩니다. GitHub Community 지원은 가장 활동적인 커뮤니티 멤버와 함께 {% data variables.product.prodname_github_community %}에 대해 조정하는 GitHub Team의 지원 엔지니어가 담당합니다. 스팸을 보고하거나, 남용을 보고하거나, 계정 액세스 관련 이슈가 있는 경우 https://support.github.com/ 에서 지원 팀에 메시지를 보낼 수 있습니다.

## 이 변경 내용은 교육 혜택에 어떤 영향을 주나요?

학생과 교사는 현재 제공되는 모든 혜택을 계속 받을 수 있습니다. GitHub는 [학생 개발자 팩](https://education.github.com/pack) 혜택의 일부로 모든 학생에게 GitHub Pro를 계속 제공할 것입니다. 또한 교사는 [교사 도구 상자](https://education.github.com/toolbox)의 일부로 GitHub Team에 계속 액세스할 수 있습니다.

## 현재 청구서로 결제하는 경우 이 변경 내용은 내게 어떤 영향을 주나요?

현재 청구서 고객인 경우 추가 정보가 필요하면 영업 담당자에게 메일을 보내세요. 영업 담당자가 누구인지 잘 모르겠으면 청구 관리자 또는 계정 소유자에게 문의하세요.

## 이 변경 내용은 미국 정부의 제재가 적용되는 지역의 사용자와 조직에 어떤 영향을 주나요?

미국의 무역 통제법 제한으로 인해 GitHub는 미국 제재 지역의 계정에 프라이빗 리포지토리 서비스 및 유료 서비스를 제공할 수 없습니다. 그러나 무료 퍼블릭 리포지토리 서비스에 대한 액세스는 계속 제공됩니다. 자세한 내용은 [GitHub 및 무역 통제](/free-pro-team@latest/github/site-policy/github-and-trade-controls)를 참조하세요.

GitHub는 제재 지역의 개발자에게 무료 프라이빗 리포지토리를 비롯한 무료 코드 협업 서비스에 대한 가능한 최대 액세스를 제공하기 위해 미국 규제 당국과 함께 계속 노력할 것입니다. GitHub는 무료 서비스 제공이 해당 지역에서 자유로운 정보 전달과 언론의 자유를 장려하는 미국의 외교 정책을 지원한다고 믿습니다.
