## {% data variables.product.prodname_dotcom_the_website %}의 새 계정 정보

{% data variables.product.prodname_dotcom_the_website %}에서 ID 역할을 하는 개인 계정을 만들거나 여러 개인 계정이 여러 프로젝트에서 협업할 수 있도록 하는 조직을 만들 수 있습니다. 계정 유형에 대한 자세한 내용은 “[{% data variables.product.prodname_dotcom %} 계정 유형](/get-started/learning-about-github/types-of-github-accounts)”을 참조하세요.

개인 계정 또는 조직을 만들 때는 계정의 청구 플랜을 선택해야 합니다. 자세한 내용은 “[{% data variables.product.company_short %}의 제품](/get-started/learning-about-github/githubs-products)”을 참조하세요.

## 새 계정에 가입

1. 새 개인 계정을 만들려면 현재 GitHub에서 로그아웃되어 있어야 합니다.
{% data reusables.accounts.create-account %}
1. 프롬프트에 따라 개인 계정 또는 조직을 만듭니다.
