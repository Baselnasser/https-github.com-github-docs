## Gist 포크하기

각 gist는 활동이 있는 포크를 나타내므로 다른 포크의 흥미로운 변경 내용을 쉽게 찾을 수 있습니다.

![Gist 포크](/assets/images/help/gist/gist_forks.png)

## Gist 복제

로컬에서 gist를 변경하고 웹에 푸시하려는 경우 gist를 복제하고 Git 리포지토리와 동일하게 커밋할 수 있습니다. 자세한 내용은 “[리포지토리 복제](/articles/cloning-a-repository)”를 참조하세요.

![Gist 복제 단추](/assets/images/help/gist/gist_clone_btn.png)

## Gist 커밋 기록 보기

Gist의 전체 커밋 기록을 보려면 gist의 상단에 있는 “수정 내용” 탭을 클릭합니다.

![Gist 수정 내용 탭](/assets/images/help/gist/gist_revisions_tab.png)

차이(diff)와 함께 gist의 전체 커밋 기록이 표시됩니다.

![Gist 수정 내용 페이지](/assets/images/help/gist/gist_history.png)
