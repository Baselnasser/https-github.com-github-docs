## 끌어오기 요청을 이슈에 연결

끌어오기 요청을 이슈에 연결하여 수정이 진행 중임을 표시하고 누군가가 끌어오기 요청을 병합할 때 이슈를 자동으로 닫으려면, 다음 키워드 중 하나를 입력하고 그 뒤에 이슈에 대한 참조를 입력합니다. 예를 들어 `Closes #10` 또는 `Fixes octo-org/octo-repo#100`입니다.

* 닫기
* closes
* closed
* 수정
* fixes
* 고정
* resolve
* resolves
* resolved

자세한 내용은 “[이슈에 끌어오기 요청 연결](/github/managing-your-work-on-github/linking-a-pull-request-to-an-issue)”을 참조하세요.

## 이슈 또는 끌어오기 요청을 중복으로 표시

이슈 또는 끌어오기 요청을 중복으로 표시하려면 “Duplicate of”를 입력한 다음 새 주석 본문에 중복되는 이슈 또는 끌어오기 요청 번호를 입력합니다. 자세한 내용은 “[이슈 또는 끌어오기 요청을 중복으로 표시](/issues/tracking-your-work-with-issues/marking-issues-or-pull-requests-as-a-duplicate)”를 참조하세요.
