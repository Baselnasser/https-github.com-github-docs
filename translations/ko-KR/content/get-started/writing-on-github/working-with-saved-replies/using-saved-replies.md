4. 목록에서 주석에 추가할 저장된 회신을 선택합니다.
   ![저장된 회신](/assets/images/help/settings/saved-replies.png)

{% tip %}

**팁:**
- 바로 가기 키를 사용하여 저장된 회신으로 주석을 자동으로 채울 수 있습니다. 자세한 내용은 “[바로 가기 키](/articles/keyboard-shortcuts/#comments)”를 참조하세요.
- 저장된 회신의 제목을 입력하여 목록을 필터링할 수 있습니다.

{% endtip %}
