{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.saved_replies %}
3. “저장된 회신 추가”에서 저장된 회신 제목을 추가합니다.
![저장된 회신 제목](/assets/images/help/settings/saved-replies-title.png)
4. “쓰기” 필드에 저장된 회신에 사용할 콘텐츠를 추가합니다. {% data variables.product.product_name %}에서 쓰기에 대한 자세한 내용은 “[기본 쓰기 및 서식 지정 구문](/articles/basic-writing-and-formatting-syntax)”을 참조하세요.
![저장된 회신 쓰기](/assets/images/help/settings/saved-replies-settings-adding.png)
5. 회신을 검토하려면 **미리 보기** 를 클릭합니다.
![저장된 회신 추가](/assets/images/help/settings/saved-replies-preview.png)
6. **저장된 회신 추가** 를 클릭합니다.
![“저장된 회신 추가” 단추](/assets/images/help/settings/saved-replies-add-button.png)
