3. “저장된 회신”에서 편집하려는 저장된 회신 옆에 있는 {% octicon "pencil" aria-label="The pencil" %}을 클릭합니다.  
![저장된 회신 편집](/assets/images/help/settings/saved-replies-edit-existing.png)
4. “저장된 회신 편집”에서 저장된 회신의 제목 및 콘텐츠를 편집할 수 있습니다.
![제목 및 콘텐츠 편집](/assets/images/help/settings/saved-replies-edit-existing-content.png)
5. **저장된 회신 업데이트** 를 클릭합니다.
![저장된 회신 업데이트](/assets/images/help/settings/saved-replies-save-edit.png)
