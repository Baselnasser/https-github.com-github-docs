[{% data variables.product.prodname_marketplace %}](https://github.com/marketplace)에서 {% data variables.product.prodname_github_apps %}, {% data variables.product.prodname_oauth_apps %}, {% data variables.product.prodname_actions %}를 비롯한 무료 및 유료 도구를 검색, 탐색 및 설치할 수 있습니다.

유료 도구를 구매하는 경우 {% data variables.product.product_name %} 구독에 대한 요금을 지불하는 데 사용하는 것과 동일한 청구 정보를 사용하여 도구 구독에 대한 요금을 지불하고 일반 청구 날짜에 하나의 청구서를 받게 됩니다. 자세한 내용은 “[{% data variables.product.prodname_marketplace %}에 대한 청구 정보](/articles/about-billing-for-github-marketplace)”를 참조하세요.

일부 도구에는 무료 14일 평가판을 선택할 수 있는 옵션이 있을 수도 있습니다. 평가판 중에 언제든지 취소할 수 있으며 요금이 청구되지 않지만 도구에 대한 액세스 권한이 자동으로 손실됩니다. 유료 구독은 14일 평가판이 종료되면 시작됩니다. 자세한 내용은 “[{% data variables.product.prodname_marketplace %}에 대한 청구 정보](/articles/about-billing-for-github-marketplace)”를 참조하세요.

## {% data variables.product.prodname_marketplace %}에서 도구 찾기

{% data variables.product.prodname_marketplace %}에서 다른 사용자가 만든 앱 및 작업을 검색, 탐색 및 설치할 수 있습니다. “[{% data variables.product.prodname_marketplace %} 검색](/search-github/searching-on-github/searching-github-marketplace)”을 참조하세요.

{% data reusables.actions.actions-not-verified %}

누구나 {% data variables.product.prodname_marketplace %}에 무료 {% data variables.product.prodname_github_app %} 또는 {% data variables.product.prodname_oauth_app %}을 나열할 수 있습니다. 유료 앱 게시자는 {% data variables.product.company_short %}에 의해 확인되며 앱의 목록은 Marketplace 배지 {% octicon "verified" aria-label="Verified creator badge" %}와 함께 표시됩니다. 확인되지 않은 앱 및 확인된 앱에 대한 배지도 표시됩니다. 해당 앱은 개별 앱을 확인하기 위해 이전 방법을 사용하여 게시되었습니다. 현재 프로세스에 대한 자세한 내용은 “[GitHub Marketplace 정보](/developers/github-marketplace/about-github-marketplace)” 및 “[앱을 나열하기 위한 요구 사항](/developers/github-marketplace/requirements-for-listing-an-app)”을 참조하세요.

## {% data variables.product.prodname_marketplace %}에서 도구 빌드 및 나열

{% data variables.product.prodname_marketplace %}에 나열할 사용자 고유의 도구를 만드는 방법에 대한 자세한 내용은 “[앱](/developers/apps)” 및 “[{% data variables.product.prodname_actions %}](/actions)”를 참조하세요.
