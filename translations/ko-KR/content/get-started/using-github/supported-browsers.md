## {% data variables.product.product_name %}에 대한 웹 브라우저 지원 정보

{% data variables.product.product_name %}는 최신 웹 브라우저를 염두에 두고 디자인됩니다. 다음 브라우저 중 하나의 최신 버전을 사용하는 것이 좋습니다.

 - [Apple Safari](https://apple.com/safari)
 - [Google Chrome](https://google.com/chrome)
 - [Microsoft Edge](https://microsoft.com/windows/microsoft-edge)
 - [Mozilla Firefox](https://mozilla.org/firefox)
 
권장 브라우저의 최신 버전을 사용하지 않거나 위에 나열되지 않은 브라우저를 사용하는 경우 {% data variables.product.product_name %} 또는 일부 기능이 예상대로 작동하지 않거나 전혀 작동하지 않을 수 있습니다.

당사에서 {% data variables.product.company_short %} 제품의 브라우저 호환성을 유지 관리하는 방법에 대한 자세한 내용은 [`github/browser-support`](https://github.com/github/browser-support) 리포지토리를 참조하세요.
## 권장 웹 브라우저에 대한 추가 지원

일부 브라우저 공급업체는 추가 지원 릴리스를 제공합니다. 당사는 {% data variables.product.product_name %}가 최신 추가 지원 릴리스에서 제대로 작동하도록 보장하기 위해 최선을 다합니다.

- Chrome의 [확장된 안정적인 채널](https://support.google.com/chrome/a/answer/9027636)
- Edge의 [확장된 안정적인 채널](https://docs.microsoft.com/en-gb/deployedge/microsoft-edge-channels#extended-stable-channel)
- Firefox의 ESR([추가 지원 릴리스](https://www.mozilla.org/en-US/firefox/organizations/))

이전 추가 지원 릴리스에서는 {% data variables.product.product_name %}가 예상대로 작동하지 않을 수 있으며 일부 기능을 사용하지 못할 수 있습니다.


## 베타 및 개발자 빌드

지원되는 브라우저의 베타 및 개발자 빌드에서 예기치 않은 버그가 발생할 수 있습니다. 릴리스되지 않은 빌드 중 하나에서 {% data variables.product.product_name %}에 버그가 발생하는 경우 동일한 브라우저의 안정적인 버전에도 버그가 있는지 확인하세요. 버그가 불안정한 버전에만 존재하는 경우 브라우저 개발자에게 버그를 보고하는 것이 좋습니다.
