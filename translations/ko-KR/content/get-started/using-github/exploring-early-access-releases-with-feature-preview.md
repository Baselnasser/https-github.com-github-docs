## {% data variables.product.prodname_dotcom %}의 릴리스 주기

{% data variables.product.prodname_dotcom %}의 제품 및 기능은 여러 릴리스 단계를 거칠 수 있습니다.

| 단계 | 설명 |
|-------|-------------|
| 알파 | 제품 또는 기능이 많이 개발되고 있으며 종종 요구 사항과 범위가 변경됩니다. 기능은 데모 및 테스트 목적으로 사용할 수 있지만 문서화되지 않을 수 있습니다. 알파 릴리스에서는 기능이 반드시 완료된 것은 아니며, SLA(서비스 수준 계약)가 제공되지 않고, 기술 지원 의무가 없습니다. <br><br> **참고**: “기술 미리 보기”로 릴리스된 제품 또는 기능은 알파 릴리스 단계에 있는 것으로 간주됩니다. 기술 미리 보기 릴리스는 위에서 설명한 것과 동일한 알파 릴리스 특성을 공유합니다.|
| 베타 | 제품 또는 기능이 더 광범위한 배포를 위해 준비됩니다. 베타 릴리스는 공개 또는 비공개일 수 있으며 문서화되어 있지만 SLA 또는 기술 지원 의무는 없습니다. |
| GA(일반 공급) | 제품 또는 기능이 완전히 테스트되고 모든 사용자에게 공개됩니다. GA 릴리스는 프로덕션에 사용할 준비가 된 것이고 관련 SLA 및 기술 지원 의무가 적용됩니다. |

## 기능 미리 보기로 베타 릴리스 탐색

베타에서 사용할 수 있는 기능 목록과 각 기능에 대한 간략한 설명을 볼 수 있습니다. 각 기능에는 피드백을 제공하는 링크가 포함되어 있습니다.
