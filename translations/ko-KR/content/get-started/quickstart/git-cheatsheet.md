사용 가능한 모든 Git 명령을 한 번에 알아보는 것은 어려운 작업이 될 수 있습니다. “[Git 참고 자료](https://training.github.com/)”를 사용하면 자주 사용하는 명령을 빠르게 참조할 수 있습니다. “Git 사용” 참고 자료는 여러 언어로 제공됩니다. 

또한 가이드, 비디오 등에 연결되는 [Git 및 GitHub 학습 리소스](/articles/git-and-github-learning-resources/) 페이지를 살펴보세요.
