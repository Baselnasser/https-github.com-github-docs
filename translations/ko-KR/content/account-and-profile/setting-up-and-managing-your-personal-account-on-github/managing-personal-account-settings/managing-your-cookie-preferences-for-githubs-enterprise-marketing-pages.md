## 엔터프라이즈 마케팅 페이지의 쿠키 기본 설정 정보

{% data variables.product.company_short %}는 일부 엔터프라이즈 마케팅 페이지에서 필수가 아닌 쿠키를 사용할 수 있습니다. 이러한 쿠키의 작동 방식을 사용자 지정할 수 있습니다. {% data variables.product.company_short %}에서 쿠키를 사용하는 방법에 대한 자세한 내용은 “[{% data variables.product.company_short %} 개인정보처리방침](/free-pro-team@latest/site-policy/privacy-policies/github-privacy-statement)”을 참조하세요.

## 쿠키 기본 설정 변경

모든 {% data variables.product.company_short %} 엔터프라이즈 마케팅 페이지에서 필수가 아닌 쿠키가 작동하는 방식을 사용자 지정할 수 있습니다.

1. 쿠키 기본 설정을 변경하려는 {% data variables.product.company_short %} 엔터프라이즈 마케팅 페이지로 이동합니다. 예를 들어 [{% data variables.product.company_short %} 리소스](https://resources.github.com/)로 이동합니다.
1. 페이지 아래쪽으로 스크롤한 다음 **쿠키 관리** 를 클릭합니다.

   ![쿠키 설정을 관리하는 단추의 스크린샷.](/assets/images/help/settings/cookie-settings-manage.png)

1. “쿠키 기본 설정 관리”에서 필수가 아닌 각 쿠키를 수락하거나 거부하려면 **수락** 이나 **거부** 를 클릭합니다.

   ![필수가 아닌 쿠키에 대해 “수락” 또는 “거부”를 선택하는 라디오 단추의 스크린샷.](/assets/images/help/settings/cookie-settings-accept-or-reject.png)

1. **변경 내용 저장** 을 클릭합니다.

   ![변경 내용을 저장하는 단추의 스크린샷.](/assets/images/help/settings/cookie-settings-save.png)
