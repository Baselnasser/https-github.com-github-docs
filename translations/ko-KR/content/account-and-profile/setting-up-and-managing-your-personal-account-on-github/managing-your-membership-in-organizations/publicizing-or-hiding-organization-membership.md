## 조직 멤버 자격의 표시 여부 변경

{% data reusables.profile.access_org %} {% data reusables.user-settings.access_org %} {% data reusables.organizations.people %}
4. 멤버 목록에서 사용자 이름을 찾습니다. 목록이 크면 검색 상자에서 사용자 이름을 검색할 수 있습니다.
![조직 멤버 검색 상자](/assets/images/help/organizations/member-search-box.png)
5. 사용자 이름 오른쪽 메뉴에서 새 표시 여부 옵션을 선택합니다.
    - 멤버 자격을 공개하려면 **퍼블릭** 을 선택합니다.
    - 멤버 자격을 숨기려면 **프라이빗** 을 선택합니다.
