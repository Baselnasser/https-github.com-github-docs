{% data variables.product.product_name %}에 렌더링된 코드의 탭 들여쓰기가 공간을 너무 많이 또는 적게 차지한다고 생각되면 설정에서 이를 변경할 수 있습니다.

{% data reusables.user-settings.access_settings %}
1. 왼쪽 사이드바에서 **{% octicon "paintbrush" aria-label="The paintbrush icon" %} 모양** 을 클릭합니다.
2. “탭 크기 기본 설정”에서 드롭다운 메뉴를 선택하고 기본 설정을 선택합니다.
