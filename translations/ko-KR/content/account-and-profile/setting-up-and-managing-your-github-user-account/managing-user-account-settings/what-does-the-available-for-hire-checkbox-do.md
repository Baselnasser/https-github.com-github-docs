사용 종료 참고: GitHub Jobs는 이제 더 이상 사용되지 않습니다. 마지막으로 채용 정보를 게시한 날짜는 2021년 5월 19일이었습니다. GitHub Jobs 사이트는 2021년 8월 19일에 완전히 종료되었으며 이제 GitHub Jobs의 지원 중단에 대한 자세한 정보가 있는 [GitHub 블로그 게시물](https://github.blog/changelog/2021-04-19-deprecation-notice-github-jobs-site/) 공지로 리디렉션됩니다.
