## <a name="about-accessibility-settings"></a>접근성 설정 정보

{% data variables.product.product_name %}에는 마우스를 사용하여 탐색하지 않고도 사이트 전체에서 작업을 수행할 수 있는 다양한 바로 가기 키가 포함되어 있습니다. 바로 가기는 시간을 절약하는 데 유용하지만 때로는 바로 가기로 인해 {% data variables.product.prodname_dotcom %}를 사용하기 어려워지고 접근성이 낮아질 수 있습니다.

모든 바로 가기 키는 기본적으로 {% data variables.product.product_name %}에서 사용하도록 설정되어 있지만 접근성 설정에서 문자 키 바로 가기를 사용하지 않도록 선택할 수 있습니다. 이 설정은 웹 브라우저에서 제공하는 바로 가기 키 또는 <kbd>Control</kbd> 또는 <kbd>Command</kbd>와 같은 한정자 키를 사용하는 {% data variables.product.prodname_dotcom %} 바로 가기에 영향을 미치지 않습니다.

## <a name="managing-character-key-shortcuts"></a>문자 키 바로 가기 관리

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.accessibility_settings %}
1. **문자 키 바로 가기 사용** 확인란을 선택하거나 선택 해제합니다.
   ![‘문자 키 바로 가기 사용’ 확인란의 스크린샷](/assets/images/help/settings/disable-character-key-shortcuts.png)
2. **저장** 을 클릭합니다.
