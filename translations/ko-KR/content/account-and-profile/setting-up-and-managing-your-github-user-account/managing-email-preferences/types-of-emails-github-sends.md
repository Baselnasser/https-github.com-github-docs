## <a name="notification-emails"></a>알림 이메일

메일을 통해 활동 알림의 일부 또는 전부를 수신하도록 선택할 수 있습니다. 자세한 내용은 “[알림 정보](/github/managing-subscriptions-and-notifications-on-github/about-notifications)”를 참조하세요. 알림 메일에는 다음이 포함될 수 있습니다.

- 관리자가 액세스할 수 있는 리포지토리의 보안 관련 활동
- 보고 있는 리포지토리에 대한 활동
- 참여 중인 대화
- @mentioned된 대화
- 참여 중인 끌어오기 요청 푸시
- 조직 또는 리포지토리의 협업 초대
- 문제 및 끌어오기 요청 열기, 주석 달기 또는 닫기 같은 사용자 고유의 활동

참여하거나 보고 있는 대화에서 받을 메일 업데이트 유형을 선택할 수도 있습니다. 자세한 내용은 “[알림 구성](/github/managing-subscriptions-and-notifications-on-github/configuring-notifications)”을 참조하세요.

## <a name="account-emails"></a>계정 메일

유료 제품 또는 기능으로 업그레이드한 경우 계정의 기본 메일 주소로 청구 영수증을 받게 됩니다. 자세한 내용은 “[청구 메일 설정](/articles/setting-your-billing-email)”을 참조하세요.

## <a name="customer-research-emails"></a>고객 조사 메일

{% data variables.product.product_name %}에서 더 나은 GitHub 구축하는 데 도움이 되도록 연구 세션에 참여할 고객을 찾는 경우가 있습니다. 해당 작업은 원격으로 수행되고 전 세계 고객에게 공개되며 다음을 포함할 수 있습니다.

- 피드백 설문 조사
- 연구 인터뷰
- 유용성 테스트 세션
- 초기 프로토타입 또는 개념 미리 보기

이와 같은 메일은 드물며 참여할지 여부를 선택할 수 있습니다. 추가적인 연구 세션 참여 기회에 관심이 있는 경우 GitHub 고객 조사 패널에 직접 추가할 수 있습니다. 자세한 내용은 “[GitHub 고객 경험 조사](https://cxr.github.com)”를 참조하세요.

## <a name="marketing-emails"></a>마케팅 메일

{% data variables.product.product_name %}은 때때로 다음과 같은 유형의 마케팅 메일을 보냅니다.

- 계정 시작에 대한 팁 및 요령
- 매력적인 프로젝트 또는 새로운 기능에 대한 사용자 지정 정보
- {% data variables.explore.explore_github %} 같이 구독한 뉴스레터

자세한 내용은 “[GitHub 마케팅 메일 관리](/articles/managing-marketing-emails-from-github)”를 참조하세요.
