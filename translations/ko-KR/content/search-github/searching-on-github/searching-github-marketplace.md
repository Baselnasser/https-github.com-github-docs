## {% data variables.product.prodname_marketplace %} 검색 정보

다음 두 가지 방법으로 {% data variables.product.prodname_marketplace %}에서 앱과 작업을 찾을 수 있습니다.

- {% data variables.product.prodname_marketplace %}에서 검색합니다.
- {% data variables.product.prodname_dotcom_the_website %} 전체에서 검색한 다음 결과를 필터링합니다.

## {% data variables.product.prodname_marketplace %}에서 검색

1. 페이지 위쪽에서 **Marketplace** 를 클릭합니다.
  ![Marketplace 링크](/assets/images/help/search/marketplace-link.png)
2. 키워드를 입력하고 **Enter 키** 를 누릅니다.
  ![{% data variables.product.prodname_marketplace %}에서 Linter 검색](/assets/images/help/search/marketplace-apps-and-actions-search-field.png)
3. 필요에 따라 왼쪽 사이드바에서 하나 이상의 옵션을 클릭하여 결과를 필터링합니다.

## {% data variables.product.prodname_dotcom_the_website %}에서 검색

언제든지 {% data variables.product.prodname_dotcom_the_website %} 전체에서 검색할 때 결과를 필터링하여 {% data variables.product.prodname_marketplace %}에서 일치하는 앱 및 작업을 볼 수 있습니다.

1. https://github.com/search로 이동합니다.
2. 키워드를 입력하고 **Enter 키** 를 누릅니다.
  ![검색 필드](/assets/images/help/search/search-field.png)
3. 왼쪽 사이드바에서 **Marketplace** 를 클릭합니다.
  ![Marketplace 사이드 메뉴 옵션이 강조 표시된 Linter에 대한 검색 결과](/assets/images/help/search/marketplace-left-side-navigation.png)
