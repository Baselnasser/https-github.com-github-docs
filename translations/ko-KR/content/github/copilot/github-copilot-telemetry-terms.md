## <a name="additional-telemetry"></a>추가 원격 분석

{% data variables.product.prodname_copilot %}을 사용하는 경우 {% data variables.product.prodname_copilot %} 확장/플러그 인은 IDE(통합 개발 환경)와 상호 작용하여 생성된 이벤트에 대한 사용 정보를 수집합니다. 이러한 이벤트에는 {% data variables.product.prodname_copilot %} 성능, 사용된 기능 및 수락, 수정 및 수락 또는 해제된 제안이 포함됩니다. 이 정보에는 [GitHub 개인정보처리방침](/github/site-policy/github-privacy-statement)에 정의된 사용자 개인 정보를 포함한 개인 데이터가 포함될 수 있습니다.

이 사용 정보는 {% data variables.product.company_short %}에서 사용되며 Microsoft 및 OpenAI와 공유되어 확장/플러그 인 및 관련 제품을 개발하고 개선합니다. 또한 OpenAI는 이 사용 정보를 사용하여 {% data variables.product.prodname_copilot %}와 관련된 다른 서비스를 수행합니다. 예를 들어 {% data variables.product.prodname_copilot %} 확장/플러그 인을 사용하여 파일을 편집하는 경우 파일 콘텐츠 조각, 제안, 제안 수정 사항이 {% data variables.product.company_short %}, Microsoft, OpenAI와 공유되고 진단 목적으로 제안 및 관련 제품을 개선하는 데 사용됩니다. {% data variables.product.prodname_copilot %}은 편집 중인 파일과 동일한 IDE 인스턴스에서 열려 있는 다른 파일 모두에서 컨텍스트에 대한 파일 콘텐츠를 사용합니다. {% data variables.product.prodname_copilot %}을 사용하는 경우 관련 파일에 대한 리포지토리 또는 파일 경로의 URL을 수집할 수도 있습니다. {% data variables.product.prodname_copilot %}은 원격 분석에서 수집된 URL, 파일 경로 또는 코드 조각을 {% data variables.product.prodname_copilot %}의 다른 사용자에 대한 제안으로 사용하지 않습니다. 이 정보는 기밀 정보로 처리되며 알아야 할 경우에만 액세스됩니다. {% data variables.product.prodname_copilot %} 확장/플러그 인에서 {% data variables.product.prodname_copilot %}의 다른 사용자에 대한 원격 분석 데이터를 수집할 수 없습니다. {% data variables.product.prodname_copilot %} 원격 분석에 대한 자세한 내용은 “[{% data variables.product.prodname_copilot %} 원격 분석 정보](/github/copilot/about-github-copilot-telemetry)”를 참조하세요. GitHub에 연락하여 기술 미리 보기에서 제거를 요청하여 이 단락에 설명된 원격 분석 및 개인 데이터 처리 작업에 대한 동의를 취소할 수 있습니다.
