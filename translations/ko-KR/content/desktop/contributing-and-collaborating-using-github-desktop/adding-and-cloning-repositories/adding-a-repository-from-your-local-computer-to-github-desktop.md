{% tip %}

**팁:** 폴더를 {% data variables.product.prodname_desktop %} 창으로 끌어 Git 리포지토리를 로컬 컴퓨터에서 GitHub Desktop으로 추가할 수 있습니다. 여러 Git 폴더를 동시에 {% data variables.product.prodname_desktop %}으로 끌면 각 폴더가 별도의 Git 리포지토리로 추가됩니다.

{% endtip %}

{% mac %}

1. **파일** 메뉴에서 **로컬 리포지토리 추가** 를 클릭합니다.
  ![로컬 리포지토리 메뉴 옵션 추가](/assets/images/help/desktop/add-local-repository-mac.png)
2. **선택...** 을 클릭하고 Finder 창을 사용하여 추가하려는 로컬 리포지토리로 이동합니다.
  ![Mac 앱의 로컬 경로 필드](/assets/images/help/desktop/add-repo-choose-button-mac.png)
4. **리포지토리 추가** 를 클릭합니다.
  ![Mac 앱의 리포지토리 추가 단추](/assets/images/help/desktop/add-repository-button-mac.png)

{% endmac %}

{% windows %}

1. **파일** 메뉴에서 **로컬 리포지토리 추가** 를 클릭합니다.
  ![로컬 리포지토리 메뉴 옵션 추가](/assets/images/help/desktop/add-local-repository-windows.png)
2. **선택...** 을 클릭하고 Windows 탐색기를 사용하여 추가하려는 로컬 리포지토리로 이동합니다.
  ![Windows 앱의 로컬 경로 필드](/assets/images/help/desktop/add-repo-choose-button-win.png)
4. **리포지토리 추가** 를 클릭합니다.
  ![Windows 앱의 리포지토리 추가 단추](/assets/images/help/desktop/add-repository-button-windows.png)

{% endwindows %}
