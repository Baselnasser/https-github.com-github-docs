{% mac %}

{% data reusables.git.remove-git-remote %}
2. [GitHub Desktop에 리포지토리를 추가합니다](/desktop/guides/contributing-to-projects/adding-a-repository-from-your-local-computer-to-github-desktop/).
{% data reusables.desktop.publish-repository %}
4. **이름** 필드에 원하는 리포지토리 이름을 입력하거나 기본 현재 로컬 리포지토리 이름을 사용합니다.
  ![이름 필드](/assets/images/help/desktop/publish-repository-name-mac.png)
5. 퍼블릭 리포지토리를 게시하려면 **이 코드를 프라이빗으로 유지** 를 선택 취소합니다.
  ![이 코드를 프라이빗으로 유지 확인란](/assets/images/help/desktop/publish-repository-private-checkbox-mac.png)
6. **조직** 드롭다운에서 리포지토리를 게시할 조직을 선택하거나 **없음** 을 선택하여 리포지토리를 개인 계정에 게시합니다.
  ![조직 드롭다운](/assets/images/help/desktop/publish-repository-org-dropdown-mac.png)
7. **리포지토리 게시** 단추를 클릭합니다.
  ![리포지토리 게시 대화 상자의 리포지토리 게시 단추](/assets/images/help/desktop/publish-repository-dialog-button-mac.png)

{% endmac %}
