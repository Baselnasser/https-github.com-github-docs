각 커밋은 다음을 보여줍니다.

 - 커밋 메시지
 - 연락처가 생성된 시간
 - 커밋한 사람의 사용자 이름 및 프로필 사진(사용 가능한 경우)
 - 커밋의 SHA-1 해시(고유 ID)

{% data reusables.desktop.history-tab %}
2. **기록** 탭에서 검토할 커밋을 클릭합니다.
![기록 탭의 커밋](/assets/images/help/desktop/branch-history-commit.png)
3. 커밋에 여러 파일이 있는 경우 개별 파일을 클릭하여 커밋에서 해당 파일의 변경 내용을 확인합니다.
![커밋의 파일](/assets/images/help/desktop/branch-history-file.png)
