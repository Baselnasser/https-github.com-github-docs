## {% data variables.product.prodname_desktop %}의 끌어오기 요청 정보
사용자 또는 협력자는 {% data variables.product.prodname_desktop %}에서 제안한 끌어오기 요청을 볼 수 있습니다. 끌어오기 요청을 사용하면 프로젝트에 대한 변경 내용을 제안하고, 피드백 및 검토를 제공하고, 변경 내용을 프로젝트에 병합할 수 있습니다. 자세한 내용은 “[끌어오기 요청 정보](/github/collaborating-with-issues-and-pull-requests/about-pull-requests)”를 참조하세요.

{% data variables.product.prodname_desktop %}에서 끌어오기 요청을 보면 기여자가 수행한 커밋 기록을 볼 수 있습니다. 커밋이 수정, 추가 또는 삭제된 파일을 확인할 수도 있습니다. {% data variables.product.prodname_desktop %}에서 원하는 텍스트 편집기에서 리포지토리를 열어 변경 내용을 보거나 추가로 변경할 수 있습니다. 끌어오기 요청의 변경 내용을 검토한 후 {% data variables.product.prodname_dotcom %}에 대한 피드백을 제공할 수 있습니다. 자세한 내용은 “[끌어오기 요청 검토 정보](/github/collaborating-with-issues-and-pull-requests/about-pull-request-reviews)”를 참조하세요.

리포지토리에서 검사를 활성화한 경우 {% data variables.product.prodname_desktop %}은 끌어오기 요청에 대한 검사 상태를 표시하고 검사를 다시 실행할 수 있습니다. 자세한 내용은 "[GitHub Desktop에서 검사 보기 및 다시 실행](/desktop/contributing-and-collaborating-using-github-desktop/working-with-your-remote-repository-on-github-or-github-enterprise/viewing-and-re-running-checks-in-github-desktop)"을 참조하세요.
