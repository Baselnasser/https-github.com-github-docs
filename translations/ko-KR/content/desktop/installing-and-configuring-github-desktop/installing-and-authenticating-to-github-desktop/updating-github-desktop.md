{% mac %}

1. **GitHub Desktop** 메뉴에서 **GitHub Desktop 정보** 를 클릭합니다.
![GitHub Desktop 메뉴 옵션 정보](/assets/images/help/desktop/desktop-menu-about-desktop-mac.png)
2. **업데이트 확인** 을 클릭합니다.
![업데이트 확인](/assets/images/help/desktop/check-for-updates.png)
3. 업데이트를 사용할 수 있는 경우 {% data variables.product.prodname_desktop %}을 종료하고 다시 실행하여 업데이트를 설치합니다.

{% endmac %}

{% windows %}

1. **도움말** 메뉴에서 **GitHub Desktop 정보** 를 클릭합니다.
![GitHub Desktop 메뉴 옵션 정보](/assets/images/help/desktop/help-about-desktop-win.png)
2. **업데이트 확인** 을 클릭합니다.
![업데이트 확인](/assets/images/help/desktop/check-for-updates.png)
3. 업데이트를 사용할 수 있는 경우 {% data variables.product.prodname_desktop %}을 종료하고 다시 실행하여 업데이트를 설치합니다.

{% endwindows %}
