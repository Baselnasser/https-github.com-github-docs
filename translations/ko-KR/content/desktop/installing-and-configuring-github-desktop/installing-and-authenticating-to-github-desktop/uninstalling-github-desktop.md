{% mac %}

1. Finder 창에서 애플리케이션 폴더로 이동합니다.
  ![Finder 창의 애플리케이션 폴더](/assets/images/help/desktop/applications-folder.png)
2. 키보드에서 **control** 키를 누른 상태에서 **{% data variables.product.prodname_desktop %}** 을 클릭합니다.
3. **휴지통으로 이동** 을 선택합니다.
  ![휴지통으로 이동 옵션](/assets/images/help/desktop/mac-move-to-trash.png)
4. 메뉴 모음의 **Finder** 드롭다운 메뉴에서 **휴지통 비우기** 를 클릭합니다.
  ![메뉴 모음의 휴지통 비우기 옵션](/assets/images/help/desktop/mac-empty-trash-menu.png)
5. 팝업 상자에서 경고를 읽고 **휴지통 비우기** 를 클릭합니다.
  ![휴지통 비우기 단추](/assets/images/help/desktop/mac-empty-trash-button.png)

{% endmac %}

{% windows %}

1. 제어판을 엽니다. 자세한 내용은 Windows 도움말의 [제어판 위치](https://support.microsoft.com/en-us/help/13764/windows-where-is-control-panel)를 참조하세요.
2. “프로그램”에서 **프로그램 제거** 를 클릭합니다.
  ![제어판 프로그램의 제거 옵션](/assets/images/help/desktop/windows-uninstall-a-program.png)
3. **{% data variables.product.prodname_desktop %}** 항목을 마우스 오른쪽 단추로 클릭한 다음 **제거** 를 클릭합니다.
  ![제거 옵션](/assets/images/help/desktop/windows-click-uninstall.png)

{% endwindows %}
