{% data variables.product.prodname_desktop %}을 설치하면 {% data variables.large_files.product_name_long %}({% data variables.large_files.product_name_short %})도 설치됩니다. {% data variables.large_files.product_name_short %}를 사용하면 {% data variables.product.prodname_dotcom %}에 파일을 푸시하여 {% data variables.large_files.max_github_size %}의 정상 제한을 초과할 수 있습니다. {% data variables.large_files.product_name_short %}에 대한 자세한 내용은 “[{% data variables.large_files.product_name_long %} 정보](/github/managing-large-files/about-git-large-file-storage)”를 참조하세요.

{% data variables.product.prodname_desktop %}과 함께 {% data variables.large_files.product_name_short %}를 사용하려면 명령줄을 사용하여 {% data variables.large_files.product_name_short %}를 구성해야 합니다. 자세한 내용은 “[{% data variables.large_files.product_name_long %} 구성](/github/managing-large-files/configuring-git-large-file-storage)”을 참조하세요.

리포지토리의 파일을 추적하도록 {% data variables.large_files.product_name_short %}를 구성한 후에는 리포지토리의 다른 파일처럼 {% data variables.product.prodname_desktop %}을 사용하여 대용량 파일에 원활하게 액세스하고 관리할 수 있습니다.
