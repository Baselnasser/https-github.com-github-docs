## 기본 편집기 구성

{% mac %}

{% data reusables.desktop.mac-select-desktop-menu %}
3. 기본 설정 창에서 **통합** 을 선택합니다.
  ![기본 설정 창의 통합 창](/assets/images/help/desktop/mac-select-integrations-pane.png)
4. ‘외부 편집기’ 드롭다운 메뉴를 사용하고 기본값으로 설정할 편집기를 선택합니다.
  ![기본 설정 메뉴 모음의 외부 편집기 메뉴](/assets/images/help/desktop/mac-editor-menu.png)
5. **저장** 을 클릭합니다.  

{% endmac %}

{% windows %}

{% data reusables.desktop.windows-choose-options %}
3. 옵션 창에서 **통합** 을 선택합니다.
  ![옵션 창의 통합 창](/assets/images/help/desktop/windows-select-integrations-pane.png)
4. ‘외부 편집기’ 드롭다운 메뉴를 사용하고 기본값으로 설정할 편집기를 선택합니다.
  ![옵션 메뉴 모음의 외부 편집기 메뉴](/assets/images/help/desktop/windows-editor-menu.png)
5. **저장** 을 클릭합니다.  

{% endwindows %}
