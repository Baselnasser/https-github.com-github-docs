## Repl.it 통합 때문에 변경되는 사항

2021년 7월 13일부터 Repl.it 통합은 사용할 수 없게 됩니다. 변경되는 사항은 다음과 같습니다.

- 이제 GitHub 클래스룸에서 새 비공개 및 공개 repl을 만들 수 없습니다.
- 이제 Always On 및 Boost 같은 Replit Hacker 혜택은 신규 GitHub 클래스룸 사용자에게 제공되지 않습니다.

## 이것은 기존 과제에는 어떤 의미일까요?
기존 GitHub 클래스룸 repl은 영원히 비공개로 유지되며, 2021년 9월 30일까지 이러한 repl을 통해 GitHub 클래스룸에 작업을 제출할 수 있습니다. 2021년 9월 이후에는 과제 추가 정보 파일에서 Replit 아이콘을 사용할 수 없게 됩니다. 통합을 사용하는 모든 추가 정보 파일에서 지침을 제거하고 학생들이 아이콘이 더 이상 작동하지 않음을 알리는 것이 좋습니다.

클래스룸에서는 비공개 repl을 만들 수는 없지만 9월 30일까지는 Replit에서 비공개 repl을 계속 만들 수 있습니다.

클라우드 IDE 통합이 클래스룸에 중요하다는 것을 알고 있으며 더 많은 옵션을 제공하기 위해 열심히 노력하고 있습니다.
