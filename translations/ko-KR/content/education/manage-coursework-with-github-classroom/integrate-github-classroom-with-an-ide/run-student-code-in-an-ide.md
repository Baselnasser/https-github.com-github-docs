## 학생 코드 및 IDE 정보

할당에 대해 IDE(통합 개발 환경)를 구성하는 경우 IDE 내에서 코드를 실행할 수 있습니다. 할당 리포지토리를 컴퓨터에 복제할 필요가 없습니다.

IDE에 대한 자세한 내용은 "[IDE와 {% data variables.product.prodname_classroom %} 통합](/education/manage-coursework-with-github-classroom/integrate-github-classroom-with-an-ide/integrate-github-classroom-with-an-ide)"을 참조하세요.

## IDE에서 학생 코드 실행 

{% data reusables.classroom.sign-into-github-classroom %} {% data reusables.classroom.click-classroom-in-list %} {% data reusables.classroom.click-assignment-in-list %}
1. 제출 오른쪽에서 **IDE 보기** 를 클릭합니다.
