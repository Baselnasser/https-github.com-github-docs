## 제출 정보

{% data variables.product.prodname_community_exchange %} 제출 프로세스 중에 리포지토리에 대한 목적, 토픽 및 제안을 선택합니다. 리포지토리가 {% data variables.product.prodname_community_exchange %}에 제출되면 이러한 세부 정보와 함께 게시됩니다. 자세한 내용은 “[GitHub Community Exchange에 리포지토리 제출](/education/contribute-with-github-community-exchange/submitting-your-repository-to-github-community-exchange)”을 참조하세요.

제출한 후에도 리포지토리와 연결된 토픽 및 제안을 편집할 수 있습니다. 해당 토픽을 변경하여 리포지토리의 목적을 업데이트할 수도 있습니다. 자세한 내용은 “[토픽을 사용하여 리포지토리 분류](/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/classifying-your-repository-with-topics)”를 참조하세요.

리포지토리와 연결된 언어는 사용되는 기본 언어이며 {% data variables.product.prodname_dotcom %}에서 자동으로 확인됩니다. 자세한 내용은 “[리포지토리 언어 정보](/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-repository-languages)”를 참조하세요.

{% data variables.product.prodname_community_exchange %} 커뮤니티는 모든 리포지토리 제출을 조정합니다.

## 제출 관리

1. {% data variables.product.prodname_global_campus %} 대시보드에서 {% data variables.product.prodname_community_exchange %} 홈페이지로 이동합니다.
1. 리포지토리 목록 위에서 **제출** 탭을 클릭합니다. ![제출 탭의 스크린샷](/assets/images/help/education/community-exchange-submissions-tab.png)
1. 필요에 따라 제출된 리포지토리를 편집합니다. 편집하려는 리포지토리 오른쪽에서 {% octicon "pencil" aria-label="The edit icon" %}을 클릭하여 리포지토리 홈페이지로 직접 이동합니다. 여기에서 리포지토리에 할당된 목적, 토픽 및 제안을 업데이트할 수 있습니다.
1. 필요에 따라 갤러리에서 제출된 리포지토리를 삭제합니다. 제거하려는 리포지토리 제출 오른쪽에서 {% octicon "trash" aria-label="The trash icon" %}을 클릭합니다.
