학교 프로젝트에 {% data variables.product.prodname_dotcom %}을 사용하면 다른 사용자와 공동 작업하고 실제 경험을 보여주는 포트폴리오를 구축할 수 있습니다.

{% data variables.product.prodname_dotcom %} 계정이 있는 모든 사용자는 {% data variables.product.prodname_free_user %}가 설치된 무제한 퍼블릭 및 프라이빗 리포지토리에서 공동 작업할 수 있습니다. 학생으로서 GitHub 학생 혜택을 신청할 수도 있습니다. 자세한 내용은 “[학생 개발자 팩 신청](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/apply-for-a-student-developer-pack)” 및 “[{% data variables.product.prodname_education %}](https://education.github.com/)을 참조하세요.

FIRST 로봇 클럽의 회원인 경우 멘토가 교육자 할인을 신청하여 팀이 무제한 사용자 및 프라이빗 리포지토리를 무료로 허용하는 {% data variables.product.prodname_team %}을 사용하여 공동 작업할 수 있습니다. 자세한 내용은 “[교육자 또는 연구원 할인 신청](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/apply-for-an-educator-or-researcher-discount)”을 참조하세요.
