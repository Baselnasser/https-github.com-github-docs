교수, 교사 및 멘토는 Campus Advisors 온라인 교육을 사용하여 Git 및 {% data variables.product.prodname_dotcom %}에 대해 완전히 숙지하고 {% data variables.product.prodname_dotcom %}를 사용하여 학생들을 가르치는 모범 사례를 배울 수 있습니다.  자세한 내용은 [Campus Advisors](https://education.github.com/teachers/advisors)”를 참조하세요.

{% note %}

**참고:** 강사는 {% data variables.product.product_location %}에서 학생용 계정을 만들 수 없습니다. 학생은 {% data variables.product.product_location %}에서 자신의 계정을 직업 만들어야 합니다.

{% endnote %}

교사는 {% data variables.product.prodname_education %}을 사용하여 소프트웨어 개발 과정을 관리할 수 있습니다. {% data variables.product.prodname_classroom %}에서는 {% data variables.product.product_name %}를 사용하여 코드를 배포하고, 피드백을 제공하고, 교과 과정을 관리할 수 있습니다. 자세한 내용은 “[{% data variables.product.prodname_classroom %}을 사용하여 교과 과정 관리](/education/manage-coursework-with-github-classroom)”를 참조하세요.

학생 또는 학교 교직원인데 학교에서 {% data variables.product.prodname_dotcom %}과 {% data variables.product.prodname_campus_program %} 학교로 파트너 관계를 맺지 않은 경우에도 {% data variables.product.prodname_dotcom %}을 사용하도록 할인을 개별적으로 신청할 수 있습니다. 자세한 내용은 “[교과목에 {% data variables.product.prodname_dotcom %} 사용](/education/teach-and-learn-with-github-education/use-github-for-your-schoolwork)” 또는 “[클래스룸 및 연구실에서 {% data variables.product.prodname_dotcom %} 사용](/education/teach-and-learn-with-github-education/use-github-in-your-classroom-and-research/)”을 참조하세요.
