{% data variables.product.prodname_campus_program %}은 학위, 졸업장 또는 인증서를 부여하는 교육 중심 기관을 위한 프리미엄 {% data variables.product.prodname_dotcom %} 액세스 패키지입니다. {% data variables.product.prodname_campus_program %}에는 다음이 포함됩니다.

- 모든 기술 및 교육 부서의 {% data variables.product.prodname_ghe_cloud %} 및 {% data variables.product.prodname_ghe_server %}에 대한 무료 액세스
- 50,000분의 {% data variables.product.prodname_actions %} 및 50GB {% data variables.product.prodname_registry %} 스토리지
- Git 및 {% data variables.product.prodname_dotcom %}를 완전히 숙지하기 위한 교사 학습
- {% data variables.product.prodname_dotcom %} 파트너의 새 기능, GitHub Education 관련 Swag 및 무료 개발자 도구에 대한 단독 액세스 권한
- {% data variables.product.prodname_student_pack %}과 같은 프리미엄 {% data variables.product.prodname_education %} 기능에 대한 자동화된 액세스 권한

교육자가 GitHub를 사용하는 방법에 대해 알아보려면 [GitHub Education 스토리](https://education.github.com/stories)를 참조하세요.

## {% data variables.product.prodname_campus_program %} 사용 약관

- 라이선스는 1년 동안 무료이며 2년마다 자동으로 무료로 갱신됩니다. 계약 조건 내에서 계속 작동하는 한 무료 라이선스를 계속 사용할 수 있습니다. [프로그램 조건](https://education.github.com/schools/terms)에 동의할 수 있는 모든 학교의 가입을 환영합니다.

- 이러한 라이선스는 학교 전체에서 사용할 수 있습니다. 이 라이선스의 사용으로 수익을 창출하지 않는 한 내부 IT 부서, 학술 연구 그룹, 협력자, 학생 및 기타 비학문 부서가 라이선스를 사용할 수 있습니다. 대학 내부에 있는, 외부에서 자금을 지원 받는 연구 그룹은 무료 라이선스를 사용하지 못할 수 있습니다.

- 모든 기술 및 교육 부서에 {% data variables.product.prodname_dotcom %}을 제공해야 하며, 학교 로고는 GitHub Education 웹 사이트에서 {% data variables.product.prodname_campus_program %} 파트너로 공유됩니다.

- 엔터프라이즈 내의 새로운 조직은 엔터프라이즈 계정에 자동으로 추가됩니다. 학교가 {% data variables.product.prodname_campus_program %}에 가입하기 전에 있었던 조직을 추가하려면 [GitHub Education 지원 팀](https://support.github.com/contact/education)에 문의하세요. 엔터프라이즈를 관리하는 방법에 대한 자세한 내용은 [엔터프라이즈 관리자 설명서](/admin)를 참조하세요. 엔터프라이즈 내의 새로운 조직은 엔터프라이즈 계정에 자동으로 추가됩니다. 학교가 {% data variables.product.prodname_campus_program %}에 가입하기 전에 존재했던 조직을 추가하려면 GitHub Education 지원 팀에 문의하세요. 


{% data variables.product.prodname_dotcom %}의 개인 정보 취급 방침에 대한 자세한 내용은 “[글로벌 개인 정보 취급 방침](/github/site-policy/global-privacy-practices)”을 참조하세요.

## {% data variables.product.prodname_campus_program %} 애플리케이션 자격

- 종종 캠퍼스 CTO/CIO, 학장, 학과장 또는 기술 책임자가 캠퍼스를 대신하여 프로그램 약관에 서명합니다.

- 학교에서 메일 주소를 발급하지 않는 경우 {% data variables.product.prodname_dotcom %}에서는 학생 개발자 팩을 학생에게 배포할 수 있는 다른 옵션을 사용하여 계정 관리자에게 연락합니다.

자세한 내용은 [공식 {% data variables.product.prodname_campus_program %}](https://education.github.com/schools) 페이지를 참조하세요.

학생 또는 학교 교직원인데 학교에서 {% data variables.product.prodname_dotcom %}과 {% data variables.product.prodname_campus_program %} 학교로 파트너 관계를 맺지 않은 경우에도 {% data variables.product.prodname_dotcom %}을 사용하도록 할인을 개별적으로 신청할 수 있습니다. 학생 개발자 팩을 신청하려면 [신청서 양식](https://education.github.com/pack/join)을 참조하세요.
