## <a name="unclear-proof-of-affiliation-documents"></a>소속 문서의 불분명한 증거

업로드한 이미지가 학교 또는 대학에 대한 현재 고용 상태를 명확하게 나타내지 못하는 경우 교직원 신분증 또는 분명한 정보가 표시된 고용 확인서의 다른 이미지를 업로드하여 재신청해야 합니다.

{% data reusables.education.pdf-support %}

## <a name="using-an-academic-email-with-an-unverified-domain"></a>확인되지 않은 도메인이 포함된 교육 기관 메일 사용

교육 기관 메일 주소에 확인되지 않은 도메인이 있는 경우 학업 상태에 대한 추가 증명이 필요할 수 있습니다. {% data reusables.education.upload-different-image %}

{% data reusables.education.pdf-support %}

## <a name="using-an-academic-email-from-a-school-with-lax-email-policies"></a>메일 정책이 느슨한 학교에서 교육 기관 메일 사용

학교의 졸업생과 은퇴한 교직원이 학교에서 발급한 메일 주소에 평생 액세스할 수 있는 경우 학업 상태를 증명하는 추가 증거를 요구할 수 있습니다. {% data reusables.education.upload-different-image %}

{% data reusables.education.pdf-support %}

학교 도메인에 대한 다른 질문이나 문제가 있는 경우 학교 IT 직원에게 Github로 문의해 달라고 요청하세요.

## <a name="non-student-applying-for-student-developer-pack"></a>학생이 아닌 사람의 Student Developer Pack 신청

교육자와 연구원은 [{% data variables.product.prodname_student_pack %}](https://education.github.com/pack)과 함께 제공되는 파트너 제품을 사용할 자격이 없습니다. 재신청 시 **교직원** 을 선택하여 학업 상태를 설명해야 합니다.
