학교 프로젝트에 {% data variables.product.prodname_dotcom %}을 사용하면 다른 사용자와 공동 작업하고 실제 경험을 보여주는 포트폴리오를 구축할 수 있습니다.

{% data variables.product.prodname_dotcom %} 계정이 있는 모든 사용자는 {% data variables.product.prodname_free_user %}가 설치된 무제한 퍼블릭 및 프라이빗 리포지토리에서 공동 작업할 수 있습니다. 학생으로서 {% data variables.product.prodname_education %} 학생 혜택을 신청할 수도 있습니다. {% data variables.product.prodname_education %} 학생 혜택 및 리소스는 교육 혜택에 액세스할 수 있는 포털인 {% data variables.product.prodname_global_campus %}에 모두 포함되어 있습니다. 자세한 내용은 “[학생으로 GitHub Global Campus에 신청](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/apply-to-github-global-campus-as-a-student)” 및 [{% data variables.product.prodname_education %}](https://education.github.com/)을 참조하세요.

Global Campus를 신청하기 전에 학습 커뮤니티가 이미 {% data variables.product.prodname_campus_program %} 학교로 당사와 파트너 관계를 맺고 있는지 확인합니다. 자세한 내용은 “[{% data variables.product.prodname_campus_program %} 정보](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/about-github-campus-program)”를 참조하세요.

학교 클럽의 회원인 경우 교사가 {% data variables.product.prodname_global_campus %}를 신청하여 팀이 무제한 사용자 및 프라이빗 리포지토리를 무료로 허용하는 {% data variables.product.prodname_team %}을 사용하여 공동 작업할 수 있습니다. 자세한 내용은 “[교사로 {% data variables.product.prodname_global_campus %}에 신청](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-teachers/apply-to-github-global-campus-as-a-teacher)”을 참조하세요.

확인된 {% data variables.product.prodname_global_campus %} 학생은 [{% data variables.product.prodname_education %} 웹 사이트](https://education.github.com)로 이동하여 언제든지 {% data variables.product.prodname_global_campus %}에 액세스할 수 있습니다. 

![학생용 {% data variables.product.prodname_global_campus %} 포털](/assets/images/help/education/global-campus-portal-students.png)

## 학생용 {% data variables.product.prodname_global_campus %} 기능

{% data variables.product.prodname_global_campus %}는 한 곳에서 {% data variables.product.prodname_education %} 혜택 및 리소스에 액세스할 수 있는 포털입니다. {% data variables.product.prodname_global_campus %} 포털에서 학생은 다음을 수행할 수 있습니다.
- 지역 캠퍼스 전문가와 연결합니다. 캠퍼스 전문가에 대한 자세한 내용은 “[Campus Experts 정보](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/use-github-at-your-educational-institution/about-campus-experts)”를 참조하세요.
- [학생 개발자 팩](https://education.github.com/pack)에서 무료 산업 도구에 대한 제안을 탐색하고 클레임합니다.
- {% data variables.product.prodname_education %} 및 학생 리더가 큐레이팅한 학생을 위해 예정된 대면 및 가상 이벤트를 확인합니다.
- 예정된 기한이 있는 [GitHub 교실](https://classroom.github.com/) 에서 과제를 봅니다.
- 최근 [Campus TV](https://www.twitch.tv/githubeducation) 에피소드를 다시 보고 커뮤니티가 어떤 관심을 가지고 있는지 알아봅니다. Campus TV는 {% data variables.product.prodname_dotcom %} 및 학생 커뮤니티 리더에 의해 생성되며 라이브 또는 주문형으로 시청할 수 있습니다.
- GitHub Community Exchange에서 학생이 만든 리포지토리를 검색합니다. 자세한 내용은 “[GitHub 커뮤니티 Exchange 정보](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/about-github-community-exchange)”를 참조하세요.

{% data variables.product.prodname_global_campus %} 학생은 다음과 같은 {% 데이터 variables.product.prodname_dotcom %} 혜택을 받습니다.
- **{% data variables.product.prodname_copilot %}**: 확인된 학생은 {% data variables.product.prodname_copilot %}에 대한 무료 구독을 받습니다. 계정 설정의 {% data variables.product.prodname_copilot %} 구독 페이지를 방문하면 무료 구독에 대한 알림이 자동으로 표시됩니다. {% data variables.product.prodname_copilot %}를 구독하고 사용하는 방법에 대한 자세한 내용은 "[{% data variables.product.prodname_copilot_for_individuals %} 구독 관리](/billing/managing-billing-for-github-copilot/managing-your-github-copilot-for-individuals-subscription#setting-up-a-trial-of-github-copilot)" 및 "[{% data variables.product.prodname_copilot %} 정보"를](/copilot/overview-of-github-copilot/about-github-copilot) 참조하세요.
- **{% data variables.product.prodname_github_codespaces %}**: {% data reusables.education.student-codespaces-benefit %} {% data variables.product.prodname_github_codespaces %}를 시작하는 방법에 대한 자세한 내용은 "[{% data variables.product.prodname_github_codespaces %} 개요"를 참조하세요](/codespaces/overview).

{% note %}
