## {% data variables.product.prodname_community_exchange %} 정보

{% data reusables.education.about-github-community-exchange-intro %}

![Community Exchange 페이지 스크린샷](/assets/images/help/education/community-exchange-page.png)

동료가 오픈 소스 기술을 배우고, 프로젝트 유지 관리자가 되고, 안전하고 검증된 커뮤니티 플랫폼 내에서 {% data variables.product.prodname_dotcom %} 포트폴리오 및 네트워크를 확장하도록 도와줄 수 있습니다.

{% data variables.product.prodname_community_exchange %}에서는 다음을 수행할 수 있습니다.
- 학생이 만든 리포지토리 검색
- 관심 있는 리포지토리에 별 표시
- 협력자가 필요한 리포지토리 제출
- 리포지토리를 제출하여 새 기술 교육
- 리포지토리 제출 관리

자세한 내용은 “[GitHub Community Exchange를 사용하여 기여](/education/contribute-with-github-community-exchange)”를 참조하세요.
