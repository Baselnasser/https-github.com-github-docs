## 요구 사항

{% data variables.product.prodname_global_campus %} 및 기타 혜택을 포함하여 {% data variables.product.prodname_student_pack %}의 이용 자격을 얻으려면 다음과 같아야 합니다.
- 현재 고등학교, 중등학교, 전문대학, 대학, 홈스쿨 또는 유사한 교육 기관과 같은 교육 과정 수료증을 부여하는 학위 또는 졸업장에 등록되어 있음
- 입증할 수 있는 학교 발급 메일 주소를 가지고 있거나 현재 학생 상태를 증명하는 문서 업로드
- [{% data variables.product.prodname_dotcom %} 개인 계정](/articles/signing-up-for-a-new-github-account)이 있음
- 만 13세 이상이어야 함

현재 학생 상태를 증명하는 문서에는 현재 등록 날짜, 수업 일정, 성적 증명서, 소속 또는 등록 확인서와 함께 학생증의 사진이 포함됩니다.

학생으로 재학하는 동안 현재 학업 상태를 주기적으로 다시 확인하라는 메시지가 표시될 수 있습니다.

{% note %}

**참고:** 한 계정에서 다른 계정으로 교육 기관 할인을 이전할 수 없습니다. 할인을 적용하려는 계정이 두 개 이상 있는 경우 개인 계정을 [병합하고](/articles/merging-multiple-user-accounts) 원하는 경우 보존된 계정의 [이름을 바꾸는 것이](/articles/changing-your-github-username) 좋습니다.

{% endnote %}

{% data variables.product.prodname_global_campus %}를 갱신하는 방법에 대한 자세한 내용은 “[만료 및 갱신](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/apply-to-github-global-campus-as-a-student/#expiration-and-renewals)”을 참조하세요.

## {% data variables.product.prodname_global_campus %}에 신청

{% data reusables.education.benefits-page %}
1. "개인"에서 **학생 혜택 받기** 를 클릭합니다.
  ![{% data variables.product.prodname_education %} 홈페이지의 "학생 혜택 받기"](/assets/images/help/education/click-get-student-benefits.png)
1. "학업 상태 선택"에서 **학생** 을 선택합니다.
  ![학업 상태](/assets/images/help/education/academic-status-student.png) 선택 {% data reusables.education.select-email-address %} {% data reusables.education.school-name %} {% data reusables.education.plan-to-use-github %} {% data reusables.education.upload-proof-status %} {% data reusables.education.submit-application %}

## 만료 및 갱신

{% data variables.product.prodname_global_campus %} 액세스가 만료되면 {% data variables.product.prodname_student_pack %}에 대한 일부 파트너 제품을 갱신할 수는 없지만 여전히 자격이 있는 경우 다시 신청할 수 있습니다. 파트너가 제공하는 시간이 지정된 제품은 설정하면 시작됩니다. 다시 신청하려면 https://education.github.com 으로 돌아가 프로필 사진을 클릭한 다음 **학업 소속 다시 확인** 을 클릭하세요.

![학업 소속 다시 확인 메뉴 옵션](/assets/images/help/education/reverify-academic-affiliation.png)

자세한 내용은 [{% data variables.product.prodname_student_pack %}](https://education.github.com/pack) 페이지를 참조하세요.

{% data variables.product.prodname_student_pack %}에 대한 무료 액세스가 만료되는 시기를 확인하려면 계정의 [청구 설정](https://github.com/settings/billing)을 방문하세요.
