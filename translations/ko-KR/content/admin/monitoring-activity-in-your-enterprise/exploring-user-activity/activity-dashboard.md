작업 대시보드는 다음 항목의 수에 대한 주간, 월간, 연간 그래프를 제공합니다.
- 새로운 끌어오기 요청
- 병합된 끌어오기 요청
- 새 문제
- 종결된 문제
- 새 문제 주석
- 새 리포지토리
- 새 사용자 계정
- 새 조직
- 새 팀

![작업 대시보드](/assets/images/enterprise/activity/activity-dashboard-yearly.png)

## 작업 대시보드에 액세스

1. 페이지 위쪽에서 **탐색** 을 클릭합니다.
![탐색 탭](/assets/images/enterprise/settings/ent-new-explore.png)
2. 오른쪽 위 모서리에서 **작업** 을 클릭합니다.
![작업 단추](/assets/images/enterprise/activity/activity-button.png)
