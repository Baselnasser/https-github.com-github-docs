{% data reusables.github-ae.github-ae-enables-you %} {% 데이터 variables.location.product_location %}에 대한 모든 고객 데이터를 저장할 위치를 선택할 수 있으며, 엔터프라이즈 구성원은 전 세계 어디에서나 {% 데이터 variables.product.product_name %}에 액세스할 수 있습니다.

엔터프라이즈를 만들 때 {% data variables.contact.contact_enterprise_sales %}에게 고객 데이터를 저장할 위치를 알릴 수 있습니다. {% data variables.product.company_short %}는 선택한 지역 외부에 엔터프라이즈 데이터를 저장하거나 이동하지 않습니다.

사용 가능한 지역에 대한 자세한 내용은 {% data variables.contact.contact_enterprise_sales %}에 문의하세요.
