## 지원 자격 정보

엔터프라이즈 계정에 대한 지원 자격이 있는 사용자는 지원 포털을 사용하여 엔터프라이즈 계정과 연결된 지원 티켓을 열고 보고 설명을 달 수 있습니다.

엔터프라이즈 소유자 및 청구 관리자는 자동으로 지원 자격을 갖습니다. 엔터프라이즈 소유자는 지원 자격을 엔터프라이즈 계정에서 소유하고 있는 조직의 최대 20명의 추가 멤버에게 추가할 수 있습니다.

## 엔터프라이즈 멤버에 지원 자격 추가

{% note %}

**참고**: 지원 자격을 추가한 후 엔터프라이즈 멤버는 티켓을 관리하기 위해 로그아웃하고 다시 로그인해야 할 수 있습니다.

{% endnote %}

{% data reusables.enterprise-accounts.access-enterprise %} {% data reusables.enterprise-accounts.settings-tab %}
3. "설정"에서 **지원** 을 클릭합니다.
![지원 메뉴 항목](/assets/images/help/enterprises/settings-support.png)
4. "지원 구성원 추가"에서 지원 자격을 부여하려는 사용자의 이름 또는 사용자 이름을 입력하기 시작합니다. 일치 항목 목록에서 해당 이름을 클릭합니다.
![지원 자격 검색 추가](/assets/images/help/enterprises/settings-support-entitlement-search.png)
5. **지원 자격 추가** 를 클릭합니다.
![지원 자격 추가 단추](/assets/images/help/enterprises/settings-support-add-entitlement.png)

## 엔터프라이즈 멤버의 지원 자격 제거

엔터프라이즈 멤버의 지원 자격을 제거할 수 있습니다. 엔터프라이즈 소유자 또는 청구 관리자의 지원 자격은 제거할 수 없습니다.

{% data reusables.enterprise-accounts.access-enterprise %} {% data reusables.enterprise-accounts.settings-tab %}
3. "설정" 아래의 사이드바에서 **지원** 을 클릭합니다.
![지원 메뉴 항목](/assets/images/help/enterprises/settings-support.png)
4. "지원 멤버"의 지원 자격을 제거하려는 사용자의 오른쪽에서 {% octicon "trash" aria-label="The trash icon" %}을 클릭합니다.
![지원 자격 제거](/assets/images/help/enterprises/settings-support-remove-entitlement.png)
