## 문제 이벤트 API 정보

문제 이벤트 API는 문제 및 끌어오기 요청의 작업에 의해 트리거되는 다양한 유형의 이벤트를 반환할 수 있습니다. 문제 이벤트 API에서 받을 수 있는 특정 이벤트에 대한 자세한 내용은 "[문제 이벤트 유형](/developers/webhooks-and-events/issue-event-types)"을 참조하세요. 문제 및 끌어오기 요청 이외의 GitHub 작업용 이벤트 API도 사용할 수 있습니다. 자세한 내용은 “[이벤트 API](/developers/webhooks-and-events/github-event-types)”를 참조하세요.
