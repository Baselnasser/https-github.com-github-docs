## 검토 요청 API 정보

끌어오기 요청 작성자, 리포지토리 소유자 및 협력자는 리포지토리에 대한 쓰기 권한이 있는 모든 사람로부터 끌어오기 요청 검토를 요청할 수 있습니다. 요청된 각 검토자는 끌어오기 요청을 검토하도록 요청하는 알림을 받게 됩니다.
