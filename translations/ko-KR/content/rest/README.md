# REST (영문)

`/content/rest` 디렉터리에는 GitHub REST API 문서가 있습니다.

* `/content/rest/guides` 및 `/content/rest/overview` 디렉터리에는 일반 문서가 포함되어 있습니다. 이러한 항목은 사람이 편집할 수 있습니다.
* `/content/rest/reference` 디렉터리에는 GitHub REST API의 각 엔드포인트 그룹에 대한 문서가 포함되어 있습니다. 이 디렉터리에 있는 콘텐츠의 대부분은 `include` 태그를 사용하여 렌더링됩니다.

  `include` 태그로 렌더링되는 콘텐츠는 `/lib/rest/static` 디렉터리에서 오는 것으로, GitHub 내부의 API 소스 코드에서 자동으로 생성되며 사람이 편집해서는 안 됩니다. 자세한 내용은 [`/lib/rest/README.md`](/lib/rest/README.md)를 참조하세요.

  **`include` 태그로 렌더링되는 콘텐츠의 변경 내용은 허용할 수 없습니다. 그러나 보려는 변경 내용을 설명하는 문제를 열 수 있습니다.**
