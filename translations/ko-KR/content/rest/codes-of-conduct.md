## 사용 규정 API 정보

사용 규정 API를 사용하여 리포지토리의 사용 규정에 대한 정보를 검색할 수 있습니다. 리포지토리의 사용 규정을 가져오려면 “[리포지토리 가져오기](/rest/reference/repos#get-a-repository)” 엔드포인트를 사용합니다.
