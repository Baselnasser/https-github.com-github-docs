## Gist 설명 API 정보

Gist 설명 API를 사용하면 gist에 대한 설명을 보고 수정할 수 있습니다. gist에 대한 자세한 내용은 "[gist와 콘텐츠 편집 및 공유](/get-started/writing-on-github/editing-and-sharing-content-with-gists)"를 참조하세요.

### Gist 설명에 대한 사용자 지정 미디어 유형

다음은 gist 설명에 지원되는 미디어 유형입니다.

    application/vnd.github.VERSION.raw
    application/vnd.github.VERSION.base64

미디어 유형에 대한 자세한 내용은 “[사용자 지정 미디어 유형](/rest/overview/media-types)”을 참조하세요.
