3. 왼쪽 메뉴에서 **새 차트** 를 클릭합니다.
   ![새 쿼리 단추를 보여 주는 스크린샷](/assets/images/help/projects-v2/insights-new-chart.png)
4. 필요에 따라 새 차트의 이름을 변경하려면 {% octicon "triangle-down" aria-label="The triangle icon" %}을 클릭하고 새 이름을 입력한 후 <kbd>Enter</kbd> 키를 누릅니다.
   ![차트 이름을 바꾸는 방법을 보여 주는 스크린샷](/assets/images/help/projects-v2/insights-rename.png)
5. 차트 위에 필터를 입력하여 차트를 작성하는 데 사용되는 데이터를 변경합니다. 자세한 내용은 “[프로젝트 필터링](/issues/planning-and-tracking-with-projects/customizing-views-in-your-project/filtering-projects)”을 참조하세요.
6. 필터 텍스트 상자 오른쪽에서 **변경 내용 저장** 을 클릭합니다.
   ![저장 단추를 보여 주는 스크린샷](/assets/images/help/projects-v2/insights-save-filter.png)
