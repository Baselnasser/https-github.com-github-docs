## <a name="deleting-a-project"></a>프로젝트 삭제

프로젝트를 삭제하여 영구적으로 제거할 수 있습니다.

{% data reusables.projects.project-settings %}
1. 페이지 맨 아래에서 **이 프로젝트 삭제** 를 클릭합니다. 
   ![프로젝트 삭제 단추를 보여 주는 스크린샷](/assets/images/help/issues/delete-project-button.png)
1. 경고를 읽은 다음, 텍스트 상자에 프로젝트 이름을 입력합니다.
   ![프로젝트 삭제 확인을 보여 주는 스크린샷](/assets/images/help/issues/project-delete-confirm.png)
1. **결과를 이해하고 있으며 이 프로젝트를 삭제합니다.** 를 클릭합니다.

## <a name="closing-a-project"></a>프로젝트 닫기

프로젝트를 닫아 프로젝트 목록에서 제거할 수 있지만 나중에 프로젝트를 다시 열 수 있는 내용과 기능을 유지할 수 있습니다. 

{% data reusables.projects.project-settings %}
1. 페이지 아래쪽에서 **이 프로젝트 닫기** 를 클릭합니다. 
   ![프로젝트 닫기 단추를 보여 주는 스크린샷](/assets/images/help/issues/close-project-button.png)
