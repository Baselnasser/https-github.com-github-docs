조직 소유의 리포지토리에서 외부 협력자가 되기 위해 보류 중인 초대가 있는 모든 사람의 목록을 볼 수 있습니다.

보류 중인 각 협력자에 대해 조직 리포지토리에 동시에 참가하는 모든 초대를 취소할 수 있습니다.

{% data reusables.profile.access_org %} {% data reusables.user-settings.access_org %} {% data reusables.organizations.people %} {% data reusables.organizations.people_tab_outside_collaborators %}
5. **X 보류 중인 협력자** 를 클릭합니다.
  !["보류 중인 협력자" 단추](/assets/images/help/organizations/pending-collaborator-list.png)
6. 초대를 취소하려는 사용자의 오른쪽에서 **초대 취소** 를 클릭합니다.
  !["초대 취소" 단추](/assets/images/help/organizations/cancel-pending-collaborators.png)
7. **보류 중인 협력자의 초대 취소** 를 클릭합니다.
  ![취소를 확인하는 단추](/assets/images/help/organizations/confirm-cancelation-of-pending-collaborators.png)
