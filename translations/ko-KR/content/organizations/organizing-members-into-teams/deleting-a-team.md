**팁:** 조직 소유자만 부모 팀을 삭제할 수 있습니다. 자세한 내용은 “[팀 정보](/articles/about-teams)”를 참조하세요.

{% endtip %}

{% data reusables.profile.access_org %} {% data reusables.user-settings.access_org %} {% data reusables.organizations.teams %}
4. 삭제하려는 팀을 선택합니다.
  ![두 팀이 선택된 팀 목록](/assets/images/help/teams/list-of-teams-selected.png)
5. 팀 목록 위에 있는 드롭다운 메뉴를 사용하고 **삭제** 를 클릭합니다.
  ![팀 표시 유형 변경 옵션이 있는 드롭다운 메뉴](/assets/images/help/teams/team-bulk-management-options.png)
6. 삭제할 팀을 검토한 다음, **이해함, 팀 삭제** 를 클릭합니다.
  ![삭제할 팀 목록 및 팀 삭제 단추](/assets/images/help/teams/confirm-delete-teams-bulk.png)
