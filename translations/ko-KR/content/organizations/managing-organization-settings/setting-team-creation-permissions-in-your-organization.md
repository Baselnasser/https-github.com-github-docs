조직 소유자는 팀 만들기 권한을 설정할 수 있습니다.

팀 만들기 권한을 설정하지 않으면 모든 조직 멤버가 기본적으로 팀을 만들 수 있습니다.

{% data reusables.profile.access_org %} {% data reusables.profile.org_settings %} {% data reusables.organizations.member-privileges %}
5. “팀 만들기 규칙”에서 **멤버가 팀을 만들 수 있도록 허용** 을 선택하거나 선택 취소합니다.
![멤버가 팀을 만들 수 있도록 허용하는 확인란](/assets/images/help/organizations/allow-members-to-create-teams.png)
6. **저장** 을 클릭합니다.
