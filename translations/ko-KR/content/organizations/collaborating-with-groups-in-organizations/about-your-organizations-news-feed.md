조직의 뉴스 피드는 조직이 소유한 리포지토리에 대한 다른 사용자의 활동을 보여 줍니다. 조직의 뉴스 피드를 사용하여 누군가가 문제 또는 끌어오기 요청을 열거나 닫거나 병합하거나, 분기를 만들거나 삭제하거나, 태그 또는 릴리스를 만들거나, 문제, 끌어오기 요청 또는 커밋에 대한 의견을 작성하거나, {% data variables.product.product_name %}에 새 커밋을 푸시하는 시기를 확인할 수 있습니다.

## 조직의 뉴스 피드에 액세스

1. {% data variables.product.signin_link %}을(를) {% ifversion ghae %}{% data variables.product.product_name %}{% else %}{% data variables.location.product_location %}{% endif %}.
2. {% data reusables.user-settings.personal_dashboard %}를 엽니다.
3. 페이지의 왼쪽 상단 구석에 있는 계정 컨텍스트 전환기를 클릭합니다.
  ![Enterprise의 컨텍스트 전환기 단추](/assets/images/help/organizations/account_context_switcher.png)
4. 드롭다운 메뉴에서 조직을 선택합니다.{% ifversion fpt or ghec %} ![dotcom의 컨텍스트 전환기 메뉴](/assets/images/help/organizations/account-context-switcher-selected-dotcom.png){% else %} ![Enterprise의 컨텍스트 전환기 메뉴](/assets/images/help/organizations/account_context_switcher.png){% endif %}
