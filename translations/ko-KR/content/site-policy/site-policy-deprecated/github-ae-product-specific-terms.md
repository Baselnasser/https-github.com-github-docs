이러한 조건은 2021년 1월 4일 이전에 제품 라이선스를 받은 고객에게 적용됩니다. 해당 날짜 이후에 GitHub 제품을 구매한 고객은 https://www.github.com/enterprise-legal 의 현재 약관으로 안내됩니다.

본 계약은 본 GitHub AE 제품 관련 약관, 고객이 수락한 일반 약관 및 주문 시 GitHub 또는 해당 계열사가 제공하는 추가 약관으로 구성됩니다.

## 1. 계정.

**계정 책임.** 고객은 최종 사용자 계정 및 콘텐츠를 제어하고 담당합니다.

**계정 보안.** 고객은 계정 로그인 자격 증명의 보안을 유지할 책임이 있습니다.

**사용 정책.** 고객의 최종 사용자는 사용 제한 정책을 준수해야 합니다.

**일시 중지.** GitHub는 고객의 중대한 위반 기간 동안 온라인 서비스 사용을 일시 중단할 수 있습니다.

**액세스** GitHub는 지원 문제 또는 보안 목적에 필요한 경우가 아니면 고객 콘텐츠에 액세스하지 않습니다.

**DMCA.** GitHub에는 저작권 침해 청구에 적용되는 DMCA(디지털 밀레니엄 저작권법) 게시 중단 정책이 있습니다.

## 2. 콘텐츠.

**콘텐츠 소유권.** 고객은 자신이 만든 콘텐츠를 소유하며, 고객이 게시하는 콘텐츠와 관련된 제3자 라이선스를 완전히 준수합니다.

**GitHub에 대한 라이선스 허여.** GitHub가 온라인 서비스를 실행하는 데 필요한 권한을 부여하는 별도의 라이선스가 고객 콘텐츠에 함께 제공되지 않는 한, 고객은 온라인 서비스 또는 지원을 제공하거나 보안상 사유로 필요한 경우, 고객 콘텐츠를 사용하고 부본을 만들 수 있는 권리를 GitHub에 부여합니다. 또한 GitHub는 법률에 따라 강제로 고객 콘텐츠를 공개해야 할 수 있습니다.

## 3. 비 GitHub 제품.
GitHub는 온라인 서비스를 통해 비 GitHub 제품을 제공할 수 있습니다. 고객이 비 GitHub 제품을 온라인 서비스와 함께 사용할 경우, 고객은 본 계약에 명시적으로 포함된 것 이상의 의무를 GitHub의 지적 재산에 부과하는 방식으로 사용해서는 안 됩니다. GitHub는 비 GitHub 제품에 대한 법적 또는 도의적 책임을 지지 않습니다. 고객의 비 GitHub 제품 사용에는 고객과 비 GitHub 제품의 게시자 사이의 계약조건(있는 경우)이 적용됩니다.

## 4. 지원 및 SLA
온라인 서비스에는 지원 및 SLA가 포함됩니다.

## 5. 데이터 보호 및 보안.
데이터 보호 계약의 조건은 온라인 서비스에 적용됩니다.

## 6. 통지.
GitHub에 대한 통지는 다음으로 보내야 합니다. GitHub, Inc. Attn: Legal Dept., 88 Colin P. Kelly St, San Francisco, CA 94107 USA.

## 7. 용어 정의.
“**사용 제한 정책**”이란 GitHub에서 구매할 때 사이트 정책에 제공된 사용 제한 정책 및 Microsoft에서 구매할 때 Microsoft 온라인 서비스 사용 약관에 제공된 사용 제한 정책을 의미합니다.

“**콘텐츠**”란 텍스트, 데이터, 소프트웨어, 이미지 및 온라인 서비스를 통해 표시되거나 다른 방식으로 제공된 기타 모든 자료를 의미합니다.

“**고객 콘텐츠**”란 고객이 생성하고 소유하거나 고객이 권리를 보유하는 콘텐츠를 의미합니다.

“**데이터 보호 계약**”이란 https://docs.github.com/github/site-policy 나와 있는 GitHub AE 데이터 보호 계약입니다.

“**디지털 밀레니엄 저작권법 게시 중단 정책**”이란 https://docs.github.com/github/site-policy 에 나와 있는 GitHub의 저작권 침해 통지를 취급하는 프로세스를 의미입니다.

“**SLA**”이란 https://docs.github.com/github/site-policy 에 나와 있는 GitHub의 온라인 서비스에 대한 가동 시간 약정을 의미합니다.

“**지원**”이란 https://docs.github.com/github/site-policy 에 추가로 설명되어 있는 GitHub의 프리미엄 플러스 지원 프로그램을 의미합니다.
