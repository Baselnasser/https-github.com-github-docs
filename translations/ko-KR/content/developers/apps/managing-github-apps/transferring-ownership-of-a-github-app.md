6. **소유권 이전** 을 클릭합니다.

![전송할 앱의 이름을 입력하는 필드](/assets/images/github-apps/github_apps_transfer_app_name.png)
8. GitHub 앱을 전송하려는 사용자 또는 조직의 이름을 입력합니다.
![전송할 사용자 또는 조직을 입력하는 필드](/assets/images/github-apps/github_apps_transfer_new_owner.png)
9. **이 GitHub 앱 전송** 을 클릭합니다.
![GitHub 앱의 전송을 확인하는 단추](/assets/images/github-apps/github_apps_transfer_integration.png)
