GitHub 앱에는 기본적으로 사용 권한이 없습니다. GitHub 앱을 만들 때 최종 사용자 데이터에 액세스하는 데 필요한 권한을 선택할 수 있습니다. 사용 권한은 추가와 제거도 가능합니다. 자세한 내용은 “[GitHub 앱의 권한 편집](/apps/managing-github-apps/editing-a-github-app-s-permissions/)”을 참조하세요.
