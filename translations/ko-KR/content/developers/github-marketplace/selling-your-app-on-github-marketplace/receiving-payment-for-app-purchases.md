유료 플랜이 포함된 앱에 대한 {% data variables.product.prodname_marketplace %} 목록이 생성되어 승인되면 재무 온보딩 프로세스의 일환으로 {% data variables.product.product_name %}에 결제 세부 정보를 제공합니다.

한 달 수익이 최소 500달러에 도달하면 {% data variables.product.company_short %}에서 전자 결제를 받게 됩니다. 이는 마켓플레이스 거래의 수입에서 {% data variables.product.company_short %}에서 운영 비용을 충당하기 위해 청구하는 금액을 뺀 금액입니다.

2021년 1월 1일 이전에 이루어진 거래의 경우{% data variables.product.company_short %}는 거래 수익의 25%를 유지합니다. 이 날짜 이후에 이루어진 거래의 경우 {% data variables.product.company_short %}애서는 5%만 유지합니다. 이 변경 내용은 2021년 1월 말부터 받은 지불에 반영됩니다.

{% note %}

**참고:** 현재 가격 책정 및 결제 조건에 대한 자세한 내용은 “[{% data variables.product.prodname_marketplace %} 개발자 계약](/free-pro-team@latest/github/site-policy/github-marketplace-developer-agreement)”을 참조하세요.

{% endnote %}
