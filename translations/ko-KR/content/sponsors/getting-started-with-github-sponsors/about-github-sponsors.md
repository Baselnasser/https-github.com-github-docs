## {% data variables.product.prodname_matching_fund %} 정보

{% note %}

**참고:** {% data reusables.sponsors.matching-fund-eligible %}

{% endnote %}

{% data variables.product.prodname_matching_fund %}는 [{% data variables.product.prodname_dotcom %} Community 지침](/free-pro-team@latest/github/site-policy/github-community-guidelines)을 홍보하는 오픈 소스 소프트웨어를 개발하는 {% data variables.product.prodname_dotcom %} 커뮤니티의 구성원에게 혜택을 주는 것을 목표로 합니다. 스폰서 조직에 대한 지불 및 조직의 결제는 {% data variables.product.prodname_matching_fund %}에 해당되지 않습니다.

{% data variables.product.prodname_matching_fund %}를 받으려면 장기적으로 사용자를 지탱할 커뮤니티를 유치할 프로필을 만들어야 합니다. 강력한 프로필 만들기에 대한 자세한 내용은 “[{% data variables.product.prodname_sponsors %}에 대한 프로필 세부 정보 편집](/sponsors/receiving-sponsorships-through-github-sponsors/editing-your-profile-details-for-github-sponsors)”을 참조하세요.

스폰서 개발자 간의 기부금은 일치하지 않습니다.
