참조 문서에서 스폰서 GraphQL API에 대한 세부 정보를 찾을 수 있습니다. 자세한 내용은 "[GraphQL 참조](/graphql/reference)"를 참조하세요. GraphQL 탐색기를 사용하여 GraphQL 호출을 빌드하는 것이 좋습니다. 자세한 내용은 [Explorer 사용](/graphql/guides/using-the-explorer)을 참조하세요.
