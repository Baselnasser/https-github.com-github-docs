{% data reusables.sponsors.beta-note-sponsors-for-companies %}

회사는 청구서 메서드를 사용하여 {% data variables.product.prodname_sponsors %} 프로그램에 참여할 수 있습니다. 이는 회사가 {% data variables.product.company_short %}와 기존 청구된 청구 관계가 있는 경우 특히 편리할 수 있습니다. 청구서로 결제하는 회사는 관심 있는 {% data variables.product.prodname_dotcom_the_website %}의 공개 프로젝트에서 오픈 소스 기여 및 작업을 측정하는 {% data variables.product.product_name %} 대시보드에 액세스할 수 있습니다.

영업 담당자에게 문의하거나 [다음으로 문의하여](https://support.github.com/contact/org-sponsors-waitlist) 이 베타 프로그램에 참가할 수 있습니다.
