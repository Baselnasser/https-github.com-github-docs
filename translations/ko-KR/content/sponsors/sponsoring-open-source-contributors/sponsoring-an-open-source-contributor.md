## 스폰서쉽 정보

{% data reusables.sponsors.sponsorship-details %}

개인 계정을 대신하여 계정을 후원하여 개인적으로 혜택을 받는 프로젝트에 투자할 수 있습니다. 여러 가지 이유로 조직을 대신하여 계정을 후원할 수 있습니다.
- 조직의 작업이 의존하는 특정 라이브러리 유지
- 조직(예: 블록체인)으로 사용하는 에코시스템에 투자
- 오픈 소스를 중시하는 조직으로서의 브랜드 인지도 개발
- 조직에서 제공하는 제품을 보완하는 라이브러리를 빌드한 오픈 소스 개발자에게 감사

신용 카드를 사용하여 {% data variables.product.product_name %}에서 계정을 후원할 수 있습니다. 조직에서 청구서로 결제하려는 경우 [청구서를 통한 {% data variables.product.prodname_sponsors %}에 대한 지불](/sponsors/sponsoring-open-source-contributors/paying-for-github-sponsors-via-invoice)에서 자세한 내용을 확인할 수 있습니다.

{% data reusables.sponsors.no-fees %} 자세한 내용은 “[{% data variables.product.prodname_sponsors %}에 대한 청구 정보](/articles/about-billing-for-github-sponsors)”를 참조하세요.

신용 카드를 사용하여 계정을 후원하면 변경 내용이 즉시 적용됩니다. {% data reusables.sponsors.prorated-sponsorship %}

특정 제한된 세금 정보를 후원된 계정과 공유할 수 있습니다. 자세한 내용은 “[세금 정보](#tax-information)”를 참조하세요.

{% data reusables.sponsors.manage-updates-for-orgs %}

스폰서쉽을 공개적으로 표시할지 여부를 선택할 수 있습니다. 일회성 스폰서쉽은 한 달 동안 계속 볼 수 있습니다. 

후원된 계정이 계층 사용을 중지하는 경우 다른 계층을 선택하거나 구독을 취소할 때까지 계층이 그대로 유지됩니다. 자세한 내용은 “[스폰서쉽 업그레이드](/articles/upgrading-a-sponsorship)” 및 “[스폰서쉽 다운그레이드](/articles/downgrading-a-sponsorship)”를 참조하세요.

후원하려는 계정에 {% data variables.product.prodname_sponsors %}에 대한 프로필이 없는 경우 계정 가입을 권장할 수 있습니다. 자세한 내용은 “[개인 계정에 대한 {% data variables.product.prodname_sponsors %} 설정](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account)” 및 “[조직에 대한 {% data variables.product.prodname_sponsors %} 설정](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization)”을 참조하세요.

{% data reusables.sponsors.sponsorships-not-tax-deductible %}

{% note %}

**참고:** {% data variables.product.prodname_dotcom %}은(는) 개발자가 자신을 나타내는 방식에 대한 책임이 없으며 {% data variables.product.prodname_dotcom %}은(는) 후원된 오픈 소스 프로젝트를 보증하지 않습니다. 클레임은 전적으로 자금을 받는 개발자의 책임입니다. 스폰서쉽을 제공하기 전에 사람을 신뢰해야 합니다. 자세한 내용은 [{% data variables.product.prodname_sponsors %} 추가 약관](/free-pro-team@latest/github/site-policy/github-sponsors-additional-terms)을 참조하세요.

{% endnote %}

## 세금 정보

귀하는 스폰서로서 스폰서 프로그램이 시작된 이후로 귀하가 후원하는 각 계정의 소유자에게 계정에 대한 스폰서쉽 지불에 대한 다음과 같은 제한된 정보를 공개할 수 있음을 인정합니다.

- 트랜잭션 날짜
- 지불된 금액
- 지불이 이루어진 국가, 주, 지방
- 비즈니스 또는 개인이 지불했는지 여부

이 정보는 이러한 스폰서쉽 지불로 인해 발생하는 세금을 납부하고 보고할 수 있도록 하는 데 필요합니다.

## 계정 후원

계정을 후원하려면 먼저 확인된 메일 주소가 있어야 합니다. 자세한 내용은 “[메일 주소 확인](/github/getting-started-with-github/verifying-your-email-address)”을 참조하세요.

1. {% data variables.product.product_name %}에서 후원하려는 사용자 또는 조직의 프로필로 이동합니다.
1. 계정에 대한 스폰서쉽 대시보드로 이동합니다.
   - 개발자를 후원하려면 개발자 이름 아래에서 **스폰서** 를 클릭합니다.
     ![스폰서 단추](/assets/images/help/profile/sponsor-button.png)
   - 조직을 후원하려면 조직 이름 오른쪽에 있는 **스폰서** 를 클릭합니다.
     ![스폰서 단추](/assets/images/help/sponsors/sponsor-org-button.png)
1. 필요에 따라 페이지 오른쪽에서 조직을 대신하여 계정을 후원하려면 **다음 자격으로 후원** 드롭다운 메뉴를 사용하고 조직을 클릭합니다.
