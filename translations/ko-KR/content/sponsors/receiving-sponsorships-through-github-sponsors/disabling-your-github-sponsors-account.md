스폰서쉽 지급을 받는 경우 GitHub 지원에 문의하여 {% data variables.product.prodname_sponsors %} 계정을 비활성화해야 합니다.

스폰서가 없는 경우 GitHub에서 {% data variables.product.prodname_sponsors %} 계정을 비활성화할 수 있습니다.

{% data variables.product.prodname_sponsors %} 계정을 다시 만들기로 결정한 경우 다시 적용해야 합니다.

또는 스폰서 프로필을 게시 취소한 다음, 다시 적용하지 않고 계정을 다시 게시할 수 있습니다. 자세한 내용은 "[{% data variables.product.prodname_sponsors %} 프로필 게시 취소](/sponsors/receiving-sponsorships-through-github-sponsors/unpublishing-your-github-sponsors-profile)"를 참조하세요.

{% data reusables.sponsors.navigate-to-sponsors-dashboard %} {% data reusables.sponsors.settings-tab %}
1. "GitHub 스폰서 계정 사용 안 함"에서 **계정 사용 안 함** 을 클릭합니다.  
  !["계정 사용 안 함" 단추](/assets/images/help/sponsors/disable-your-account-button.png)
2. 확인하려면 **GitHub 스폰서 계정 사용 안 함** 을 클릭합니다.
  !["GitHub 스폰서 계정 사용 안 함" 단추](/assets/images/help/sponsors/disable-github-sponsors-account-dialog.png)
