## 회계 호스트를 사용하여 {% data variables.product.prodname_sponsors %} 지급을 받는 정보

후원을 받을 수 있도록 {% data variables.product.prodname_sponsors %} 프로필에 등록하는 경우 사용자가 제어하는 Stripe Connect 계정에서 또는 이미 작업 중인 회계 호스트를 통해 결제를 받을지 선택할 수 있습니다.

{% data reusables.sponsors.bank-info-fiscal-host-reminder %}

{% data variables.product.prodname_sponsors %}은 다음 회계 호스트를 지원합니다.

- [오픈 소스 집단](https://oscollective.org/)
- [NumFOCUS](http://www.numfocus.org/)
- [오픈 컬렉티브 유럽](https://opencollective.com/europe)
- [소프트웨어 지하](https://softwareunderground.org/)
- [소프트웨어 자유 보존](https://sfconservancy.org/)

## 회계 호스트 사용 선택

후원하려는 계정에 대한 "대기 목록 가입" 프로필 설정 페이지로 이동하면 설정한 계정 유형에 따라 "지급액을 받는 방법" 또는 "조직 은행 계좌" 섹션이 표시됩니다. {% data variables.product.prodname_sponsors %} 프로필에 등록하는 방법에 대한 자세한 내용은 "[개인 계정에 대해 {% data variables.product.prodname_sponsors %} 설정](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account)" 또는 "[조직에 대한 {% 데이터 variables.product.prodname_sponsors %} 설정](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization)"을 참조하세요.

1. "지급을 받는 방법" 또는 "조직 은행 계좌"에서 **회계 호스트 사용** 또는 **이 조직에서 회계 호스트 사용** 옵션을 선택합니다.

   ![은행 계좌와 회계 호스트 중에서 선택하는 옵션의 스크린샷](/assets/images/help/sponsors/user-waitlist-payout-options.png)
2. **재정 호스트 선택** 드롭다운을 사용하여 사용할 회계 호스트를 선택합니다.
3. "회계 호스트 프로젝트 프로필 URL" 아래의 텍스트 상자에 회계 호스트 사이트의 프로필 페이지에 URL을 입력합니다. 이렇게 하면 실제로 회계 호스트에 등록되어 있는지 확인하는 데 도움이 됩니다.
  
  ![회계 호스트의 프로젝트 프로필 URL 텍스트 상자 스크린샷](/assets/images/help/sponsors/fiscal-host-project-profile-field.png)
