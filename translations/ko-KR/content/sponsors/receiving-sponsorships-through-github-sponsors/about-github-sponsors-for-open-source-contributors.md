## {% data variables.product.prodname_sponsors %}에 참가

{% data reusables.sponsors.you-can-be-a-sponsored-developer %} 자세한 내용은 “[개인 계정에 대한 {% data variables.product.prodname_sponsors %} 설정](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account)”을 참조하세요.

{% data reusables.sponsors.you-can-be-a-sponsored-organization %} 자세한 내용은 “[조직에 대한 {% data variables.product.prodname_sponsors %} 설정](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization)”을 참조하세요.

{% data variables.product.prodname_sponsors %}에 참가한 후 기여하는 오픈 소스 리포지토리에 스폰서 단추를 추가하여 {% data variables.product.prodname_sponsors %} 프로필 및 기타 자금 조달 플랫폼의 가시성을 높일 수 있습니다. 자세한 내용은 “[리포지토리에 스폰서 단추 표시](/articles/displaying-a-sponsor-button-in-your-repository)”를 참조하세요.

스폰서쉽에 대한 목표를 설정할 수 있습니다. 자세한 내용은 “[스폰서쉽 목표 관리](/sponsors/receiving-sponsorships-through-github-sponsors/managing-your-sponsorship-goal)”를 참조하세요.

## 스폰서쉽 계층

{% data reusables.sponsors.tier-details %} 자세한 내용은 “[개인 계정에 대한 {% data variables.product.prodname_sponsors %} 설정](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account)”, “[조직에 대한 {% data variables.product.prodname_sponsors %} 설정](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization)” 및 “[스폰서쉽 계층 관리](/sponsors/receiving-sponsorships-through-github-sponsors/managing-your-sponsorship-tiers)”를 참조하세요.

누구나 작업을 쉽게 지원할 수 있도록 월별 및 일회성 계층을 포함하여 다양한 스폰서쉽 옵션을 설정하는 것이 가장 좋습니다. 특히 일회성 결제를 통해 사람들은 재정이 정기적인 결제 일정을 감당할 수 있을지 여부에 대해 걱정하지 않고도 귀하의 노력을 보상할 수 있습니다.
