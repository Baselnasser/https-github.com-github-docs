## {% data variables.product.prodname_sponsors %}의 지급 정보

Stripe Connect를 통해 지급된 경우 {% data variables.product.prodname_sponsors %}에서만 지급을 관리할 수 있습니다. 스폰서 개인 계정의 ACH 이체 또는 전신 송금을 통해 지급되는 경우 스폰서 오픈 소스 기여자 대시보드에 **지급** 탭이 표시되지 않습니다. 스폰서 조직은 지급을 위해 Stripe Connect만 사용할 수 있습니다. 자세한 내용은 "[오픈 소스 기여자의 {% data variables.product.prodname_sponsors %} 정보](/sponsors/receiving-sponsorships-through-github-sponsors/about-github-sponsors-for-open-source-contributors#sponsorship-payouts)"를 참조하세요.
