## 리포지토리에 {% data variables.product.prodname_discussions %} 사용 또는 사용 안 함

{% data reusables.discussions.enabling-or-disabling-github-discussions-for-your-repository %}
1. 토론을 사용하지 않도록 설정하려면 “기능”에서 **토론** 선택을 취소합니다.
