사용자가 레이블을 검색할 때 찾을 수 있도록 공용 리포지토리에서 문제에 `good first issue` 레이블을 적용할 수 있습니다. 레이블별로 검색하는 방법에 대한 자세한 내용은 “[문제 및 끌어오기 요청 검색](/search-github/searching-on-github/searching-issues-and-pull-requests/#search-by-label)”을 참조하세요.

{% data variables.product.prodname_dotcom %}은(는) 알고리즘을 사용하여 각 리포지토리에서 가장 접근 가능한 문제를 확인하고 {% data variables.product.prodname_dotcom_the_website %}의 다양한 위치에 표시합니다. `good first issue` 레이블을 추가하면 문제가 표시될 가능성이 높아질 수 있습니다.
