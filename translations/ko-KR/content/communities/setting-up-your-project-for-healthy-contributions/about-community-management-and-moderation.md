## 커뮤니티 관리 및 조정 정보

{% data reusables.policies.github-community-guidelines-and-terms %}

또한 {% data variables.product.prodname_dotcom %}는 리포지토리 유지 관리자가 {% data variables.product.prodname_dotcom_the_website %}의 자체 커뮤니티에서 동작의 더 높은 기준을 정의하고 적용할 수 있는 도구를 제공합니다.

조직 관리자는 조직 구성원을 조정자로 지정할 수 있습니다. 이렇게 하면 해당 구성원이 조직 담당자만이 사용할 수 있었을 조정 기능에 액세스할 수 있습니다. 자세한 내용은 “[조직의 조정자 관리](/organizations/managing-peoples-access-to-your-organization-with-roles/managing-moderators-in-your-organization)”를 참조하세요.

커뮤니티에 대한 행동 기준을 수립하면 신속하고 공정한 방식으로 문제성 행동에 대응하여 즐겁고 생산적인 환경을 도모할 수 있습니다. 효과적인 조정을 통해 커뮤니티 내에서 신뢰를 구축하고 참여자가 안전하고 편안함을 느낄 수 있습니다.

## 커뮤니티 조정 도구

{% data variables.product.prodname_dotcom %}의 도구는 광범위한 조정 환경 없이도 커뮤니티의 관리 및 조정 작업을 간소화합니다.

리포지토리 유지 관리자는 프로젝트 참여에 대한 지침을 작성하고 프로젝트의 리포지토리에서 공유할 수 있습니다. {% data variables.product.prodname_dotcom %}는 커뮤니티 상태 파일을 작성하고 게시하는 데 도움이 되는 커뮤니티 프로필 체크리스트를 제공합니다. 프로젝트에 대한 잠재적 참여자는 커뮤니티 프로필 체크리스트를 검토하여 커뮤니티의 기준에 대해 알아보고 참여 여부를 결정할 수 있습니다. 자세한 내용은 “[건전한 참여를 위한 프로젝트 설정](/communities/setting-up-your-project-for-healthy-contributions)”을 참조하세요.

{% data reusables.community.tools-for-moderating %}

참가자는 언제든 {% data variables.contact.github_support %}에 문제성 콘텐츠를 보고할 수 있습니다. 조직의 퍼블릭 리포지토리에 대해 보고된 콘텐츠를 옵트인하는 경우 참여자는 문제성 동작이 포함된 댓글을 리포지토리 유지 관리자에게 직접 보고할 수도 있습니다. 자세한 내용은 “[조직의 리포지토리에서 보고된 콘텐츠 관리](/communities/moderating-comments-and-conversations/managing-reported-content-in-your-organizations-repository)”를 참조하세요.
