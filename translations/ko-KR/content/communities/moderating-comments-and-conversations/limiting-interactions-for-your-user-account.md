## <a name="about-temporary-interaction-limits"></a>임시 상호 작용 제한 정보

개인 계정에 대한 상호 작용을 제한하면 개인 계정이 소유한 모든 퍼블릭 리포지토리에 대한 임시 상호 작용 제한을 사용할 수 있습니다. {% data reusables.community.interaction-limits-restrictions %}

{% data reusables.community.interaction-limits-duration %} 제한 기간이 지나면 사용자는 퍼블릭 리포지토리에서 정상적인 작업을 계속할 수 있습니다.

{% data reusables.community.types-of-interaction-limits %}

사용자 차원의 활동 제한을 사용하도록 설정하면 개별 리포지토리에서 상호 작용 제한을 사용하거나 사용하지 않도록 설정할 수 없습니다. 개별 리포지토리의 활동 제한에 대한 자세한 내용은 “[리포지토리에서 상호 작용 제한](/communities/moderating-comments-and-conversations/limiting-interactions-in-your-repository)”을 참조하세요.

사용자를 차단할 수도 있습니다. 자세한 내용은 “[개인 계정에서 사용자 차단](/communities/maintaining-your-safety-on-github/blocking-a-user-from-your-personal-account)”을 참조하세요.

{% data reusables.user-settings.access_settings %}
1. 사이드바의 “액세스” 섹션에서 **{% octicon "report" aria-label="The report icon" %} 조정** 을 선택한 다음 **상호 작용 제한** 을 클릭합니다.
