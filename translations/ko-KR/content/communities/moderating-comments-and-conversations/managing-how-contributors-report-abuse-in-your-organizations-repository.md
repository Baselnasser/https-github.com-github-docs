리포지토리에 대한 관리자 권한이 있는 모든 사용자는 참여자가 리포지토리에서 남용을 보고하는 방식을 관리할 수 있습니다.

조직에서 소유한 퍼블릭 리포지토리에 대해 보고된 콘텐츠를 사용하거나 사용하지 않도록 설정할 수 있습니다.

{% data reusables.repositories.navigate-to-repo %} {% data reusables.repositories.sidebar-settings %} {% data reusables.repositories.sidebar-moderation-reported-content %}
4. “보고된 콘텐츠”에서 **협업자 및 이전 참여자의 콘텐츠 보고서 수락** 을 선택하거나 선택 취소합니다.
