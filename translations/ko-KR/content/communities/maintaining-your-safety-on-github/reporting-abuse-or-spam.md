## 문제 또는 끌어오기 요청 보고

1. 보고하려는 문제 또는 끌어오기 요청으로 이동합니다.
2. 문제 또는 끌어오기 요청의 오른쪽 위 모서리에서 {% octicon "kebab-horizontal" aria-label="The horizontal kebab octicon" %}을 클릭한 다음 **콘텐츠 보고** 를 클릭합니다.
  ![댓글 보고 단추](/assets/images/help/repository/menu-report-issue-or-pr.png) {% data reusables.community.report-content %}

## 댓글 보고

1. 보고하려는 댓글로 이동합니다.
2. 주석의 오른쪽 위에서 {% octicon "kebab-horizontal" aria-label="The horizontal kebab octicon" %}을 클릭한 다음 **콘텐츠 차단** 을 클릭합니다.
![댓글 보고 옵션이 있는 케밥 메뉴](/assets/images/help/repository/menu-report-comment.png) {% data reusables.community.report-content %}

## 템플릿 선택기에서 연락처 링크 남용 보고

1. 보고하려는 연락처 링크가 포함된 리포지토리로 이동합니다.
2. 리포지토리 이름 아래에서 {% octicon "issue-opened" aria-label="The issues icon" %} **문제** 를 클릭합니다.
3. 템플릿 선택기의 오른쪽 아래 모서리에서 **남용 보고** 를 클릭합니다.
  ![남용 보고 링크](/assets/images/help/repository/template-chooser-report-abuse.png)
4. 연락처 양식을 작성하여 {% data variables.contact.contact_support %}에게 연락처 링크의 동작에 대해 알리고 **요청 보내기** 를 클릭합니다.
