{% data variables.product.product_name %}은(는) 쿠폰을 적용하기 전에 계정에 대해 결제하는 경우 환불을 발급할 수 없습니다. 또한 잘못된 계정에 적용되는 경우 사용된 쿠폰을 양도하거나 새 쿠폰을 제공할 수 없습니다. 쿠폰을 사용하기 전에 올바른 계정에 쿠폰을 적용하고 있는지 확인합니다.

{% data reusables.dotcom_billing.coupon-expires %}

{% data variables.product.prodname_marketplace %} 앱의 유료 플랜에는 쿠폰을 적용할 수 없습니다.

## 개인 계정에 대한 쿠폰 사용

{% data reusables.dotcom_billing.enter_coupon_code_on_redeem_page %}
4. “쿠폰 사용”에서 *개인* 계정의 사용자 이름 옆에 있는 **선택** 을 클릭합니다.
   ![단추 선택](/assets/images/help/settings/redeem-coupon-choose-button-for-personal-accounts.png) {% data reusables.dotcom_billing.redeem_coupon %}

## 조직에 대한 쿠폰 사용

{% data reusables.dotcom_billing.org-billing-perms %}

{% data reusables.dotcom_billing.enter_coupon_code_on_redeem_page %}
4. “쿠폰 사용”에서 쿠폰을 적용할 *조직* 옆에 있는 **선택** 을 클릭합니다. 아직 존재하지 않는 새 조직에 쿠폰을 적용하려면 **새 조직 만들기** 를 클릭합니다.
   ![단추 선택](/assets/images/help/settings/redeem-coupon-choose-button.png) {% data reusables.dotcom_billing.redeem_coupon %}
