## 개인 계정에 대한 {% data variables.product.prodname_copilot %}의 평가판 설정

{% data variables.product.prodname_copilot %}을 사용하기 전에 평가판 또는 구독을 설정해야 합니다.

{% data reusables.copilot.signup-procedure %}

## {% data variables.product.prodname_copilot %} 구독 수정

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.billing_plans %}
1. “{% data variables.product.prodname_copilot %}” 섹션에서 **편집** 드롭다운을 선택합니다.
   - 월별 청구 주기를 사용하는 경우 **연간 청구로 변경** 을 선택합니다.
   - 연간 청구 주기를 사용하는 경우 **월별 청구로 변경** 을 선택합니다.

   ![편집 드롭다운이 강조 표시된 청구 페이지의 {% data variables.product.prodname_copilot %} 섹션 스크린샷](/assets/images/help/copilot/copilot-settings-edit-dropdown.png)
   

## {% data variables.product.prodname_copilot %} 구독 취소

언제든지 {% data variables.product.prodname_copilot %} 구독을 취소할 수 있습니다. 60일 평가판 기간 중에 취소하는 경우 요금이 청구되지 않습니다. 60일 평가판이 끝나기 전에 {% data variables.product.prodname_copilot %} 구독을 취소하지 않으면 평가판을 설정할 때 선택한 청구 주기에 따라 요금이 청구됩니다.

{% data reusables.user-settings.access_settings %} {% data reusables.user-settings.billing_plans %}
1. “{% data variables.product.prodname_copilot %}” 섹션에서 **편집** 드롭다운을 선택합니다.
   - 유료 구독이 있는 경우 **{% data variables.product.prodname_copilot %} 취소** 를 클릭합니다.
   
   ![취소 {% data variables.product.prodname_copilot %} 옵션이 강조 표시된 청구 페이지의 {% data variables.product.prodname_copilot %} 섹션 스크린샷](/assets/images/help/copilot/copilot-billing-edit-dropdown.png)

   - 60일 평가판 기간 내에 있는 경우 **평가판 취소** 를 클릭합니다.
   
   ![평가판 취소 옵션이 강조 표시된 청구 페이지의 {% data variables.product.prodname_copilot %} 섹션 스크린샷](/assets/images/help/copilot/copilot-cancel-trial.png)

2. “{% data variables.product.prodname_copilot %}” 모달에서 **{% data variables.product.prodname_copilot %} 취소** 를 클릭합니다.

   ![{% data variables.product.prodname_copilot %} 내가 이해한 모달 취소, {% data variables.product.prodname_copilot %} 취소 단추 강조 스크린샷](/assets/images/help/copilot/copilot-cancel-modal.png)

   - 60일 평가판 기간 내에 있는 경우 이해합니다를 클릭하고 **{% data variables.product.prodname_copilot %} 평가판을 취소합니다**.

   ![{% data variables.product.prodname_copilot %} 내가 이해한 평가판 모달 취소, 평가판 취소 단추 강조 스크린샷](/assets/images/help/copilot/copilot-trial-cancel-modal.png)
