## 엔터프라이즈에 대한 청구서 정보

{% data reusables.billing.about-invoices-for-enterprises %} 자세한 내용은 “[엔터프라이즈에 대한 청구 정보](/billing/managing-billing-for-your-github-account/about-billing-for-your-enterprise)”를 참조하세요.

엔터프라이즈 계정의 구독 및 사용량을 보는 방법에 대한 자세한 내용은 “[엔터프라이즈 계정의 구독 및 사용량 보기](/billing/managing-billing-for-your-github-account/viewing-the-subscription-and-usage-for-your-enterprise-account)”를 참조하세요.

## 현재 청구서 보기

{% data reusables.enterprise-accounts.access-enterprise-on-dotcom %} {% data reusables.enterprise-accounts.settings-tab %} {% data reusables.enterprise-accounts.billing-tab %}
4. “현재 청구서”에서 **청구서 보기** 를 클릭합니다.
  ![청구서 링크 보기](/assets/images/help/business-accounts/view-invoice-link.png)

## 현재 청구서 지불

{% data reusables.enterprise-accounts.access-enterprise-on-dotcom %} {% data reusables.enterprise-accounts.settings-tab %} {% data reusables.enterprise-accounts.billing-tab %}
4. “결제 기한”에서 **청구서 지불** 을 클릭합니다.
  ![청구서 지불 링크](/assets/images/help/business-accounts/pay-invoice-link.png)
5. “청구서 지불”에서 신용 카드 정보를 보안 양식으로 입력한 다음, **청구서 지불** 을 클릭합니다.
  ![청구서 확인 및 지불](/assets/images/help/business-accounts/pay-invoice.png)

## 현재 청구서 다운로드

{% data reusables.enterprise-accounts.access-enterprise-on-dotcom %} {% data reusables.enterprise-accounts.settings-tab %} {% data reusables.enterprise-accounts.billing-tab %}
4. “빠른 작업”에서 **현재 청구서 다운로드** 를 클릭합니다.
  ![현재 청구서 링크 다운로드](/assets/images/help/business-accounts/download-current-invoice.png)

## 결제 기록 보기

{% data reusables.enterprise-accounts.access-enterprise-on-dotcom %} {% data reusables.enterprise-accounts.settings-tab %} {% data reusables.enterprise-accounts.billing-tab %}
4. 과거 청구 활동의 요약을 보려면 “청구”에서 **과거 청구서** 탭을 클릭합니다. ![결제 기록 보기 탭](/assets/images/help/business-accounts/view-payment-history.png)
