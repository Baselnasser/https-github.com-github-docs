**팁**: {% data variables.product.prodname_dotcom %} 할인은 다른 유료 제품 및 기능에 대한 구독에는 적용되지 않습니다.

{% endtip %}

## 개인 계정에 대한 할인

{% data variables.product.prodname_free_user %}가 있는 학생 및 교직원을 위한 무제한 퍼블릭 및 프라이빗 리포지토리 외에도 확인된 학생은 {% data variables.product.prodname_student_pack %}을 신청하여 {% data variables.product.prodname_dotcom %} 파트너로부터 추가 혜택을 받을 수 있습니다. 자세한 내용은 “[학생으로 {% data variables.product.prodname_global_campus %}에 신청](/free-pro-team@latest/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/apply-to-github-global-campus-as-a-student)”을 참조하세요.

## 학교 및 대학 할인

검증된 교직원은 교육 또는 학술 연구를 위해 {% data variables.product.prodname_team %}을 신청할 수 있습니다. 자세한 내용은 “[교사용 {% data variables.product.prodname_global_campus %}](/free-pro-team@latest/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-teachers)”을 참조하세요. 또한 학생들을 위한 교육 자료도 요청할 수 있습니다. 자세한 내용은 [{% data variables.product.prodname_education %}](https://education.github.com/)을 방문하세요.

## 비영리 조직 및 도서관 할인

{% data variables.product.product_name %}은(는) 무제한 프라이빗 리포지토리, 무제한 협력자 및 적격 501(c)3(또는 이에 상응하는) 조직에 대한 전체 기능 집합이 있는 조직과 도서관에 무료로 {% data variables.product.prodname_team %}을 제공합니다. [비영리 페이지에서](https://github.com/nonprofit) 조직에 대한 할인을 요청할 수 있습니다.

조직에 이미 유료 구독이 있는 경우 비영리 할인이 적용되면 조직의 마지막 트랜잭션이 환불됩니다.
