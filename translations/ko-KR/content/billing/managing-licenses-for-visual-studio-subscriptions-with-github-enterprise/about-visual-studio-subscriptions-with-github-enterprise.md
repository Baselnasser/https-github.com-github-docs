## {% data variables.visual_studio.prodname_vss_ghe %} 정보

{% data reusables.enterprise-accounts.vss-ghe-description %} {% data variables.visual_studio.prodname_vss_ghe %}는 Microsoft 기업계약 약관에 따라 Microsoft에서 사용할 수 있습니다. 자세한 내용은 [{% data variables.product.prodname_vs %} 웹 사이트의 {% data variables.visual_studio.prodname_vss_ghe %}](https://visualstudio.microsoft.com/subscriptions/visual-studio-github/) 를 참조하세요.

라이선스의 {% data variables.product.prodname_enterprise %} 부분을 사용하려면 {% data variables.product.prodname_dotcom_the_website %}에 있는 각 구독자의 개인 계정이 {% data variables.product.prodname_dotcom_the_website %}에서 엔터프라이즈가 소유한 조직의 구성원이어야 합니다. 이를 위해 조직 소유자는 메일 주소로 새 구성원을 조직에 초대할 수 있습니다. 구독자는 {% data variables.product.prodname_dotcom_the_website %}에서 기존 개인 계정으로 초대를 수락하거나 새 계정을 만들 수 있습니다.

{% data variables.visual_studio.prodname_vss_ghe %}의 설정에 대한 자세한 내용은 "[{% data variables.visual_studio.prodname_vss_ghe %} 설정](/billing/managing-licenses-for-visual-studio-subscriptions-with-github-enterprise/setting-up-visual-studio-subscriptions-with-github-enterprise)"을 참조하세요.

## {% data variables.visual_studio.prodname_vss_ghe %}에 대한 라이선스 정보

{% data variables.visual_studio.prodname_vss_ghe %}에 대한 라이선스를 구독자에게 할당하면 구독자는 {% data variables.product.prodname_dotcom_the_website %}의 개인 계정으로 엔터프라이즈의 조직에 가입하여 라이선스의 {% data variables.product.prodname_enterprise %} 부분을 사용합니다. {% data variables.product.prodname_dotcom_the_website %}에서 엔터프라이즈 구성원의 개인 계정에 대해 확인된 이메일 주소가 {% data variables.product.prodname_vs %} 계정에 대한 구독자의 UPN(사용자 기본 이름)과 일치하는 경우 {% data variables.product.prodname_vs %} 구독자는 {% data variables.visual_studio.prodname_vss_ghe %}에 대해 하나의 라이선스를 자동으로 사용합니다.

{% data variables.product.prodname_dotcom %}에 대한 엔터프라이즈 라이선스의 총 수량은 표준 {% data variables.product.prodname_enterprise %} 라이선스의 합계와 {% data variables.product.prodname_dotcom %}에 대한 액세스가 포함된 {% data variables.product.prodname_vs %} 구독 라이선스의 수입니다. 엔터프라이즈 구성원의 개인 계정이 {% data variables.product.prodname_vs %} 구독자의 메일 주소와 일치하지 않는 경우 {% data variables.product.prodname_vs %} 구독자에 대해 개인 계정에서 사용하는 라이선스를 사용할 수 없습니다.

{% data variables.product.prodname_enterprise %}에 대한 자세한 내용은 “[{% data variables.product.company_short %}의 제품](/github/getting-started-with-github/githubs-products#github-enterprise)”을 참조하세요. {% data variables.product.prodname_dotcom_the_website %}의 계정에 대한 자세한 내용은 “[{% data variables.product.prodname_dotcom %} 계정 유형](/github/getting-started-with-github/types-of-github-accounts)”을 참조하세요.

{% data variables.location.product_location %}에서 엔터프라이즈에서 사용할 수 있는 {% data variables.product.prodname_enterprise %} 라이선스 수를 볼 수 있습니다. 보류 중인 초대 목록에는 아직 엔터프라이즈에서 하나 이상의 조직의 구성원이 아닌 구독자가 포함됩니다. 자세한 내용은 “[엔터프라이즈 계정의 구독 및 사용량 보기](/billing/managing-billing-for-your-github-account/viewing-the-subscription-and-usage-for-your-enterprise-account)” 및 “[엔터프라이즈의 사용자 보기](/admin/user-management/managing-users-in-your-enterprise/viewing-people-in-your-enterprise#viewing-members-and-outside-collaborators)”를 참조하세요.

{% tip %}

**팁**: “[엔터프라이즈 계정의 구독 및 사용량 보기](/billing/managing-billing-for-your-github-account/viewing-the-subscription-and-usage-for-your-enterprise-account#viewing-the-subscription-and-usage-for-your-enterprise-account)”의 6단계에서 엔터프라이즈 라이선스 사용량으로 CSV 파일을 다운로드하는 경우 “이름” 또는 “프로필” 열에 대한 값이 누락된 모든 구성원은 아직 엔터프라이즈 내 조직에 가입하라는 초대를 수락하지 않은 것입니다.

{% endtip %}

{% data variables.visual_studio.prodname_vss_admin_portal_with_url %}에서 구독자에 대한 보류 중인 {% data variables.product.prodname_enterprise %} 초대를 볼 수도 있습니다.
