## 추가 스토리지 및 대역폭 구매

추가 스토리지 및 대역폭은 단일 데이터 팩으로 제공됩니다. 데이터 팩 1개 비용은 {% data variables.large_files.pack_monthly_price %}이고, 월별 할당량으로 {% data variables.large_files.pack_monthly_bandwidth %} 대역폭과 {% data variables.large_files.pack_monthly_storage %} 스토리지를 제공합니다. 필요한 만큼 데이터 팩을 구매할 수 있습니다. 예를 들어 150GB 스토리지가 필요한 경우 데이터 팩 3개를 구입합니다.

{% data variables.large_files.product_name_short %} 데이터 팩 구매는 {% data variables.product.product_name %}의 다른 유료 기능 또는 제품과 별개입니다.
