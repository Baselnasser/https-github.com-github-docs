1. Gib unter „Ignored Terms" (Ignorierte Begriffe) eine Komma-getrennte Liste von Begriffen ein, die in den Titeln von Pull Requests erscheinen können. Geplante Erinnerungen werden keine Pull Requests einschließen, welche eine oder mehrere dieser Begriffe in ihren Titeln beinhalten.
