1. Unter "Minimum staleness" (minimale Veraltung) gib die Zeit seit der letzten Aktivität auf einem Pull Request in Stunden ein. Geplante Erinnerungen werden keine Pull Requests beinhalten, deren letzte Aktivität aktueller war als dieses Zeitspanne.
