Wenn ein Kunde seinen Plan aktualisiert und bei der Zahlung ein Fehler auftritt, stellt GitHub den vorherigen Zustand seines {% data variables.product.prodname_marketplace %}-Abonnements wieder her. GitHub sendet auch eine E-Mail an den Kunden, um ihn über den Fehler zu informieren und ihm zu erlauben, seinen Kauf erneut zu versuchen. Du erhältst einen Webhook mit der `changed`-Aktion, der dich zum Wiederherstellen des vorherigen Plans auffordert.
