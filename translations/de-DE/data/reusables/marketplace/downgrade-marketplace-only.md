Das Kündigen einer App oder das Herabstufen einer App auf kostenlos wirkt sich nicht auf deine [anderen kostenpflichtigen Abonnements](/articles/about-billing-on-github) auf {% data variables.product.prodname_dotcom %} aus. Wenn du alle kostenpflichtigen Abonnements auf {% data variables.product.prodname_dotcom %} beenden möchtest, musst du jedes kostenpflichtige Abonnement separat herabstufen.
