1. Klicke mit der rechten Maustaste auf den Commit.

    - Wenn ein Commit nur über ein Tag verfügt, klicke auf **Tag TAGNAME löschen**.
  ![Auswählen des Menüelements zum Löschen von Tags](/assets/images/help/desktop/select-delete-tag.png)
    - Wenn ein Commit mehrere Tags aufweist, zeige auf **Tag löschen**, und klicke dann auf das Tag, das du löschen möchtest.
  ![Zeigen auf das Menü „Tag löschen“](/assets/images/help/desktop/delete-tag-multiple.png)
