Nachdem du {% data variables.product.product_name %} erworben hast, wirst du gebeten, eine E-Mail-Adresse und einen Benutzernamen für die Person anzugeben, die das Unternehmen initialisieren soll. Dein dedizierter technischer Kontomanager in {% data variables.contact.enterprise_support %} erstellt ein Konto für den Unternehmensbesitzer und sendet diesem eine E-Mail, damit er sich bei {% data variables.product.product_name %} anmeldet und die Initialisierung abschließt.
