Schreibberechtigung für die Überprüfungs-API ist nur für GitHub Apps verfügbar. OAuth-Apps und authentifizierte Benutzer können die Überprüfungsausführungen und Überprüfungssammlungen anzeigen, aber nicht erstellen. Wenn du keine GitHub App erstellst, bist du möglicherweise an der [Status-API](/rest/reference/commits#commit-statuses) interessiert.
