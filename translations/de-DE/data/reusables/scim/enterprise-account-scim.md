Du kannst diese Implementierung von SCIM nicht mit einem Unternehmenskonto oder mit einer {% data variables.product.prodname_emu_org %} verwenden. Wenn dein Unternehmen für {% data variables.product.prodname_emus %} aktiviert ist, musst du eine andere Implementierung von SCIM verwenden. Andernfalls ist SCIM auf Unternehmensebene nicht verfügbar. Weitere Informationen findest du unter [Bereitstellung von SCIM für {% data variables.product.prodname_emus %} konfigurieren](/admin/identity-and-access-management/managing-iam-with-enterprise-managed-users/configuring-scim-provisioning-for-enterprise-managed-users).
