**Warnung:** Es wird dringend empfohlen, keine Platzhalter-DNS-Einträge wie z. B. `*.example.com` zu verwenden. Ein Platzhalter-DNS-Eintrag würde es jedem ermöglichen, eine {% data variables.product.prodname_pages %}-Website auf deinen Subdomänen zu hosten, auch wenn diese verifiziert sind. Weitere Informationen findest du unter [Überprüfen deiner benutzerdefinierten Domäne für {% data variables.product.prodname_pages %}](/pages/configuring-a-custom-domain-for-your-github-pages-site/verifying-your-custom-domain-for-github-pages).

{% endwarning %}
