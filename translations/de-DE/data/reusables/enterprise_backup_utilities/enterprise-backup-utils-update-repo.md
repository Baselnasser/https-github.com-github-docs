1. Zum Aktualisieren auf die neueste Projektreleaseversion verwende den `stable`-Branch, indem du den `git checkout stable`-Befehl ausführst.

  ```shell
  git checkout stable
  ```
  
  Falls du eine bestimmte Projektversion verwenden möchtest, kannst du alternativ den folgenden Befehl ausführen, wobei du `X.Y.Z` durch die gewünschte Releaseversion ersetzt.

  ```shell
  $ git checkout vX.Y.Z
  ```
