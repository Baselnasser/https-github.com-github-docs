Jeder, der an einem Open-Source-Projekt mitwirkt und ein Bankkonto in einer unterstützten Region hat, kann ein „unterstützter Entwickler" werden. Zu Beiträgen zählen beispielsweise Fehlerberichte, Issue-Triage, Code, Dokumentationen, Leitung, Geschäftsentwicklung, Projektverwaltung, Mentoring und Design. Wenn Du in einer Region lebst, die noch nicht von {% data variables.product.prodname_sponsors %} unterstützt ist, kannst Du Dich in die Warteliste eintragen, um an der Beta-Version von {% data variables.product.prodname_sponsors %} teilzunehmen.
