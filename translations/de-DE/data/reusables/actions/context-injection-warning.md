**Warnung:** Beim Erstellen von Workflows und Aktionen solltest du immer überprüfen, ob dein Code gegebenenfalls nicht vertrauenswürdige Eingaben von Angreifern ausführen kann. Bestimmte Kontexte sollten als nicht vertrauenswürdige Eingaben behandelt werden, da ein Angreifer seine eigenen schädlichen Inhalte einfügen könnte. Weitere Informationen findest du unter „[Grundlegendes zum Risiko von Skript-Einfügungen](/actions/learn-github-actions/security-hardening-for-github-actions#understanding-the-risk-of-script-injections)".
