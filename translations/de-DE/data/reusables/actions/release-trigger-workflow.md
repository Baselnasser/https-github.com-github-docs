Jedes Mal, wenn du auf {% data variables.product.product_name %} ein neues Release erstellst, kannst du einen Workflow auslösen, um dein Image zu veröffentlichen. Der Workflow im folgenden Beispiel wird ausgeführt, wenn das `release`-Ereignis mit dem `created`-Aktivitätstyp ausgelöst wird. Weitere Informationen zum `release`-Ereignis findest du unter [Ereignisse, die Workflows auslösen](/actions/reference/events-that-trigger-workflows#release).
