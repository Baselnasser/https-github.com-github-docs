Wenn du kurzlebige Runner verwendest und automatische Updates deaktiviert hast, solltest du vor dem Upgraden von {% data variables.product.product_location %} zunächst deine selbstgehosteten Runner auf die Version der Runneranwendung upgraden, die deine upgegradete Instanz ausführen wird. Wenn du ein Upgrade für {% data variables.product.product_location %} durchführst, bevor du die kurzlebigen Runner upgradest, können deine Runner offline gehen. Weitere Informationen findest du unter [Upgraden von {% data variables.product.product_name %}](/admin/enterprise-management/updating-the-virtual-machine-and-physical-resources/upgrading-github-enterprise-server).
