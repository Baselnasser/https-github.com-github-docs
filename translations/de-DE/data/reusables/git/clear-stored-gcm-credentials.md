- Wenn die Ausgabe `manager-core` lautet, verwendest du Git Credential Manager. Um die Anmeldeinformationen zu löschen, führe den folgenden Befehl aus.

    ```shell{:copy}
    git credential-manager reject https://github.com
    ```
