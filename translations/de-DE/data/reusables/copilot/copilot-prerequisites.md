- {% data variables.product.prodname_copilot %} kann von geprüften Studenten und Open Source-Betreuern kostenlos verwendet werden. 
- Wenn du kein Student oder Open Source-Betreuer bist, kannst du {% data variables.product.prodname_copilot %} mit einer einmaligen 60-Tage-Testversion kostenlos ausprobieren. Nach dem kostenlosen Test benötigst du ein kostenpflichtiges Abonnement für die weitere Nutzung. Du musst Abrechnungsinformationen angeben, um eine kostenlose Testversion zu nutzen. Weitere Informationen findest du unter [Informationen zur Abrechnung für {% data variables.product.prodname_copilot %}](/billing/managing-billing-for-github-copilot/about-billing-for-github-copilot).
