Für jede Arbeitsauftrag für eine Gruppe erstellt {% data variables.product.prodname_classroom %} automatisch ein einzelnes gemeinsam genutztes Repository, auf das das Team zugreifen kann. Das Repository kann leer sein, oder du kannst das Repository aus einer Vorlage mit Startcode, Dokumentation, Tests und anderen Ressourcen erstellen. Das Repository gehört zum Konto deiner Organisation in {% data variables.product.product_name %}, und {% data variables.product.prodname_classroom %} gewährt Zugriff auf Teams, die Lernende erstellen oder denen sie beitreten, wenn sie den Arbeitsauftrag annehmen.
