Die Inaktivitätsschwelle ist die Zeitspanne welche ein Benutzer inaktiv sein muss um als ruhend betrachtet zu werden. Die Inaktivitätsschwelle beträgt standardmäßig 90 Tage, du kannst sie aber für {% data variables.product.product_location %} anpassen.
