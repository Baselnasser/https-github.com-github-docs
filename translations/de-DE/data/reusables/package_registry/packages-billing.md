Die {% data variables.product.prodname_registry %}-Nutzung ist für öffentliche Pakete kostenlos. Für private Pakete erhält jedes Konto auf {% data variables.product.product_location %} eine bestimmte Menge ein kostenloses Kontingent für Speicher und Datenübertragung, abhängig von dem Produkt, das mit dem Konto verwendet wird. Jede Nutzung, die über die enthaltenen Kontingente hinausgeht, wird durch Ausgabenlimits gesteuert.
