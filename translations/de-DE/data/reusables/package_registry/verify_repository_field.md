1. Überprüfe das Feld `repository` in der Datei *package.json* deines Projekts. Das Feld `repository` muss mit der URL für Ihr {% data variables.product.prodname_dotcom %}-Repository übereinstimmen. Wenn ihre Repository-URL beispielsweise `github.com/my-org/test` lautet, sollte das Repositoryfeld `https://github.com/my-org/test.git` lauten.
