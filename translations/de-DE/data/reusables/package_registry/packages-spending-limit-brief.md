Wenn du ein Kunde mit monatlicher Abrechnung bist, gilt für dein Konto ein Standardausgabenlimit von 0 US-Dollar (USD). Dadurch wird die zusätzliche Nutzung von Speicher oder Datenübertragungen verhindert, sobald die enthaltenen Beträge erreicht sind. Wenn die Abrechnung für dein Konto auf Rechnung erfolgt, gilt für dein Konto ein unbegrenztes Standardausgabenlimit.
