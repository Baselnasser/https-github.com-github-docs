Standardmäßig können Benutzer in öffentlichen Repositorys Reviews übermitteln, die einen Pull Review genehmigen oder Änderungen anfordern. Organisationsbesitzer und Repositoryadministratoren können einschränken, wer Pull Request-Reviews zur Genehmigung durchführen oder Änderungen anfordern kann. Weitere Informationen findest du unter [Verwalten von Pull Request-Reviews in deiner Organisation](/organizations/managing-organization-settings/managing-pull-request-reviews-in-your-organization) und unter [Verwalten von Pull Request-Reviews in deinem Repository](/repositories/managing-your-repositorys-settings-and-features/managing-repository-settings/managing-pull-request-reviews-in-your-repository).
