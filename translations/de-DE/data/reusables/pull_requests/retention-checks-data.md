Überprüft, ob Daten archiviert werden, die älter sind als 400 Tage. Im Rahmen des Archivierungsprozesses erstellt {% data variables.product.prodname_dotcom %} einen Rollupcommitstatus, der den Status aller Prüfungen für diesen Commit darstellt. Dies hat zur Folge, dass das Mergefeld in jedem Pull Request mit erforderlichen archivierten Prüfungen blockiert ist, und du die Prüfungen erneut durchführen musst, bevor du den Pull Request mergen kannst.
