Wenn eine Organisation, in der Du Mitglied bist, E-Mail-Benachrichtigungen auf eine genehmigte E-Mail-Domäne beschränkt, musst Du eine E-Mail-Adresse in dieser Domäne verifizieren, um E-Mail-Benachrichtigungen über Aktivitäten in der Organisation zu erhalten. Weitere Informationen findest du unter [Einschränken von E-Mail-Benachrichtigungen auf eine genehmigte Domäne](/articles/restricting-email-notifications-to-an-approved-domain).
