Du kannst keine E-Mail-Adressen von Diensten für Wegwerf-E-Mail-Adressen überprüfen (dies sind Dienste, mit denen du E-Mails an einer temporären Adresse empfangen kannst, die nach einer bestimmten Zeit abläuft). Wenn deine E-Mail-Adresse privat bleiben soll, kannst du eine von {% data variables.product.prodname_dotcom %} bereitgestellte `noreply`-E-Mail-Adresse verwenden. Weitere Informationen findest du unter [Festlegen deiner Commit-E-Mail-Adresse auf {% data variables.product.prodname_dotcom %}](/articles/setting-your-commit-email-address#setting-your-commit-email-address-on-github).
