Um einen CVSS-Score zuzuweisen, wähle „Schweregrad mithilfe von CVSS bewerten“ aus, und klicke auf die entsprechenden Werte im Rechner. {% data variables.product.product_name %} berechnet den Score gemäß dem [Common Vulnerability Scoring System Calculator](https://www.first.org/cvss/calculator).
