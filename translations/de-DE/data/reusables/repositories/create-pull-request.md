1. Klicke zum Erstellen eines Pull Requests, der überprüft werden kann, auf **Pull Request erstellen**.
Verwende zum Erstellen eines Pull Request-Entwurfs das Dropdownmenü. Wähle **Pull Request-Entwurf erstellen** aus, und klicke anschließend auf **Pull Request-Entwurf**. Weitere Informationen zum Entwerfen von Pull Requests findest du unter [Informationen zu Pull Requests](/articles/about-pull-requests#draft-pull-requests). ![Schaltfläche „Pull Request erstellen“](/assets/images/help/pull_requests/pullrequest-send.png)
