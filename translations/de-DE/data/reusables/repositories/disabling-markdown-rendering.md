Wenn du eine Markdowndatei anzeigst, kannst du oben in der Datei auf {% octicon "code" aria-label="The code icon" %} klicken, um das Rendern von Markdown zu deaktivieren und stattdessen den Quelltext der Datei anzuzeigen.

![Anzeigen von Markdown als Quelle](/assets/images/help/writing/display-markdown-as-source.png)

Durch das Deaktivieren des Markdownrenderings kannst du die Funktionen der Quellenansicht nutzen, wie z. B. Zeilenverknüpfungen, die beim Anzeigen von gerenderten Markdowndateien nicht verfügbar sind.
