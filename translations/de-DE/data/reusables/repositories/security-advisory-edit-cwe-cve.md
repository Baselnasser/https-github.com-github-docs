1. Füge Common Weakness Enumerators (CWEs) für die Arten von Sicherheitslücken hinzu, die diese Sicherheitsempfehlung behandelt. Eine vollständige Liste der CWEs findest du in der [Common Weakness Enumeration](https://cwe.mitre.org/index.html) von MITRE.
1. Wenn du bereits über einen CVE-Bezeichner verfügst, wähle „Ich verfüge über einen vorhandene CVE-Bezeichner“ aus, und gib den CVE-Bezeichner in das Textfeld ein. Andernfalls kannst du später einen CVE-Bezeichner bei {% data variables.product.prodname_dotcom %} anfordern. Weitere Informationen findest du unter [Informationen zu {% data variables.product.prodname_security_advisories %}](/github/managing-security-vulnerabilities/about-github-security-advisories#cve-identification-numbers).
