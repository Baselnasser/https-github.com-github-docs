Es wird empfohlen, einer begrenzten Anzahl von Mitgliedern in jeder Organisation eine Organisationsbesitzerrolle zuzuweisen, die vollständigen administrativen Zugriff auf diese Organisation bietet. Weitere Informationen findest du unter [Rollen in einer Organisation](/organizations/managing-peoples-access-to-your-organization-with-roles/roles-in-an-organization).

Für Organisationen, in denen du über Administratorberechtigungen verfügst, kannst du auch den Zugriff auf die einzelnen Repositorys anhand von detaillierten Berechtigungsstufen anpassen. Weitere Informationen findest du unter [Repositoryberechtigungsstufen für eine Organisation](/organizations/managing-access-to-your-organizations-repositories/repository-permission-levels-for-an-organization).
