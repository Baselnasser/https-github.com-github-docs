{% data variables.product.prodname_pages %} ist ein statischer Websitehostingdienst, der HTML-, CSS- und JavaScript-Dateien direkt aus einem Repository verwendet und auf einer Website veröffentlicht. Du kannst {% data variables.product.prodname_pages %} für deine Unternehmensmitglieder auf Organisationsebene aktivieren oder deaktivieren. Weitere Informationen findest du unter [Konfigurieren von {% data variables.product.prodname_pages %} für dein Unternehmen](/admin/configuration/configuring-your-enterprise/configuring-github-pages-for-your-enterprise) und [Informationen zu {% data variables.product.prodname_pages %}](/pages/getting-started-with-github-pages/about-github-pages).
