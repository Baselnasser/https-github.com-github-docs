Mit den Webhook-REST-APIs kannst du Repository, Organisation und App-Webhooks verwalten. Du kannst diese API verwenden, um Webhook-Lieferungen für einen Webhook aufzulisten oder eine individuelle Lieferung für einen Webhook abzurufen und erneut auszuführen, der in eine externe App oder einen Dienst integriert werden kann. Du kannst die REST-API auch verwenden, um die Konfiguration des Webhooks zu ändern. Du kannst beispielsweise die Nutzlast-URL, den Inhaltstyp, die SSL-Überprüfung und das Geheimnis ändern. Weitere Informationen finden Sie unter

- [Repository-Webhooks-REST-API](/rest/reference/webhooks#repository-webhooks)
- [Rest-API für Organisations-Webhooks](/rest/reference/orgs#webhooks)
- [{% data variables.product.prodname_github_app %}-Webhooks-REST-API](/rest/reference/apps#webhooks)
