Beim nächsten Klonen einer HTTPS-URL, die eine Authentifizierung verlangt, wirst du von Git zur Anmeldung über ein Browserfenster aufgefordert. Möglicherweise wirst du zunächst aufgefordert, eine OAuth-App zu autorisieren.{% ifversion not ghae %} Wenn dein Konto oder deine Organisation eine [zweistufige Authentifizierung](/github/authenticating-to-github/securing-your-account-with-two-factor-authentication-2fa) erfordert, musst du auch die 2FA-Aufforderung erfüllen.{% endif %}
