Wenn {% data variables.product.prodname_dependabot %} Pull Requests auslöst, können diese Pull Requests für _Sicherheits_- oder _Versionsupdates_ gelten:

- _{% data variables.product.prodname_dependabot_security_updates %}_ sind automatisierte Pull Requests, mit denen Du Abhängigkeiten mit bekannten Sicherheitsrisiken aktualisieren kannst. 
- _{% data variables.product.prodname_dependabot_version_updates %}_ sind automatisierte Pull Requests, mit denen Deine Abhängigkeiten auf dem aktuellen Stand gehalten werden, auch wenn sie keine Sicherheitsrisiken aufweisen. Um den Status von Versionsupdates zu überprüfen, navigiere zur Registerkarte „Erkenntnisse“ Deines Repositorys, dann zum Abhängigkeitsdiagramm und {% data variables.product.prodname_dependabot %}.
