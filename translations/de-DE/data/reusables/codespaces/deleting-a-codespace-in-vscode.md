Du kannst Codespaces aus {% data variables.product.prodname_vscode_shortname %} löschen, wenn du gerade nicht in einem Codespace arbeitest.

{% data reusables.codespaces.click-remote-explorer-icon-vscode %}
1. Klicke unter GITHUB CODESPACES mit der rechten Maustaste auf den Codespace, den du löschen möchtest.
1. Klicke auf **Codespace löschen**.

   ![Löschen eines Codespace in {% data variables.product.prodname_dotcom %}](/assets/images/help/codespaces/delete-codespace-vscode.png)
