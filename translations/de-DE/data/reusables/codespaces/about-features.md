Features sind eigenständige Einheiten von Installationscode und Entwicklungscontainerkonfigurationen, die für eine Vielzahl von Basiscontainerimages konzipiert sind. Du kannst Features verwenden, um deinem Codespaceimage schnell Tools, Runtimes oder Bibliotheken hinzuzufügen. Weitere Informationen findest du auf containers.dev unter [Verfügbare Features](https://containers.dev/features) und [Featurespezifikation](https://containers.dev/implementors/features/).

Du kannst Features zu einer `devcontainer.json`-Datei von {% data variables.product.prodname_vscode_shortname %} oder aus deinem Repository auf {% data variables.product.prodname_dotcom_the_website %} hinzufügen.
