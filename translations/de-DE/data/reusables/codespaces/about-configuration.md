Du kannst eine Standard-Codespace-Konfiguration für Dein Repository erstellen, welche die Umgebung jedes neuen Codespace bestimmt, den jemand für dieses Repository erstellt. Die Konfiguration definiert einen Entwicklungscontainer, der Frameworks, Tools, Erweiterungen und Portweiterleitung enthalten kann.
