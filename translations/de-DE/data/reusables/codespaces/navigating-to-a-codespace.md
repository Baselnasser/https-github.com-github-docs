## Navigieren zu einem vorhandenen Codespace

1. {% data reusables.codespaces.you-can-see-all-your-codespaces %}
1. Klicke auf den Namen des Codespace, in dem du entwickeln möchtest.

   ![Name des Codespaces](/assets/images/help/codespaces/click-name-codespace.png)

Alternativ kannst du alle deine Codespaces für ein bestimmtes Repository anzeigen, indem du zu diesem Repository navigierst und **{% octicon "code" aria-label="The code icon" %} Code** auswählst. Im Dropdownmenü werden alle aktiven Codespaces für ein Repository angezeigt.
