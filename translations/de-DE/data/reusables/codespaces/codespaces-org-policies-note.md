**Hinweis**: Codespacerichtlinien gelten nur für Codespaces, die deiner Organisation in Rechnung gestellt werden. Wenn ein einzelner Benutzer einen Codespace für ein Repository in deiner Organisation erstellt und dieser Codespace nicht unter die Abrechnung für deine Organisation fällt, ist er nicht an diese Richtlinien gebunden. Informationen dazu, wie du auswählst, wer Codespaces erstellen darf, die deiner Organisation in Rechnung gestellt werden, findest du unter [Aktivieren von {% data variables.product.prodname_github_codespaces %} für deine Organisation](/codespaces/managing-codespaces-for-your-organization/enabling-github-codespaces-for-your-organization#choose-who-can-create-codespaces-that-are-billed-to-your-organization).

{% endnote %}
