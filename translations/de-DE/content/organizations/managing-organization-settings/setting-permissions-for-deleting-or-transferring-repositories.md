Inhaber können die Berechtigungen zum Löschen oder Übertragen von Repositorys innerhalb einer Organisation festlegen.

{% data reusables.profile.access_org %} {% data reusables.profile.org_settings %} {% data reusables.organizations.member-privileges %}
5. Aktiviere oder deaktiviere unter „Repositorys löschen und übertragen“ das Kontrollkästchen **Mitgliedern das Löschen oder Übertragen von Repositorys für diese Organisation erlauben**.
![Kontrollkästchen zum Zulassen des Löschens von Repositorys durch Mitglieder](/assets/images/help/organizations/disallow-members-to-delete-repositories.png)
6. Klicken Sie auf **Speichern**.
