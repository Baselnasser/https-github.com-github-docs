## Zulassen der Erstellung von Diskussionen für Benutzer*innen mit Lesezugriff oder Aufheben der Genehmigung

Standardmäßig können Organisationsmitglieder mit Lesezugriff Diskussionen erstellen, wenn Repositoryadministrator*innen oder Organisationsbesitzer*innen Diskussionen für ein Repository im Besitz der Organisation aktivieren.

{% data reusables.profile.access_org %} {% data reusables.profile.org_settings %} {% data reusables.organizations.member-privileges %}
5. Aktiviere oder deaktiviere unter „Repositorydiskussionen“ die Option **Benutzern mit Lesezugriff das Erstellen von Diskussionen erlauben**.
  ![Kontrollkästchen für das Zulassen der Erstellung von Diskussionen durch Personen mit Lesezugriff](/assets/images/help/discussions/toggle-allow-users-with-read-access-checkbox.png)
6. Klicken Sie auf **Speichern**.
  ![Schaltfläche „Speichern“ für Diskussionseinstellungen](/assets/images/help/discussions/click-save.png)
