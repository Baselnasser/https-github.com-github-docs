Alle verfügbaren Git-Befehle auf einmal zu lernen ist eine Herausforderung. Mit „[Git-Spickzetteln](https://training.github.com/)“ erhältst du eine Kurzübersicht über häufig verwendete Befehle. Der Spickzettel zur Verwendung von Git ist in mehreren Sprachen verfügbar. 

Schau dir außerdem unsere Seite mit [Git- und GitHub-Lerninhalten](/articles/git-and-github-learning-resources/) an, die Links zu Leitfäden, Videos und mehr enthält.
