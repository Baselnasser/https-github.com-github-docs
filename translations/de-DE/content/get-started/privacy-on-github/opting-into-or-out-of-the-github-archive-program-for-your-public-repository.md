Wenn du das {% data variables.product.prodname_archive %} für ein Repository beendest, wird das Repository von allen langfristigen Archiven ausgeschlossen, die zukünftig auf {% data variables.product.prodname_dotcom %} erstellt werden können. Außerdem senden wir an alle unsere Partnerorganisationen eine Aufforderung, das Repository aus ihren Daten zu entfernen.
