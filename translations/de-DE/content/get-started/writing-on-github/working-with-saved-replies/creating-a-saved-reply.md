3. Gib unter „Add a saved reply“ (Eine gespeicherte Antwort hinzufügen) einen Titel für Deine gespeicherte Antwort ein.
![Titel der gespeicherten Antwort](/assets/images/help/settings/saved-replies-title.png)
4. Gib im Feld „Write“ (Schreiben) den Inhalt ein, den Du für die gespeicherte Antwort verwenden möchtest. Weitere Informationen zum Schreiben von {% data variables.product.product_name %} findest Du unter „[Grundlegende Schreib- und Formatierungssyntax](/articles/basic-writing-and-formatting-syntax)“.
![Schreiben einer gespeicherten Antwort](/assets/images/help/settings/saved-replies-settings-adding.png)
5. Um Deine Antwort zu überprüfen, klicke auf **Vorschau**.
![Hinzufügen einer gespeicherte Antwort](/assets/images/help/settings/saved-replies-preview.png)
6. Klicke auf **Gespeicherte Antwort hinzufügen**.
![Schaltfläche „Gespeicherte Antwort hinzufügen“](/assets/images/help/settings/saved-replies-add-button.png)
