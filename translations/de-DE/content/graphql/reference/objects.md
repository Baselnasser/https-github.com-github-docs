## Informationen zu Objekten

[Objekte](https://graphql.github.io/graphql-spec/June2018/#sec-Objects) in GraphQL stellen die Ressourcen dar, auf die Du zugreifen kannst. Ein Objekt kann eine Liste von Feldern enthalten, die speziell eingegeben werden.

Das Objekt [`Repository`](/graphql/reference/objects#repository) weist beispielsweise ein Feld namens `name` auf, bei dem es sich um ein `String` handelt.
