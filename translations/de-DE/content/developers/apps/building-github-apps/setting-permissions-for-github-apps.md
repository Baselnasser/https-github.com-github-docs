GitHub-Apps verfügen standardmäßig nicht über Berechtigungen. Wenn du eine GitHub-App erstellst, kannst du die Berechtigungen auswählen, die für den Zugriff auf Endbenutzerdaten erforderlich sind. Berechtigungen können auch hinzugefügt und entfernt werden. Weitere Informationen findest du unter [Bearbeiten der Berechtigungen einer GitHub-App](/apps/managing-github-apps/editing-a-github-app-s-permissions/).
