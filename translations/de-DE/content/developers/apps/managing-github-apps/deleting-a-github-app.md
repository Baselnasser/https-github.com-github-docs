4. Wähle die GitHub-App aus, die du löschen möchtest.
![App-Auswahl](/assets/images/github-apps/github_apps_select-app.png) {% data reusables.user-settings.github_apps_advanced %}
6. Klicke auf **GitHub-App löschen**.
![Schaltfläche zum Löschen einer GitHub-App](/assets/images/github-apps/github_apps_delete.png)
7. Gib den Namen der GitHub-App ein, um zu bestätigen, dass du sie löschen möchtest.
![Feld, um den Namen der GitHub-App zu bestätigen, die du löschen möchtest](/assets/images/github-apps/github_apps_delete_integration_name.png)
8. Klicke auf **Ich kenne die Folgen, diese GitHub-App löschen**.
![Schaltfläche zum Bestätigen der Löschung deiner GitHub-App](/assets/images/github-apps/github_apps_confirm_deletion.png)
