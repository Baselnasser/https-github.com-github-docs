Unternehmen können eine Rechnungsmethode verwenden, um am {% data variables.product.prodname_sponsors %}-Programm teilzunehmen. Dies ist dann besonders bequem, wenn dein Unternehmen bereits über eine Abrechnungsbeziehung mit {% data variables.product.company_short %} verfügt. Unternehmen, die per Rechnung bezahlen, erhalten Zugriff auf ein {% data variables.product.product_name %}-Dashboard, das Open-Source-Beiträge und Aktivitäten in allen öffentlichen Projekten auf {% data variables.product.prodname_dotcom_the_website %} misst, an denen sie interessiert sind.

Du kannst diesem Betaprogramm beitreten, indem du mit deinem Vertriebsmitarbeiter sprichst oder [uns kontaktierst](https://support.github.com/contact/org-sponsors-waitlist).
