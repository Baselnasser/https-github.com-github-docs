## Informationen zur API für die OAuth-Autorisierung

Du kannst diese API verwenden, um den Zugriff zu verwalten, über den OAuth-Anwendungen für dein Konto verfügen. Du kannst auf diese API nur über die [Standardauthentifizierung](/rest/overview/other-authentication-methods#basic-authentication) anhand deines Benutzernamens und Kennworts und nicht über Token zugreifen.

Wenn für dich oder deine Benutzer die zweistufige Authentifizierung aktiviert ist, stelle sicher, dass dir die [Arbeit mit der zweistufigen Authentifizierung](/rest/overview/other-authentication-methods#working-with-two-factor-authentication) vertraut ist.
