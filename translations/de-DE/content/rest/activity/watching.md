## Informationen zur Überwachungs-API

Mit der Überwachungs-API können Benutzer*innen Benachrichtigungen für Aktivitäten in einem Repository abonnieren. Weitere Informationen zum Festlegen von Lesezeichen für ein Repository findest du unter [Versehen mit einem Stern](/rest/reference/activity#starring).
