4. 选择要转让其所有权的 GitHub 应用程序。
![应用选择](/assets/images/github-apps/github_apps_select-app.png) {% data reusables.user-settings.github_apps_advanced %}
6. 单击“转让所有权”。
![用于转让所有权的按钮](/assets/images/github-apps/github_apps_transfer_ownership.png)
7. 输入要转让的 GitHub 应用程序的名称。
![用于输入要转让的应用名称的字段](/assets/images/github-apps/github_apps_transfer_app_name.png)
8. 输入要将 GitHub 应用程序转让到的用户或组织的名称。
![用于输入要转让到的用户或组织的字段](/assets/images/github-apps/github_apps_transfer_new_owner.png)
9. 单击“转让此 GitHub 应用”。
![用于确认转让 GitHub 应用的按钮](/assets/images/github-apps/github_apps_transfer_integration.png)
