在您的应用程序上架信息满足 {% data variables.product.prodname_marketplace %} 要求后，您可以提交该应用程序。 关于应用上架要求的概述，请参阅“[上架应用的要求](/developers/github-marketplace/requirements-for-listing-an-app)”。

1. 在应用上架信息页面的左上角，单击“概述”。 有关查找应用上架信息页面的信息，请参阅“[起草应用上架信息](/developers/github-marketplace/drafting-a-listing-for-your-app)”。
  
  ![Marketplace 上架信息草稿的选项概述](/assets/images/marketplace/edit-marketplace-listing-overview.png)

2. 若要提交已完成的应用上架信息，请单击“请求发布”。

  ![“将应用发布到市场”检查列表，底部有提交按钮](/assets/images/marketplace/publish-your-app-checklist-and-submission.png)

上架专家将联系您并提供更多信息。
