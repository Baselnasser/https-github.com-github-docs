## <a name="following-people"></a>关注他人

当你在 {% data variables.product.prodname_dotcom %} 上关注某人时，你将在个人仪表板上收到关于其公共活动的通知。 如果你关注的某人创建新存储库、用星号标记存储库或关注其他用户，则活动将显示在仪表板中。 

单击某人个人资料页上的“关注”可关注他们。

![关注用户按钮](/assets/images/help/profile/follow-user-button.png)

有关详细信息，请参阅“[关注他人](/get-started/exploring-projects-on-github/following-people)”。

## <a name="watching-a-repository"></a>关注仓库

可以订阅存储库来关注它并接收其中活动的通知。 当所有者更新存储库时，你将在个人仪表板中看到其更改。 有关详细信息，请参阅“[查看订阅](/github/managing-subscriptions-and-notifications-on-github/viewing-your-subscriptions)”。

单击存储库顶部的“监视”可监视存储库。

![关注仓库按钮](/assets/images/help/repository/repo-actions-watch.png)

## <a name="communicating-on--data-variablesproductproduct_name-"></a>{% data variables.product.product_name %} 上的通信

{% data variables.product.product_name %} 提供内置的协作通信工具，使你能够在构建出色的软件时与社区进行密切互动。 例如，可以通过创建拉取请求，或使用问题跟踪 bug 或建议存储库中的想法，从而为其他用户的项目做出贡献。 还可以通过与你的团队讨论来寻求新的想法。 

有关这些工具的概述，请参阅“[在 {% data variables.product.prodname_dotcom %} 上进行通信的快速入门](/github/collaborating-with-issues-and-pull-requests/quickstart-for-communicating-on-github)”。

## <a name="participating-in-organizations"></a>参与组织

组织是共享帐户，其中业务和开源项目可同时跨多个项目进行协作。 所有者和管理员可建立具有特殊权限的团队、拥有公共组织资料以及跟踪组织内的活动。 有关详细信息，请参阅[关于组织](/articles/about-organizations/)。

在仪表板中，单击仪表板左侧的用户名的下拉菜单。 在此处，可以查看你作为成员的组织并轻松地在组织之间切换。

![切换帐户上下文下拉列表](/assets/images/help/overview/dashboard-contextswitcher.png)

### <a name="exploring-other-projects-on--data-variablesproductprodname_dotcom-"></a>在 {% data variables.product.prodname_dotcom %} 上探索其他项目

可以在 {% data variables.product.prodname_dotcom %} 的“浏览”页上发现全新且有趣的项目。 可以用星号标记感兴趣的项目，以便今后轻松找到它们。 访问你的星标页面以查看所有加星标的项目。 有关星标的详细信息，请参阅“[使用星标保存存储库](/get-started/exploring-projects-on-github/saving-repositories-with-stars)”。

仪表板源根据你的兴趣（包括你加星标的存储库、关注的人员以及为公共存储库做出的贡献）显示项目。 在仪表板中，可以查看热门项目、主题和收藏集锦。 

若要浏览更多项目，请参阅“{% data variables.explore.explore_github %}”。
