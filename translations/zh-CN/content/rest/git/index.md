## 关于 Git 数据库

REST API 使你能够在 {% data variables.product.product_name %} 上的 Git 数据库中读取和写入原始 Git 对象，并列出和更新你的引用（分支头部和标记）。 有关使用 REST API 与 Git 数据库交互的详细信息，请参阅“[Git 数据 API 入门](/rest/guides/getting-started-with-the-git-database-api)”。
