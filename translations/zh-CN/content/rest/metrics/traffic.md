## 关于存储库流量 API

对于您具有推送权限的仓库，仓库流量 API 提供对仓库图中所示信息的访问权限。 更多信息请参阅“[查看仓库的流量](/repositories/viewing-activity-and-data-for-your-repository/viewing-traffic-to-a-repository)”。
