## 关于代码空间用户机密 API

Codespaces 用户机密 API 允许用户创建、列出和删除机密（如云服务的访问令牌），以及将机密分配给用户有权访问的存储库。 这些机密在运行时可供代码空间使用。 更多信息请参阅“[管理代码空间的加密机密](/codespaces/managing-your-codespaces/managing-encrypted-secrets-for-your-codespaces)。。
