{% tip %}

提示：你可以通过拖放文件夹到 {% data variables.product.prodname_desktop %} 的方式，将 Git 存储库从本地计算机添加到 GitHub Desktop。 如果同时将多个 Git 文件夹拖入 {% data variables.product.prodname_desktop %}，则每个文件夹将添加为一个单独的 Git 仓库。

{% endtip %}

{% mac %}

1. 在“文件”菜单中，单击“添加本地存储库” 。
  ![添加本地存储库菜单选项](/assets/images/help/desktop/add-local-repository-mac.png)
2. 单击“选择...”，并使用查找器窗口导航到要添加的本地存储库。
  ![Mac 应用中的“本地路径”字段](/assets/images/help/desktop/add-repo-choose-button-mac.png)
4. 单击“添加存储库”。
  ![Mac 应用程序中的“添加存储库”按钮](/assets/images/help/desktop/add-repository-button-mac.png)

{% endmac %}

{% windows %}

1. 在“文件”菜单中，单击“添加本地存储库” 。
  ![“添加本地存储库”菜单选项](/assets/images/help/desktop/add-local-repository-windows.png)
2. 单击“选择...”，并使用 Windows 资源管理器导航到要添加的本地存储库。
  ![Windows 应用中的“本地路径”字段](/assets/images/help/desktop/add-repo-choose-button-win.png)
4. 单击“添加存储库”。
  ![Windows 应用程序中的“添加存储库”按钮](/assets/images/help/desktop/add-repository-button-windows.png)

{% endwindows %}
