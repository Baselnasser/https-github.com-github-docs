在还原到上一个提交时，还原本身也是提交。 原提交仍会保留在仓库的历史记录中。

{% tip %}

提示：在还原多个提交时，最好按照从最新到最旧的顺序还原。 如果以其他顺序还原提交，可能会出现合并冲突。

{% endtip %}
