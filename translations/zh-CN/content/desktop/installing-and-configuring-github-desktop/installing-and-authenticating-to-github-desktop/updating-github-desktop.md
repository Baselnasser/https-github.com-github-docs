{% mac %}

1. 在 GitHub Desktop 菜单中，单击“关于 GitHub Desktop” 。
![“关于 GitHub Desktop”菜单选项](/assets/images/help/desktop/desktop-menu-about-desktop-mac.png)
2. 单击“检查更新”。
![“检查更新”按钮](/assets/images/help/desktop/check-for-updates.png)
3. 如果更新可用，请退出并重新启动 {% data variables.product.prodname_desktop %} 以安装更新。

{% endmac %}

{% windows %}

1. 在“帮助”菜单中，单击“关于 GitHub Desktop” 。
![“关于 GitHub Desktop”菜单选项](/assets/images/help/desktop/help-about-desktop-win.png)
2. 单击“检查更新”。
![“检查更新”按钮](/assets/images/help/desktop/check-for-updates.png)
3. 如果更新可用，请退出并重新启动 {% data variables.product.prodname_desktop %} 以安装更新。

{% endwindows %}
