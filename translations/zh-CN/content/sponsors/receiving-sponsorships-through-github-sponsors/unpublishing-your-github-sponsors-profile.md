如果您有赞助者，您需要联系 GitHub Support 帮助您取消发布 {% data variables.product.prodname_sponsors %} 帐户。

取消发布您的 {% data variables.product.prodname_sponsors %} 帐户后：
- 您将不再收到赞助付款。
- 您可以请求重新发布您的个人资料。

{% data reusables.sponsors.navigate-to-sponsors-dashboard %} {% data reusables.sponsors.settings-tab %}
3. 在“取消发布 GitHub 发起人个人资料”下，单击“取消发布个人资料”。
  ![“取消发布个人资料”按钮](/assets/images/help/sponsors/unpublish-profile-button.png)
4. 要确认，请单击“Unpublish GitHub Sponsors account（取消发布 GitHub Sponsors 帐户）”。
  ![“取消发布 GitHub 发起人个人资料”按钮](/assets/images/help/sponsors/unpublish-profile-dialog.png)
