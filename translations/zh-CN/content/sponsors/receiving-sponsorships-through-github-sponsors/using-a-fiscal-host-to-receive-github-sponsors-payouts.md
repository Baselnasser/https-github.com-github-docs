## 关于使用财务托管接收 {% data variables.product.prodname_sponsors %} 资金

注册 {% data variables.product.prodname_sponsors %} 个人资料以便获得赞助时，你可选择是通过自己控制的 Stripe Connect 帐户接收资金，还是通过已与你合作的财务托管接收资金。

{% data reusables.sponsors.bank-info-fiscal-host-reminder %}

{% data variables.product.prodname_sponsors %} 支持以下财务托管：

- [Open Source Collective](https://oscollective.org/)
- [NumFOCUS](http://www.numfocus.org/)
- [Open Collective Europe](https://opencollective.com/europe)
- [Software Underground](https://softwareunderground.org/)
- [Software Freedom Conservancy](https://sfconservancy.org/)

## 选择使用财务托管

导航到希望获得赞助的帐户的“加入等待列表”个人资料设置页面后，你将看到“接收资金方式”或“组织银行帐户”部分，具体取决于要设置的帐户类型。 有关如何注册 {% data variables.product.prodname_sponsors %} 个人资料的详细信息，请参阅“[为个人帐户设置 {% data variables.product.prodname_sponsors %}](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-personal-account)”或“[为组织设置 {% data variables.product.prodname_sponsors %}](/sponsors/receiving-sponsorships-through-github-sponsors/setting-up-github-sponsors-for-your-organization)”。

1. 在“接收资金方式”或“组织银行帐户”下，选择“使用财务托管”或“此组织正在使用财务托管”选项 。

   ![用于在银行帐户和财务托管之间进行选择的选项的屏幕截图](/assets/images/help/sponsors/user-waitlist-payout-options.png)
2. 使用“选择财务托管”下拉列表，选择要使用的财务托管。
3. 在“财务托管项目资料 URL”下的文本框中，键入财务托管网站上你的个人资料页的 URL。 这有助于我们验证你是否确实已在财务托管注册。
  
  ![你的财务托管的项目资料 URL 的文本框屏幕截图](/assets/images/help/sponsors/fiscal-host-project-profile-field.png)
