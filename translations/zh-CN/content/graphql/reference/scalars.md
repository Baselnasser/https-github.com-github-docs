## 关于标量

[标量](https://graphql.github.io/graphql-spec/June2018/#sec-Scalars)是基元值：`Int`、`Float`、`String`、`Boolean` 或 `ID`。

调用 GraphQL API 时，必须指定嵌套子字段，直到仅返回标量。

有关详细信息，请参阅“[GraphQL 简介](/graphql/guides/introduction-to-graphql#field)”。
