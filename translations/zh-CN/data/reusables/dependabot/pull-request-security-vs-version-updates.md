当 {% data variables.product.prodname_dependabot %} 提出拉取请求时，这些拉取请求可以是安全更新或版本更新： 

- {% data variables.product.prodname_dependabot_security_updates %} 是自动拉取请求，可帮助你更新已知漏洞的信赖项。 
- {% data variables.product.prodname_dependabot_version_updates %} 是自动拉取请求，即使它们没有任何漏洞，也会保持更新依赖项。 要检查版本更新的状态，请依次导航到仓库的 Insights（见解）选项卡、Dependency Graph（依赖关系图）、{% data variables.product.prodname_dependabot %}。
