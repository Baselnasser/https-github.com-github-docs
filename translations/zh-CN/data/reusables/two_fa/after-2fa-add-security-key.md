配置 2FA 后，使用基于时间的一次性密码 (TOTP) 移动应用 {% ifversion fpt or ghec %}，或通过短信 {% endif %} 添加安全密钥，例如指纹读取器或Windows Hello。 启用安全密钥身份验证的技术称为 WebAuthn。 WebAuthn 是 U2F 的继承者，适用于所有现代浏览器。 有关详细信息，请参阅“[WebAuthn](https://webauthn.guide/)”和“[是否可以使用](https://caniuse.com/#search=webauthn)。”
