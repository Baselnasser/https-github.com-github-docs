您可以对仓库启用或禁用 {% data variables.product.prodname_discussions %}。

{% data reusables.repositories.navigate-to-repo %}
1. 在存储库名称下，单击 {% octicon "gear" aria-label="The gear icon" %}“设置”。
![存储库设置按钮](/assets/images/help/discussions/public-repo-settings.png)
1. 在“功能”下，选择“讨论”。
    ![“功能”下用于启用或禁用存储库的 {% data variables.product.prodname_discussions %} 的复选框](/assets/images/help/discussions/select-discussions-checkbox.png)
