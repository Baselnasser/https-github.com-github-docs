注意：如果你是安全研究人员，应直接联系维护人员，要求他们创建安全通告，或在你不管理的存储库中代表你发布 CVE。 但是，如果为存储库启用了私人漏洞报告，则可以自行私下报告漏洞。 有关详细信息，请参阅“[私下报告安全漏洞](/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)”。
