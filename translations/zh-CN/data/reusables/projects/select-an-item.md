1. 选择该项。 要选择多个项，请执行下列操作之一：
     - 按住 <kbd>Command</kbd> (Mac) 或 <kbd>Ctrl</kbd> (Windows/Linux) 并单击每个项。
     - 选择某项，然后按 <kbd>Shift</kbd>+<kbd>↑</kbd> 或 <kbd>Shift</kbd>+<kbd>↓</kbd> 以选择所选项上方或下方的其他项。
     - 选择一个项，然后按 <kbd>Shift</kbd> 并单击另一个项以选择两个项之间的所有项。
     - 选择一行或一项后，按 <kbd>Command</kbd>+<kbd>A</kbd> (Mac) 或 <kbd>Ctrl</kbd>+<kbd>A</kbd> (Windows/Linux)，以选择板布局中某一列的所有项或表格布局中的所有项。
