对于每个组的作业，{% data variables.product.prodname_classroom %} 自动为团队创建一个单独的共享仓库进行访问。 这个仓库可以是空的，您也可以从模板仓库创建这个仓库，加入启动代码、文档、测试和其他资源。 仓库属于您在 {% data variables.product.product_name %} 上的组织帐户，{% data variables.product.prodname_classroom %} 授予学生在接受作业时创建或加入的团队访问权限。
