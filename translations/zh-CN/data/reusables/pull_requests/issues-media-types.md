### 自定义媒体类型

以下是此 API 支持的媒体类型：

    application/vnd.github.raw+json
    application/vnd.github.text+json
    application/vnd.github.html+json
    application/vnd.github.full+json

有关媒体类型的详细信息，请参阅“[自定义媒体类型](/rest/overview/media-types)”。
