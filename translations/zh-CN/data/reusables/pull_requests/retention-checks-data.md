检查超过 400 天的数据是否已存档。 在存档过程中，{% data variables.product.prodname_dotcom %} 创建汇总提交状态，表示该提交的所有检查的状态。 因此，具有需要存档检查的任何拉取请求中的合并框都将处于阻止状态，你需要重新运行检查，然后才能合并拉取请求。
