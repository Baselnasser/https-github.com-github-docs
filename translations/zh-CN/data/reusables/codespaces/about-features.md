功能是指安装代码和开发容器配置的独立单元，其设计目的是跨各种基本容器映像工作。 可以使用功能将工具、运行时或库快速添加到 codespace 映像。 有关详细信息，请参阅 containers.dev 上的[可用功能](https://containers.dev/features)和[功能规范](https://containers.dev/implementors/features/)。

可以从 {% data variables.product.prodname_vscode_shortname %} 或 {% data variables.product.prodname_dotcom_the_website %} 上的存储库向 `devcontainer.json` 文件添加功能。
