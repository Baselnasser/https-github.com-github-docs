如你当前未在 codespace 中工作，可从 {% data variables.product.prodname_vscode_shortname %} 内删除 codespace。

{% data reusables.codespaces.click-remote-explorer-icon-vscode %}
1. 在“GitHub Codespaces”下，右键点击你要删除的 codespace。
1. 单击“删除 codespace”。
