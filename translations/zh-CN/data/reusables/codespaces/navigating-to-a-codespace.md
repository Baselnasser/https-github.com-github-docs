## 导航到现有代码空间

1. {% data reusables.codespaces.you-can-see-all-your-codespaces %}
1. 单击您要在其中开发的代码空间的名称。

   ![Codespace 的名称](/assets/images/help/codespaces/click-name-codespace.png)

或者，你可以通过导航到该存储库并选择“{% octicon "code" aria-label="The code icon" %} 代码”来查看特定存储库的任何 codespace。 下拉菜单将显示存储库中所有处于活动状态的 codespace。
