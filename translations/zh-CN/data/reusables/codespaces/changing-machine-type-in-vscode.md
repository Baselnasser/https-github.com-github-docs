1. 在 {% data variables.product.prodname_vscode_shortname %} 中，使用 <kbd>Command</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> (Mac) or <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> (Windows/Linux) 打开命令面板。
1. 搜索并选择“代码空间：更改机器类型”。

   ![搜索分支以创建新的 {% data variables.product.prodname_codespaces %}](/assets/images/help/codespaces/vscode-change-machine-type-option.png)
1. 如果没有在 codespace 中遵循这些说明，请单击要更改的 codespace。

   ![搜索分支以创建新的 {% data variables.product.prodname_codespaces %}](/assets/images/help/codespaces/vscode-change-machine-choose-repo.png)

   如果要在 codespace 中遵循这些说明，则更改将应用于正在使用的 codespace。
1. 选择你要使用的计算机类型。
1. 如果要更改为具有不同存储容量的计算机类型，则会出现一个提示，该提示询问你是否要继续。 阅读提示并单击“是”接受。
