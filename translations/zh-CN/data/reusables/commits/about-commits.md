类似于保存已经编辑的文件，提交会记录对分支中一个或多个文件的更改。 Git 将为每个提交分配唯一的 ID，称为 SHA 或哈希，用于识别：

- 具体的更改
- 进行更改的时间
- 更改创建者

在进行提交时，必须包含简要描述更改的提交消息。
