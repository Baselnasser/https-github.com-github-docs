### 支持的 GPG 密钥算法

{% data variables.product.product_name %} 支持多种 GPG 密钥算法。 如果您尝试添加使用不支持的算法生成的密钥，可能会发生错误。

- RSA
- ElGamal
- DSA
- ECDH
- ECDSA
- EdDSA
