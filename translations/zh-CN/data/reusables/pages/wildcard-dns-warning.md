警告：强烈建议勿使用通配符 DNS 记录，例如 `*.example.com`。 通配符 DNS 记录将允许任何人在你的其中一个子域上托管 {% data variables.product.prodname_pages %} 站点，即使这些子域会进行验证也是如此。 有关详细信息，请参阅“[验证 {% data variables.product.prodname_pages %} 的自定义域](/pages/configuring-a-custom-domain-for-your-github-pages-site/verifying-your-custom-domain-for-github-pages)”。
