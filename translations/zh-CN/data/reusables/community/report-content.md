1. 你可能会看到“向存储库管理员报告”或“向 {% data variables.product.prodname_dotcom %} 支持报告滥用情况”的选项 。 如果没有，请跳至下一步。
    - 要向 {% data variables.product.prodname_dotcom %} 支持报告内容，请单击“向 {% data variables.product.prodname_dotcom %} 支持报告滥用情况”。
    - 要向存储库维护员报告内容，请使用“选择原因”下拉菜单选择原因，然后单击“向存储库管理员报告” 。 您的报告已提交。
  ![“报告内容”模式](/assets/images/help/repository/report-content-modal.png)
2. 完成“报告内容”表单，然后单击“发送请求”。
  ![“报告内容”表单](/assets/images/help/repository/report-content-to-support-form.png)
