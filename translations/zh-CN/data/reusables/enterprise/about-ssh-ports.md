每个 {% data variables.product.product_name %} 实例都通过两个端口接受 SSH 连接。 站点管理员可以通过 SSH 访问管理 shell，然后运行命令行实用工具、故障排除并执行维护。 用户可以通过 SSH 进行连接，以访问和写入实例存储库中的 Git 数据。 用户对实例没有 shell 访问权限。 有关详细信息，请参阅以下文章。

- [网络端口](/admin/configuration/configuring-network-settings/network-ports)
- [访问管理 shell (SSH)](/admin/configuration/configuring-your-enterprise/accessing-the-administrative-shell-ssh)
- [关于 SSH](/authentication/connecting-to-github-with-ssh/about-ssh)
