本文面向经典 GitHub 搜索体验。 如果你正在使用新的代码搜索和代码视图 beta 版本并启用了新功能，则搜索代码的语法和 GitHub.com 搜索界面将有所不同。 但是，非代码搜索类型的搜索语法将保持不变。

若要了解当前处于有限公共 beta 版本的新代码搜索和代码视图，请参阅“[关于 GitHub 代码搜索（beta 版本）](/search-github/github-code-search/about-github-code-search)”。 若要试用新的代码搜索和代码视图 beta 版本，请加入[等待列表](https://github.com/features/code-search-code-view/signup)。
