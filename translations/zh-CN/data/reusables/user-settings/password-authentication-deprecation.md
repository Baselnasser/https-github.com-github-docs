当 Git 提示输入密码时，请输入个人访问令牌 (PAT)。 或者，可以使用 [Git 凭据管理器](https://github.com/GitCredentialManager/git-credential-manager/blob/main/README.md)等凭据帮助程序。{% ifversion not ghae %} Git 的基于密码的身份验证已被删除，以支持更安全的身份验证方法。{% endif %} 有关详细信息，请参阅“[创建个人访问令牌](/github/authenticating-to-github/creating-a-personal-access-token)”。
