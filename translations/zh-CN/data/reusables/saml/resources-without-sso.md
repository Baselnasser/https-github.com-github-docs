以某些方式访问公共存储库时不需要 IdP 身份验证：

- 查看 {% data variables.product.prodname_dotcom %} 上的存储库概述页面和文件内容
- 复刻存储库
- 通过 Git 执行读取操作，例如克隆存储库

通过其他方式访问公共存储库（例如查看问题、拉取请求、项目和发布）时需要身份验证。
