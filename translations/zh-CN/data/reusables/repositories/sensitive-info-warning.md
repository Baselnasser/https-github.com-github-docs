警告：从不 `git add``commit` 或 `push` 敏感信息到远程存储库。 敏感信息包括但不限于：

- 密码
- SSH 密钥
- [AWS 访问密钥](http://docs.aws.amazon.com/AWSSimpleQueueService/latest/SQSGettingStartedGuide/AWSCredentials.html)
- API 密钥
- 信用卡号
- PIN 号码

有关详细信息，请参阅“[从存储库中删除敏感数据](/articles/removing-sensitive-data-from-a-repository)”。

{% endwarning %}
