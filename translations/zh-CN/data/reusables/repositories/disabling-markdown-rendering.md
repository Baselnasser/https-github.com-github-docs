查看 Markdown 文件时，可单击文件顶部的 {% octicon "code" aria-label="The code icon" %} 以禁用 Markdown 呈现并改为查看文件的源。

![将 Markdown 显示为源](/assets/images/help/writing/display-markdown-as-source.png)

禁用 Markdown 呈现使你能够使用源视图功能，例如行链接，这在查看呈现的 Markdown 文件时不可用。
