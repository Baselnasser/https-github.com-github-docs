1. 要创建可供审查的拉取请求，请单击“创建拉取请求”。
若要创建草稿拉取请求，请使用下拉列表并选择“创建草稿拉取请求”，然后单击“草稿拉取请求” 。 有关草稿拉取请求的详细信息，请参阅“[关于拉取请求](/articles/about-pull-requests#draft-pull-requests)”。![“创建拉取请求”按钮](/assets/images/help/pull_requests/pullrequest-send.png)
