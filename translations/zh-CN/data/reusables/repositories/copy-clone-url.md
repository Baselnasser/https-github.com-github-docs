1. 在文件列表上方，单击 {% octicon "download" aria-label="The download icon" %}“代码”。
  ![“代码”按钮](/assets/images/help/repository/code-button.png)
1. 复制存储库的 URL。

   - 要使用 HTTPS 克隆存储库，在“HTTPS”下单击 {% octicon "clippy" aria-label="The clipboard icon" %}。 
   - 要使用 SSH 密钥克隆存储库，包括组织的 SSH 证书颁发机构颁发的证书，请单击“SSH”，然后单击 {% octicon "clippy" aria-label="The clipboard icon" %}。 
   - 要使用 {% data variables.product.prodname_cli %} 克隆存储库，请单击“{% data variables.product.prodname_cli %}”，然后单击 {% octicon "clippy" aria-label="The clipboard icon" %}。
  ![用于复制 URL 以使用 GitHub CLI 克隆存储库的剪贴板图标](/assets/images/help/repository/https-url-clone-cli.png)
