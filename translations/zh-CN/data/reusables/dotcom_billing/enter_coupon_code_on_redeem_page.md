1. 访问“[兑换优惠券](https://github.com/redeem)”页面。
2. 在“输入优惠券代码”框中，键入优惠券代码，然后单击“兑换”。
   ![“兑换优惠券”框](/assets/images/help/settings/redeem-coupon-box.png)
3. 如果您尚未登录，可以在此页面上登录，或者新建一个帐户来使用您的优惠券。
