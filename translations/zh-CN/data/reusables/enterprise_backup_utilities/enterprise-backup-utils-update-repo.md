1. 若要更新到最新项目发行版本，请通过运行 `git checkout stable` 命令使用 `stable` 分支。

  ```shell
  git checkout stable
  ```
  
  或者，若要使用特定项目版本，请运行以下命令，将 `X.Y.Z` 替换为所需的发行版本。

  ```shell
  $ git checkout vX.Y.Z
  ```
