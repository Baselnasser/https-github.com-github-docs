Web 挂钩可通过不同的内容类型传递：

- `application/json` 内容类型直接将 JSON 有效负载作为 `POST` 请求的正文传递。
- `application/x-www-form-urlencoded` 内容类型将 JSON 有效负载作为表单参数发送，称为 `payload`。

选择最符合你需要的预设。
