创建 IP 允许列表后，可以启用允许的 IP 地址。 启用允许的 IP 地址时，{% data variables.product.company_short %} 会立即强制实施 IP 允许列表中的任何已启用条目。

在启用你的 IP 允许列表之前，可以检查你的允许列表是否允许来自特定 IP 地址的连接。 有关详细信息，请参阅“[检查是否允许使用 IP 地址](#checking-if-an-ip-address-is-permitted)”。
