### 使用限制

团队同步功能存在使用限制。  超过这些限制将导致性能下降，并可能导致同步失败。

- GitHub 团队中的成员人数上限：5,000
- GitHub 组织中的成员人数上限：10,000
- GitHub 组织中的团队数上限：1,500
