配置文件可以位于正在分析的存储库中，也可以位于外部存储库中。 使用外部存储库可以在一个位置为多个存储库指定配置选项。 引用位于外部存储库中的配置文件时，可以使用 OWNER/REPOSITORY/FILENAME@BRANCH 语法。 例如， _octo-org/shared/codeql-config.yml@main_ 。
