从下拉菜单中选择合适的原因很重要，因为这可能会影响到是否继续将查询纳入未来的分析。 {% ifversion comment-dismissed-code-scanning-alert %}（可选）可对关闭操作进行注释，记录警报关闭情况的上下文。 关闭操作注释将添加到警报时间线，可在审核和报告期间用作理由。 可使用代码扫描 REST API 检索或设置注释。 注释包含在 `alerts/{alert_number}` 终结点的 `dismissed_comment` 中。 有关详细信息，请参阅“[代码扫描](/rest/code-scanning#update-a-code-scanning-alert)”。
{% endif %}
