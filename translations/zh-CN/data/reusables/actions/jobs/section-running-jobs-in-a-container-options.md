使用 `jobs.<job_id>.container.options` 配置其他 Docker 容器资源选项。 有关选项列表，请参阅“[`docker create` 选项](https://docs.docker.com/engine/reference/commandline/create/#options)”。

{% warning %}

**警告：** 不支持 `--network` 选项。

{% endwarning %}
