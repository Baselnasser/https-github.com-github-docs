如果客户升级其计划但付款失败，GitHub 会将其 {% data variables.product.prodname_marketplace %} 订阅还原到以前的状态。 GitHub 还会向客户发送一封电子邮件，通知他们失败的情况，并允许他们重新尝试购买。 你将会收到带有 `changed` 操作的 Webhook，要求你还原到以前的计划。
