# 词汇表

以下文件包括 [Crowdin 词汇表](https://support.crowdin.com/glossary/)：

* `external.yml` 包含面向客户的词汇表条目。
  * `external.yml` 中的字符串支持 Liquid 条件。 请参阅 [contributing/liquid-helpers.md](/contributing/liquid-helpers.md)。
* `internal.yml` 包含仅供翻译人员使用的条目。 这些术语显示在 Crowdin UI 中，为译员提供关于所翻译内容的附加上下文信息，以及相应术语的推荐本地化字符串。
* `candidates.yml` 包含可能位于内部或外部词汇表中但尚未定义的术语。
