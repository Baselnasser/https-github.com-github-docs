アプリケーションのリストが{% data variables.product.prodname_marketplace %}の要件を満たせば、アプリケーションをサブミットできます。 アプリケーションのリストに関する要件の概要については、「[アプリケーションをリストするための要件](/developers/github-marketplace/requirements-for-listing-an-app)」を参照してください。

1. アプリケーション掲載ページの左上にある **[概要]** をクリックします。 アプリケーション掲載ページの探し方に関する情報については、「[アプリケーションのリストのドラフト](/developers/github-marketplace/drafting-a-listing-for-your-app)」を参照してください。
  
  ![Marketplaceのリストのドラフトの概要オプション](/assets/images/marketplace/edit-marketplace-listing-overview.png)

2. 完成したアプリの一覧を送信するには、 **[公開の要求]** をクリックします。

  ![下に提出ボタンの付いた、[Marketplace にアプリを公開する] チェックリスト](/assets/images/marketplace/publish-your-app-checklist-and-submission.png)

オンボーディングの専門家が後ほど連絡し、追加情報をお伝えします。
