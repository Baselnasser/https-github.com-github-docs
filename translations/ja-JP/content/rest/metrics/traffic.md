## リポジトリ トラフィック API について

ユーザーがプッシュ アクセス権を持つリポジトリの場合、リポジトリ トラフィック API を使うと、リポジトリ グラフで提供される情報にアクセスできます。 詳細については、「[リポジトリへのトラフィックを表示する](/repositories/viewing-activity-and-data-for-your-repository/viewing-traffic-to-a-repository)」を参照してください。
