## リポジトリ自動リンク API について

詳細については、「[外部リソースを参照する自動リンクの構成](/github/administering-a-repository/configuring-autolinks-to-reference-external-resources)」を参照してください。

自動リンク API を使うには、読み取りまたは書き込みアクセス権を持つリポジトリ管理アクセス許可が {% data variables.product.prodname_github_apps %} に必要です。
