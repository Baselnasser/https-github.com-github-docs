## 行動規範 API について

Codes of Conduct API を使用して、リポジトリの行動規範に関する情報を取得できます。 リポジトリの行動規範を取得するには、"[リポジトリの取得](/rest/reference/repos#get-a-repository)" エンドポイントを使います。
