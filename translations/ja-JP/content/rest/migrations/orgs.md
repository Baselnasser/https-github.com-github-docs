## Organization の移行 API について

移行 API を使用できるのは、認証済みの Organization オーナーのみです。 詳細については、「[organization のロール](/organizations/managing-peoples-access-to-your-organization-with-roles/roles-in-an-organization#permission-levels-for-an-organization)」および「[その他の認証方式](/rest/overview/other-authentication-methods)」を参照してください。
