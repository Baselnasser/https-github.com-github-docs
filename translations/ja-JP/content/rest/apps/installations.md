## {% data variables.product.prodname_github_app %} インストール API について

"_インストール_" とは、アプリをインストールしたユーザーまたは組織のアカウントを指します。 インストールとして認証し、特定のリポジトリへのアクセスを制限する方法については、「[インストールとして認証を行う](/apps/building-github-apps/authenticating-with-github-apps/#authenticating-as-an-installation)」を参照してください。

Organization のすべての GitHub App インストールを一覧表示するには、「[Organization のアプリ インストールの一覧表示](/rest/orgs/orgs#list-app-installations-for-an-organization)」を参照してください。
