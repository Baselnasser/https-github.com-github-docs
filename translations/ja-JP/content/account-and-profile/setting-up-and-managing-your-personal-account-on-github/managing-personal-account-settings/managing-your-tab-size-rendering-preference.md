{% data variables.product.product_name %} でレンダリングされたコードのタブによるインデントが大きすぎるか、小さすぎると感じられる場合、設定でそれを変更できます。

{% data reusables.user-settings.access_settings %}
1. 左側のサイドバーで、 **[{% octicon "paintbrush" aria-label="The paintbrush icon" %} 外観]** をクリックします。
2. [Tab size preference] の下でドロップダウン メニューを選択し、基本設定を選択します。
   ![タブ サイズの基本設定ボタン](/assets/images/help/settings/tab-size-preference.png )
