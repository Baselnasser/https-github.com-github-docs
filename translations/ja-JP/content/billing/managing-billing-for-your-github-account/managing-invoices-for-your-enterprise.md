## Entepriseの請求書について

{% data reusables.billing.about-invoices-for-enterprises %} 詳細については、「[Enteprise の支払いについて](/billing/managing-billing-for-your-github-account/about-billing-for-your-enterprise)」を参照してください。

Enteprise アカウントのサブスクリプションと利用状況を表示する方法については、「[Viewing the subscription and usage for your enterprise account (Enterprise アカウントのサブスクリプションと利用状況を表示する)](/billing/managing-billing-for-your-github-account/viewing-the-subscription-and-usage-for-your-enterprise-account)」を参照してください。

## 現在の請求書を表示する

{% data reusables.enterprise-accounts.access-enterprise-on-dotcom %} {% data reusables.enterprise-accounts.settings-tab %} {% data reusables.enterprise-accounts.billing-tab %}
4. [現在の請求書] で **[請求書の表示]** をクリックします。
  ![[請求書の表示] リンク](/assets/images/help/business-accounts/view-invoice-link.png)

## 現在の請求書の支払い

{% data reusables.enterprise-accounts.access-enterprise-on-dotcom %} {% data reusables.enterprise-accounts.settings-tab %} {% data reusables.enterprise-accounts.billing-tab %}
4. [支払期限] で、 **[請求書の支払い]** をクリックします。
  ![[請求書の支払い] リンク](/assets/images/help/business-accounts/pay-invoice-link.png)
5. [請求書の支払い] で、セキュリティで保護されたフォームにクレジットカード情報を入力し、 **[請求書の支払い]** をクリックします。
  ![確認して請求書を支払う](/assets/images/help/business-accounts/pay-invoice.png)

## 現在の請求書をダウンロードする

{% data reusables.enterprise-accounts.access-enterprise-on-dotcom %} {% data reusables.enterprise-accounts.settings-tab %} {% data reusables.enterprise-accounts.billing-tab %}
4. [クイック アクション] で **[現在の請求書のダウンロード]** をクリックします。
  ![[現在の請求書のダウンロード] リンク](/assets/images/help/business-accounts/download-current-invoice.png)

## 支払い履歴を表示する

{% data reusables.enterprise-accounts.access-enterprise-on-dotcom %} {% data reusables.enterprise-accounts.settings-tab %} {% data reusables.enterprise-accounts.billing-tab %}
4. 過去の支払いアクティビティの概要を表示するには、[支払い] の **[過去の請求書]** タブをクリックします。![支払い履歴タブの表示](/assets/images/help/business-accounts/view-payment-history.png)
