## Repl.it 統合による変更点

2021 年 7 月 13 日をもって、Repl.it 統合は利用できなくなりました。 変更点は次のとおりです。

- GitHub Classroom 内から新しいプライベートとパブリックの repls を作成できなくなりました。
- Always On や Boosts などの Replit Hacker の特典は、新しい GitHub Classroom ユーザーには使用できなくなりました。

## これによる既存の課題への影響
既存の GitHub Classroom の repls は永久に非公開のままとなります。2021 年 9 月 30 日までは、これらの repls を介して GitHub Classroom に作業を送信できます。 2021 年 9 月以降、課題 README の Replit アイコンは機能しなくなります。 統合を使用して README から指示を削除し、アイコンが機能しなくなることを学生に通知することをお勧めします。

Classroom 内からプライベートの repls を作成することはできなくなりますが、9 月 30 日まで Replit でのプライベートの repls 作成は引き続きできます。

クラウド IDE の統合は Classroom にとって重要であるため、より多くのオプションを提供できるように努力しています。
