## 送信について

{% data variables.product.prodname_community_exchange %} 送信プロセス中に、リポジトリの目的、トピック、オファーを選びます。 リポジトリが {% data variables.product.prodname_community_exchange %} に送信されると、これらの詳細と共に発行されます。 詳しくは、「[GitHub Community Exchange へのリポジトリの送信](/education/contribute-with-github-community-exchange/submitting-your-repository-to-github-community-exchange)」を参照してください。

送信後も、リポジトリに関連付けられているトピックとオファーを編集できます。 また、対応するトピックを変更することで、リポジトリの目的を更新することもできます。 詳細については、「[トピックを使用したリポジトリの分類](/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/classifying-your-repository-with-topics)」を参照してください。

リポジトリに関連付けられている言語は、使用される第 1 言語であり、{% data variables.product.prodname_dotcom %} によって自動的に決定されます。 詳しくは、「[リポジトリ言語について](/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-repository-languages)」を参照してください。

{% data variables.product.prodname_community_exchange %} コミュニティでは、すべてのリポジトリの送信をモデレートします。

## 送信の管理

1. {% data variables.product.prodname_global_campus %} ダッシュボードから、{% data variables.product.prodname_community_exchange %} ホーム ページに移動します。
1. リポジトリ リストの上にある **[送信]** タブをクリックします。![[送信] タブのスクリーンショット](/assets/images/help/education/community-exchange-submissions-tab.png)
1. 必要に応じて、送信されたリポジトリを編集します。 編集するリポジトリの右側にある {% octicon "pencil" aria-label="The edit icon" %} をクリックして、リポジトリのホームページに直接移動します。 そこから、リポジトリに割り当てられた目的、トピック、オファーを更新できます。
1. 必要に応じて、ギャラリーから送信されたリポジトリを削除します。 削除するリポジトリ送信の右側にある {% octicon "trash" aria-label="The trash icon" %} をクリックします。
