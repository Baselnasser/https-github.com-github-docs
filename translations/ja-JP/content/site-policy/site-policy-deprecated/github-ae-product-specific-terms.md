_これらの契約条件は、2021 年 1 月 4 日より前に製品のライセンスを取得したお客様に適用されます。その日以降に GitHub 製品を購入したお客様は、最新の契約条件として https://www.github.com/enterprise-legal をご覧ください。_

本契約は、GitHub AE 製品固有の条件、お客様が承諾した一般条件、および注文時に GitHub またはその関係会社が提示する追加条項で構成されます。

## 1. アカウント。

**アカウントの責任。** お客様は、エンド ユーザーのアカウントとコンテンツを管理し、責任を負うものとします。

**アカウントのセキュリティ。** お客様は、ご自身のアカウントのログイン資格情報のセキュリティを維持する責任を負うものとします。

**ポリシーの使用。** お客様のエンド ユーザーは、利用規定を遵守する必要があります。

**一時停止。** GitHub は、お客様が重大な違反を犯している期間、本オンライン サービスの利用を停止することができます。

**アクセス。** GitHub は、サポート案件やセキュリティ上の目的で必要な場合を除き、お客様のコンテンツにアクセスすることはありません。

**DMCA。** GitHub には、著作権侵害の申し立てに適用されるデジタル ミレニアム著作権法 (DMCA) テイクダウン ポリシーがあります。

## 2. コンテンツ。

**コンテンツの所有権。** お客様は、作成したコンテンツを所有し、お客様が投稿するコンテンツに関するサードパーティのライセンスを完全に遵守するものとします。

**GitHub へのライセンス付与。** お客様のコンテンツに本オンライン サービスの運営に必要な権利を GitHub に付与するライセンスが別途付属している場合を除き、お客様は、本オンライン サービスやサポートの提供、またはセキュリティ上の理由により必要に応じてお客様コンテンツを使用して付随するコピーを作成する権利を GitHub に付与するものとします。 また、GitHub は、法律によりお客様のコンテンツの開示を強制される場合があります。

## 3. GitHub 以外の製品
GitHub は、本オンライン サービスを通じて GitHub 以外の製品を利用できるようにすることがあります。 お客様が本オンライン サービスで GitHub 以外の製品を利用する場合、お客様は、本契約に明示的に含まれる義務以上の義務を GitHub の知的財産に負わせるような方法で利用することはできないものとします。 GitHub は、GitHub 以外の製品について一切の責任または義務を負いません。 お客様による GitHub 以外の製品の利用については、お客様と GitHub 製品以外のパブリッシャーとの契約条項 (存在する場合) に準拠します。

## 4. サポートと SLA。
本オンライン サービスには、サポートと SLA が含まれます。

## 5. データ保護とセキュリティ。
本オンライン サービスには、データ保護契約の条項が適用されます。

## 6. 通知。
GitHub への通知は、GitHub, Inc. Attn: Legal Dept., 88 Colin P. Kelly St, San Francisco, CA 94107 USA の住所に送付する必要があります。

## 7. 定義。
「**利用規定**」とは、GitHub から購入する場合は、サイトポリシーに掲載されている利用規約、Microsoft から購入する場合は、Microsoft オンライン サービス条件に掲載されている利用規約を意味します。

「**コンテンツ**」とは、本オンライン サービスを通じて表示またはその他の方法で利用可能になるテキスト、データ、ソフトウェア、画像、およびその他の素材を意味します。

「**お客様のコンテンツ**」とは、お客様が作成、所有、またはお客様が権利を有するコンテンツを意味します。

「**データ保護契約**」とは、 https://docs.github.com/github/site-policy に掲載されている GitHub AE データ保護契約を意味します。

「**デジタル ミレニアム著作権法テイクダウン ポリシー**」とは、 https://docs.github.com/github/site-policy に掲載されている著作権侵害の通知を処理する場合における GitHub の手続きを意味します。

「**SLA**」とは、 https://docs.github.com/github/site-policy に掲載されている本オンライン サービスに対する GitHub の稼働時間に関する誓約事項を意味します。

「**サポート**」とは、 https://docs.github.com/github/site-policy でさらに説明されている GitHub の Premium Plus サポート プログラムを意味します。
