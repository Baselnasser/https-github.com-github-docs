## 1. 企業向け利用規約

評価版にアクセスするお客様には、GitHub.com の法人アカウントが必要です。 Enterprise Cloud のお客様の評価は、本規約および Corporate Terms of Service ([https://docs.github.com/articles/github-corporate-terms-of-service](/articles/github-corporate-terms-of-service) に見い出せます) (合わせて、以下「**評価契約**」という) に準拠するものとします。 ただし、[Enterprise Cloud 追加契約](/articles/github-enterprise-cloud-addendum)に関連する補足条項は、お客様が購入を Enterprise Cloud の有償サブスクリプションにアップグレードするまでは適用 **されない** ものとします。   本評価契約で使用されている用語のうち定義のないものについては、企業向けサービス使用条件で定義されています。

## 2. 契約期間および解約

*2.1 期間* 「評価契約発効日」とは、お客様が本評価契約の条件に同意した日付です。 本契約は、評価契約発効日に開始し、14 日間有効です (以下「**評価期間**」と言う)。

*2.2 解除* どの当事者も、いかなる理由でも、理由なく本評価契約を解約することができます。 セクション 2.3 は、本評価契約を解約するいずれかの当事者の判断に適用されます。 お客様が本評価契約の条項に違反した場合、本評価契約は通知を必要とせず直ちに解約となります。

*2.3 解約の効果* 本評価期間の期限が切れると、サブスクリプション ライセンスは自動的に終了します。お客様は Enterprise Cloud を使用する権利を失い、お客様のアカウントのコンテンツ、機能、または容量にアクセスできなくなる場合があります。 コンテンツのコピーの取得については、企業向け利用規約で「解約の効果」セクションを参照してください。
