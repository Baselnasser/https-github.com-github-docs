GraphQL API の使用を始めるには、「[GraphQL の概要](/graphql/guides/introduction-to-graphql)」を参照してください。

Sponsors GraphQL API の詳細については、リファレンスドキュメントをご覧ください。詳細については、「[GraphQL リファレンス](/graphql/reference)」を参照してください。 GraphQL エクスプローラーを使用して GraphQL 呼び出しを構築することをお勧めします。 詳細については、「[Explorer の利用](/graphql/guides/using-the-explorer)」を参照してください。
