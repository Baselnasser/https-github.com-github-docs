スポンサーがいる場合は、{% data variables.product.prodname_sponsors %} アカウントを非公開にするときに GitHub Support に問い合わせて支援を受ける必要があります。

{% data variables.product.prodname_sponsors %} アカウントが非公開になると、以下のようになります。
- スポンサーシップの支払いを受け取ることはなくなります。
- プロフィールの再公開をリクエストできます。

{% data reusables.sponsors.navigate-to-sponsors-dashboard %} {% data reusables.sponsors.settings-tab %}
3. [GitHub Sponsors プロファイルの公開取り消し] で、 **[プロファイルの公開取り消し]** をクリックします。
  ![[プロファイルの公開取り消し] ボタン](/assets/images/help/sponsors/unpublish-profile-button.png)
4. [Unpublish GitHub Sponsors account] をクリックして確認します。
  ![[GitHub Sponsors プロファイルの公開取り消し] ボタン](/assets/images/help/sponsors/unpublish-profile-dialog.png)
