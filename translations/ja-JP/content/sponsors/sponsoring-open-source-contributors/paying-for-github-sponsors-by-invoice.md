{% data reusables.sponsors.beta-note-sponsors-for-companies %}

企業は、請求書の方法を使用して、{% data variables.product.prodname_sponsors %} プログラムに参加できます。 これは、企業に {% data variables.product.company_short %} との既存の請求書による課金関係がある場合に特に便利です。 請求書による支払いを行っている企業は、関心のある {% data variables.product.prodname_dotcom_the_website %} の公開プロジェクト全体でのオープン ソースの貢献と活動を測定する {% data variables.product.product_name %} ダッシュボードにアクセスできます。

このベータ プログラムに参加するには、営業担当者にお問い合わせいただくか、[こちらにお問い合わせください](https://support.github.com/contact/org-sponsors-waitlist)。
