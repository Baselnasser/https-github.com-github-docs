## {% data variables.product.product_name %} の Web ブラウザーのサポートについて

{% data variables.product.product_name %} は、最新の Web ブラウザーを考慮して設計されています。 次のいずれかのブラウザーの最新バージョンをご使用になることをお勧めします。

 - [Apple Safari](https://apple.com/safari)
 - [Google Chrome](https://google.com/chrome)
 - [Microsoft Edge](https://microsoft.com/windows/microsoft-edge)
 - [Mozilla Firefox](https://mozilla.org/firefox)
 
推奨されるブラウザーの最新バージョンを使わない場合、または上記以外のブラウザーを使った場合、{% data variables.product.product_name %} または一部の機能が期待どおりに、またはまったく動作しない可能性があります。

{% data variables.product.company_short %} の製品で、どのようにブラウザーの互換性が維持されているかの詳細については、[`github/browser-support`](https://github.com/github/browser-support) リポジトリを参照してください。
## 推奨 Web ブラウザーの延長サポート

一部のブラウザー ベンダーは、延長サポート リリースを提供しています。 {% data variables.product.product_name %} が最新の延長サポート リリースで適切に機能するよう、最善を尽くしています。

- Chrome の [Extended Stable チャンネル](https://support.google.com/chrome/a/answer/9027636)
- Edge の[拡張安定チャネル](https://docs.microsoft.com/en-gb/deployedge/microsoft-edge-channels#extended-stable-channel)
- Firefox の[延長サポート リリース](https://www.mozilla.org/en-US/firefox/organizations/) (ESR)

以前の延長サポート リリースでは、{% data variables.product.product_name %} が期待どおりに動作しなかったり、一部の機能が使用できなかったりする可能性があります。


## ベータ版および開発者版

サポートしているブラウザのベータ版や開発者版では予期しないバグに出くわす可能性があります。 このようなリリースされていないビルドで、{% data variables.product.product_name %} 上のバグに出くわした場合、同じブラウザの正式バージョンでも存在するかどうか確認してください。 もしバグが非公式バージョンに限って存在する場合、バグをブラウザのデベロッパーに報告することを検討してください。
