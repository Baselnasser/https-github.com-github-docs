## {% data variables.product.prodname_dotcom_the_website %} の新しいアカウントについて

個人アカウントを作成できます。このアカウントは、{% data variables.product.prodname_dotcom_the_website %} または組織の ID として機能します。組織を使用すると、複数の個人アカウントが複数のプロジェクトで共同作業を行うことができます。 アカウントの種類の詳細については、「[{% data variables.product.prodname_dotcom %} アカウントの種類](/get-started/learning-about-github/types-of-github-accounts)」を参照してください。

個人アカウントまたは組織を作成する場合は、アカウントの課金プランを選択する必要があります。 詳細については、「[{% data variables.product.company_short %} の製品](/get-started/learning-about-github/githubs-products)」を参照してください。

## 新しいアカウントにサインアップする

1. 新しい個人アカウントを作成する場合は、現在 GitHub からサインアウトしていることを確認します。
{% data reusables.accounts.create-account %}
1. プロンプトに従って、個人アカウントまたは Organization を作成してください。
