## Pull RequestをIssueにリンクする

修正が進行中であることを示すために pull request を Issue にリンクしたり、誰かが pull request をマージしたときに Issue を自動的に閉じるには、次のいずれかのキーワードに Issue への参照を追加入力します。 たとえば、`Closes #10` または `Fixes octo-org/octo-repo#100` です。

* 閉じる
* closes
* closed
* 解決策
* fixes
* 固定
* resolve
* resolves
* resolved

詳細については、「[pull request を Issue にリンクする](/github/managing-your-work-on-github/linking-a-pull-request-to-an-issue)」を参照してください。

## Issue または Pull Request を重複としてマークする

IssueあるいはPull Requestに重複としてマーク付けをするには、新たなコメントの本文に"Duplicate of"に続けて重複のIssueあるいはPull Requestの番号を入力してください。 詳細については、「[Issue または Pull Request を重複としてマークする](/issues/tracking-your-work-with-issues/marking-issues-or-pull-requests-as-a-duplicate)」を参照してください。
