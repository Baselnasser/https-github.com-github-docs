3. [Add a saved reply] の下に返信テンプレートのタイトルを追加します。
![返信テンプレートのタイトル](/assets/images/help/settings/saved-replies-title.png)
4. [Write] フィールドに、返信テンプレートに使用するコンテンツを追加します。 {% data variables.product.product_name %} での記述の詳細については、「[基本的な書き方とフォーマットの構文](/articles/basic-writing-and-formatting-syntax)」を参照してください。
![返信テンプレートの記述](/assets/images/help/settings/saved-replies-settings-adding.png)
5. 返信を確認するには、 **[プレビュー]** をクリックします。
![返信テンプレートの追加](/assets/images/help/settings/saved-replies-preview.png)
6. **[返信テンプレートを追加]** をクリックします。
![[返信テンプレートを追加] ボタン](/assets/images/help/settings/saved-replies-add-button.png)
