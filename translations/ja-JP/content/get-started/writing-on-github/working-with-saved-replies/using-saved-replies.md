2. 希望する Issue またはプルリクエストをクリックします。
3. 返信テンプレートを追加するには、コメント フィールドで、{% octicon "reply" aria-label="The mail reply" %} をクリックします。
   ![[返信テンプレート] ボタン](/assets/images/help/writing/saved-replies-button.png)
4. リストから、コメントに追加したい返信テンプレートを選択します。
   ![返信テンプレート](/assets/images/help/settings/saved-replies.png)

{% tip %}

**ヒント:**
- キーボードショートカットを使用して、返信テンプレートをコメントに自動入力できます。 詳細については、「[Keyboard shortcuts](/articles/keyboard-shortcuts/#comments)」 (キーボード ショートカット) を参照してください。
- 返信テンプレートのタイトルを入力して、フィルタリングすることができます。

{% endtip %}
