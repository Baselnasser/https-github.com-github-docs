使用可能なすべての Git コマンドを一回で覚えるのは大変です。 「[Git チート シート](https://training.github.com/)」を使用すると、頻繁に使用されるコマンドを簡単に参照できます。 「Git を使用する」のチートシートは、複数の言語で提供されています。 

さらに、ガイドやビデオなどへのリンクを示す [Git と GitHub の学習リソース](/articles/git-and-github-learning-resources/)のページをご覧ください。
