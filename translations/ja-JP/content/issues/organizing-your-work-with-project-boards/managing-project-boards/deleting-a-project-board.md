{% data reusables.projects.project_boards_old %}

{% tip %}

**ヒント**: 完了した、または不要になった {% data variables.projects.projects_v1_board %} へのアクセスを維持し、その内容へのアクセスが失われないようにするには、削除する代わりに [{% data variables.projects.projects_v1_board %} をクローズする](/articles/closing-a-project-board)ことができます。

{% endtip %}

1. 削除する {% data variables.projects.projects_v1_board %} に移動します。
{% data reusables.project-management.click-menu %} {% data reusables.project-management.click-edit-sidebar-menu-project-board %}
4. **[プロジェクトの削除]** をクリックします。
![[プロジェクトの削除] ボタン](/assets/images/help/projects/delete-project-button.png)
5. プロジェクトボードの削除を確認するには、 **[OK]** をクリックします。
