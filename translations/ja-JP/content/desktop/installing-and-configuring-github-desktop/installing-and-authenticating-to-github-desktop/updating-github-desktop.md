{% mac %}

1. **[GitHub Desktop]** メニューの **[GitHub Desktop について]** をクリックします。
![[GitHub Desktop について] メニューオプション](/assets/images/help/desktop/desktop-menu-about-desktop-mac.png)
2. **[更新プログラムの確認]** をクリックします。
![[更新プログラムの確認] ボタン](/assets/images/help/desktop/check-for-updates.png)
3. アップデートが利用可能な場合、アップデートをインストールするには、{% data variables.product.prodname_desktop %}を終了して、再起動します。

{% endmac %}

{% windows %}

1. **[ヘルプ]** メニューの **[GitHub Desktop について]** をクリックします。
![[GitHub Desktop について] メニューオプション](/assets/images/help/desktop/help-about-desktop-win.png)
2. **[更新プログラムの確認]** をクリックします。
![[更新プログラムの確認] ボタン](/assets/images/help/desktop/check-for-updates.png)
3. アップデートが利用可能な場合、アップデートをインストールするには、{% data variables.product.prodname_desktop %}を終了して、再起動します。

{% endwindows %}
