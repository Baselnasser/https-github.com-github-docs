{% mac %}

1. Finder ウィンドウで Applications フォルダに移動します。
  ![[Finder] ウィンドウの Applications フォルダー](/assets/images/help/desktop/applications-folder.png)
2. キーボードの **Control** キーを押しながら、 **{% data variables.product.prodname_desktop %}** をクリックします。
3. **[ごみ箱に移動]** を選択します。
  ![[ごみ箱に移動] オプション](/assets/images/help/desktop/mac-move-to-trash.png)
4. メニュー バーで、 **[Finder]** ドロップダウン メニューを使用し、 **[ごみ箱を空にする]** をクリックします。
  ![メニュー バーの [ごみ箱を空にする] オプション](/assets/images/help/desktop/mac-empty-trash-menu.png)
5. ポップアップ ボックスの警告を読んで、 **[ごみ箱を空にする]** をクリックします。
  ![[ごみ箱を空にする] ボタン](/assets/images/help/desktop/mac-empty-trash-button.png)

{% endmac %}

{% windows %}

1. [コントロール パネル] を開きます。 詳細については、Windows ヘルプの「[コントロール パネルの場所](https://support.microsoft.com/en-us/help/13764/windows-where-is-control-panel)」を参照してください。
2. [プログラム] で、 **[プログラムのアンインストール]** をクリックします。
  ![コントロール パネルの [プログラムのアンインストール] オプション](/assets/images/help/desktop/windows-uninstall-a-program.png)
3. **{% data variables.product.prodname_desktop %}** という名前のエントリを右クリックし、 **[アンインストール]** をクリックします。
  ![[アンインストール] オプション](/assets/images/help/desktop/windows-click-uninstall.png)

{% endwindows %}
