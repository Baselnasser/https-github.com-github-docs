{% tip %}

**ヒント:** フォルダを {% data variables.product.prodname_desktop %} ウィンドウにドラッグすると、Git リポジトリをローカル コンピューターから GitHub Desktop に追加することができます。 {% data variables.product.prodname_desktop %}に同時に複数のフォルダをドラッグする場合、各フォルダが別々のGitリポジトリとして追加されます。

{% endtip %}

{% mac %}

1. **[ファイル]** メニューで、 **[ローカル リポジトリの追加]** をクリックします。
  ![[ローカル リポジトリの追加] メニュー オプション](/assets/images/help/desktop/add-local-repository-mac.png)
2. **[選択]** をクリックし、[Finder] ウィンドウで、追加するローカル リポジトリに移動します。
  ![Mac アプリ内の "ローカル パス" フィールド](/assets/images/help/desktop/add-repo-choose-button-mac.png)
4. **[リポジトリの追加]** をクリックします。
  ![Mac アプリ内の [リポジトリの追加] ボタン](/assets/images/help/desktop/add-repository-button-mac.png)

{% endmac %}

{% windows %}

1. **[ファイル]** メニューで、 **[ローカル リポジトリの追加]** をクリックします。
  ![[ローカル リポジトリの追加] メニュー オプション](/assets/images/help/desktop/add-local-repository-windows.png)
2. **[選択]** をクリックし、Windows エクスプローラーで、追加するローカル リポジトリに移動します。
  ![Windows アプリ内の "ローカル パス" フィールド](/assets/images/help/desktop/add-repo-choose-button-win.png)
4. **[リポジトリの追加]** をクリックします。
  ![Windows アプリ内の [リポジトリの追加] ボタン](/assets/images/help/desktop/add-repository-button-windows.png)

{% endwindows %}
