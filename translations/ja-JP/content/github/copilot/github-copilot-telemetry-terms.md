## <a name="additional-telemetry"></a>追加の製品利用統計情報

{% data variables.product.prodname_copilot %} を使用する場合、{% data variables.product.prodname_copilot %} 拡張機能/プラグインは、統合開発環境 (IDE) との対話によって生成されたイベントに関する使用状況の情報を収集します。 これらのイベントには、{% data variables.product.prodname_copilot %} のパフォーマンス、使用された機能、および承認された提案、変更されて承認された提案、または無視された提案が含まれます。 この情報には、[GitHub プライバシーに関する声明](/github/site-policy/github-privacy-statement)で定義されている、ユーザーの個人情報を含む個人データが含まれる場合があります。

この使用状況の情報は、{% data variables.product.company_short %} によって使用され、Microsoft および OpenAI と共有され、拡張機能/プラグインおよび関連製品の開発および改善に使用されます。 OpenAI では、この使用状況の情報を使用して、{% data variables.product.prodname_copilot %} に関連するその他のサービスも実行されます。 たとえば、{% data variables.product.prodname_copilot %} 拡張機能/プラグインを有効にしてファイルを編集すると、ファイル コンテンツ スニペット、提案、修正候補が {% data variables.product.company_short %}、Microsoft、および OpenAI と共有され、提案と関連製品を改善するための診断目的で使用されます。 {% data variables.product.prodname_copilot %} は、編集中のファイルと、同じ IDE インスタンスで開かれている可能性のある他のファイルの両方で、コンテキストのファイル コンテンツに依存します。 {% data variables.product.prodname_copilot %} を使用している場合は、関連するファイルのリポジトリまたはファイル パスの URL を収集することもあります。 {% data variables.product.prodname_copilot %} では、利用統計情報で収集されたこれらの URL、ファイル パス、スニペットは、{% data variables.product.prodname_copilot %} の他のユーザーの提案として使用されません。 この情報は機密情報として扱われ、関係者以外極秘でアクセスされます。 {% data variables.product.prodname_copilot %} 拡張機能/プラグインから、{% data variables.product.prodname_copilot %} の他のユーザーに関する利用統計情報を収集することは禁止されています。 {% data variables.product.prodname_copilot %} の利用統計情報の詳細については、「[{% data variables.product.prodname_copilot %} 利用統計情報について](/github/copilot/about-github-copilot-telemetry)」を参照してください。 お客様は、GitHub に連絡し、技術プレビューからの削除を要求することにより、本項に記載されている利用統計情報および個人データ処理操作への同意を取り消すことができます。
