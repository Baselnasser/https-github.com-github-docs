1. 新しいリポジトリの所有者を選び、リポジトリ名を入力し、希望のプライバシー設定を選んで、 **[テンプレートからリポジトリを作成する]** をクリックします。
1. 新しく作成されたリポジトリのメイン ページで、 **[{% octicon "code" aria-label="The code icon" %} コード]** ボタンをクリックして、 **[Codespaces]** タブを選びます。

   ![[New codespace] ボタン](/assets/images/help/codespaces/new-codespace-button.png)

   このタブが表示されない場合、{% data variables.product.prodname_github_codespaces %} を使うことはできません。 {% data variables.product.prodname_github_codespaces %} へのアクセスについて詳しくは、「[Codespace を作成する](/codespaces/developing-in-codespaces/creating-a-codespace#access-to-github-codespaces)」をご覧ください。

1. **[Codespaces]** タブで、 **[メインで codespace を作成する]** をクリックします。
