通常、2 コアから 32 コアの任意のリモート マシン上で codespace を実行できます。 これらはそれぞれ、異なるレベルのリソースと支払い層を持っています。 詳しくは、「[{% data variables.product.prodname_github_codespaces %} の支払いについて](/billing/managing-billing-for-github-codespaces/about-billing-for-github-codespaces)」をご覧ください。

既定では、codespace を作成するときに、有効なリソースが最も少ないマシンの種類が使われます。
