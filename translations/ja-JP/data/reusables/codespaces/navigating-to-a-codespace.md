## 既存の codespace に移動する

1. {% data reusables.codespaces.you-can-see-all-your-codespaces %}
1. 開発する codespace の名前をクリックします。

   ![Codespace の名前](/assets/images/help/codespaces/click-name-codespace.png)

または、リポジトリに移動して、 **{% octicon "code" aria-label="The code icon" %} Code** を選ぶことで、特定のリポジトリの codespace を表示することもできます。 ドロップダウン メニューには、リポジトリのアクティブな codespace がすべて表示されます。
