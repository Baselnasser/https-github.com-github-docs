IP 許可リストのエントリを編集できます。 有効なエントリを編集すると、変更がすぐに適用されます。 

エントリの編集が終了した後で、リストを有効にしたら、許可リストが特定の IP アドレスからの接続を許可するかどうかを確認できます。
