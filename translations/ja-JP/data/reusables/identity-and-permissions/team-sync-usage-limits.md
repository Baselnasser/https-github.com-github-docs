### Usage limits (使用状況の制限)

チームの同期機能には使用制限があります。  これらの制限を超えると、パフォーマンスが低下し、同期エラーが発生する可能性があります。

- GitHub Team のメンバーの最大数: 5,000
- GitHub Organization のメンバーの最大数: 10,000
- GitHub Organization の Team の最大数: 1,500
