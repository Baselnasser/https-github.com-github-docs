7. レプリケーション用のキーの組を生成するには、プライマリ アプライアンスの IP アドレスを指定して `ghe-repl-setup` コマンドを使用し、それが返す公開キーをコピーします。
  ```shell
  $ ghe-repl-setup <em>PRIMARY IP</em>
  ```
