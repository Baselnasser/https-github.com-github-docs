1. **[Report to repository admins]\(リポジトリ管理者に報告する\)** オプションや **[{% data variables.product.prodname_dotcom %} Support に不正使用を報告する]** オプションが表示される場合があります。 もし表示されなければ、次のステップに飛ばしてください。
    - コンテンツを {% data variables.product.prodname_dotcom %} Support に報告するには、 **[{% data variables.product.prodname_dotcom %} Support に不正使用を報告する]** をクリックします。
    - コンテンツをリポジトリ保守担当者に報告するには、 **[Choose a reason]\(理由の選択\)** ドロップダウン メニューを使って理由を選択してから、 **[Report to repository admins]\(リポジトリ管理者に報告する\)** をクリックします。 レポートは送信されました。
  ![[Report content]\(レポートのコンテンツ\) モーダル](/assets/images/help/repository/report-content-modal.png)
2. [Report content]\(レポートのコンテンツ\) フォームの入力を完了して、 **[Send request]\(要求を送信する\)** をクリックします。
  ![[Report content]\(レポートのコンテンツ\) フォーム](/assets/images/help/repository/report-content-to-support-form.png)
