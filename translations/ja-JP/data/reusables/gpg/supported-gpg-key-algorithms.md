### サポートされているGPGキーのアルゴリズム

{% data variables.product.product_name %}はいくつかのGPGキーアルゴリズムをサポートします。 サポートされていないアルゴリズムで生成されたキーを追加しようとすると、エラーが生じることがあります。

- RSA
- ElGamal
- DSA
- ECDH
- ECDSA
- EdDSA
