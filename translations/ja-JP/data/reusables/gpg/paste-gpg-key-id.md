1. GitでGPG署名キーを設定するには、以下のテキストを使いたいGPGキーIDに置き換えて貼り付けてください。 この例では、GPG キー ID は `3AA5C34371567BD2` です。
  ```shell
$ git config --global user.signingkey <em>3AA5C34371567BD2</em>
```
