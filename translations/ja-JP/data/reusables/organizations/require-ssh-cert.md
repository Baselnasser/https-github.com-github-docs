1. 必要に応じて、メンバーに SSH 証明書の使用を要求するには、 **[SSH 証明書を要求する]** を選択し、 **[保存]** をクリックします。
    ![[SSH 証明書を要求する] チェックボックスと [保存] ボタン](/assets/images/help/organizations/require-ssh-cert.png)

   {% note %}

   **注:** SSH 証明書を要求する場合、その要件は承認されたサード パーティ統合や {% data variables.product.prodname_actions %}{% ifversion fpt or ghec %} や {% data variables.product.prodname_codespaces %}{% endif %} などの {% data variables.product.prodname_dotcom %} 機能には適用されません。それらは {% data variables.product.prodname_dotcom %} エコシステム内の信頼できる環境です。

   {% endnote %}
