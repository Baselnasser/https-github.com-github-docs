1. {% data variables.product.product_location %} で、Organization のメイン ページへ移動します。
1. Organization 名の下で、{% octicon "gear" aria-label="The gear icon" %} **[設定]** をクリックします。
![Organization の設定ボタン](/assets/images/help/discussions/org-settings.png)
1. [ディスカッション] で、 **[この Organization のディスカッションを有効にする]** を選択します。
1. Organization のディスカッションのソース リポジトリとして使用するリポジトリを選択します。
  ![Organization のディスカッションを有効にする設定](/assets/images/help/discussions/enable-org-discussions.png)
1. **[保存]** をクリックします。
