リポジトリ内の C/C++、C#、または Java コードに非標準のビルド プロセスがあると、`autobuild` が失敗するおそれがあります。 ワークフローから `autobuild` ステップを削除し、手動でビルド ステップを追加する必要があります。 リポジトリのGoのファイルで展開するものを指定したい場合には、ビルドのステップを追加しなければなりません。
