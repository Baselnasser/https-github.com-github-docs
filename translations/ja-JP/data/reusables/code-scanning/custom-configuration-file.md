構成ファイルは、分析するリポジトリ内、または外部リポジトリ内に格納できます。 外部リポジトリを使用すると、1 つの場所の複数のリポジトリに対して構成オプションを指定できます。 外部リポジトリにある構成ファイルを参照する場合は、 _OWNER/REPOSITORY/FILENAME@BRANCH_ 構文を使用できます。 たとえば、 _octo-org/shared/codeql-config.yml@main_ です。
