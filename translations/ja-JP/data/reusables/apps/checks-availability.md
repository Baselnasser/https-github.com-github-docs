Checks APIのための書き込み権限は、GitHub Appsだけが利用できます。 OAuth App及び認証されたユーザは、チェック実行とチェックスイートを見ることができますが、作成することはできません。 GitHub アプリを構築しない場合は、[Statuses API](/rest/reference/commits#commit-statuses) に関心があるかもしれません。
