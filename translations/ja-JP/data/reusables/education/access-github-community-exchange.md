{% data variables.product.prodname_community_exchange %} にアクセスするには、 https://education.github.com で自分の {% data variables.product.prodname_global_campus %} ダッシュボードに移動します。

認定教育機関の学生または教職員の場合は、{% data variables.product.prodname_global_campus %} 内での {% data variables.product.prodname_community_exchange %} へのアクセスが含まれる {% data variables.product.prodname_education %} 特典を申し込むことができます。

- {% data variables.product.prodname_education %} にまだ参加していない学生の場合は、[学生申請フォーム]( https://education.github.com/discount_requests/student_application)を使って申請します。 詳しくは、「[学生向け {% data variables.product.prodname_global_campus %} について](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/about-github-global-campus-for-students)」を参照してください。

- {% data variables.product.prodname_education %} にまだ参加していない教師の場合は、[教師申請フォーム]( https://education.github.com/discount_requests/teacher_application)を使って申請します。 詳しくは、「[教師として {% data variables.product.prodname_global_campus %} に応募する](/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-teachers/apply-to-github-global-campus-as-a-teacher)」を参照してください。
