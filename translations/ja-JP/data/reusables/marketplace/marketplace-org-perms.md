{% tip %}

  **ヒント**: アプリに Organization レベルのアクセス権が必要な場合は、Organization の所有者だけがそのアプリの購入、インストール、キャンセルと、Organization のアプリの課金の管理を行えます。 アプリケーションがOrganizationレベルのアクセスを必要としないなら、リポジトリ管理者がアプリケーションのインストールとアンインストールを行えます。

{% endtip %}
