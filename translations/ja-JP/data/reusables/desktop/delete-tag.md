1. コミットを右クリックしてください。

    - コミットにタグが 1 つしかない場合は、 **[Delete Tag TAG NAME]** \(TAG NAME のタグを削除\) をクリックします。
  ![タグを削除のメニュー項目の選択](/assets/images/help/desktop/select-delete-tag.png)
    - コミットにタグが複数ある場合、 **[Delete Tag...]** \(タグの削除...\) の上にカーソルを重ね、削除するタグをクリックしてください。
  ![[Delete Tag]\(タグの削除\) メニューにカーソルを重ねる](/assets/images/help/desktop/delete-tag-multiple.png)
