[Git for Windows](https://git-for-windows.github.io/) など **別のターミナル プロンプトを使用している場合は**、ssh-agent を有効にします。
```shell
# start the ssh-agent in the background
$ eval $(ssh-agent -s)
> Agent pid 59566
```
