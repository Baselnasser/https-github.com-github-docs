Git からパスワードの入力を求められたら、個人用アクセス トークン (PAT) を入力します。 または、[Git Credential Manager](https://github.com/GitCredentialManager/git-credential-manager/blob/main/README.md) などの資格情報ヘルパーを使用することもできます。{% ifversion not ghae %}より安全な認証方法を優先して、Git のパスワードベースの認証が削除されました。{% endif %}詳細については、「[個人用アクセス トークンの作成](/github/authenticating-to-github/creating-a-personal-access-token)」を参照してください。
