1. 最新のプロジェクト リリース バージョンに更新するには、`git checkout stable` コマンドを実行して `stable` ブランチを使用します。

  ```shell
  git checkout stable
  ```
  
  または、特定のプロジェクト バージョンを使用するには、次のコマンドを実行し、`X.Y.Z` を目的のリリース バージョンに置き換えます。

  ```shell
  $ git checkout vX.Y.Z
  ```
