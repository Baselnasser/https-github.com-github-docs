1. 必要に応じて、既にレビューされて承認された pull request に対するスケジュールされたリマインダーをオフにするには、 **[承認済みの pull request を無視する]** を選択します。 次に、 **[1 つ以上の承認で無視]** をクリックして、プル要求を無視する必要がある承認の数を選択します。
