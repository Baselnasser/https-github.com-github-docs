Markdown ファイルを確認するときは、ファイルの上部にある {% octicon "code" aria-label="The code icon" %} をクリックすると、Markdown レンダリングを無効にして、代わりにファイルのソースを表示することができます。

![Markdown とソースとして表示する](/assets/images/help/writing/display-markdown-as-source.png)

Markdown レンダリングを無効にすると、ライン リンクなどのソース ビュー機能を使用できます。これは、レンダリングされた Markdown ファイルを表示する場合には使用できません。
