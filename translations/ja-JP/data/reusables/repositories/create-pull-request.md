1. レビューの準備が完了している pull request を作成するには、 **[pull request の作成]** をクリックします。
ドラフトの pull request を作成するには、ドロップダウンを使用して、 **[ドラフトの pull request の作成]** を選択し、 **[ドラフトの pull request]** をクリックします。 ドラフトの pull request の詳細については、「[pull requests について](/articles/about-pull-requests#draft-pull-requests)」を参照してください。![[pull request の作成] ボタン](/assets/images/help/pull_requests/pullrequest-send.png)
