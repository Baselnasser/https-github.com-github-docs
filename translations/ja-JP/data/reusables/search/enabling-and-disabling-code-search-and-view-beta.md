ベータ版への参加の順番待ちリストから承諾されると、メールが届きます。 アクセス権が付与されると、アカウントで新しいコード検索とコード ビューのベータ版が自動的に有効になります。

{% data variables.product.prodname_dotcom_the_website %} のベータ版はいつでも無効または有効にすることができます。 この設定は、コード検索とコード ビューの両方に適用されることに注意してください。

{% data reusables.feature-preview.feature-preview-setting  %}
1. [新しい Code Search とコード ビュー (ベータ版)] の右側にある **[有効]** または **[無効]** をクリックします。
