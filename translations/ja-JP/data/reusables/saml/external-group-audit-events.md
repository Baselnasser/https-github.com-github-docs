| アクション | 説明
|------------------|-------------------
| `external_group.delete` | Okta グループが削除されたときにトリガーされます。 詳細については、「[チームへの Okta グループのマッピング](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams)」を参照してください。
| `external_group.link` | Okta グループが {% data variables.product.prodname_ghe_managed %} チームにマップされたときにトリガーされます。 詳細については、「[チームへの Okta グループのマッピング](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams)」を参照してください。
| `external_group.provision` | Okta グループが {% data variables.product.prodname_ghe_managed %} でチームにマップされたときにトリガーされます。 詳細については、「[チームへの Okta グループのマッピング](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams)」を参照してください。
| `external_group.unlink` | Okta グループが {% data variables.product.prodname_ghe_managed %} チームからマップ解除されたときにトリガーされます。 詳細については、「[チームへの Okta グループのマッピング](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams)」を参照してください。
| `external_group.update` | Okta グループの設定が更新されたときにトリガーされます。 詳細については、「[チームへの Okta グループのマッピング](/admin/authentication/configuring-authentication-and-provisioning-with-your-identity-provider/mapping-okta-groups-to-teams)」を参照してください。
