1. シングル サインオン ダイアログの下部にある **[回復コードを使用する]** をクリックし、シングル サインオンをバイパスします。

   ![回復コードを入力するためのリンクのスクリーンショット](/assets/images/help/saml/saml_use_recovery_code.png)

2. [Recovery Code] フィールドにリカバリコードを入力します。

   ![回復コードを入力するためのフィールドのスクリーンショット](/assets/images/help/saml/saml_recovery_code_entry.png)

3. **[確認]** をクリックします。

   ![回復コードを検証するためのボタンのスクリーンショット](/assets/images/help/saml/saml_verify_recovery_codes.png)

回復コードは使用後に無効になることにご留意ください。 リカバリコードは再利用できません。
