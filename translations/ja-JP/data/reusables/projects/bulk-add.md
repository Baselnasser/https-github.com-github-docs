1. 必要に応じて、リポジトリを変更するには、ドロップダウンをクリックしてリポジトリを選びます。 特定の issue や pull request を検索することもできます。
  ![[リポジトリ] ドロップダウン示すスクリーンショット](/assets/images/help/projects-v2/add-bulk-select-repo.png)
1. 追加する issue と pull request を選びます。
  ![追加する issue と pull request が選ばれているところを示すスクリーンショット](/assets/images/help/projects-v2/add-bulk-select-issues.png)
1. **[選択した項目の追加]** をクリックします。
  ![[選択した項目の追加] ボタンを示すスクリーンショット](/assets/images/help/projects-v2/add-bulk-save.png)
