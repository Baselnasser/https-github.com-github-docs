プロジェクトの説明と README を設定して、プロジェクトの目的を共有したり、プロジェクトの使用方法を説明したり、関連するリンクを含めたりすることができます。

{% data reusables.projects.project-settings %}
1. プロジェクトに簡単な説明を追加するには、[説明の追加] でテキスト ボックスに説明を入力し、 **[保存]** をクリックします。
  ![[説明の追加] 設定を示すスクリーンショット](/assets/images/help/projects-v2/edit-description.png)
1. プロジェクトの README を更新するには、[README] の下のテキスト ボックスに内容を入力します。
   - Markdown を使用して README の書式を設定できます。 詳細については、「[基本的な書き方とフォーマットの構文](/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)」を参照してください。
   - テキスト ボックスと変更のプレビューを切り替えるには、{% octicon "eye" aria-label="The preview icon" %} または {% octicon "pencil" aria-label="The edit icon" %} をクリックします。
  ![プロジェクトの README の編集を示すスクリーンショット](/assets/images/help/projects-v2/edit-readme.png)
1. README に変更を保存するには、 **[保存]** をクリックします。
  ![プロジェクトの README の [保存] ボタンを示すスクリーンショット](/assets/images/help/projects-v2/save-readme-button.png)

プロジェクトに移動し、右上にある {% octicon "sidebar-expand" aria-label="The sidebar icon" %} をクリックすると、プロジェクトの説明と README を表示してすばやく変更できます。
