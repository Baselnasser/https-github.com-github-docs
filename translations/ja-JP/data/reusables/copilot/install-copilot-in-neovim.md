1. Neovim で {% data variables.product.prodname_copilot %} を使用するには、{% data variables.product.prodname_copilot %} プラグインをインストールします。 プラグイン マネージャーからプラグインをインストールすることも、直接インストールすることもできます。
   - vim-plug や packer.nvim などのプラグイン マネージャーを使用する場合は、プラグイン マネージャーを使って `github/copilot.vim` をインストールします。 詳しくは、プラグイン マネージャーのドキュメントを参照してください。 たとえば、[vim-plug](https://github.com/junegunn/vim-plug) または [packer.nvim](https://github.com/wbthomason/packer.nvim) のドキュメントを参照できます。
