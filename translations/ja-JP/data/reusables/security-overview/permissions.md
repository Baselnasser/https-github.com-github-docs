Organization 所有者とセキュリティ マネージャーは、Organization {% ifversion ghec or ghes > 3.4 or ghae-issue-6199 %} のセキュリティの概要にアクセスして、Enterprise レベルのセキュリティの概要から Organization のリポジトリを確認できます。 Enterprise 所有者は、Enterprise レベルのセキュリティの概要を使って、Enterprise の Organization {% endif %} 内にあるすべてのリポジトリを確認できます。 Teamのメンバーは、Teamが管理者権限を持つリポジトリのセキュリティの概要を見ることができます。
