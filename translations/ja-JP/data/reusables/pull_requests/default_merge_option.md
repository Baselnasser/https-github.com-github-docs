{% data variables.product.product_location %} の pull request で既定の **[pull request のマージ]** オプションをクリックすると、機能ブランチからのすべてのコミットがマージ コミット内でベース ブランチに追加されます。 pull request は、[`--no-ff` オプション](https://git-scm.com/docs/git-merge#_fast_forward_merge)を使用してマージされます。

pull request をマージするには、リポジトリの[書き込みアクセス許可](/articles/repository-permission-levels-for-an-organization/)が必要です。
