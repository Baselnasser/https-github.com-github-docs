Enterprise で OIDC SSO を使うと、{% data variables.product.prodname_dotcom %} は IdP の条件付きアクセス ポリシー (CAP) の IP 条件を自動的に使って、メンバーが IP アドレスを変更したとき、および個人アクセス トークンまたは SSH キーが使われるたびに、{% data variables.product.prodname_dotcom %} でのユーザー操作を検証します。
