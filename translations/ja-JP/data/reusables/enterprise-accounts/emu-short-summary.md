{% data variables.product.prodname_emus %} は、エンタープライズ メンバーとリソースをより細かく制御できる {% data variables.product.prodname_ghe_cloud %} の機能です。 {% data variables.product.prodname_emus %} では、{% data variables.product.product_name %} にユーザーが独自のアカウントを作成する代わりに、ID プロバイダー (IdP) を通じてすべてのメンバーをプロビジョニングおよび管理します。 チーム メンバーシップは、IdP 上のグループを使用して管理できます。 {% data variables.product.prodname_managed_users_caps %} は企業に制限されており、コードをプッシュしたり、共同作業を行ったり、企業外のユーザー、リポジトリ、組織とやり取りすることはできません。 詳細については、[{% data variables.product.prodname_ghe_cloud %} ドキュメントの「{% data variables.product.prodname_emus %}](/enterprise-cloud@latest/admin/authentication/managing-your-enterprise-users-with-your-identity-provider/about-enterprise-managed-users){% ifversion not ghec %}」を参照してください。{% else %}。{% endif %}
