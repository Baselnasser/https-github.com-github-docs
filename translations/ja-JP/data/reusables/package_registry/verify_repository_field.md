1. プロジェクトの *package.json* の `repository` フィールドを確認します。 `repository` フィールドは、{% data variables.product.prodname_dotcom %} リポジトリの URL と一致している必要があります。 たとえば、リポジトリの URL が `github.com/my-org/test` である場合、repository フィールドは `https://github.com/my-org/test.git` でなければなりません。
