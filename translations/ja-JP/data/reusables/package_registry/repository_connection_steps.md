1. パッケージのバージョンで、 **[リポジトリの接続]** をクリックします。
  ![パッケージのランディング ページでリポジトリ ボタンを接続する](/assets/images/help/package-registry/connect-repository.png)

  ![リポジトリ選択のポップアップ ウィンドウ内の [リポジトリの接続] ボタン](/assets/images/help/package-registry/select_a_repo.png)
