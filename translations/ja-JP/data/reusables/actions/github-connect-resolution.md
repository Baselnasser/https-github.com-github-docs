ワークフローが、アクションが格納されているリポジトリを参照してアクションを使用する場合、{% data variables.product.prodname_actions %} は最初に {% data variables.product.product_location %} でリポジトリを検索します。 リポジトリが {% data variables.product.product_location %} に存在せず、{% data variables.product.prodname_dotcom_the_website %} への自動アクセスが有効になっている場合、{% data variables.product.prodname_actions %} は {% data variables.product.prodname_dotcom_the_website %} でリポジトリを検索します。
