環境を参照するワークフロー ジョブが実行されると、`environment` プロパティに環境の名前が設定された展開オブジェクトが作成されます。 ワークフローが進行すると、`environment` プロパティに環境の名前、`environment_url` プロパティに環境の URL (ワークフローで指定されている場合)、および `state` プロパティにジョブの状態が設定された展開状態オブジェクトも作成されます。
