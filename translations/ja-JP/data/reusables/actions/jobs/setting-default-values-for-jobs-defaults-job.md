`jobs.<job_id>.defaults` を使用して、デフォルト設定の `map` を作成します。これは、ジョブ内のすべてのシェルに適用されます。 ワークフロー全体に対してデフォルト設定を設定することもできます。 詳細については、「[`defaults`](/actions/using-workflows/workflow-syntax-for-github-actions#defaults)」を参照してください。
