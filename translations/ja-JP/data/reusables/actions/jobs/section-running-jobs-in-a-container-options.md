`jobs.<job_id>.container.options` を使用して、追加の Docker コンテナー リソース オプションを構成します。 オプションのリストについては、「[`docker create` オプション](https://docs.docker.com/engine/reference/commandline/create/#options)」を参照してください。

{% warning %}

**警告:** `--network` オプションはサポートされていません。

{% endwarning %}
