外部認証プロバイダーのアカウントを持たない一部のユーザーに対して認証を許可する場合は、{% data variables.product.product_location %} でローカル アカウントへのフォールバック認証を許可できます。 詳しくは、「[プロバイダー外のユーザーのためのビルトイン認証の許可](/admin/identity-and-access-management/managing-iam-for-your-enterprise/allowing-built-in-authentication-for-users-outside-your-provider)」を参照してください。
